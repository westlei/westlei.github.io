# 冬日序曲

# 讳疾忌医

最近有些疲惫。

可能是要入冬了吧，总是不想起床，不想工作，但又不得不起床，不得不工作。

我的心态就是挺差的。

想起之前我写的功能参与联调，拉着几个部分的人一点点去对问题，改参数，打中间变量，打堆栈，改上下游工具，直到终于打通所有环节，model proc time出现。算是来到这里做出的第一丢丢成就，但也没多喜悦，还有很多困难在等着。

某人曾经问过我四个问题。有的可以简单的给出答案，而有的不能。突然想起了某个微博关注的po主曾经发过一段话，大概是她曾患有过抑郁症，而如今甜美的爱情让她再也不用看见那些药片。

感情于我而言，大约就像是药片一般。治愈苦涩的生活，并为之添加一些期盼。我知道药片能愈疗伤痛，却有些讳疾忌医。明知病不可自愈，却仍抱有些许幻想。

何况，药片应当是甜的。

# 风雪江至

今天的上海好冷，似乎是下雪了。那位长者终究没有等来下一个春天，在这初雪的日子里永远的离开了人世间。西方有老妪，东方有仙翁，千里共角力，万世作永恒。属于那个时代的人物逐渐退场，空旷的舞台上，只剩一群拙劣的滑稽戏演员，追随着莫名其妙的所谓正确。

想起上一次在上海看雪，已经是四年前了。时光仿佛一个莫比乌斯环，在彼时彼刻与此时此刻间流转。沿着莫比乌斯环走到一个点的另一面，一切相似却又截然不同。

彼时彼刻，生活中还没有无处不在的健康码和核酸亭，周末的夜晚去KTV唱到深夜然后直接坐上火车来一场说走就走的旅行也没有人会管你。崇拜长者的亚文化方才兴起，外面的世界也没有封锁的那么彻底。

还记得那天漫步外滩时突然飘起的漫天大雪，还记得姑苏甪直的白墙黑瓦与雪后初霁。还记得那年冬天唱过的那一支歌，那一个人，还有许许多多与此相关的，回不来的记忆 。

只如今/茫茫大雪之中等着谁回顾/明知无人回顾/谁能初心不负

# 天际海城

社畜是一向不怎么用QQ的。

一则是新的社会关系全在微信，二来旧的朋友大多也不再看QQ，社交软件没有人社交了，那自然也就不会再用了。

然而微信的朋友圈自是不可以乱发的，一来社畜间的关系比较淡泊，二来说多了难免给人抓些把柄，给人适时参一本之类的，即使无甚大事，也难免恼火，于是索性就不怎么发了。

而我尚未完全接纳自己身为社畜的现实，但也不认为自己仍在学习。姑且认为自己是在一个准gap&混日子的状态，等待自己找到下一个适合自己的身份。

我对生活的一向态度无非无事就开摆，有事开香槟。上进这两个字从四年前就开始跟我没有关系了，做事面子和乐子总得占一样。其中吃力不讨好的大部分是为了面子，轻松又捡漏的基本上图一乐子。严格意义上我也没再努过力，否则日子过的应该没现在好。

我个人在选择方向这样的能力上大概是有一点问题的，大凡我越是努力工作而急于求成的，越是常常一败涂地。反之若是随手而作无心插柳，反而能得到些意想不到的成果。

现在的生活并不快乐，实实在在的压力较之以前多了很多，但是虚空的压力比之以前明显小了。我只需要搞定眼下的一切，而不需要去猜测不可知的明天又要去做啥。周末依然是在四处游荡中度过，但游荡的感觉似乎又没啥意思。无可排解的无聊充斥着生活，而又无从改变。

古镇，湖泊，浦江，外滩，不熟悉，也不陌生。三个月没下深渊，打了一把也没意思。去看明日方舟的番，意识到自己打方舟从来没看过剧情，甚至不知道乌萨斯和龙门、罗德岛和黑钢企鹅物流这些是啥关系。

于是打开都市天际线，开了一张海岛新图。这图每隔十来年一场大海啸，把城市的建成区冲刷的干干净净，人口从几万直接掉到三位数，不过两三年后一切又能恢复原样就是了。人生大抵也是如此，在灾难面前，一切准备都没有意义，幸免于难，只能靠运气。熬过灾难，两三年后就是重新开始的光明。

# 倾落伽蓝

一个非常稻妻风味儿的任务。非常传统的逛街活动和塞各种乱七八糟的背景文案。网上的评价非常两极分化，一边说很好一边说很糟。我的评价是故事不怎么样，纯粹为了清理一堆伏笔而构造一套逻辑。以此为由让他进池，我持保留意见。

但是米桑总有一些出人意料的结合实事，在这刚刚过去的两个星期里，我们见证了一代伟人的谢幕，见证了从左满舵到右满舵的转折。当一些属于白色的记忆被抹杀在历史的长河里，如同那些三十多年前的往事，哪怕在他的生平里也只敢一笔带过。未来，谁又能成为降临者，守护历史的真相呢？

今日，「大雪」，也是2022年的最后一场月圆。终于可以从咫尺之外的虹桥出发，去向肆意的彼方。而我，又在害怕什么呢？

还是来一局七圣召唤吧。

# 后疫情时代与姑苏行记

# 天平山·木渎古镇

大约是江南层林尽染的时节了。姑苏城外的天平山脚，如织的游人穿行于如火的林间。阳光透过秋叶映入水色，照出一片绚烂的金红。尽管岁已初冬，但秋的华彩仍未落幕。况且，山间的秋叶，此时仍未红尽。

江南的古镇大抵相仿，一条古街遍布了琳琅的网红商品，一条溪流跨越古镇带来小桥流水人家。甪直、南浔、塘栖、朱家角乃至平江路、山塘街，皆是如此，木渎也不例外，但木渎的感觉稍有不同。似乎是古镇与居民区交错的缘故，来来往往的人，大多忙于生计，而非一般的游人。

# 洞庭西山·明月湾

当时光在岁月里凝结成诗，化作太湖上的烟云一缕。大榕树下的石桥边，小船轻轻敲打着波浪。青石板街与白墙黑瓦，通向湖边那一座小小的码头。十三年前的记忆与现实交叠，重又组合成现今。

三载的疫情似乎未曾在这里留下许多痕迹，仿佛这世外的一隅从未受过白色的洗礼。明月湾古村，一如当年明月，仍旧光亮如新。所谓悠然，即似午后的太湖般水平如镜，波澜不惊。

# 观前街·平江路

玄妙观前的街市，似乎又回到了昔日的车水马龙。熙熙攘攘的人群热热闹闹的挤在街上，感慨久违的放开。北寺塔到拙政园这段路似乎总是在施工，夜晚时分的这片人烟甚是稀少，只有一家蜜雪冰城仍然开着。

转入平江路沿河而下，人流重又渐渐的密了起来。不知何时兴起的城市路牌挂满了每一个景点的大街小巷，所谓一条平江路，半座姑苏城。传统意义上护城河所盘绕的这片姑苏古城，其精华大约也就在于此了。

# 一篇本来没有打算写的年终总结

其实这几年的生活，在《疫病往事（疫病劫灰往世书）》里已经写的差不多了。但也许，抛开疫情不谈，关于这一年，还有一些小小的故事。

QQ空间里早于2019年9月的内容已经被我删了若干遍，但依然时常会有一些往日的留存时不时跳出来。昔日的列表已经清的面目全非，昔日所记下的故人也早就失去了联系。或许，可能有例外，但哪怕是例外，也并不例外。四年了，我一直以为多少也算是个朋友，但事实上的距离，就像我曾经认为的那样远，且从来都不曾更近。

在那个秋天以后，我的列表里少了一位现实世界可能的朋友，多出来了一位远在异国他乡的网友。我羡慕她毅然选择梦想的勇气，却也感叹于疫情之下命运的波折。如果是我，我会有这样的勇气吗？

结束了15个月的深圳之旅。深圳很美，但是缺少一些我觉得需要的东西，譬如人情。做云上的服务很累，但比累更焦虑的是迷茫。高度混乱的需求，以及永远不知道自己在干嘛的感觉，这种感觉很难受。

或许我可能早就只是行尸走肉了，但仍仿佛自己是个人的样子。夜色里，迷雾里，百鬼笙歌里。夜色未尽，我又何必醒。鬼是没有影子的，原来早就已经奠定了结局。

也记得翡翠湖的一干朋友们，但终究大家毕业的毕业，离散的离散。天各一方的人们，大约再也不会有机会聚齐。曾经的朋友不再联系，会难受吗？也许，人事代谢就是如此。有的人，只有短暂的相遇，却已走完了这辈子相同的旅途。

沙井吧，百度贴吧里的新躺平吧。人间躺平地，宝安找沙井。自从三和人才市场整顿以后，三和大神四散而去，而后重又聚集在深圳西北角的沙井。当龙华开始了高速发展，宝安沙井的若干工厂给了落魄者们最后的生存空间。也许再下去，就是东莞长安。

新的一年逐渐开始失望，逐渐觉得无论画饼或者如何，不过这样。就仿佛是梦，梦里的一切都带不走，梦里的一切也不会带来现实的杀伤。所以梦里的he和be，都没有意义。

但我还是期待恋爱的。哪怕是梦，美好的记忆也是可以留下的。我想追寻我可以预期的长久，可我的现实生活却并没有让我能够期许长久。我应该去给自己找一个未来，一个自己想要的未来。

哪怕所谓的未来是梦，那也是好的。就像四月的海岛之旅，景色美的仿佛另一个世界。我想，如果我再去一次，哪怕是同样的地方，同样的天气，我也不会再有同样的感受了。所谓的旅行，行的不过是心境。

终于，一切尘埃落定。我选择了海思，去做我所习惯的嵌入式开发工作。追逐即将退潮的芯片风口，是明智还是愚钝？这都不重要，眼下的机遇，就像是晚霞，错过了，就只剩下无边黑暗。

星与月的交织，编成夜空的圆舞曲。小王子终究会返回他的星球，我们也终究会失去我们的青春。但玫瑰依然在哪里，等待着那个人的归来。
