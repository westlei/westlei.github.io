# 容颜

我喜欢你的容颜 那动人心魄的娇艳

那天与地之间 你是最美的一抹亮色

一见钟情不需要理由 就像我之于你

就在 这个难忘的季节和难忘的春天

这或许就是 一种叫喜欢你的感觉吧

# 梦落

因为音乐 因为剧情 因为风景 而喜欢剑三

从苍山洱海到昆仑之巅 看大漠的冷月高悬

从龙泉冰河到东海茫茫 看扬州城似锦繁花

独自一人行走在江湖的自由 就像是一场梦


谁念这天地玄黄 算不到动如参商

第四个皈依后 沉默让树下的人感殇

你眼中有柔情千种 如脉脉春风冰雪也消融

原来拈花一笑成了孽 令你这般误尽华年

若踏错 一步间背离终点 永不还 此生永不还

只如今 茫茫大雪之中等着谁回顾


喜欢抒情 喜欢古风 醒了很久也还是很感动

喜欢小鞠的歌 喜欢她独特的嗓音 治愈心灵

花与雪相遇 停住的四季 编织冬日的序曲

唱着 未来 可期

# 人生如寄

曾经 特别喜欢眉间雪

只道是纵明知无人回顾 自己初心不负便好

总记得那个冬天的日子 那个落雪的季节

一个人走过外滩和西湖 看古镇甪直七里山塘

转专业最无助的时节 埋藏心里最深沉的感动

那一首歌 那一些人 活在记忆中 却离开了现实


后来才明白 真正的人生 如寄

是不是再多念想捱不过 时光利刃

终究化成寒暄 三两声

曾经亲近的人们最终都成了记忆里的过去

曾经感动的故事都只留下自己一个人的回忆

再多的念想都捱不过时光的利刃

所能剩下的 最多不过是寒暄三两声

许多人怪我清列清的太频繁 清的太绝情

但我更不愿看见一些名字 渐渐消失了痕迹

也许终究还是要渐行渐远的 那又何必流连

一个人走了 他再也不回来 就如同寄出的信

即使再次遇见 也不过是寒暄三两声 而已


听人说后来我们都变成 江湖传闻

茶楼酒肆里说完一生

江湖里渐渐多了些故事 亦真亦假 亦是亦非

细打听才方知那故事主角 似乎是我的名字

浪迹江湖的岁月短暂 却在江湖留下了印迹

也再没有听说过前尘往事 终究断绝了历史

既然如此 那一切的一切 便都随他去吧


你只道昔年情谊皆封存 何必有悔恨

愿前路安稳 不再为旧事执念深

# 拖延

只要你sleep够久 你的任务就可以不用做

总会有Interrupted Exception e让你解脱

即使一切变故都没有发生 世界正常运转

直到最终的ddl来临 实在避无可避的时候

你也可以选择 从容地抛出new Date();

让ddl的倒计时 重新回到那一切的起点

但其实 任务早已经不是原来的任务

而线程 却依然还是原来的那个线程

sleep的这段光阴 叫做挥霍的岁月

在挥霍的岁月里 即使悄无声息淘汰了你

你也不会发觉 这个时代已经不再是属于你

# 百鬼笙歌

生命里总有着许多不可承受之痛

在这生老病死的凡世间 一切皆有可能发生

不可预期的生和死 不可预期的种种天灾和人祸

还有更加不可预期的人心

就像这波澜诡橘的岁月里 疫情的风云变换

就像这危机重重的金融市场 花样的熔断

抑郁在所难免 而死亡成为了一种解脱

可是 在那人间以外的世界里 你就一定能幸福吗

你在第二层 你以为我在第一层

我说我在第五层 可我其实在负十八层

人其实也可以是鬼 鬼其实也可以是人


百鬼笙歌里 趁着它 夜未尽。。。

# 纵月人间

老带学生往往经常性的会遇到连锁打击

毕竟当你一倒霉起来 都是一件接着一件的

于是乎就会让人觉得说

年纪轻轻的 说话总是有一种很沧桑的感觉

这是生活 这又不是生活 生活才不是这样的呢

要不就是我喜欢这种生活 可谁会喜欢呢

挫折会来 也会过去 热泪会流下 也会收起


有时她枕着梦/有风流来相逢

说着爱慕情衷许多种/最后偏又来去匆匆

有时她沉入梦/等久别的重逢

若有一点心动需珍重/深情有始无终

INFP人格 不易动感情 又极易动感情

沉沦在纠结和挣扎里 害了别人也害了自己

# 芒种

“一想到你我就/空恨别梦久/烧去纸灰埋烟柳”

又到了一年一度的芒种时节，也就是曾经，每一年高考的时候。在大学的下半场里，高考已经成为了一个渐渐在记忆里淡却的词，但它也终是应该铭记的吧，毕竟，是上一个阶段的结束，也是这一个阶段的开始。也许我还是应该感谢碑亭巷口长江路边奋斗的日子，感谢玄武湖下西家大塘落幕的钟声，感谢当初的运气和选择，至少有了一个不错的故事开始。

看见表白墙上说：“在我发出消息的一刻，我就没想过还能跟你做朋友”。我早已清楚了这句话，于是便没有勇气去想。这种犹豫贯穿了后来的三年时光，也许这便是我性格里最大的弱点：内心深处的怯懦，和不愿改变现状的保守。可是，当现状本就脆弱地无法保持，那若不去改变，便只能看着它土崩瓦解，而且是以你最不想看到的局面瓦解。

可是这样的经历，却让我在之后的日子里更加保守，更加小心翼翼。我总是天真的以为，既然没有表白，那就不会失败，也许还能继续做着朋友，直到一切时机都成熟的时候便自然圆满了。可是，人总是难以一帆风顺，每一个引人注目的果实总会勾起许许多多觊觎的目光。时机永远会在成熟之前被意外破坏，果实在总在你的犹豫之中归于他人之手。世界不是一个人的世界，成功却只有一个人能成功。

成年人的世界里没有不劳而获，韭菜的奋斗和后浪的“拼搏”并不分贵贱。我愿意为我想要的东西奋斗，却总是在自己的欲望面前犹疑，可成功需要的运气来自于时机。如果没有把握住去年的一连串机遇，可能就没有我经历中唯一能看的几笔。就像今年的危机，又有谁能事先想到这样的开局？敢于承担竞赛失败的后果，却不敢成为情感危机的败例。其实本就一无所有，却为什么不愿意。

就仿佛那研究生的愿望，心中的白月光。尽管很难，但也想要去尝试一下。可是随着时间的推移，日益忙碌的课程让我信心游移。也许不去动摇选择坚定结局会是胜利，但胆怯的内心总想让我选择放弃。生活艰难，网课尤甚。实验不会，作业手残。可能这终是每一个大三学子必须面对的过程，在这赤裸裸的现实面前，理想或者妥协，没有第三种选择，也就是说，无可逃避。

据说，避免失败的方法有这么几项：1.不要回忆过去，2.不要好为人师，3.坚定目标前行。我已经失去了前两种机会，于是便只剩了最后这么一个可能性。目标是什么？目标对我的诱惑有多大？实现目标又应该做些什么？但愿我能解决这三个困扰了许久的问题，也但愿感情的魔咒就此结束，不再有下一次，看见自己感觉到喜欢的姑娘换上了情头，却不是和自己。

“暧昧就是所有人都以为你们在一起了，只有你清楚的知道你们之间的距离。”所有的暧昧里，总有人要迈过那一步距离，不然一切看上去的美好，都终会烟消云散。但前提是，得有暧昧。
