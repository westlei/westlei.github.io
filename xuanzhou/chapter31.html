# 生之无奈

一个月来的第四次闪现，地点是小城仪征。或许这一次旅途结束以后，我暂时不会再有出门的念头了。旅行是为了缓解自己的无聊，可旅途本身却让自己变得更加无聊。简单算了下，三月开学到现在花掉了四千多点，去掉两千不到的日常里面杂七杂八的开支，大概有两千多花在了旅途上。值得吗？似乎也不算很值的样子。我确实看到了江南春日里别样的风景，但那终究只是一段段零散的记忆罢了。

列表充斥着悍不畏死的卷王，毅然决然的在保研加分一线冲锋陷阵。也有手握好几篇SCI的大佬，在学术道路上一路凯歌。但更多的人都只是需要考研的普通人，胜者只是被导师push的学术萌新，而败者也不过是考研二百五十余万炮灰中普普通通的一份子罢了。关于考不考研和怎么选专业的事情，每个人都有每个人自己的想法。生化环材们吐槽自己的所学脱离现实，找不到工作。机械土木们吐槽自己的所学层次太低，看不到希望。但其实哪有那么多高大上的工作需要去做呢？当每一个人都在追求更高的目标的时候，卷就无可避免的降临了。

工作永远是有限的，那只能让更高级的人来做更低级的工作。宇宙机和宇宙金的热门不是偶然，他们共同的特点是生产力完全依赖于人而非物。你只需要一台电脑就可以写出给千千万万人服务的代码，就可以参与金融市场的惨烈搏杀。当一个人的工作成本越低的时候，他所能拿到的报酬也就越高，但同时他被压榨的也就越狠。即便如此，宇宙机和宇宙金也走向了两个极端。宇宙机依靠超额的时长来榨取日益微薄的利润，从而维持整个行业的相对高薪。而宇宙金则为了维护行业的绝对高薪，而不断拉升自己的门槛，阻挡外人的进入。但渐渐的，金融的工作强度越来越大，计算机的门槛开始越来越高。两者终将在内卷中走向同一结果，大约类似于如今的医学，门槛与成本均极高，但收益不足。

那么，我们再回到大学本身。研究型大学，培养的是研究型的人才。研究型人才读硕士博士一路深造，天经地义。于是当我们读完本科发现，我们所学的东西全部是为了我们进一步深造而准备的。寄希望于通过我们学到的东西去找工作，难度必然很大。当我们按部就班的读完四年，在内卷中失败丢失了保研名额，其实我们就已经是被学校所放弃的人了。但是惯性总会驱动着我们考研，因为对于绝大多数人来说除了读研别无他法。一个从未学到过本专业赖以谋生的技能，甚至连自己的毕业设计都难以完成的人，怎么去从事本专业相关的工作呢？

当金融绑架了市场，房价便成为了悬在当代人头上的一把利刃。尽管事实而言房价并不是人生唯一的意义，但对于大多数人而言，房依然是使人隐隐作痛的心病。自从城市化快速发展以来，大量的商品房拔地而起，但大家都普遍忽略了一个现实：老旧的无学区的住宅，价值只会越来越低，甚至趋近于0。因此即使是住宅置换，依然需要掏出百万级以上的差价。我亲戚家卖掉了北京三套地段尚可甚至还有学区的老旧住房，才成功换得一套全新的商品楼盘。房地产的泡沫只是表象，即使未曾戳破，他也只是一个洗劫财富的工具。等到中国的城市化基本完成的时候，我们的房子也基本上到了价值基本归零的时候，那时的拆迁，也就意味着你的资产被剥削一空。中国不像西方，你对你的房屋没有完整的所有权，你无法自行赋予其新的价值，只能板等房屋随着自身的老化而凋敝，而失去价值。于是乎新房永远都供不应求，这样的话房价也就永远不可能跌落。

似乎这么说起来的话，人生好像过于悲观了。但是人生本来就是苦的，甜蜜的人生只是少数人的幸运，而大多数人注定劳碌无为。我害怕去幻想所有美好的事物，害怕去寻觅爱情。我期待，却又忐忑。其实所有的无疾而终，都只不过是临阵退缩罢了。当我自己都不敢去相信自己的时候，又有谁敢去相信你呢？终于，我应当是看开了。能力不够的时候就不应该去想乱七八糟的，又不敢冲，又舍不得，那不就是徒增烦恼罢了嘛。

# 彩金·宣州

从大二到大四，每当我心情好或是不好的时候，我都会骑上我的老伙计，然后来到这里。就在湖边上，依靠在自行车上，吹一会儿湖风，看一会儿湖水。

这里的湖水相当清澈，忠实的书写着天空的模样。天空湛蓝的时候，湖水也是透亮。天空阴沉的时候，湖水也是深青。阳光洒落的时候，湖面一片的波光粼粼。夕阳西下的时间，又是一片火红的苍凉。待到星光闪烁，夜色下的湖边伸手不见五指，遥看另一侧城市的点点灯火，又仿若是另一个世界。

站在北岸大堤上举目四望，湖畔尽是山野之所，间或有点点村庄若隐若现。东侧的荒山之后，图书馆的顶楼清晰可见。西面是唯一的出入口，小径联通着现实和远方。北侧的工地上一栋栋高楼拔地而起，描绘着城市的方向。这里是城市，也是郊野。我们看见城市的喧嚣，却听见郊野的寂静。

这里离学校确然是极近的。初时自北门跋山涉水而来，仅1300余米。若骑车而来，则需绕行柏枧山路，约6.5km。待创业路修筑，步行之路阻断，但骑行仅需3.5km。如今文景路修筑，骑行至湖边仅2.2km，十分钟即可抵达。

我有过许多喜欢的人。她们或文静，或吵闹，或温柔，或霸道，或俏皮，或理智。我喜欢过每一个不一样的她们，实意或真心。但终究她们都淹没在了茫茫人海里。或许记得，或许忘记。考研似乎是终点，又似乎只是终点。考研后的我，终究失去了一切期待的我，只会将一切沉默以待。

湖水沉默地倾听者一切。我无数次想这么一直从大堤上走下去，走进湖水里，永远消失在湖水里。但我终究还是选择回去面对现实。别人眼中的我，并不是真正的我。没有人知道什么是真正的我。于是也就没有什么会让我开心起来。所谓的龙岗开心故事集，大约也是梦里都不会有的东西吧。

起风了，乌云来了，要下雨了，且归去吧。

到五一的最后一天，阴雨后的宣城又一次绽放出蓝天和白云。骑行出发，穿过植物园，躺在新港的大草坪上。回忆充斥脑海，仿佛现在仍是大二的模样，我还在模电数电电磁场的包围里不得喘息，还在为单片机比赛而做着努力。那时的我失落到了谷底，没有人会叫我大佬，也没人会觉得我是。前尘忆梦，一切仿佛就在昨天，但却然已经不是昨天。现在的我，只不过是离社畜生活只剩三个月的咸鱼而已。

再一次穿过敬亭山的环山南路，在极速下坡的刺激中释放所有的失落。每一次当我心情不好的时候，我都会来到这条路上，让我的老伙计载着我以自行车的最高马力冲下山坡。也无数次想过，只要稍微偏离一下，我也就可以获得长久的安息。但或许，总有什么牵绊着我，流连在这个世界上。但我依然不曾找到，那究竟是什么。

下山不久，便来到了宣城中央绿地。曾经的这里只是一个荒凉的水库，而今却已完全变了模样。音乐喷泉的炫目灯光下，人群熙攘。在失去的两年时光里，宣城还是我熟悉的宣城，却也不再是我熟悉的宣城。回忆越来越多，却又在忘却中越来越少。就像列表里一个个太久不曾联系的老朋友，已经不知道该如何交谈。就像一个个我仍然熟悉的名字，但熟悉的可能也只剩了名字。

# 光·雨

一不小心就过去了半个五月，连绵的雨季让时光的流逝都显得有些安静。每天只能工作三小时的摸鱼论文写作机器，在奋战了三天后也生产出了六千多字的学术垃圾。眼前的生活似乎快要结束了，离开一所不那么像大学的大学，去向下一个似乎更像是大学的地方。过去的东西只能让他过去，而未来的东西还是要留给未来。毕竟，故事进行到这里，也差不多快要收场了。幕落，人去楼空，如此而已。

我仍未知道那个给我送奶茶的人是谁，但我并没有什么兴趣去知道了。会记得我很久以前写的一句话的人，或许会是旧友，但必定不属于亲朋，我不喜欢藏在暗处的东西。藏在暗处的东西，就像我自己的感情线一样，要么见光就死，要么无影无踪。姑且就当没有发生过。毕竟，这种注定已经没有结果的东西，是谁也不想去接受的。

关于努力和卷两个词的对比而言，努力和卷之间其实没有什么关系，我们可以说一个人在努力的卷，但也可以说一个人在努力摸鱼。努力是形容词或名词，形容某人做某事很卖力或者卖力做事这样一个过程。而卷是动词，应该与之对标的是奋斗这样的动词。从我的视角看来，奋斗和卷的内核有近似之处，但奋斗的本质是超越自己，是积极的；卷的本质是战胜别人，是消极的。超越自己不一定战胜别人，但自己获得了确确实实的进步；战胜别人不一定超越自己，可能是让大家都陷入无意义的工作。

从这种定义的角度上来讲，我和卷之一字几乎毫无关系。我所做的每一件事，几乎都只是单纯为了我自己的获得感。我对别人的好奇仅仅只在于一种上帝视角下的观望，而从不会把自己代入同一个棋局，即使我身在棋局之中也一样。

但这样的后果也就是，我的内心会总是感觉很孤独。我对所有的一切都不会有太多的代入感，我的生活日常只不过是一部持续更新的互动舞台剧。我会希望有各种各样的新剧情，对日复一日沉闷而繁琐的日常并没有足够的兴趣。既然是剧情，那么意料之外的结局总是常见的。因此尽管我仍会对未来有所期待，但我所期待的更多的则是一种对新鲜感的期待。

甘雨的剧情里，我很喜欢关于绝云间的孤独与璃月港的孤独那一段。应该没有人喜欢孤独，但相较于绝云间的孤独，人潮如织的璃月港的孤独，才是更深的孤独。因此我会选择独自旅行，独自去看我视角里不一样的风景。我期待一个旅行的伴侣，但旅行者，终究应该是孤独的。

橙光游戏《遇龙》的主题曲《遇萤》，讲述的是一场美丽的邂逅与错过。而我的人生里，不会有这般凄清的错付，只会有，故事里落幕的一场戏。当故事回到现实，终究也不会有人去奠祭。

# 浔·觅

当岁月的指针在光阴的催促下游走，迎面而来的是一年一度五二零的时光。记忆里依然残存的是一地金黄，但迷情的紫罗兰色却已然占领了那一片河谷。沉浸在阴雨连绵里的五月，稀松平常的每一天，只有在夜色里，才会获得一点不多的安宁。感谢，每一个夜色里守候我的灵魂。

大学的生涯就像北门外体育馆旁边的那片河谷，度过了大一大二金黄色的春天，经历了大三那年疫情缺失的遗憾，终于在大四的时候望见了薰衣草的紫色绚烂。足够浪漫，只不过是不属于我的浪漫。五二零，实验室五个人，两个没有打农药，零个人在学习，如此而已。

然而学长讲堂的时候看见一个小姐姐，感觉蛮可爱的。不过又跟我有什么关系呢？

就想起2019年的5月，当我第一次来到南浔的时候，这里的美让我震撼。湛蓝的天空下，小桥、流水、扁舟，古朴的建筑装点着水乡的浪漫。许是离乌镇、西塘、同里都太近的缘故，声名不显的南浔并不像其他几个古镇一样有着太多的人流。那时的我，曾经想过，如果我再一次来南浔，那么，定是要带着那所爱的人一起。

一晃，便到了两年之后的2021年。依然孑然一身的我，又一次来到了这里。经历过了无可奈何，便知道，浪漫不过是表面之上炫目的华彩。或许绚烂，但终究在华光褪去之后，不再有鲜丽的颜色。既已习惯孤独，那就不被孤独所束。我依然孤独，但似乎，也并不孤独。

人生而孤独，行于世间，无所依仗与救赎。祈愿于人海，欲寻求一人与之相合。然人海茫茫，所遇者众，只投缘者无多。人所以为人，盖心有九窍。而心生七面，善恶难辨，是故人不可尽知人，不可尽信人。

资本不眠，而浪漫不死。浪漫依文化而生，与文明共存。凡文明所在之处，皆有浪漫为注脚。俗世艰难，唯有喜欢。喜欢廉价，但是是所有一切里，最高雅的廉价。

人生有起有落，投资可赢可输，但浪漫不渝。INFP的喜欢是病态的喜欢，是非常执着的喜欢，但也是怯懦的喜欢。我可能就是所谓的回避型依恋吧，下意识就会回避情感，就会回避喜欢的人。我害怕亲密关系，会去执着的远离亲密关系。适度的亲近，对我来说，就够了。

所以不论在哪里，朋友总是越来越多，而后越来越少。太久不联系的人，消失在了我的记忆里，也消失在了对方的记忆里。直到很久以后，有人提及，可能才会想起，哦，当时曾经有过这么个人啊。

于是最后的最后，一无所有的我的灵魂，带上一无所有的我的躯体，在天地间独行。我渐渐忘了什么是浪漫本应有的意义。或许，世间的一切都很浪漫，除了人。

毕业论文的致谢，是我人生里写的第一篇致谢，也可能将是最后一篇。我感谢了带给我归属感和依恋感的一切，没有学校，没有家庭，没有出身，只有这整整一年来的点滴温暖。忘却所有不愉快的部分，才能赢得更好的结局。

一切的一切都有两面性，就纵使是魔都也不例外。上海不仅是陆家嘴和外滩，也有青浦的小桥流水。但浪漫，永不缺席。
