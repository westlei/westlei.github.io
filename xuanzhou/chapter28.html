# 年度总结

——一个loser的考研失败自述

# 前言

2020的一切，就像是一场梦，醒了很久还是很感动，只不过是自我感动罢了。这一年，过得熟悉而又陌生，一切都不再是从前的样子。不似现在，不见未来。我看不见光，却一直埋头追寻影子。可影子终究只是一个自己的投影，当自己丢失了自己，影子便也随之自然而然的消失了。光与影的变换，是大自然展现的一切里最美的部分，也是最动人的部分。

当你在逆境里，光让自己往前走就已经要竭尽全力的时候，看太远只会让自己掉沟里。不过既然自己已经在沟里了，那么怎么其实都无所谓了。所谓无所谓有，无所谓无，虽然听起来是那么一点自欺欺人的意味，但是真到了这个份上，那也无法可想。

从现在的角度来看，2020的失败最主要的原因是因为反复横跳的心态以及深入感情漩涡的无奈。其实结局本身不算特别意外，就是结局的内容略微超过了预期。数学大约比国家线低上了几分，这是已经预测到的失败主因，但专业课的失败却着实出乎了意料。为专业课投入了最多的精力和时间，也对它报以了最高的期待，也给予了我最重的打击。这个结局，也就像2020年的每一个故事，透支了一切希望，把冰冷的现实砸在脸上。大凡我的预期里存在最坏的结果的时候，现实往往会比我预期的最低点更坏些。

那么，便以时间轴的方式来复盘一年的落魄境遇，来看看，失败的人都是怎么度过这样一年的。

# 开年与旅居

2020年的开始平平无奇。赶在2019年的最后一天完成了科三和课设，于是2020年的开局便似乎无事可做。草草结束了四门毫无意义的考试以及驾考的科四，拿着新出炉的驾照开始了上路初体验，两名体验人员表示还活着真好。

简单拿上一点行李，我便开始了去往深圳的旅途。本想着不久之后就会回来，不成想再归来时却已然盛夏。走的那天雨很大，大约也预示了这一年的种种不顺。我遇上了一个很漂亮的女孩，尽管萍水相逢，但短暂的接触和交谈，却让我下定了决心，并找到了一个合宜的去处，考研的历程也就拉开了帷幕。想来我应该还是要感谢她的。尽管一切到最后还是失败了，但起码为我树立了一个目标，并带给我了一点努力的动力。

二零二零年的第一次合肥之行，是在雨中漫步天鹅湖畔。在这里，我收到了小美赛出乎意料的F奖结果。虽然这个奖对我来说意义有限，但从某种意义上，这是2019年的最后一抹余晖，也是2020昏暗的开始。掀过这一页，过去是2019的辉煌，前方是2020的惨淡。所有值得称颂的成功与骄傲，就像新桥机场前未化尽的积雪，终究会消散于人间。

深圳的风景很美，从宝安到南山，从盐田到大鹏，蔚蓝的海带给了深圳清新的风。呼吸着一月的南方温润的空气，忧心于现在和未来。期待了很久的爱情公寓5，少了曾经的纯真，多了几分世俗和现在。虽然不是最好的答卷，但终究也是回忆的终结。

突如其来的疫情以迅雷不及掩耳之势火速席卷了全国，并很快就蔓延到了全世界。在所有人都不曾准备好的时候，一切迈入了一个新的纪元，新冠纪元。在这个纪元里，一切都发生了改变，互联网终于完完全全渗透进了生活的每一个角落，我们看见了一个足不出户的世界，一个齐心抗疫的奇迹。人们逐渐熟悉了线上办理一切，熟悉了口罩和健康码，熟悉了无处不在的体温监测。政府的能力和效率在这一刻被无限放大，也许这只是一个开始，也许这寓意着未来。

# 花开与花谢

待到三月中旬疫情有所好转，我才终于回到了南京。这大约是2017年之后时隔三年再一次看到南京的春天。这个春天天气相当好，玄武湖的樱花绚烂地绽放着，南理工的二月兰少了游人光顾，更加浪漫。灯会的华彩在疫情之后重又映入大家的眼帘，依然夺目，依然炫彩纷呈。

考研也算是正式开始了，坚持每天背英语、看专业课，数学按部就班的走。网课让一切的安排变得更宽松了起来，也有空去水一水蓝桥杯，蹭一蹭学弟的大创什么的，生活闲适而自在。在网上认识了一个很温柔可爱的学妹，也算是挺投缘的吧。我确然是心动了，可当什么也没有发生的时候，她便和其他人在一起。似乎应当是了结了念想，却也未成想，这给后来的一切又埋下了些许引子。

网课本身的压力渐渐重了。时至五月，学期已然过半。压力渐渐大了起来，也曾经在下雨的夜里一气之下顶着雨骑过半个紫金山，最后在玄武湖边枯坐，寻求清醒。也曾从大行宫一路骑行到仙林，然后从汇通路坐上地铁深入灵山，去探访南京曾经最荒凉的地铁站。早晨出发经麒麟、仙林和宝华山来到镇江西津渡，中午横渡长江来到扬州瘦西湖东关街，傍晚乘着双层列车回到南京城。春天的日子过得自在也充实，或许那个时候的我，对考研还是很有信心的吧。

2020年的梅雨季节特别漫长，从六月中一直连绵了一个多月。反常的气候引发了长江的大洪水，长江的水位线刷新了历史以来的最高点。但似乎除了景观栈道的淹没，这场大洪水却未曾给疫情阴霾下的中国带来多么惨痛的损失。但也正是从梅雨时节开始，我的心态也逐渐趋于恶化了。考试，课设，答辩，一环套一环，压的人喘不过气，令人无所适从。考研的进度也就很明显的进入了停滞。但还好，有朋友们的互相支持，让我的心态没有足够的恶化。让我起码能够支撑着继续下去。

梅雨逐渐过去，烈日重新笼罩了南京城。随着学期的结束，考研的状态逐渐回到了之前的水平。进度仍在追赶，虽然追赶的很艰难，但是总有所收获。感情依然微妙的酝酿着，我有所期待，却又不敢期待。试图接近，却又疏远。这或许也就是我的矛盾之处。

在七月的最后一天，我花了一个白天，回到宣城，探访半年未见的学校。天很蓝，石臼湖很美，传说中的天空之镜不过如此。白云掩映的彩金湖泛着波光，蓝色的天与金色的复兴号相映生辉。一切都很美好，但我不配享受这些美好。无论如何，生活总得继续。

也许人生就像是一次骑行。穿过中山门看石狮子的威严，穿过汉府街看总统府人潮汹涌。路过颐和路体验民国风情，来到下关看历史与现代交融。骑上长江大桥的漫漫引桥会很吃力，但是骑上大桥上看到的江景却又是与众不同的壮阔。最终你还是要骑下大桥，在下坡路上万劫不复。

互联网+的校赛拿到了两金一银，但与我并没有太多的关系。我所负责的只有半本计划书和一个项目的基础。但这也正说明了凡是我所努力参与的一切，都会取得反比例的回报。越是尽力而为，越是输的凄惨。越努力，越幸运，只不过幸运的可能不会是我。

# 秋风与寒霜

渐渐的，返校的日子终于也来临了。稀里糊涂参加的夏令营没有什么结果，投出去的简历也死在了笔试或面试中。考研的进度在这一时期无限的逼近停滞，勉强结束了专业课的挣扎却也终究无力继续巩固数学。或许失败的结局在这时已经埋下，亦或是失败的种子早已发了芽。就像一场看起来热闹的生日会，热闹都是别人的，属于我的部分里面，更多的还是一个人的孤独。

感情在现实的接触后，逐渐偏离了方向。尽管她真的不是我喜欢的类型，但或许我真的就迷上了这种莫名其妙的好感。可我已经自身难保了，自然也无力去承载一份喜欢。也许我确实有过喜欢吧，但路是我自己选的，既然无法可想，那也无法可行。

国庆前后去了两趟合肥，去见了好些许久未见的朋友，也见了些从未见过的朋友。不管个人的处境多么艰难，但我还是喜欢跟这些朋友在一起的时光。仿佛就能远离现实，仿佛就能逃避现实。姜子牙好看又不好看，我会为妲己的境遇而感动，却又为整个狐族的境遇而无奈。我只记得，在那个大雨的夜里，金寨路高架上没有车，我开的很快。

整个十月过的平平无奇。刷题，看书，小毕设。政治一千题惨不忍睹到肖八勉强能看，专业课真题从大片不会到基本都懂。可数学依然没有太多起色，依然是低位上徘徊。逐渐没有信心，最终放弃了最初的理想，选择了一条看上去好走的路。却殊不知，这是一声为自己敲响的丧钟。

晚风中骑行在金寨路上，从翡翠湖到合柴，再到罍街宁国路。我终于吐露了内心，却也终究不能再如何了。秋日的天空显得高远，站在巢湖边上，漫无边际的湖水好似海洋。在中庙许下的愿望，终究也没能兑现，但也总算是维持了些许内心的安宁吧。

感情终究是起了些波纹。她分手了，她似乎想找我倾诉，可我选择了逃避。这个时候了，不该有的想法似乎就是不应该存在。但是心乱了，那就是乱了。也许这就是我最大的弱点，容不得一点感情的动荡。

于是她也看出了我的犹豫不决，巧妙地试探了我的关心。关心终究是掩盖不了的，但也不能说明什么。所有的科目都陷入了困境，或许应该换换心情，准备一下中期答辩吧，也就遇到了些很有意思的人。

# 雪落无声

一切随着十二月的到来而恶化了。心态在崩溃的边缘徘徊，一度感觉自己发烧，又一度失眠到天亮。她有了新的男朋友，是一个我很讨厌的人。我下定决心去切断与她的联系，可我还是忍不住会去想起。她从来也不曾与我在一个世界里，永远是我自作多情。

这个时候的我，已经觉得自己没有太多的希望了。教室一次一次的搬，每搬一次，心态就跌落几分。数学反正于事无补，那么便努力去学一学专业课，或许专业课能救起我的数学呢？但我忘记了，专业课的成败，并不只是取决于自己而已。

就这么惶惶不可终日到了上考场的时候。也许是上考场前满列表的支持，也可能是学妹临考之前的鼓励，考研的第一天居然过的很顺利，可考数学前我还是失眠了。平心而论，今年的卷子并不难，但计算量还是让我焦头烂额，不过好歹知道方法能写上点东西。一切都让本来濒死的我又回复了些信心，可惜只不过续了两个月又跌回了谷底。

再之后一两天里发生的一切，我依然不愿去回想。或许我一开始并没有多少恶意，但等到整个考研期间积攒的怒气喷发，一切便不可收拾了起来。我依然不愿意去想起她看我的眼神，她说的话。一切就像外面的大雪一样冰冷，冷到我不得不想去逃避。

翡翠湖的雪景很美。八人团的再一次相聚时隔了十四个月，但大家依然还是开心的模样。也许，2020的时光就定格在这里吧。

# 后记

2020的故事终究还是以失败落幕了。其实我预期到了，只不过没有预期到落幕的这么难看。专业课的压分使得我的成绩直接定格在了一个很低的位置，而数学又为我钉下了棺材板上的最后一颗钉子。

今年的考研形势无非是内卷出奇和哀鸿遍野，我至今也没有看见任何一个列表大佬晒出自己的喜报。我所能知道列表里大约很稳的也就是一个390+的哥们儿，但他的结果配得上他的努力。

从目前的情况而言，目前的前路有这么几条。可以出国，但疫情下海外的一切充斥着不确定，以致于这个选项并不是一个很好的解。要么二战，但是2020的一切给我留下的心理阴影让我有些犹豫不决。家里的意见是考虑进体制，毕竟也不缺钱，走体制混日子也不错。个人依然对大厂抱有一丝期待，尽管2020被大厂接二连三的pass，但我觉得只要准备再充分一点总有机会。以及如果我选择躺平，还有四条路之外的最后一条——到我实习的地方接着混日子。

也许loser就是我这样子的吧。
