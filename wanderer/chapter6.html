# 匆匆岁月流逝

南京地铁更新了2024版的毕业祝福：

岁月匆匆，惟愿总相逢。南京地铁祝福毕业生毕业快乐，愿大家去更远的地方，见更亮的光。

相聚别离，都是刚好。南京地铁祝福毕业生毕业快乐，愿大家在未来的日子里，前程似海，一路繁花。

从南京南站一路向南，大大小小的学校数不胜数。南航、河海、金肯学院、南航金城学院、南工浦江学院、江苏二师院、南财红山学院、南京机电学院。如果从高淳继续向南，再有六十多公里，还有隐蔽在山间小城之中的牛职院。似乎，这两天也正好是牛职院的离校日。

每到一个临近学校的地铁站，毕业生祝福语便会播报一遍。但毕业于我而言，已经是一个有些古老的名词。三年之前，依稀记得是个有点下雨的日子，从牛职院驱车向北，跨越石臼湖大桥，越过这一座座大学。而随着今天这一批人的离校，在校期间与我有所交集的人也就全部离开了。再回去，似乎也没有什么大的意义了。

毕业以后这么些年，在不同的岗位间周转，在各地研究所里游荡。说混的好也算混的还好，有说得过去的工作和说得过去的收入，起点已经可以比肩不少硕士。说混的不好也混得不好，完全看不到上升空间的工作状态和一片混乱的生活状态让我收获了十分糟糕的精神状态和健康状态。曾经还想说或许有什么契机可以改善一下，如今的我只觉得维持现状就很难了。

闲暇之时，看看大江大湖大海，也不失为一种生活的慰藉。毕竟，除此以外，我的生活只剩下双眼无神的麻木工作与看管人扮傻子了。当然最近也沉迷于剑网三，不过游戏总是一阵一阵的，过段时间估计就咕咕了，就像我不知道多久没打开过的帕鲁一样。

昨晚做了一个非常离谱的梦。梦见我从坂田打车去深圳站(罗湖)赶高铁，然后等了很久很久，高德才有人接单。显示是距离我12公里，从平湖过来。等他来的时候，已经快来不及了。然后我问师傅从平湖过来这么远接单，师傅说他是从塘厦来的，更远。

接上我师傅就往布吉开，我跟师傅说应该走坂银通道，然后师傅完全不听，一直开到了沙湾。这时候差不多到发车点了，于是我跟司机大吵了一架，就感觉血压噌噌的往上升高。最后师傅把我丢在了丹平快速路上，我汗流浃背的醒来了。

昨天下午小转了一下绍兴。南京出发到杭州西两个小时，杭州西坐地铁到鲁迅故里又得两个小时。雨后的小城里，屋檐依然滴答滴答的落着水。闷湿潮热的空气里散发着臭豆腐与黄酒交杂的发酵气味，熏的人有些迷离。

仓桥直街跟江浙沪其它地方的古镇水街并没有太多的不同，最大的不一样大概就是别的地方卖奶茶，到这里全变成了黄酒，连冰棒都是黄酒冰棒。转一圈再到鲁迅故里的时候已经是晚上，免费的预约票早已售罄。门口看见有家面馆，想想还是吃一顿。杭州的面总的来说我还是喜欢的，虽然说浇头感觉怪怪的，不如苏式面也不如南京的面。

# 风雨夏日

24年也过去半年了。

昨天喝了点酒，度数不高，但是喝完第二天早上起来就有点头疼。出去溜了个五公里多点的小圈，脑袋就昏昏的了，果然又是一个躺着半梦半醒度过的下午。

明天就是24H2了，上半年什么都没做，就这么浑浑噩噩的过了。去年底接的项目，半年下来就写了半个demo，倒是进了一些奇奇怪怪的工作组，跟我刚入职时那群级别高资历老的leader们排在了一起。只能说没想到在外面转了一圈回来还是面对这些人，不过位置倒是从弟中弟混成了合伙人的代理人。但是级别没升工资没涨，所以事实上还是弟中弟中弟。

在事业毫无长进的同时，生活也还是同样的混沌着。健康状态几乎没有任何改观，看病检查做了一半觉得麻烦不做了，智齿拔了以后天天卡各种东西，腹泻和呕吐也还是每周必有的常客。对象依然是看不见摸不着的虚空概念，半年里线下打过交道的女生除了同事和前同事好像应该就只有一个，工作以外除了看v就是到处转悠。

铁道抽了阮梅花火知更鸟加流萤，但是练度烂的主线任务都过不去。原神几乎啥也没抽，娜维娅后面就希格雯，因为剧情烂的没有想玩的欲望。雀魂有活动偶尔打，没活动登陆都懒得登。农药的话因为领导打所以我偶尔也打，靠着两个英雄上了波王者，但是基本上也打不动了。最近沉迷剑网三，但是热情消得很快，已经懒得去开地图主线了。

上半年一共26个周末+节假日，含7个单休，三个三休，两个长假。出南京玩的话大概是23趟，可以说每个双休以上都有至少一天不在南京。不过长途只有西安和珠三角，剩下都是长三角的周边城市，其中苏州四次，上海三次，杭州两次。大部分出行没有明确目的，甚至也不一定会造访多少景点，只是出于一种对生活的逃避和厌烦而出行。就像跟朋友说的一样，如果出门没带脑子，那我的双腿会自动把我带到南京南站。

剪了26个plog发阿B，加起来浏览量2500出头，主打一个粗制滥造只给自己看。同步到阿抖的精简版24个，加起来浏览量530，主打一个仅好友观看但没有好友。QQ动态和朋友圈大约各100多条，频率大约是1.6-1.7天一条。微博发了25条，频率在一周一条。welink水群记录估计五万条起步，大概每天上班都要花半天以上的时间闲聊。简单的说，废话是很多的，能聊天的人是很少的。拿夜宵的次数是40次，也就是说加班仅40天，别的日子都是六七点之间下班。

这堆数字好像能说明点什么，但事实上什么都说明不了。大概就是说，现在的我挺闲的，但是很无聊，生活中充斥着无所事事。当然如果你说我一定要做点啥的话，那我肯定是抗拒的。生活就是这样，维持现状总是好的。

毕竟，社畜生活的第三年结束了。

在华为兜兜转转，深圳、上海、南京，三个城市，三个部门，做不一样的内容，但是差不多的方向。看了看个税app，税前收入差不多刚好一百万，但是存下来的不到五十万。剩下的五十万去哪了，我也不知道。

三年过去，跟入职伊始没有任何不同，待遇，级别，能力，什么都一样。最大的区别，年龄增长了三岁，心态像张锡峰般颓废。活着很好，死了也没事，甚至前半句都可以省掉，只留下死了也没事。

犹豫和纠结才是一切失望和落寞的根源，打败自己的也只是虚妄的幻想。这句话大约是我18年的时候写的，但六年后看着似乎也没什么问题。只能说以前的我的文字水平，跟现在完全不在一个水平上，不读书使人愚钝而庸俗。

但是社畜生活的第四年开始了。

# 赛博桑海宁

之前突然意识到公司WiFi的出口是上海华为云机房，所以上班的时候我依然是赛博上海宁，在各大平台的IP归属都是上海。

在上海待的一年，应该说大多数的时候都不太愉快。但是从华为管理最松散的部门到管理最严苛的部门，对华为底层逻辑的理解更加深刻了一些。我大概不太适合华为，但在目前的情况下，也没什么更好的去处。

去年这个时候，有些期待，有些感慨。去年毕业的那批人贯穿了我大学的下半场，乃至我工作以后的生活。今年又到了毕业季了，但是今年毕业的各位只能让我感到陌生。不知道明年的话又是一个什么样的景象了，或许我那时候也毕业了呢？

这个月的账单有些离谱了，钱如落花流水般散落不知归处。流萤和知更鸟一起也就花了648+128，乱七八糟的东西也没买多少，赶紧把原计划当礼物的东西拿去转手卖了回回血，换一件没那么贵的当礼物。当然其实不送也可以，毕竟我一直也没有送礼的习惯来着，但恍惚记得我好像说过我要送了，再不送也不好来着。

在城市化如火如荼的中国，每座城市都有一些边缘的郊野片区，依然停留在城市化早期的乡村集镇状态。连接这些边缘村镇的道路大多修建于上世纪，路宽十米左右的双向单车道，也就是都市天际线里最基本的那种公路。

由于多年的发展，道路两边混杂着各种各样的低矮居民楼、商铺与乡镇工厂。由于这些区域地处边远，不具备拆迁的价值，所以周边景观水平一直停滞在上世纪八十年代，完全看不到一点现代化的气息。

上海的川南奉公路就是一条非常符合这一说法的老路，始建于1935年的它目前全长大约在60公里，从川沙的华夏高架路一直到奉贤南桥路。向西对接南亭公路至松江亭林，并可进一步通过亭枫公路连接到沪嘉边界的枫泾。向北对接东川公路至濒临长江的外高桥高东板块，构成一道与郊环高速基本平行的浦东-浦南大外环。

东川公路-川南奉公路及其贫困带的存在，使得川南奉公路至东海之间出现了巨大的发展空白区，空白区内只剩下了农田、堆场、垃圾场、火葬场。所以大家一般说，上海是没有海的。空白区里最大的建筑是知名的浦东机场，而机场航站楼到川南奉公路直线距离也有四公里，因此机场成为了一个巨大的孤岛，从航站楼到最近的川南奉公路上的江镇居民区都得11km。

川南奉公路在大团镇与四团镇的交界处上跨S2沪芦高速并由南北向转为东西向，同时由浦东新区进入奉贤区。这一段的川南奉公路同时也作为老南汇区与临港新区的事实边界(规划中临港的西界一直扩到黄浦江转角到奉贤海湾碧海金沙)。从市区方向穿过川南奉公路以后向东/南约七公里，到书院/泥城/平安，才能重新看出些发达片区的影子，而临港中心的滴水湖片区，则要向下更远一些。严格来说，临港更像是一座独立的城市，与曾经的老南汇并没有什么直接的关联。

南奉公路的奉贤段直接的穿过了四团、奉城、青村、南桥四镇的正中心，也几乎穿过了整个奉贤的繁华地带。四团以南的平安/芦潮港随着特斯拉超级工厂的入驻而稍有繁华，但奉城以南的燎原就只有海湾森林公园这样一个相当原生态的保护区。青村以南有上海最大的人造沙滩碧海金沙，但经营不甚良好，南桥以南则是上海交通最不方便的大学城海湾大学城，属于若干年前规划失败的遗产。发展不过川南奉公路，是地理因素的决定，也是规划不善的现实。

# ZZZ

绝区零是一个说实话不是很能玩的下去的游戏，探洞探的让人非常困倦。但他把距离保底的抽数明明白白的写出来，才会让人说在极度不想玩的情况下，硬是去肝那么点菲林抽鲨鱼妹。如果这个保底歪了，那我大概率是直接卸载游戏再也不碰了。

跟抽卡游戏一样，人生也是需要正反馈的。华为大概每半年一个升级窗，就好像一次抽卡，你把你半年的兢兢业业放进去抽，看看能抽到升级还是加薪。每一次抽都一无所有，玩家的游戏体验就会变得稀烂，以致于出现不想玩，想退游的情绪。

可惜正如一个米哈游账号只能玩一个绝区零存档，一个人一辈子正常情况下也只能进一次华为。这就导致删档重开面临的压力大大增加了，沉没成本会让你舍不得退游删档，但是非气四溢的号又很难拯救。此时陷入的局面就是两难之间。

感情的问题比这来的会更复杂一些，但总体的逻辑并没有什么区别。当一段感情迈入死胡同以后，割舍与否都是一个困难的选择。无论花多少心思去弥补，渐行渐远的人之间裂痕已经产生，便没那么容易消失。

但如何避免自己陷入这样的局面呢？当机立断的割舍是一种办法，但从一开始就不要陷入这样的局面才是更好的选择。濒临24岁的老社畜并不是完全不渴望爱情，而是感觉持有一段走不下去的感情从一开始就是一个错误的选择。

最终，选择一切都不开始也就变成了选择两手空空。两手空空让我摆脱了感情的困扰，也让我变成了精神层面上完完全全一无所有的穷光蛋。活着挺好，死了也没事。这就是最适合我当下的，多么美好的精神状态啊...

不过呢，由于我并没有什么感情生活，所以我的日常生活并不需要花很多的米在培养感情这方面。因此我的金钱主要的去处只有两个：文娱和出行。虽然这并不能解释我为什么花掉了好几十万，但是能解释我为什么会有一堆时尚小垃圾，以及车票本为什么满了。

文娱类开支主要包括两个数字，一个叫648，一个叫138。648是某些二次元游戏的充值消费，138是某些主播的上舰消费。在我的现实生活还能支撑起我的精神时，主要的消费都在前者，比如氪金抽心海。在我的现实生活已经摇摇欲坠，精神世界彻底混沌以后，主要消费集中在后者，开了估计有十几人次的舰长。

二游与虚拟主播的共性是虚拟感，二游的虚拟感是让你进入一个想象中对你还不错的世界，从而获得慰藉。虚拟主播则是在虚拟中度过现实，屏幕对面的皮套人大约是个精神状态比你还糟糕的暴躁老姐，听她时而幼教、时而下头、时而歌唱、时而骂人。对方的精神状态经常比你要烂，比如某个喜提11k舰长数的三十高龄老v；对方的身体可能比你更烂，比如看到一个转氨酶高达2700(正常人40左右)而被送去住院的小V。你看着他们比你更惨，从比烂中获得慰藉。

出行类则来自我对中国铁路12306的长期大力支持。毕业前三年才能用完三页的票本，在上海的一年就能用完三页。同时曾经以绿皮深夜档为主的行程如今也变成了白日高铁四处漂泊。曾经宣城到南京绿皮转动车加补票、南京到青岛一趟车买四张票，这种行程如今体力大约支撑不起，但我甚至还妄想体验从敦煌坐绿皮速通拉萨(最终未能成行)。打纪念票也渐渐增加，之前只打过一个宣城高铁首发纪念票，后来又打了重庆站停运纪念票、广东城际接管莞惠纪念票。

然而就像褪色到完全看不出印记的车票一样，大部分的旅途在我的脑海里淡化模糊，甚至于我需要找找对应日期的照片才能想起我这一天去了哪里做了什么。旅行的本质还是逃避，一种基于南京南站/上海虹桥/深圳北站的快速逃逸式逃避。逃避真的有用吗？只能说我能勉强逃过周末的空虚，但依然逃脱不了上班时的痛苦。就像熬夜一样，越熬第二天越难受，但明天的事自有明天的我来解决，这也是不错的心理欺骗。

可以说，上班对我来说是一直是一件相当痛苦的事情。

一般来说，正常人上班的痛苦普遍来自于事业上的停滞、工作内容的焦虑以及早起和加班的绝望感。一想到多年毫无长进的工资和稳定垫底的级别，有的人，比如我，就像死了一样。工作的动力是一点也没有的，自然工作的产出自然也是听天由命的。

如果把上班的痛苦具象的描述一点，那大概就是上班的旅途本身。家的海拔大概是十米，而公司的海拔大概是五十米。自然，四十米落差的存在就导致我无论是骑车、步行还是使用任何并非点对点的交通工具，上班的路上都要进行一个坡的爬。尤其是在骑车上班的时候，濒临终点的一百米总是喘的像死了一样，我的心率也一度达到过196的恐怖数值。

可一晃已经来南京九个多月了，这条上班路也已经走了两百多天了。现在上坡虽然依然大喘气，但是心率也只不过162而已。熟悉的路途让我也不怎么再会早起，大概看着时间差不多就行。工作上虽然还是个废物，但是不怎么焦虑，也不怎么加班。毕竟工资毫无长进，级别稳定垫底，那努力有什么必要的意义呢？

于是我偶尔也开始会去期待一点生活之外的东西，比如说符合前LGBT时代一般普通人价值观里所存在的那种爱情。

但我的生活之中甚至没有任何一点异性的影子了。

八月了。

# 山海彼岸

海的那一边是什么？

海的那一边还是海。蓝色的海，绿色的海，黄色的海，灰色的海。

两天1300多km的行程，相对于我之前的行程来说，绝对算不上很远。但从疲惫感来说，这次的行程超过任何一次，以至于回来以后感觉还不如去工位加两天班来的舒服。大约是因为从一开始就没有一个规划的原因，一路上充斥着奇奇怪怪的意料之中与意料之外，也给我带来了足够多相当新奇的体验。

但我好像没有什么可以分享的人，也失去了写下这些东西的欲望。新奇、浪漫、感慨或者恐惧，都是仅属于那一刻的感受，描述不出，也体会不了。crj900小飞机的飞行，落日时分高空之中的摩天轮。小电驴环岛悠哉悠哉的兜风，在能见度不足五十米的大雾里骑在大桥机动车道上看后面车来车往的恐惧。由于大量航班延误而人满为患的三江码头，由于返程客流太多而吵吵嚷嚷的李柱山码头，高温天里空无一人的石柱海滩、东海渔村。当然还有晚上在酒店的海景房吃着外卖看瞳子姐姐的三周年，单纯是因为店里人太多还要等位。

平心而论，舟山的景点相当不适合一个人去，因为交通困难而且收费高昂。所以主要的景点这次基本上都没去，比如东沙的蓝眼泪，比如普陀山的大佛。可我并没有专业相机去捕获来自大海的一点蓝光，也没有什么需要去向大佛进行祈福的诉求。错过了所有的日出，因为我起不来，但是即使起来了看见的也会是大雾弥漫。没有在最好的位置看日落，但摩天轮上的日落和东海大桥的日落也相当不错。或许一个人出行本身就是一种完全没法考虑性价比的事情吧，尤其是时间还不够的情况下。

经常有人会说抑郁症怎么怎么样，我现在觉得抑郁症可能就是一种精神层面的炎症罢了。发炎的本质是一种机体自我的防御反应，来自于各种致炎因子的共同作用。有的人是急性，反应剧烈；有的人是慢性，看不出什么特征。大部分的消炎药主要是抗生素，消除了导致炎症的外因，也就能治愈了。而对外界因素引起的抑郁而言，这同样有效。

慢性抑郁就像慢性肠胃炎，平时没什么感觉，但经常动不动就胃疼一下，并且很难去有效治疗。我感觉我或许就是这样一个阶段，动不动就会低落一下，但又难以治疗。厌恶社交，不想认识任何新的朋友，也不想去接触那些很久没接触过的人。世界大抵就像虚环，谁都可以是主角，每个角色都有自己的生活，除了我。
