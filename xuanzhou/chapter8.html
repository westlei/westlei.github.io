# 岁月神偷

# 七月七日晴

一年的光阴，不过是一晃而逝的岁月。曾经的伤痕，早已在岁月的击打之中抹平。回顾过往的时候，早已分不清楚，哪些是真的，哪些是假的。可能除了我整个世界都是假的，也有可能只有我这个人是假的。不知真假的岁月，只剩迷茫。

学习日渐失去了动力，整天浑浑噩噩地过着。打着各种各样的游戏，却不知为什么要打；学着乱七八糟的东西，却不知为什么要学。似乎期待着什么，却又不期待着什么。如今的生活，犹如一潭死水，失去了活力。

不再相信任何的所谓“我喜欢”，也不再期待任何的所谓“我想要”。所有想的，要么去做了，要么放弃了。自觉远离了所有我喜欢的人，不想为对方，更不想为自己带来烦恼。只剩困惑，焦虑，麻木，不安。世界，本不是彩色。

可能从我决定离开的那一刻开始，快乐也就不复存在了吧。回想这一年，难得快乐一点的时光也就是难得的几次出行。暂时逃离了压抑和痛苦，只是为了让自己更坚强些。路是自己选择要走的，我没有理由去畏惧和逃避。

一个人走过了冬夜的西湖，去看了那外滩上的一场大雪。唯美的南浔和甪直，辉映着观前山塘的夜色。拙政园里的意趣，金鸡湖畔的夜晚；徽杭古道的清风，钱塘江上的华彩。随影，随行，唯一朝心动，无片刻犹疑。

当她离开了我的世界，也带走了我的最后一抹灿烂的青春岁月。理想终究向现实低下了高贵的头颅，在虚幻的游戏世界里拜托自我。彻底地放纵自我，彻底地麻痹自我。能推则推能拖则拖，却发现大二已经走到了尽头。

七月七日晴，忽然期盼着大雪。不敢睁开眼，希望是我的幻觉。我站在外滩边，眼睁睁看着雪，覆盖我来的那条街。

# 暑热

暑假的时光在云顶之弈间飞逝，转眼三下乡结束一周了，却还没找回该学习的感觉。新的日历揭下，日子逐渐好了起来，也许2018的黑暗只是短暂的一瞬，2019的前途仍然泛着微弱的光明。

三下乡的三天，看学弟们自由自在的生活，突然心生艳羡。似是因着转专业的缘故，大学生涯虽已过半，却仿佛刚刚开始。新的朋友和新的故事，在伤痛的过去里抹上了一层薄薄的壳，一击就碎，但也聊胜于无。

未来仍然迷离着，但也许找到了那么一点希望。没有在深渊里越陷越深，已经是莫大的幸运。不想再有意无意伤害任何一个本心善良的人，疏远才是最好的成全。不后悔，因为没有开头的故事，不配结尾。

已经逐渐不在意过往的提及，也不在意失落的曾经。半年的底谷里，学会了不信任每一个人。我选择顺从那脆弱而敏感的内心，顺从那所有我的心选择亲近的人。它已容不下，再一次的伤害。

# 繁星

夜幕下的繁星，高悬在浩瀚的宇宙间，看人世百年，不过一瞬之间。地面上的凡人，为不曾明了的未来操劳，看满天繁星，不过一闪而逝。下半场的号角拉开了帷幕，大佬们开始了渐趋于白热化的厮杀，迷茫的大多数仍在学海里挣扎。多愁善感的夜，总有些想念，想念清风，想念明月，想念那片星的海洋。但也只有失去，才会让人更加的珍惜。

逐渐忘记那一开始的目标和理想，转而追求那每一种不同的体验。在江浙沪的山水间漫步，在实验室的器材间穿行。沉迷在虚妄的社交网络里，只有在旅游和竞赛的时候，才会放下手中的一切，去寻求更极致的快乐。不再去后悔每一次选择，当你走过了一个路口的时候，它就已经永远消失在了这个世界里。时间一旦远去，就不会再回来，就像有的人，一旦错过，就不再。

# 宣州行记

余自金陵而下宣州两载有余，伤于心，困于命，渐至于今日之败。每思之，夜不能寐，两载故事犹然在目。初向学，名每列于众人之先，与诸友相偕，或舞文弄墨，或游山涉水，心康而体健。后渐迷于色，心有所指而意不足，久而知无所成，遂生去意。既去，课业多艰，适情有所困，学无所成，心焦而身颓，遂一败涂地。既败，闷闷而不乐，郁结于心，头益秃，体益肥。后迷于赛事，辛勤数月，略有斩获。然学业益艰，向学之心益减，思及往后，则惶惶不可终日，唯以暴食并通宵游乐而自慰。愿改过而自新，然无力为之。是夜，复思及此，遂书此文，示来者，且以学业为念，勿蹈敝人之覆辙。

# 释然

那个有些遥远的夏天后，生命的颜色些许的黯淡了。没有那么多希望，也没有那么多值得。没有了心动的感觉，消散了人事的纷争。也许，只是单纯的老了而已。也许，是岁月终究磨平了心灵，把未来照进现实的侧影。

冬天若即若离，在各种各样的选择之中犹豫，在各色各样的劳碌间挣扎。爱之一词沦为禁忌，就连喜欢两个字也不敢轻易说出。仿佛一开口，所有美好的东西就消散的无影无踪。

曾经的我，给每一个好友都写过一段话，也许他们并不知道，也不会知道。在心中占据一席之地的人们，渐渐走散，消失在时间的路口，消失在列表的尽头。以前总觉得失去的总会珍惜，可如今，什么都不剩的时候，反而释然了。

没有梦想，孤独的走在不为人知的沙场，在水天一色里流浪。就好像那一个个夜晚的旅途，不知何时开始，累了才会结束。形形色色的人，有一天会遇见，有一天会消失。能相逢于这个偌大的世界之中，本就是一种缘分。记得那最后一眼，殊不知一眼万年。

曾经，有过一个大雪的夜。唱了最喜欢的歌，却再也找不见，不归的旅人。

# 若梦

当久不曾见过的学姐突然提起了鹅叔，不禁回忆起昔日的历历。那年种下的因果里早已书写了悲剧，只不过在两年之后补上了一个结局。从此喜欢上了白酒的香气，在悲伤的寒夜里酗酒成瘾。让高度的酒精流进脆弱的胃里，留下凌晨醒来独自痛苦的记忆。

当空间里突然提及了纷飞的大雪，却又唤起了去年的如今。来宣城的第一个大雪的夜里，与那也许再也见不到的人一起唱过的歌曲。凌晨四点的西子湖滨和雪花飞舞的外滩烟云，雪后清寂的甪直古镇和夜色璀璨的山塘风景。也许并不是喜欢的感觉，但无可磨灭的是那一份独特的回忆。在那艰难的岁月里，如同一杯温热甘甜的可可，融化在了我的心里。

时光恍然若梦，或成或败，也算见过了许多大多数人见不到的风景，经历了许多大多数人没有过的经历。在某一个瞬间，也曾感动过，收获过。回望悠然岁月，似也不虚此行。

# 一期一会

# 轮盘

人生有如一个无止尽的轮盘，三载一期，一期一会。见过了胜利，也目睹了流离。通晓了秘密，也没有了好奇。迈进一个又一个新的阶段，却也总逃不脱这一期一会的铁律。转眼，又是新的一期。这次所会的，却又是谁呢？

# 叶子

叶子落了。无从知晓它是什么时候落下的，也无从知晓它的下落如何了。它就是去了。也许它曾经有过许多故事，听过风的歌唱，到过云的酒席。筵席的过往，云烟般消逝。空灵的歌唱，未闻的久长。只有误会，在时光的发酵里越陷越深。岁月的痕迹，终究在那时光里不见了。只知道，一切都过去了。

# 风铃

每当海风吹过，岸边的风铃发出悦耳的清响。喜欢听那风铃响起的声音，尽管它的主人并不是我。清脆的叮当声，一声一声拨在那断裂的心弦上。仿佛穿越千年的伤痛，得到了温暖的轻拂。迷茫的时候，只有你，能拨开我眼前的云雾，重见那阳光洒落。

# 霜露

寒冬的清晨，霜露静谧地覆盖了大地。独自行在那寒夜里，陪伴那霜露，从生到灭。倾听它的故事，却走不进它的世界里。那是一片纯洁的白，来的快，去的也快。贪恋它的美，只会让它更快地消逝。且随着它的本性，任它成长，任它自由飞翔。

# 茉莉

茉莉的清香，花的芬芳。溢出了花瓶，充盈了世界。给所在的地方一缕清新，把污浊的凡世短暂清理。好看的你，一朝风景，带来了好心情。折一支茉莉，插入那天地。闻花未名，听花无语。恬淡的芬芳，不浓烈的香，却不经意把光彩带给全世界。

# 流星

流星划破天空，与你相遇。是千年难遇的时机，还是百世轮回的缘尽。乱入了你的世界，我很抱歉。生如流星，在宇宙间走走停停，被每一颗星影响，也影响每一颗星。一许流星划破天空，带着耀眼的光芒，让人沉醉，令人倾心。向流星许愿，不愿过去，只愿未来。

# 朝阳

当旭日从东方的海面上升起时，心中充满了羡慕。那自带的光环是多么的耀眼夺目，一瞬间就把一切陷落在尘埃里。它的卓越，似是与生俱来。出生便是焦点，吸引着万众的目光。卑微如我，仰视着那冉冉升起的朝阳，溢于言表的，那欣赏和钦慕的神情。

# 月出

深夜的天空，是明月的天下。傍晚时分的东方天空，那一轮明月初上，似是未曾圆满，略显稚嫩。但不可否认的玲珑可爱，给那初升月光别样的色彩。小幸运，小喜欢。可爱的人，总能受到更多的青睐。

# 月明

子夜时分，明月高悬，褪去了稚嫩，一副清冷圣洁的模样。凌然乎天地，做午夜的王者。清辉下的倔强，明朗下的高傲。也许，是你的努力打动了我。也许，是你的能力震撼了我。也许，一切本没有也许。优秀，不因黑夜而丧失光彩，不因无人欣赏而不再绚烂。

# 落雨

初秋的雨，凄清而又美满。细雨霖霖，滴在那青青草地，滴在那秋叶林里。一朝相遇，惊艳于那凄清的高雅，震撼于那博学的淡漠。冷的恰到好处的水滴，和着雨音降临人世间。雨落升烟，缭绕着雾霭的迷茫，渐渐飘落，入那心房。

# 从一

万千星辉，从零而始，从一而终。眉间的雪，今世的缘，终在那理想和现实间化为了永别。从一而至，从一而终。不曾开始，那便没有尽头。羡慕那坚持的勇气，感叹那不改的初心。不是喜欢，只是赞叹。无所畏惧，直面未来。

# 匆匆

匆匆乍见，匆匆一别。匆匆过客，匆匆那年。便是缘，今生最美的遇见。2019，十个印象最深的女生。让我从深渊重新踏上大地，感恩过去，迎接未来。感恩2019，遇见了许许多多的人，一路的扶持，一路的鼓励。若未来可期，最需要感激的，莫过于你们。
