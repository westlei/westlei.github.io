# 日薄西山

# 仓皇

看着学弟学妹们踏入了转专业的不归路

听见那远处在争夺奖学金名额的号角声

预备考研的钟声在前方一遍一遍地敲响

似乎没完没了的比赛掩盖了黯淡的迷茫

总是会无意识地伤害到自以为亲近的人

总是会忘记自己究竟需要的是什么东西

总是会在同样的错误之中循环而又往复

总是会栽在自己所珍视的每一件事物上

别人的故事里，有着快乐，也有着悲伤，

但也许你只看到了快乐，却看不见悲伤

自己的故事中，有着选择，却没有悔过，

你为你做下的一切负责，永远不能逃离


我累了

但我想逃离这一切时，才发现无处可逃

# 明天

从愚昧之巅走向绝望之谷的谷底

从人山人海迈向灯火阑珊的流离

逐渐找到一点前进的方向和动力

却有人说我在做的一切毫无意义

不曾想过前方的光明出现在哪里

仿佛一个只会怼人还暴躁的笨逼

终究会在挣扎的末尾去寻求放弃

那故事的开始所谓追寻又是何必


十八岁的最后一天

厌倦眼前的一切 期待未知的明天

# 输赢

当失败的铁拳一记又一记地落在我的头顶

当现实的利刃一道又一道地戳进我的心灵

仿佛神魂仍在那梦里，不曾醒，也不愿醒

仿佛宣州秋日的细雨，不曾停，也不必停

就让我一直在梦里，无需面对现实的困境

就让我还活在过去，依然拥抱美好的曾经

若再来一次，必绝不犹豫，总好过，困于情

若再来一次，必坚持自己，何必想，输与赢

# 抑郁

终于似乎忙完了手上的事情 只感觉累

不知道是连绵十天的课伤尽了元气

还是突如其来的寒潮冻结了热情

也可能是单纯的心生倦意

最近抑郁症的话题逐渐火热

我也在想 是什么支撑我走到现在

好像过去的绝大多数时间都不尽如人意

但似乎也没有糟糕的极点的地步

一年多噩梦般的经历 失落在渺茫的岁月里

迈着颤抖的步伐 从天堂走向谷底

似乎活跃的表象 掩盖着日渐封闭的自己

心灰意冷 为现在所拒绝 为过去所抛弃

也许说 曾经努力的动力 多么荒唐可笑

再看来 再荒诞的缘由 总胜过于没有

为何会去死亡 似是 找不到活着的理由

无所牵挂 无所念想 无所欲望 无所寻求

心空灵 神沉寂 心痛的感觉 崩溃的瞬间

感谢忙碌 感谢忙碌让我避开了空虚的想

感谢酒精 感谢酒精在难受的时候麻醉了我

也感谢艰难的日子里陪伴过我的每一个人

既已注定了众叛亲离 何不开始就选择自闭

至少 小确丧的日子 没有心痛的抑郁

# 陷落

梦终究陷落在天亮的时候

就像每个故事都有始有终

像不曾报以过的零星期待

像晚霞终究会落下成夜幕

都过去了 一切都过去了

失落而迷离的情感

并一切的休闲享乐

都过去了 一切都过去了

不愿回忆那些往事

不曾请求所谓谅解

如同所谓酒不醉人人自醉

迟来的答案与残酷的现实

这世上本没有一见钟情

更多的都只是一厢情愿

可能，这是最好的一天

也许，这是最坏的一天

# 冬来

今年的冬天，似乎比往年要来得更早些。

还算温暖的白天过后，清冷的夜晚降临。

读着花呗账单数字，内心多了些许寒意。


实验和作业如一把把尖刀，戳进了心口。

但随之而来的无尽考试，才是透心的凉。

看起来，这个冬天并不比上一个更好些。

# 天气之子

这个世界已经足够疯狂了

那我再疯狂一点又如何？

我可以失去晴天，但我不能失去你


但愿每一个世界都不会被淹没

但愿每一份爱情都可以被祝福

但愿每一对情人都能终成眷属


祈愿这艰难的岁月能够过去

祈愿最后能守得云开见月明

祈愿美丽善良的人都有好运


鸟居沦落了 天空犹在

江户淹没了 不变是爱

庙宇坍毁了 神还是神

# 散落

不知道该不该留的人 那就不留了

不知道该不该回的话 那就不回了

不知道该不该做的事 那就不做了

不知道该不该活下去 那该怎么办

想找一个人 好好说说话

却发现 整个列表都找不到了

当过去全部散于云烟

这世间里便似乎没有了我的位置

不想再去帮任何一个人

也不想再去操任何一份心

把世界活成NPC的自娱自乐

# 往昔

转眼 一年 想念515的曾经

想念天边夕阳 想念大雾弥漫

想念瑞雪皑皑 想念白云一片

想念魔都 想念甪直 想念姑苏

但 过去的终究已经过去

只剩照片里 依稀的过往 依稀的快乐

在那电信科的一开始

其实 岁月静好 云淡风轻
