# 坠入深渊

# 深渊

当你凝视着深渊时，深渊也在凝视着你。

从不曾向现实低头的人，从深渊之上走过，无所觉察，亦不曾沦落。屈服于命运的人，俯视深渊，被无尽的深邃吸引，陷入其中，便无法自拔。

深渊永无止境。当你向命运低头的那一刻，就是陷入深渊的开始。待到愈陷愈深时，即使恍然醒悟，却已不得脱身，只能沉沦。

也有的人在短暂的沉沦后重又振作起来，无视那深渊的凝视，抬头远望，从深渊中爬起。但更多的人，却无法抗拒深渊的凝视，而坠入了深渊。

凝视着你的，不是深渊本身，而是那无数已经沉沦在深渊里的灵魂。

时光匆匆，从来也不曾有梦，一切成空。

茫茫白雾，笼罩天地的孤独，尽是虚无。

爱过恨过，不留遗憾的错过，只未悔过。

无论如何，自己做出的选择，也许值得。

# 雪月

雪，落在了宣州大地。纷纷扬扬的精灵，飞舞着，盘旋着。白茫茫的大地与呼啸的寒风，大雪绘就出梦幻的冬。纵几多相似，纵几重相思。尽化作那该当忘却的故事里，永远不再的记忆。

没有散场的灯光，可大幕已悄悄拉上。没有播完的结局，也无人能够想象。只有时光，仍在流淌。

只如今，茫茫大雪之中等着谁回顾，明知无人回顾。谁能初心不负。

# 无声

没有目标的旅途，随心所欲的行动，抛下所有牵挂和束缚。即刻出发，风霜雨雪间看凌晨四点西子湖畔灯光点点。总有惊喜，雪花飘飘里观外滩上下城隍庙间魔都风情。梦里水乡，姑苏内外走过甪直古镇观前繁华山塘热闹。漂泊的时光里，释放那不羁的灵魂。单调的岁月里，需要那放纵的旅程。茫茫大雪之中，不求谁能回顾，不问初心何处。只不负自己，便是极好。

还依稀记得那句：或许不知梦的缘故，流离之人追逐着幻影。不知梦的人，流离着，追逐着一片又一片的幻影。忘却了过往，也迷失了心情。迷茫和混沌是2018的主旋律，在群聊里丢弃了自我，方寸荧屏抽尽了灵魂。如吸毒成瘾，戒不去，消不得。

飘渺的感情不知归宿，只不过是灵感的来由。再也写不出那些词句，只因失却那份心情。划地三尺只为转世灵魂换你的生辰，诵一段因果，结来世的红绳。宿命注定的，相聚相散，相合相分。月光偷偷打量，可怜缘分，岁月的年轮，再诚恳，也渡不过红尘。相识是幸运，离别是宿命，莫去强求。

真正彻彻底底的将心置空，容不下任何身影。不曾有过喜欢人的能力，在疯狂中放纵自己。因一个人，爱一座城，又厌一座城。直至故事的最后，却重又恋上这座城。一座古镇，一条小河，沉淀了记忆，放松了心灵。午后，似晴似阴的时光里，漫步在青石板路上，身旁是小桥流水，渐忘却了时光。

不曾喜欢现代的奢华，偏爱那份古典的优雅。不再相信一见钟情的童话，也不会去迷恋任何有趣的灵魂。也许依然有喜欢的人，但，已与我无关。忆起上海滩的那一场大雪，下得那么深，下得那么认真……

# 岁末

2018，即将走到尾声。从开始到结束，一事无成，一无所获。只剩了匆匆逝去的日子，还有那匆匆逝去的年华。

还忆得彩金湖畔的碧水青天。一池碧水，守望着宁静的天地。一山之隔，隔绝了牛头冲的熙熙攘攘，断开了尘世的一缕喧嚣。默然，相守，寂静，喜欢。目视湖心那片轻浮的落叶，也许有所喜欢，便是心灵最好的慰藉。

还忆得摩天轮下的碧绿草地。柔和的阳光撒在脸上，微风轻抚，忘却世间多少烦恼，留下一个甜蜜而美好的梦。既无所念，便忘却那敬亭宛陵，心所往，唯新港彩金。

还记得年初，那场纷纷扬扬的大雪。洁白的天地里，金陵褪去了浮华，只剩一份优雅。故事已经结束，却仿佛未曾开始。似乎淡却的伤感之中，又添了几分不一样的东西。帝都风华，洗去已久的尘埃，回归豁然与平和。

待到一年将了时，一场大雪不期而遇。纷纷扬扬的白，落在宣州大地，洗落一片心空。凌晨的杭州，寒风微雪，卷过沉睡的西湖。雪落的上海，美如仙境，还上海滩一片安详。甪直古镇，小桥流水，尘世的喧嚣不再。姑苏，又回到了记忆中那个美丽的城。曾因人而喜欢，也曾因人而厌倦，终于在大雪飘飞的时节，回归了从前。

爱那风，也爱那云。爱那雪，也爱那月。观风云变幻，看雪月交辉。不再望那一山一湖，也不再记得湖中曾有过，那飘飞心上的一片树叶。

梦里梦到醒不来的梦，红线里被软禁的红。所有刺激剩下疲乏的痛，再无动于衷。

岁月的尽头处，一切不复当初。光阴的流淌里，长梦不曾结束。

2018的最后，匆匆回顾，想一切如故，不曾如故。

# 忆·金庸先生

问世间、情为何物，只教生死相许？

天南地北双飞客，老翅几回寒暑。

欢乐趣，离别苦，就中更有痴儿女。

君应有语，渺万里层云，千山暮雪，只影向谁去？

武侠的世界，有着国仇家恨，有着快意江湖。

时代在笔下风云变幻，

驰骋沙场保家卫国的豪迈里，却也有着儿女情长的温柔。

襄阳城的血战见证了侠之大者的为国为民，

绝情谷的一跃也把那份痴情演绎到了极致。

可惜，这一切都不在了。

金庸先生走了，属于武侠的时代也已经没落了。

# 纪念日

——纪念侵华日军南京大屠杀八十一周年

八十一载的岁月，不曾磨尽伤痛，不曾流干泪水。

理应铭记的历史，仍然沾满鲜血，仍然留下伤痕。

一二·一三，记得这个沉重的日子。

古都金陵，白骨成堆，血染长江水。

三十万同胞，在日寇的屠刀下死去。

铭记历史，只是为了，更好的前行。

不要以为战争有多么遥远，

也许和平的终结比我们想的更快些。

至今不愿正视历史的日本，

还有虎视眈眈向中国下手的美利坚。

珍惜现在的岁月静好，珍惜先烈们为我们争取来的和平。

勿忘国耻，不要原谅，砥砺奋发，振兴中华。

# 江城子·宣州雪

清辉一片雪纷扬。弃过往，却情伤。

宣州多事，一载叹时光。

从此相逢不相识，勿思量，只相忘。

浩瀚山野总苍茫。身彷徨，心流浪。

物是人非，叹世事无常。

天涯沦落又何妨，拾行囊，向前方。

# 岁月尽头

迈过了18岁的台阶，推开了电信科的大门。

忘却了伤感的故事，放下了过往的风尘。

心底里一直忘不掉的影子，丢不下的几分倔强。

终于丢盔弃甲，终于化作风尘。

曾以为喜欢山，却发现偏爱的是山巅的云彩。

曾以为喜欢湖，却发现偏爱的是湖畔的清风。

像风一样自由，像云一样起舞。无所谓喜欢，只是偏爱几重。

来到过凌晨四点的西湖畔，也见过夜晚八点的敬亭山。

击毁了骄傲，余下了放纵。沙雕的生活，也不是不好。

却见，雪下得那么深，下得那么认真。
