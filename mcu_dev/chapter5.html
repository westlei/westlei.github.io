# 从单片机到嵌入式开发——(5) 外部中断

date: 2021-01-16

author: 西风镭

location: 南京，苏宁睿城慧谷中心

截止本文为止，我们所使用的程序都是一个无限循环不间断运行的程序。但在我们实际的使用之中，我们需要考虑到各种突发情况。在突发情况下，单片机不应当按照默认的程序继续运行，而需要处理完相应的情况之后，方能恢复运转。

这个时候我们就需要用到对于所有单片机都十分重要的一个功能：中断。中断，顾名思义，中断你目前的工作状态，然后执行其他工作。执行完之后，回到原先的位置，按照原来的方式继续运行。

中断需要一定的触发条件，通过单片机设定的中断源执行中断子程序，执行完之后需要恢复现场。

标准的51单片机有5个中断源：两个外部中断、两个定时器/计数器中断。具体如下：

优先级 中断源 介绍
1st INT0——外部中断0 由P3.2端口引入，低电平或下降沿引起。
2nd INT1——外部中断1 由P3.3端口引入，低电平或下降沿引起。
3rd T0——定时器/计数器0中断 由T0计数器计满回零引起。
4th T1——定时器/计数器1中断 由T1计数器计满回零引起。
5th TX/RX串口中断 由串口通信引起。

首先我们介绍的是外部中断。外部中断可以使用硬件触发和软件触发。如图是硬件触发外部中断的效果。

如图，两个按键分别连接的是外部中断源INT0对应的P3.2端口和INT1对应的P3.3端口。正常运行时流水灯自右向左顺次点亮，当外部中断0触发时，流水灯瞬间全部熄灭，当外部中断1触发时，流水灯瞬间全部点亮。

该程序的代码如下：

#include <reg52.h>
void delay50ms(void)   //误差 0us
{
   unsigned char a,b;//unsigned char类型
   for(b=190;b>0;b--)
      for(a=130;a>0;a--);
}
void main(void)
{
   IT0 = 1;
   IT1 = 1;
   EX0 = 1;
   EX1 = 1;
   EA = 1;
   P1=0xfe;
   while (1)
   {
      P1 = P1 << 1;
      if (P1 == 0x00)
      {
		 delay50ms();
		 P1 = 0xfe;
      }
      delay50ms();
   }
}
void int0() interrupt 0
{
   P1 = 0xff;
   delay50ms();
}
void int1() interrupt 2
{
   P1 = 0x00;
   delay50ms();
}

如上所示，使用外部中断的时候需要进行以下工作：

首先设置中断寄存器。

与外部中断相关的寄存器主要是ITn和EXn(n=1/2)。EXn表示相应中断源的中断允许开关，1表示开。ITn表示相应中断源的触发方式，1表示上升沿(0->1)，0表示下降沿(1->0)。将两个中断源的中断允许开关均打开，触发均设为上升沿（松开按键触发）。

然后打开总中断，设置总中断开关EA为1。

接下来正常书写主函数部分。

中断函数的书写只需在函数声明处加入interrupt n，即可表示中断处理函数。标号n与中断源的对应关系如下：

标号 中断源
0 INT0——外部中断0
1 T0——定时器/计数器0中断
2 INT1——外部中断1
3 T1——定时器/计数器1中断
4 USART——串口中断

这样就完成了一个普通的硬件触发外部中断程序。但是硬件触发外部中断也有一定的弊端，因为没有去抖过程，所以存在误触发的可能性。此外，硬件触发外部中断本身条件过于局限。因此，我们更常用的触发方式是软件触发。

#include <reg52.h>
sbit key = P2^0;
sbit in0 = P3^2;
sbit in1 = P3^3;
void delay5ms(void)   //误差 0us
{
   unsigned char a,b;//unsigned char类型
   for(b=19;b>0;b--)
      for(a=130;a>0;a--);
}
void delay50ms(void)   //误差 0us
{
   unsigned char a,b;//unsigned char类型
   for(b=190;b>0;b--)
      for(a=130;a>0;a--);
}
void main(void)
{
   unsigned char mode = 0;
   IT0 = 1;
   IT1 = 1;
   EX0 = 1;
   EX1 = 1;
   EA = 1;
   P1=0xfe;
   while (1)
   {
      P1 = P1 << 1;
      if (P1 == 0x00)
      {
          delay50ms();
          P1 = 0xfe;
      }
      if(key == 0)
      {
          delay5ms();
          if(key == 0)
          {
              mode++;
              if(mode == 1) in0 = 0;
              if(mode == 2) in1 = 0;
              if(mode == 3) mode = 0;   
	          while(key == 0);
	          in0 = 1;
	          in1 = 1;
	       }
      } 
      delay50ms();
   }
}
void int0() interrupt 0
{
   P1 = 0xff;
   delay50ms();
   delay50ms();
}
void int1() interrupt 2
{
   P1 = 0x00;
   delay50ms();
   delay50ms();
}
