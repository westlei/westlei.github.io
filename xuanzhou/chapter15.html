# 恍然

听闻翡翠湖下起了大雪，突然忆起去年的时光

时间过得居然那么快啊，转瞬间就又过了半年

还想像曾经的毫无顾虑，想抛下负担却已太难

梦里梦见形形色色的人，可曾还记得我的存在

待到那花开时那夏日间，到翡翠湖畔再醉一回


梦里梦上的岁月，与现实交错。

恍然未醒的旅程，不曾化解的谜题。

# 凄影

尘封太久的记忆在一瞬间突然惊起

八年间认识的人在奇怪的地方相遇

初中同学仍在喋喋不休地说着坏话

捕风捉影地去影射我的每一举一动

昔日随风而逝的情缘似乎仍是当年

重见的日子却从来没想过会是怎样

又仿佛回到了平安喜乐的大一时节

去掉不应有的念想生活便轻松愉悦

所有一切终究落到美赛的帷幕之下

但当地铁一号线驶过了南京站之后

人走了 便只剩了自己 往迈皋桥而去

是大梦里的醉客 还是现实中的凄影

# 南海故梦

2月13日，大雨，有雷电。记忆的闸门猛然打开，往日的故事终究可以付诸笔下。那生命里短暂的过客，终究在时光的洗礼间消失无影，只留下，一点残破的记忆。

记忆的起点，应当是五年前的冬天吧。在那时，他是那数学的王者，她则是数学的弃儿，他们的数学成绩之和刚刚够得到满分。偶然之下他们结识，他为她补习数学，她为他补习英语。眼见她的数学有了起色，英语却跌到了与他一般的水平。不知什么时候，他对她有了莫名的好感，但随之而来的，却是分班的时刻。

顺理成章地，他去了理科班，她去了文科班。繁重的学习压力下，他无暇他顾，可还是会时不时关注她，打听她的消息。也曾跨过半个城市，只为期待她的出现。听说，她有了男票，有了自己的生活。他终于意识到，他们之间，就如同那高考分数的鸿沟难以逾越，只能祝愿她，前程似锦，余生幸福。

淮水之阳，她的故乡，一望成欢，即是淮阳。两年之前，似是今日，又是明日，也无妨，终究是放下了。非是故事，也不值得。既然已经离开了那座城市，一切，说不定会好起来。

当他初遇那山间的小城时，便被那靓丽的绝色惊艳了。她像一颗璀璨的明珠，吸引着无数的追求者，也让更多的人望而却步。当先前的一切尽数放下，那个春天里，他不禁动了心思。可随之而来的，是一道艰难的选择题。斟酌之后，他选择了离开，让所有的花火随着转专业而熄灭。误会逐渐产生和蔓延，却也不曾想要解释，这或许便是放弃的代价。后来，没有了后来，一切的一切，不过是罪有应得。

大雨如注。又是一年冬天，为一个偶然相遇的人而改变了选择，这又是否值得？

# 二·二九

四年前 紧张而放松地面对小高考 不知道未来

今天 紧张而懈怠地面对考研 依然不知道未来

四年后 却不知道会面对什么样的 未知的未来

也许我考上了中意的学校 也遇上了 中意的她

也许我被社会残酷地鞭打 在苦痛里 拼命挣扎

ISFP的人格 成功并不容易 理想终会输给现实

但愿岁月不要对我太过残酷 四年后 初心不负

因着一个人 而去拥有一个目标 本不是个笑话

就像一见倾心般的美好 不应破坏 也不应打扰

政治英一数学一 专业课九六一 它们都不简单

但是既然你喜欢 那我便努力 纵你我本无关系

# 沉寂

当世界陷于了沉寂，春天的到来也变得悄无声息。停滞的岁月里，今天与明天毫无区别，一切的一切索然无趣。未可名的现在，不可知的未来。白色恐怖下的天地里少了一方烟火气，人烟消散了，城市也不再有着城市的模样。只剩了碧海和蓝天，和空空荡荡的大街小巷。

果实熟透了，吃了才不会生病。时机成熟了，努力才不会白费。昔日若是没有迈出那步，也许境遇便大有不同。选择了一条比较艰难的路，便要学会接受路上的一切。不论是好是坏，是苦是累。纵使有更好的选择，也要把这条路走过。尽管，未来很可能会去后悔。

一个人真正想要什么，不取决于他说了些什么，而取决于他做了些什么。面对目标，谁先低头谁就输了。也许当初的高考分数在考研面前显得卑微，也许自身的能力在理想面前显得狼狈，但这都没有关系。为了心中的那一片美好而奋斗，即使输到一无所有，也值得。

明天，不曾开学，却将开始上课。一切陌生而不适即将发生，一切的一切...就像爱情，也许会在某个无法预知的时机来临吧。

再到那东海之上的小岛 听浪涛的喧嚣

未来也近了又远了 渐渐忘记那位姑娘


Optimistic Award; Fabulous Award;

Mysterious Award; Helpless Award;

It seems good, but I choose Sleep Award.

# 似是而非

但我还是有了些熟悉的感觉

也许是确实有那么几分神似

可那便是我所喜欢的样子吧


其实客居的日子本就是这么无聊

就像单片机和嵌入式的网课一般

一潭死水般毫无生气也毫无活力

也包括每一个见或未见的小姐姐

# 旅途

二零二零的深圳旅途终究走向落幕

一切的改变皆源自突如其来的变故

单片机嵌入式的作业变得愈发恐怖

网课的岁月朴实无华且枯燥如经书

大型群友线上聚会只为把那法来乳

考研词汇所剩无多前方可见那尽处

没有电脑只得论文写在手机备忘录

智能车的规则姗姗来迟却改动无数

魔幻现实如同小说 情节离奇难以预估

岁月静好时光匆匆 不见未来成功之路
