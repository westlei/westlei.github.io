# 十里春风

# 四月人间

人间的四月天，又降临了。而不一样的是，这个四月，我，在宣州。

四月，春到了它最绚烂的时节，那青涩的花蕾，终于开出了它的那一份辉煌。多少萌发在那秋末冬初的爱情，也在此时得到了满意的结果。因高数而相识，因平淡而相知，优秀的人们长相厮守，携手度过，那美好的一生。

六年前的春天，青涩的我第一次遇见那心动的人。年少的时光不懂爱情，也许只是几分青春的悸动。鲜花盛开的四月，望着那可爱的脸庞，心中渐渐泛起，那一圈又一圈的波澜。

三年前的春天，阳光下，那樱花烂漫的日子。可爱的女孩撩起那内心的向往，樱花灿烂映衬那美丽的脸庞。人间的四月天，遇上你，是我的缘。远远望着那念想的人，而时光却带领着走向了那不同的前程。在那奋进的时光里，她那青春靓丽的身影，给我以无尽的前进动力。不必再挂念，那已成过去的一切。

终究，又到了那四月天。在这皖南的小城宣州，遇见许许多多有意思的人。在百花丛里留恋，在万花筒里拣选。遇见那好看的皮囊，高傲而冷酷的外表下有着有趣的灵魂，但那相距千里的，是傲娇的元神。确认过眼神，这不是对的人。继续寻找那心心念念的伴侣，也许终究，能遇上那个人。

也许，还有那可爱的松鼠，还有那酷帅的女郎。但终究，若即若离，似近实远。相识寥寥，相知无觅。恋，那清纯的外表；爱，那干净的灵魂。羡慕那帅哥与学霸的结合，空叹自己那难填的心房。也许，那相识或不相识的人中还有那心仪的人，只是未曾发觉。听，那柔和婉转的闽语，也是，不可多得的享受吧。

飘然的春雨中，且抛却那些所谓学霸和大佬的身份，以一个朋友的身份，看看身边，那每一个不同的人。

帅气潇洒而又好脾气的他活的潇洒而纯粹，不为任何俗事烦忧，但在别人有困难时尽力帮助。深谙为人之道的他深得大家的喜爱。没有人排斥他，大家都愿意信任，这样一个有着萌萌哒内心的青年。

一个能力出众而又阳光帅气的青年才俊，让无数女生为之倾倒。曾经的他也许受过挫折和失败，但百折不挠的他总会一次又一次的前进。超凡的文学修养，卓越的数学能力，还有，那无时无刻不在的迷之自信。遇上了对的人，愿他幸福一生。

总有再多的事情放不下，再多的烦恼忘不掉，看见单纯的他，总能暂时抛下。也许，他身上仍有着未脱去的天真无邪，仍有着一份足够纯洁的内心。简单的面对大千世界，世界其实很简单。

一个能力出众的人，无论面对多少事情，他总能安排好一切，并给自己足够多的放松时机。没有什么能完全难倒他，只有他愿不愿意去做。时间总是有的，在他的身上反映的淋漓尽致。

迷之自恋的他，总觉得自己英俊潇洒而健壮，然而起初的他并不是这样。可贵的是，虽然自恋，但他仍相当的努力，不会失去内心的碧树。骚年，努力是总会有回报的。

活跃在每一个地方，为各种不同的事而操劳。抑制不住的好奇心让他的八卦欲望根本停不下来，而讨喜的性格也使得他不至于受到排斥。这样，也挺好的。

也许，这就是我眼中的朋友们吧。他们都很善良，也很有才华。这样一个美好的四月天里，遇上你们，是我的荣幸。不必再去记挂着去寻找伴侣，与你们为伴，单身的日子，也是快乐的。也许，真的不需要那么一个伴侣，来慰藉自己空虚的心灵。

# 十里春风

不要太奢望时光让回忆尘封，有些事情不若落花流水抑或山南水北，难得消逝，难得迷惘，难得顾盼流连。时光悠然，奢望那回忆尘封其中；落花流水，祈愿那世事消散凡尘；山南水北，妄想那时空淡却思慕。难得消逝，那灰尘覆盖的过往；难得迷惘，那形影模糊的人物；难得顾盼流连，那爱过错过的青春。不必在意过往，不必思念曾经。毕竟，那丁酉年，已经成为了过去。

四天时光，三座城市。北京的繁华，南京的热闹，最终都将归于那宣州城里的清寂荒凉。寒假就这么戛然而止，一切都将重归寻常。也许曾有梦想，也许曾有向往，也许曾想成为那不同寻常。但终究放不下过去，看不到前方，逃避着自我来回奔忙。愿意尝试，但不愿意挑战；愿意探索，但不愿意坚持。大佬的生涯注定无缘，爱情的美好注定无觅。阳春三月，那江南最美好的时光里，孤独守望，那小小的幸福。

一切的最后，重归于山水的宣州。寒冬似已远去，而那江南最美丽的时节将要来临。花开花谢，春去春来，每一次都是不一样的风景，每一次都是不一样的心情。夕阳西下之时，前路仍有那晚霞的最后一缕光明。在高处，压抑不住的那坠落的欲望；在低处，抑制不了那堕落的冲动。明天，又是新的一天了。

当那刺激战场的硝烟散尽，世事已归于那一败涂地的喧嚣。收拾心情整理时光，不再让往事回忆那不尽忧伤。宣州夜幕过，上元雨落终。找寻已消亡无处的自我，唤醒那消磨殆尽的真实。珍重那阳光下敬亭山里无拘无束的日子，那，是历经坎坷而重又焕发生机的青春。重整心情，开始迎接，那江南大地最美好的，三月春光。再莫谈，如风往事。

# 星河漫天

阳光明媚。那宣州城里，春初的岁月却已浮现出了盛夏的痕迹。宛陵依旧。浪涛拍岸，诉说那千年古韵的变迁。彩金犹然。风舒云淡，漠视那鸡犬相闻的田园。植物园里，那勃发的生机，讲述那属于春天的故事；新港城中，那喧嚣的盛景，书写那布满纸鸢的天空。这是春天来临前的温暖，这是狂风暴雨前的安宁。春色美好，却似昙花一闪即逝。莫去感怀，莫去追忆，雨后天晴，那是更美好的春光。

那星河的光芒已阴霾笼罩，不复追逐与寻找。那明月的炫影仍普照大地，何必为此而着迷？无处不在的是几多风雨，化解不得的是七情六欲。宣州苦雨淡却了记忆，唯有那皎白月光的幻影笼罩大地。春，也许曾经来过，也许终将到来，但如今，它不在。还好，那春将至未至的日子里，有你相伴。

阴云起，风呼啸。雨落宣州，须臾寒冬复来。一切的不幸，一切的迷茫，都是头文件里那已经包含的沧桑。选择的分支无尽无终，但每一处都是难能回首的凄惶。只修正错误却忽视那无止无休的警告，执行的后果是那混乱崩溃的记号。屏幕在绚烂的光华后化为灰暗，终究放弃。有心于工科，最终还是回归现实。既已选择了商学，那便一往直前，不再想着未来三年投入信工的怀抱。安于理，修于文，迷茫远望，无尽前方。

阳光灿烂，那宣州的下午。迷茫之中，那敬亭的某处。茫茫列表之中，多少人是那相逢不识的陌路者，多少人是那不曾交集的过客。做那空间的看客，为每一条说说点赞。坐看那是是非非，纠纠缠缠，一晃皆是，过眼云烟。毕竟，交集不过了了，相知不过几人。但，若是你喜欢上一个人，她的每一点变化，你都会看在眼里。这时，别人都变成了常数，而她才是，那唯一的变量。也许，她曾出现；也许，她今犹在。

# 三维世界

人群熙熙攘攘，表象下是孤独。观那大佬互相吹捧，见那能人碾压多数。无能似我，理应沉默。我眼中与你很熟，你觉得你我陌路。摸不清说话的分寸，探不到玩笑的尺度。似乎熟络，其实生疏。人与人，本就不过如此。

江南的烟雨来了又去，洗涤尽那三月的宣州大地。霍金的时代去而不来，与浩瀚的宇宙星河同在。爱上那三月的江南，湖光潋滟百花盛开；心怀那春日的宣州，叶上柳梢蝶绕花丛。也许，不应再像那冬天时一般堕落，不应再想那曾经的春景与春情。不用幻想那惊天动地的表白，不必担忧那不知真假的一跃。淡然面对一切，我只是我自己，不是大佬，不是大神，只是人群中那最大多数的大多数。

也许是那爱与梦的缘故，每在这江南烟雨的时光里都是怅然的迷茫。天阴色的阳光穿不透那寒冷的风，永远也到不了我的心房。花仍灿烂着，但那没有蓝天的背景，却是压抑着冬的深沉。许是病了，许是倦了，许是忆起了曾经，许是厌弃了现在。没有奋进的动力，也没有堕落的勇气。算不上坚强的壳已经磨灭，保护不住那脆弱而敏感的心。谁仍亲近？谁已疏离？孤寂的雨夜里，戏只剩了自己，把那痛苦埋葬在无梦的夜里。

这不是我想要的生活，绝不。整日里为了各种本与我无关的事情忙忙碌碌，然后在贪图享乐的时光中水群与堕落。书看的不多却满腹无端的感慨，钱花的不少只为了无谓的放纵。也许我喜欢上了一个人，也许我喜欢上了一个人。迷失在微经的课堂里，困惑着程序与高数的谜题，也许窗外是三月江南的好风景，但也没有那欣赏的心情。恍然发现，许久未入，宣州城里。

当z轴遇上了x-y的平面，枯燥的一维世界变得立体。变化万端的方向之中，不再是单调的前进后退。但那跨越维度的相会，终究是昙花一现的精彩。过了那O点的相遇，便只剩下，那渐行渐远的无奈。

# 一意孤行

人世间的一切悲剧，本质上都是由于当事人能力的不足。然而，能力不足的我，却已经没有了寻求改变的动力。在那清幽的山野中一意孤行，在那高楼的顶层里遗世独立。没有选择的能力，便倒在那纠结里。幸福=效用/欲望，但当欲望处在无尽的变量里，何来幸福的意义？

我，只是一个时不时伪装成大佬的菜鸡，遇见问题只能临时学习。也许目标并不清晰，但根本没有完成的动力。每当有人问我问题，只要我力所能及，那便抛下一切去解疑，顺便把那疑惑来学习。但除此以外，便不再有任何动力，躺倒在地，犹如未僵的尸体。

无理想，无目标；无追求，无向往。也许，寻找一个伴侣，能让那学海之上，绽放出绚烂的花火。但，那心中的碧树告诉我，这是不可能的。我，完全不是一个优秀的人，没有资格，去挑选一个伴侣。胆怯的人啊，怕面对旅途的艰辛，却不去想那错过之后的遗憾，与惋惜。我承认，对她有过那么一些好感。但是，她太优秀了，已经远离了我的世界，那便，只能目送着远去。

也许，只有在这样一个日子，才会去说上一些似真似假的话吧。就全当，是愚人节的玩笑吧。自欺欺人，骗不过自己，也骗不过别人。就像，所谓的匿名，和坦白说。都知道是谁，却以为，人皆不知，徒作笑柄而已。

忽然发现，自己已经走在了一条不归路上，并渐行渐远。似乎一直沉浸在梦里，从未醒来，正眼看过眼前的世界。做着一场忧伤的梦，用精神的鸦片放纵自我。丧失了节制的意识，钱如流水般远去。学习已经荒废，身形正在憔悴。无心赏敬亭春色，无心看宛陵春水，新港不往，彩金未向。只剩灵魂无觅的肉体，在水群的光阴里逃避，不敢去正视真实的自己。这又是何必？春风十里，不如有你。你既厌弃，我应远离。三月在迷茫中过去，浑浑噩噩的四月又已来临。梦不曾惊醒，也许是忘了，有人还关注着自己。

感谢那昨日将我从梦里点醒的人，感谢那些一直相信着我的人，感谢一直陪伴着我的哥们儿，感谢那些曾经夜晚相聚的温馨。也许现在是风雨，但相信前途，还有一片光明。春风十里，不如你，那深夜里陪我喝酒买醉的哥们儿；春风十里，不如你，那傻傻笨笨的糊涂女儿；春风十里，不如你，那些一直容忍着我的人们；春风十里，不如你，那在表白墙上表白的陌生人；春风十里，不如你们。

# 初心

不要假装自己努力，因为结果不会陪你演戏。

不要试图忘记过去，因为念旧才能勇于前进。

纵使伪装大佬一名，可是成绩依旧会出卖你。

纵使否认失败过去，可是记忆终究难忘彻底。

抛弃学习拥抱吃鸡，到了考前求生希望难觅。

抛弃往事拥抱当今，到了午夜梦回心伤无尽。

那年花开月圆金陵，倾尽思慕迷茫不解前途。

如今花谢月落宣州，无关风月时光梦幻往昔。

三月春来遍野花开，时光悠然轻言思慕无再。

石城匆归所爱无回，心无片叶只愿初心犹在。

# 守望

感觉生活很没有意义，感觉世界充斥着滑稽。

世事艰难前路无光亮，夜空是那靛青的微茫。

昏暗的路上惨白的光，不知何处照耀着前方。

星河无影惟天地苍凉，夜风吹去是四海八荒。

没有不破的爱情幻想，不如在夜里随风流浪。

孤行向南到湖边怅惘，声色光影照水面金黄。

栈桥春影心不曾荡漾，落雨终归入浩浩大江。

夜幕之下忽心羡叶杨，再多曾经亦当忘则忘。

不曾有过的年少轻狂，理不清楚的满腹忧伤。

世上有多少山高水长，何必把过去念念不忘。

几多故事里几多怨望，几多伤情时几多迷茫。

有故事的人已成班长，抛下过去与幸福守望。

摸不清何处心之所往，岁月静好再扬帆起航。
