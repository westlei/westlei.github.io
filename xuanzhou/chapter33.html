# 毕业纪行

似乎 又到了一年一度的毕业季了

这似乎是作为看客身份旁观的第三个毕业季

满学校熙熙攘攘的人群 摆着pose拍着毕业照

倾吐着关于毕业的不知道是伤感还是喜悦

我不是他们中的一员 自然也无法理解

我只不过是说走就走的旅人而已

自然 作为旅程一部分的毕业 也不过尔尔

我不会觉得我还能遇到哪个谁

我终于完全失去了任何一点的念想


灵魂和身体 总要有一个在路上

就像诗和远方 总要有一个在眼前

御风而行的吟游诗人 从不拘于世俗

喜欢喝酒 喜欢醉酒的感觉 唉嘿

摸鱼什么的 可是生活的全部乐趣啊

尽管以我的能力 不应当如此摸鱼才是

不过 以普遍理性而论 飞比跑快吧

甚少还有人知晓 三年前牛头冲的模样


图书馆开业于2018年的2月28日

敬亭一楼的临时图书馆尘封至今

哈啰单车入驻宣城是在2018年的3月中

几乎没人记得红色的宣城公共自行车了

万达广场开业于2018年的6月28日

从此去星隆国际的人变得越来越少了

海底捞开业则是2021年1月的事情了

火车站东广场开通于2018年7月27日

老火车站在18年11月的爆破中灰飞烟灭

原地建成的西广场开通于20年6月28日

学校西边的创业路开通于2019年的8月

宛陵科创城大楼交付也是那个时间前后

北门对面翡翠华庭在2017年还是荒地

而交房并开启商业街也是2021年的事

体育馆开业也已经是2021年5月了


或许宣城、宣城校区对我来说最多的记忆

更多还是这些城市、学校发展的点点滴滴吧

和人相关的东西 总是变得太快了 太模糊了

与物相关的历史 才是会长久能够记得的吧

# 前程既往

尽管其实很不愿意承认

但是不得不说 四年确然已经到头了

最近几天 依然不想收拾东西 也不想准备什么

天天还是去实验室 一如既往 待到十点

沿着湖边一路走一路聊 到南区转一转

然后再沿着湖另一边绕回北区的宿舍

就好像毕业真的还有很久 很久


我对大学的情感其实并没有多么深重

所谓离别的季节似乎也并没有多么伤感

但总感觉如果离开了就好像少了什么东西

就好像是一种习惯 离开了总觉得空落


经常远行的人啊 当我走进综合实验楼一的大门

当我迈步进入406实验室那熟悉的角落工位

就仿佛像回到了家里 回到了真正的家里

从大二一直到大四 我一直都驻守在这里

从竞赛到课设 从考研到面试 一桌一椅一人

从来都是这里 从来都在这里 从来不曾离去

一批一批的人走了 一批一批的人来了

406会所里永远回荡着快活的空气

单片机 嵌入式 安卓开发 网站构建

只有你想不到的 没有你学不到的

说起来留恋是有的 但是也没有特别留恋

别离什么的总是常有的 人与人 人与物

但若是不常在 或许也就不会过于难过

人和人之间 缘分尽了也就尽了吧 江湖嘛

不过你要是硬要续上 其实也不是不行

但众所周知 断开的线要想重新连上 就很难了


遗憾或许也是有的 而且确实有很多

每一个我喜欢过的人对我的影响都很深远

尽管从来也不曾获得过 甚至都不敢想过

我分不清什么是喜欢 什么是见色起意

但我永远也不够勇敢 只会辜负别人的好感

我变得颓废 不再受得起别人越界的所谓友谊

我敞开大门 却把自己牢牢的关在门里

但无论你怎么看待我

在记忆消逝之前 我永远喜欢你

我喜欢 又怎样

说不出口 便是乌有

如此而已


就突然觉得好像 也没什么好说的了

有缘今生还会相遇 无缘那就来世再说吧

# 影忆 (Cover：音频怪物 琴师)

毕业之前的最后一个假期来临了

许多人的耳畔仍是四六级的听力声

而在海风吹拂中醒来的我

看见的却不是阳光沙滩

而是一片迷雾封锁着的海滨

那么 便越过崂山 北上鳌山湾

去那旁人所不曾想要去的地方


海风召唤将我目光吸引

背起行囊即刻前行

沙滩碧浪点缀湛蓝海滨

心如止水一般安宁


恍惚入了夜色星光下生雾气

繁华灯影尽皆淹没朦胧里

走过台东的夜市花火已凋敝

就像在十分遥远的过去


潮水涨涨落落有周期

仿佛海在弹奏乐曲

独自望着远方一片静寂

可容我再收敛下思绪


四载大梦一晃间便散尽

独余袅袅几许余情

仍记得那些许翩然身影

心底深处默默刻印


似海边的风铃般悦耳动听

像晴朗夜空里的月明

清晨君又要向着哪里而行进

只见明月在悄悄的消隐


转眼间所有人都向四方散去

不知何时失了音信

命运交汇你我初次相遇

从此便渐渐远了距离


深夜列车奔行在这滨海大地

却又不知还有多少前缘未定

终究还是变成了不愿提及

就像起伏不定情绪

即使把过去全部都想起

也改变不了一点一滴


人生起起落落是天意

时间面前无能为力

岁月的车轮轰鸣着驶离

只剩被碾压在地的你
