# 夏夜星河

# 通缩时代

在我的个人视角上，未来一段时间，世界的前景是很灰暗的。世界经济大通胀的时代要结束了，下面是大通缩的开始。虽然这个结论并不是很符合感知，但是我觉得，现在的情况确实是通货紧缩的时代到来了。

通货，也就是市面上的货币。通货膨胀意味着市面上货币量增加，而通货紧缩则意味着市面上货币量减少。在市面上物资数量不变的情况下，如果货币增加，则意味着物价上涨；货币减少，则意味着物价下跌。在21世纪的世界，主权货币的主要来源是主权债务。如果贷款政策宽松、存款准备金率下调、贷款利率降低，市面上发放的贷款就会增多，货币就会增加；反之货币就会减少。但是政策只是一个方面，贷款的发放还要取决于市场需求和市场预期，而这个情况则会反映在长期存款利率上。如果市场需求大，预期向好，则长期存款利率会有所上升，反之存款利率则会明显下降。目前政策上是鼓励贷款的，但是贷款却明显的发不出去了。钱没有一个可预见能盈利的流向，导致贷款的风险明显上升，而长期存款的利率就变得很难看了。

在过去的二十年里，由于对外贸易导致的需求增加，中国一直处在持续的大水漫灌之中，通货膨胀维持在一个比较夸张的水平。在这样一个背景下，只要有能力得到钱的人，不管做什么投资都取得了超额的回报。但是，在2015年，全行业的投资回报率，尤其是第二产业的投资回报率就见顶了。以前投一分钱能赚两分甚至十分，但当全球的需求都已经饱和，恶性竞争就开始了。为了防止市面上的货币过度抬高物价，部分行业开始了局部放水。首当其冲的便是房价，2015年以后，各地的房价翻了数倍，早早参与贷款买房的人在红利之中一飞冲天。另一方面是4G时代到来导致的移动互联网产业大发展，BAT等巨头把计算机行业供上了神位。但这两者的本质都是金融创新，是金融定向放水的后果。部分行业的繁华背后是全行业的落寞，四大天坑两大护法面临着待遇低下和失业的困扰。刚刚入门经济学的我，在股票市场分析时本能的觉得经济陷入了某种困局，而当时的我并不足以理解到底发生了什么，但我可以润。

全行业的萧条在2018年以后日渐显著，在疫情后成为了公认的现实。教培行业的突然死亡让人们认清了经济危机背景下体制外的危险，A4纸的背后是一个行业的灭顶之灾。大家开始追求稳定，追求公职，希望在体制的庇护下能够度过危机。但问题在于，体制只是社会的缩影，体制的结构失衡比体制外更加严重。当危机来临时，体制内的底层更难获得安稳。受制于体制内各种有形和无形的限制，体制内的底层有可能过的更加艰难。通货紧缩了，谁手上都没有钱。向体制外加紧剥削的后果只能是体制外躺平摆烂，彻底交不出钱。然而，老爷们的钱和福利肯定砍不了，所以老爷们要去砍谁的呢？

危机是全球化的，大洋彼岸某国的大水漫灌影响的是全世界。润出去可以改变你在食物链里的格局，让你从绝对底层变成相对底层，但改变不了作为底层受害的事实。纵观历史，大通缩周期唯一的解决方案，就是制造需求增量，让预期变得乐观。增量需求的来源，要么是一个阵营或体系被打倒所带来的资产倾销，比如二十年前；要么是大家一起洗牌带来的资产毁灭，比如…

这也就意味着，总有人要成为代价。我们现在所能做的一切，是让自己在剥削序列里的地位不要垫底，并把自己押注在可能取得胜利的那一边。当然，也有可能，这个游戏本来就没有胜者。

# 七夕又至

明天又是七夕节了。

上一个七夕节，我去了趟珠海情侣路，去广州珠江边看了看璀璨的广州塔。广州塔上粉色的七夕字样很是浪漫，海心沙岛的夜景也很绚烂。那时的我刚来不久，还有许多对新生活的憧憬和幻梦。而如今，在坂田基地工作了一年的我，经过了266个工作的日子、133个加班的夜晚的我，厌倦了这哪怕太阳最烈的午后不开灯依然暗无天日的工位的我，失望于这日复一日看似有所不同实则毫无区别的生活，却又不知道有什么选择。

有些东西其实是没有悬念的。比如当战车热热闹闹的驶过海沧大桥时，也就意味着这次不会有什么大的药。人家出的不过一张尖，你可以出个二，也可以出张王，甚至可以来个小炸。吃瓜群众们想看的，无非是会有些什么小的乐子。但你什么也没出，这就令人有些失望了。安全运营也是一样，人家红队没心情在这啃硬骨头，也就出点三三四四七七八八这样的小牌，我们也就对等的压一压。唯一的兴奋点也就人家出了张皮蛋，结果我们还反手一个王。

一潭死水的生活，总归是需要改变的。怎么改变呢？那就，旅行吧。几乎每座繁华的大城都有这么一个地方，水泽之畔，灯火辉煌，游人如织。北京什刹海，上海外滩，重庆洪崖洞，南京夫子庙，苏州山塘街，杭州龙翔桥，温州白鹿洲，合肥包公园，长沙橘子洲，广州海心沙，深圳深圳湾，哈尔滨防洪塔…各色灯光映照水面，混成一片地上天国般的景象。旅行，无非从一处到另一处，但来来回回，不过如此。陌生的城与城，对我而言，也并没有多少不同。

那么，还是得看人吧。生命中来来去去的过客，谁能治愈我的精神内耗呢？别看二舅了，多看看周劼吧。起码有句话他说的很对，只有没着落的小镇做题家才去卷北上广深，卷到最后也没几个能留得下来。看看人二舅就算说是在北京部队里混的风生水起，最后还不是滚回老家了。要么一步到位润出去，要么回老家去做人下人。憋屈的活在北上广深，活到一个无力继续待下去的年纪，终究还是要走的。这大抵就是，生存在这片土地上的宿命。

在我决定人生的目的地之前，爱情，还是不应该奢望的好。

# 立秋时节

夏天已经过去了一半了。

可能是因为前几天刚刚下过雨的缘故，今天的深圳并不是很热。三十一二度的天气，出门走走并不会有热的离谱的感觉。加之风还挺大，总的来说还是比较舒服的。

我很喜欢西丽这一片的。骑车从留仙大道过深圳北站，拐进学苑大道丽水路，转进沙河西路。北出白芒关可以去石岩、光明，西走留仙洞可以去宝安、前海，南下大沙河直达深湾、蛇口，东进珠光路去往侨城、福田。当然，即使不骑车的时候，也可以在这边走走，绿树成荫，人烟稀少。吵吵嚷嚷的坂田待的厌了，来此也不失为一种消遣。

进入八月以来， 越来越多的朋友来到了深圳。熟悉的，不熟悉的，闻名多年但不曾见面的。可能是常被认为是社牛的缘故，我认识许多林林总总的人。但事实上我只是一个web服务，没人访问的时候，并不会主动去访问别人。大部分的时候，充当的是一个nginx的角色，把合适的请求转发给对应的人。剥开社牛的外壳，我应当是社恐。

星空的璀璨，在于它的变幻。生如流星，在宇宙间走行。流星划破天空，带着耀眼的光芒，让人沉醉，令人倾心。但当它离去，只有无可奈何。回首望去，陪伴你的，却只有一轮明月，高挂天空。

# 菊厂纪行III

终于，离开“华为技术有限公司”的日子开始了倒计时。虽然说还不知道流程要走多久，但总归算是有了一个盼头。这一年的生活不是太坏，但也不算太好。我逐渐迷失了自己的方向，在若干的杂务里趋于倦怠，在疫情的反复中趋于躺平。好在这无尽迷茫与混沌的旅途之中，有人一直支撑着我，一直安慰着我。

最近两天我又一次见识到了人类物种的多样性。我的各项行为被安排上了一个全新并且听起来非常合理的理由——骗P。我仔细反思了一下确实是这么回事，都已经做到这份上了为啥不去再努努力骗个P啥的呢？毕竟坊间传闻，公司里的前辈们做这种事儿也不是一次两次了，那自然我也可以这么做。

但是口嗨归口嗨，口嗨是不需要带脑子的，真的去骗单纯漂亮妹妹这种事我大抵还是做不出来的。出于对所有经手的事物负责的态度，我很难允许自己很快速的投入一段恋爱又快速的结束，哪怕不是以骗的态度。当我完全一无所有的时候，我也不需要什么顾虑，但当我有了真正需要去牵挂的另一半的时候，我也不愿意轻易去放手。

我期许的感情，是信任与责任的统一。没有高度的信任，再深的感情也是虚无飘渺的幻影。而充分的责任，才能为感情扎下牢固的根基。偶像剧男主的剧本注定不属于我，我只希望我遇见的那个人，能陪我一起走过前程的风雨，能照亮彼此的彩虹。
