# 光年之外

# 终成眷属

——忘不掉。

——真的那么爱么?

——不爱。

——忘不掉什么?

答案，我自己也不知道。

情人节，愿列表里每一对秀恩爱的人都幸福，愿所有的有情人都能终成眷属。

# 宿命轮回

人世间的一切纷扰，终逃不过宿命的安排。如牵丝的傀儡，永坠爱恨轮回，却不知为谁而悲。不如做那世末的歌者，跨千年霜雪，看漫天流萤。只愿下一个雪落的时节，能道声：初心不负。

四季轮回，仿佛一个永远不会停止的循环，在大自然的程序里，无限地重复着。有无限的机会试错，也有无限的机会改过。只要服从着头文件里写好的自然法则，一切便永远不会结束。

牛头冲的第二个春天开始了，一切仿佛都不再是曾经的模样。云掩盖的世界，落魄了沧桑。只见得风吹浪涌，日落斜阳。湖光悠然，不曾有过恋过的印象；浮华喧嚣，不再勾起，挣扎的迷茫。

晚霞的末尾，仍泛着微光。不甘的梦魇，欲书写倔强。渐见得，夜幕初降。往事，终究遗忘。

# 迷情紫色海

三百一十七天的时光缓慢地从岁月里溜走，如同一个转了一圈的圆。还记得那时看见薰衣草的梦幻，还记得那最后一次与高中的朋友相见。唯独忘了，那也是一个故事的起点。

午后的河海大学泛着周末的慵懒，悠然的时光里却与一条消息相遇。未知的世界里未知的谜题，未解的心结下未解的焦虑。未尝揣测出用意，未尝去明晰事理。就在这未知里，一切不期而遇。

时光缓缓而行。意外的参与，意外的获奖，意中的背离，意中的远去。意外的反目，意外的绝情，意中的失落，意中的自弃。戏如人生，人生如戏。从高峰到低谷，只是短短的一瞬而已。

倘若没有那次比赛，故事又会走向哪里？度过了萧瑟，跨过了绝望，新的故事，出现在新的起点。也许这一次，不再有过去的悲伤。也许这一次，不用再经历绝望。

又回到了当初选择的时光。而这一次，我选择拒绝。放弃心中的喜爱，去追赶自己的曾经。也许，心无旁骛的时光里，一切都将慢慢变好。

就像，那片盛开的薰衣草，那片梦幻的紫色海洋。

# 因果循环

扪心自问，若是时间重新回到2017年9月7日我踏入牛头冲的那一刻，一切重头再来，我未必能比现在过得更好。所有的一切都是一环扣一环的结果，种下的因结出的果，又为下一段因果埋下了种子。所作出的任何一点改变，都将会带来无法预知的影响。

严格说来，其实在第一年里，我所经历的一切都足够的幸运。优秀而融洽的室友，松散但和睦的班级，不多但知心的朋友，不好也不坏的成绩。在这一切的加持和助力之中，我收获了我大学四年里可能再也拿不到的奖项，也完成了大学四年该完成的基本任务。

但随着弱点的逐步显现，一切恶的因终究种下，并渐渐开出了恶的花朵。不够自律的我不沉迷游戏却也抵制不住手机，课程的难度也渐渐超过了我能应付的水平。不善做人的我逐渐开始失去朋友，同时又把舔狗的失败做到了影响恶劣的程度。一切潜移默化的开启了转折。

转专业是一个不知是对是错的决定，我在选择我比较擅长的计算机的同时也要捡起我最不擅长的物理，放弃把一手好牌打的一团糟的过去的同时也意味着手中的牌要重洗。急功近利的选了数量远超能力的课程，后果就是面对挂科的惨剧。生活混乱，精力也趋于崩溃。

但生活仍然在按着既定的思路继续，每一个要放弃的时候最后都挺了过去。嘴上羡慕别人有着女朋友的柔情，心里却早就有了自知之明。既然成绩不好就在其他方面努力，既然不会做人就别往人群中挤。不能享受生活那就被生活享受，过好当下未来才能来临。

假如我年少有为不自卑，懂得什么是珍贵。那些美梦，没实现，我终将后悔。所幸，我还年少，或许还可以有为。

# 魔鬼与天使

时光过去了太久，久到已经忘记了过去，也忘却了未来。曾经说说里写下的那些人，也渐渐的淡漠了联系。重视感情，但又没有感情。害怕孤独，却又拥抱孤独。凭一时意气，注一世孤生。

抗拒不了所有美丽的事物，沉醉湖光山色，也迷离倩影依稀。会本能去亲近，也会尽力去付出。无偿帮助的一切，不过是因为心底的那份偏爱。我是魔鬼中的天使，也是天使中的魔鬼。

曾经的我还有过那么几分骄傲，有那么一点卑微的清高，但如今却已失去了自负的本钱。做一条舔狗也没什么不好，自有那属于舔狗的快乐鼓舞着。只愿有朝一日，海底月是天上月，眼前人是心上人。

重又想起去年初夏，将欲转专业的时光。那时也许不甚快乐，却也无所忧伤。那时的我有所喜欢，也有所成就，拥有我如今失却的一切。再回首时，一切都已不复从前。

也许，我终有一天会彻底忘记她们吧。

# 公主与神灯

阿拉丁最后拥有了公主，公主则拥有了苏丹的权力。精灵拥有的梦寐以求的自由，贾方也成为了他想成为的最强者。正派和反派都拥有了他们想拥有的一切，却也同时失去了一些其他的东西。

也许，你想得到什么，那你也必须要付出些什么。The more you want, the more you lose.

有的时候我们总觉得我们自己很聪明，但聪明的同时却失去了更重要的善良，单纯和智慧。不那么聪明的人往往能凭着那么一点笨拙和朴实吸引着更聪明有才的人为他效力，最终一切应有尽有。聪明过头的人则往往会在不经意间为自己挖下大坑，最终落得一个悲惨的结局。

但，纵使是神灯也有他所不能实现的事情，比如爱情。想要获得爱情，终究还是需要那份勇敢，纯真，正直和率性。这，便是我不可望不可即的吧。

# 浑浑噩噩

光阴的指针逐渐指向了大学的中点，回忆愈发地厚重了起来，希望却消失在飘渺的虚无之间。日益单调和乏味的生活，已掀不起一丝一毫的波澜。生命之中的过客，相逢在那岁月之中，本已是一种缘分，随遇而安，随喜赞叹，不去记起，也不去挂怀。

可岁月静好的表象之下，却往往有着无人知晓的过往，与那无可言说的忧伤。那失眠到彻夜未必想起，生命之水喝醉未必想起，却总会在一个意想不到的时间里，因着那平凡的岁月而唤起。让我在那脆弱的时候，懂得了心痛这个词真真实实的在生理存在。一个背影，一个名字，一个熟悉的场景，或者，譬如，一个日期。这一切的一切都会告诉你，什么是爱过。

随着时间的推迟，流离的情感会渐渐地消失，但直到下一个所爱慕的人出现之前，她仍将全方位地影响着一切。颓废，失落，悲伤，绝望。不再走心的岁月，浑浑噩噩的光阴。不敢去想以后如何，只想着当前的得过且过。默然，相守，寂静，喜欢。是卑微的感觉。

忘却曾经模样

湖中的叶片落在心房

舔狗仍怀揣着虚无的念想

所做与所为不得欣赏

何必赴火蹈汤

初心已亡 神思迷惘 梦境微凉 落发成霜

寄情虚妄 换得失望 对错无妨 不过惆怅
