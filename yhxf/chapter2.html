# 南国新景

# 故园千里

清缴违规补课的号角吹了许久，也终于是看见了南京方面的文件的落地。打着防疫的旗号清缴“违规”补课，也算是很有才了。文件内容也很符合南京的一贯精神，充满了官样文章，没有一点内容细节。XX就进一步XXXX做好XXXX工作提出XXXX，听起来很有道理，其实等于一个字没说。万事靠“领会”，没有悟性，还是算了。在这样一个地方，或许只有公务员才是一个不错的出路吧，毕竟，没有别的什么高薪而稳定的工作了。

南京是个完全没有互联网基因的城市。甚至于南京的民营经济，完全就是一片彻底的萧条。在国企垄断一切的南京城里，新百、中央、华润、金鹰。。。生活的方方面面背后，是国有体制下的升斗小民，是家长里短的烟火气息，是盘根错节的关系网络。一场疫情让全国重新认识了南京，可南京本就如此。

深圳诚然是个挺好的地方，但或许并不是一个长久之计。公务员体系大约却然是不适合我这种厌恶沉闷的人的。 宇宙的尽头是什么，宇宙没有尽头。我相信变数，但不相信未来。改变必定存在，但不一定是好的改变。

我印象里我弃坑前的王者荣耀还是S12赛季，那时候的它被称为电子海洛因。现在的农药似乎已经到了S24赛季，但他的绰号变成了精神鸦片。从海洛因到鸦片，是好了还是坏了，犹未可知。十二个赛季不打王者的我，可能已经完全无法理解新版本了。或许我也不需要理解吧。

算起来好像我光大学四年打过的游戏也不少，王者荣耀，绝地求生/和平精英，英雄联盟，钢铁雄心IV，我的世界，炉石传说，明日之后，明日方舟，部落冲突，皇室战争，元气骑士，阴阳师，原神，剑网三。。。但基本上都在退坑和回坑之间游荡，没有哪个一直玩下去的。大三以后打游戏的时间似乎也越来越少了，生活日益消减了趣味，人也日益颓废了起来。

我大概就是这样一个无趣的人吧。我不喜欢热血沸腾的球类竞赛，也不喜欢人潮汹涌的繁华街巷。我喜欢在僻静的山野听风，喜欢看海边的浪花翻腾起舞。我喜欢音乐，喜欢阅读，喜欢灵魂的安静与清明。但长久如此，未免寂寞，于是乎又喜欢在互联网的掩护下去水一水群，从而排解凄清的孤独。我奢望有人能走近我，但若是真的有人走近，我却只会戒备着越走越远。

绝云间待久了，总觉得世上的一切凡人与我都有着莫名的隔阂。尽管我可能是很多学弟学妹所认识的学长中最亲切的那批，但对于普遍大多数而言，却是最疏远的那一份子。因为一视同仁，所以亲疏不论。期待一直都在，但终究会化作灰白。灰白，非灰非白，不似色彩。

许是因为近视与脸盲的缘故，我甚少见到令我足够心动的plmm，自然也不会考虑出击。我似乎早已不再相信爱情，但仍在试图寻觅所谓的心仪之人。苏州姑娘的温婉，扬州姑娘的灵动，南京姑娘的平易近人，常州姑娘的小家碧玉。我的心里永远会有那么一两个人，但也最多只容得下一两个人。

或许未来的某一天，等我真的遇见了谁，我可能会去她的地方，寻觅她的足迹。但我想，这一天，终究是无可在现实中实现的吧。或许，比起码农，我更应该算是一个，长情的冷漠诗人吧。工作是工作，生活是生活。而生活剥离开工作，剩下的，却只能是，空空如也的感情，与脆弱的自己。

# 谁的七夕

2019年秋天，遇见一个女孩，人特别好，也很漂亮。不过那时候碰见了一个更喜欢的人，然后就凑了一队人马去打比赛了。但是很不幸，出师未捷，我更喜欢的那个女生被她们电气一个阔少给泡走了，有点心灰意冷，就把整个项目扔出去了。后来这个项目拿到了互联网+校金，但跟我关系不大了。

然后紧接着就是20年疫情，疫情的时候网恋。众所周知网恋一般没什么好下场，最后发现对方是个海王，同时吊着好多鱼，考研最后一个多月心态完全爆炸。还是那个女孩来安慰我。我觉得很对不住，她一直对我特别好，我一直在喜欢别人，然后还是一直失败，反复失败。

抽烟喝酒蹦迪的女生看不上不抽烟不喝酒不逛夜店洁身自好的男生，但她们同样也不是这些男生的目标。海王容易找到海王，因为海王更换对象的频率总是更快一些，人总是更趋近于寻找自己的同类。你想要找什么样的人，你首先应该变成什么样的人。

孤独的城市里，无聊的周末时光，即将开始，无所事事。不再努力试图打破孤独，而是选择去融入。白月光和朱砂痣，一个高悬空中无力企及，一个点在心口不愿触碰。那便该如何如何吧。七夕，不知是谁的七夕，但必定不是我的七夕。

# 经济社会和历史周期律

最近总有一些老年人，以一己之力让一座座城市坠入疫情的泥潭。当我们抱怨坏人变老的时候，不妨从历史的角度来看看这个问题。或许，宇宙的终点是宿命，在宿命论的维度里，一切都能够找到答案。一个人的15-25岁，对一个人的一生，可以说是有决定性的影响的。因此，一个人在这十年间所处的时代背景，基本决定了这个人一生奋斗的难度系数。时代的力量决定了大多数人的平均数，而个人的选择与努力，只能决定你是人群的上限还是下限。他们是被历史裹挟的不幸的一份子，绝大多数是人生的苦难儿。

历史，是自然的历史，也是人的历史。历史的大周期是自然气象周期， 每一个全球气温上升的时期往往也是盛世来临的时期，反之则是乱世。一个上升下降的周期大约也就是200多年，恰恰是一个王朝的长度。历史的小周期，大抵也不过是资本的寿命周期。农业文明里，最大的资本，也就是生产力，是人。自然状况下人口增加到一定限度，就会超过当地的承载能力达到饱和，然后就只有加剧内卷和被迫迁徙两种手段。当大多数地方的人口都饱和的时候，战乱、饥荒就不期而至，人口被大量屠戮，一切回归到原点。农业社会的低效使得生产的矛盾点集中于生产力和资源承载力，生产需要的资源一旦超过资源承载力，则必将陷入动乱。

工业社会下的资本变成了机器。人的生产能力和效率，与机器相比，太过于低下。资本积累阶段努力生产，提高生产力。但生产力达到一定程度的时候就会陷入消费瓶颈，然后又到了只能价格战加剧内卷和开拓新市场的阶段。当全球的消费都达到了瓶颈的时候，经济大危机不期而至，大量公司破产，生产力急剧降低，一切回到原点。工业社会的效率使得生产对资源的依赖大大降低，消费能力和生产能力成为了新的矛盾点。生产力的发展永远是一个螺旋上升的过程，消费促进生产，生产反作用于消费。

一个大的经济周期大致对应一个人的一生，出生到成长到凋敝死亡，这个时间的长度也恰好接近于人的寿命中主要做出贡献的长度——50到60年。这也就是康德拉季耶夫周期理论。第一次工业革命标志着人类进入了工业文明，随之而来的第一次康波周期从1780年代持续到1840年代。第一次工业革命的推广造就了第二轮周期，从1840年代持续到1890年代。第二次工业革命带来的第三轮周期从1890年代持续到1950年代，而第二次工业革命的全球化余波带来的所谓第三次工业革命从1950年代延续到1990年代。真正的第三次工业革命或许应该是真正意义上的全球化到来和通信大发展，从1990年代到2040或50年代。每一个周期包括20年左右的繁荣、10-20年的衰退和战争、10年左右的萧条和10左右的回升（也可能是战争）。

周期中的衰退期和回升期都可能爆发战争，但战争与战争又有所不同。衰退期的战争只是衰退期背景下矛盾的调解和释放，比如第一次世界大战和越南战争，本质是地缘矛盾冲突的释放，战争各方都得不到明显的好处。而复苏期的战争则是完全为了收割财富以进入下一轮繁荣，比如第二次世界大战被美国收割殆尽的老欧洲，以及冷战决胜局被收割一空的苏联。全球的繁荣总是建立在某个大国被收割的血泪之中，而一切会在萧条期的尽头图穷匕见。

最幸运的一代人生于萧条期，长于复苏期，然后在人生的黄金时期迈入了繁荣期，凭借积蓄在新的衰退期和萧条期里依然过的不错，这一批人也就是65-75出生的，我们的父辈。生于复苏期的人们如果能尽早抓住机会，在繁荣期中攒下积蓄，日子也能过的不错。90后的日子只能说是一代不如一代，只能在内卷的潮水里厮杀。最悲惨的一代人则是衰退期出生的人，由于往往衰退期也会伴随婴儿潮，所以衰退期出生的人也是最多的。他们在应当接受教育的时候遇上了萧条期，家庭困难会使得他们接受的教育大打折扣。他们走入社会时遇上了动乱的复苏期，一无所有的他们很容易沦为社会底层。等到机会来临的繁荣期他们一无所有，最终沦为被所有人讨厌的人。

10后的两级分化已经大到了可怕的地步，中考录取率持续降低，而少数牛人卷到飞起。在广大中西部乡镇，到处都是无人管教，从小沉迷手机游戏和刷短视频的孩子们。未来的他们，可能就会像现今的那一批老人一样，沦为被社会所排斥的对象。本轮的繁荣期已经在2010年前后结束了，而新冠疫情的来临则正式意味着大萧条的到来。至于新的科技革命的繁荣期曙光，大约至少需要2035年才能出现。未来一段时间的日子，总归是难过的。

万事万物总归是有规律的。某种意义上，我们的世界可以理解为一个大的计算机仿真程序。既然是仿真，就可以预测。尽管不一定准确，也不一定有效，但总归是可以对未来有所参考的。

# 宿命南国

仿佛是与这个南国城市的宿命，又或者只是我个人陷入的梦魇。每次来到这里，都会受失眠和多梦的困扰。本以为早就已经忘记的东西，又一件件的在凌晨的梦里浮现。过客许许多多，我以为能留下印象的却没有几个。但夜深人静时，总有一些逝去已久的记忆泛起。

大概是这么多年的过往，使得我习惯了作为一个弱者的无助。尽管后来的我在这个世界逐渐找到立锥之地，但是潜意识里的我依然从未相信过我自己。我也不会相信任何所谓有能力的人，且不说他们会不会帮我，他们的能力对于解决我的问题而言多半没什么帮助。我也不希望任何人毫无保留的信任我，因为我的能力不可能让一切尽善尽美。何况就像昨天某人说的一样，尽管我很多时候会对别人特别好，但他们多半不会记着你的好，甚至还可能有人觉得你有所图谋。

我并不怎么喜欢工作，但同样也不怎么期待假期。就像大四最后尽管没有什么事情，我还是会每天在实验室待到十点，或者是现在即使在工位上无所事事摸鱼，我也会在工位混到八点半之后再离开。一个人的日子，或许只有在远行的途中才不够无聊吧。某种意义上，深圳这样一个有足够多荒野的地方，倒也适合我。

尽管你可能不相信，但我真的觉得，我配不上你。

七夕到了。

有一个我一直回避着的问题，也算是得到了答案。尽管是一个很早之前就已经猜到了的答案，我只是一直不敢去按下确认键罢了，但终于当我能够按下确认的时候，其实一切也没有怎样。唔，毕竟，永远在给自己加戏，怎么也不像一个好习惯啊。

我总是习惯于把自己埋没在阴暗中。尽管在我意识不到的时间里，我自身的光芒已经越来越亮，足以去指引一些人。但我还是会把自己的核隐藏在黑暗中，期待有人能够走进暗面的我，期待有人会拯救我。但后来发生的一切，终究击碎了那层薄薄的阴暗的壳。我无力去继续装作鸵鸟，但我却早就习惯了沙子底下的视角。

你是那么一个乐观开朗、积极向上的人啊。每天早晨打开朋友圈的我，都期待着熟悉的名字浮现。你的坚持，也让我的坚持有了意义。如果我成功了，或许半年前的我就会去按下那个确认键，不论怎样，都不留遗憾。

可惜我失败了。失败的我，怎么也配不上优秀的你。无数次欲言又止，终究还是觉得，离开才是最好的选择。我没有怎么去打扰你的生活，你应该值得更好的。但或许，总还是有那么一点不甘心吧。有些东西，能挽回的还是要主动挽回，可从来都不曾存在过的东西，又怎么去挽回呢？

终究还是什么样的人才会更吸引什么样的人吧。尽管某个人给我带来了相当的伤害，但我依然记得她的一切，一切对我的好。她是我见过，最能理解我、对我最好的人。但有些东西，既然不可能重来，那么倒也不必挂怀了。

工作一晃也过了半个月了。最开始的短暂不适期逐渐过去，对现在的工作也开始逐步熟稔。相比于其他人而言，我的工作似乎清闲许多。没有很大的困难，也不需要多少加班的时间和精力。现在的我，一份清闲的工作，一份舒适的工资。好像也不需要担心什么，也不需要去忧虑什么。

我终于也能够去正视七夕，正视一个人的岁月。过往的一切纷繁复杂的故事，细说起，也不过是笑话罢了。我正视我的平凡，但也正视我的不平凡。有缘则续，无缘则止。安于自我，安于现实。我无从改变过去，但我可能往前看。

某个人曾经建议我，往后的日子，可以叫《龙岗开心故事集》。但生活不仅仅是龙岗，也是整个深圳，整个大湾区。或许，后面的故事，不如叫《粤海新风》吧。

但愿，我可以摆脱过去的悲伤和沉郁。那不如，从买一辆车开始，随风而去吧。
