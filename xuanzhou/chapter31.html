# 梦亦无欢

当山峰没有棱角的时候

当江河湖水消失不见

当冬天的雷声震耳欲聋

当夏天的雪晶莹透彻

那是一个伸手不见五指的白天

那是到了故事的最后

男女主角决裂 老死不相往来

你可以不喜欢你不喜欢的东西

但你得允许他的存在

你可以喜欢你喜欢的人

但你也得接受她可能与别人在一起

情人节的故事未必都是单纯的悲剧或喜剧

或许看似悲剧的结局才是最好的喜剧

当我看见列表里一个又一个十分优秀的人

倾吐着所谓的单身的情人节的烦恼

她们所渴望的是更加优秀而温暖的怀抱

却与我没有任何一点点关系

哪怕曾经喜欢 哪怕有过心动

但我清楚 我这样的 不配

又是一年二月十四号了

这也算是一个对我来说意义特殊的日子

一次次的开始和结局轮回里

我总会把这一天作为心理的锚点

割舍掉那所有过去的情感纠葛

新的春天 新的机缘

以及可能的 新的 喜欢的人

所爱的人 或许就是梦里也不会存在吧

# 小城雨后

说好的雨未曾相遇 便寻思远赴溧水

却在溧水城里遭遇了一场大雨

雨后进山 又是另一番风景

无色无想 无相无形

是无想 故无想

何必想 不必想

五十公里的距离 似乎已在另一个城市

小城的夜色景象 仿佛那年盛夏的溧阳

夜幕拉开 灯彩四射 河上风光 梦里水乡

未知风动 未知意浓 也不解 是否心动

也许我想得太多 但我只是不愿意看到

本来可以过的更好的人 因为我 而受苦

# 是非妄想

爱 经过的地方

我再多愿望再多的想象 填不满空荡

爱 美丽的皮囊

有太多欲望有太多迷茫 你留的时光

爱 固执的假象

有许多勉强有许多假装 有许多的狂妄

爱 他孤独流浪

总不停的猜总不停的想 在某个地方

大年初五 似乎也应该到了迎财神的时候了

但说到底 我对钱财本身没有那么大的兴趣

虽说没有钱万万不能 但钱本身也不是万能

我这辈子不会很有钱 但应该也不穷困潦倒

总有着太多的想法 不会实现也不想去实现

或是因为一点胆怯 或是因为不愿承担失败

我承认起码到目前为止咋说我都是个loser

也许有人觉得我说这话听起来好像个coser

或许吧 面具戴的太久早就已经摘不下来了

好像只有眼前短暂的生命才是我的生命

或许是啥也不干干脆的睡上一个白天

或是刷上两个小时海灯节任务什么的

什么都很无聊 但总得想办法耗着时间

甚至不想去思考十天后就要见面的成绩

甚至不想去思考下周就要发布的新手机

甚至不想去思考后天上班要去做的工作

甚至不想去思考。。。

是真的不想去思考嘛？

只是单纯的害怕而已吧。

是真的不想去追求嘛？

只是单纯的害怕而已吧。

是真的不想去脱单嘛？

只是。。。

经过的地方 美丽的皮囊 固执的假象

以及 孤独 以及 流浪

以及 所有的一切的 妄想


是未曾设想 未出新年 花开已然绚烂

红白梅花相配 玉兰点映 更有早樱初绽

春天 似乎比预期来得要更早一些

满怀期待 又不再期待 奢望和幻想 梦魇

是梦中人 是梦中亦不相配之人

故事注定不会有结果 且随意去猜测

时日无多 她总值得一个更好的人

我总以为我看见了希望 可事实上总是自己多想

久而久之我也习惯了这样 这种感觉就是很丧

# 不必多想

既然本质上只不过是自己想太多了

那就没有什么好留恋惋惜的了

虽然有的时候错过确实可惜

但是还是错付的话更可笑一些

世界是自己的 和他人毫无关系

所以没有自己 其实也就没有了整个世界

听起来怪怪的 但唯心主义的话是这样子

恋爱的冲动很容易让人迷失掉所谓的自我

好好吃 好好玩 好好追逐理想 虽然没有

多掌握一点知识技能 万事不求人

其实我好像已经差不多做到了这一步

但不是因为技能点高 而是没有了需求

人就是这么一个奇怪的东西 怪无聊的

每一个人都觉得自己很失意 在低谷 爬不起

一切都没意思 前途多舛 自己又很菜的样子

谁不是这样子呢 只有主角的生活才是生活

我这种npc 最大的存在意义就是指引一下关卡

每次我秒回别人的时候 都觉得自己很卑微

但我就是很卑微啊 就像一个小雪人

谁都可以踩两脚 踢两下 虽然可能踢不动

雪人没有心 但也是别人用心打造出来的

寒冷的天气让他坚强 但他也知道雪总会化

就像我本人好像也是一个矛盾的集合体

可以坐在湖边看一个小时太阳渐落

也可以凌晨四点去爬敬亭山期待日出

消失在没有人看得见的角落

无声无息 无影无踪 无人知晓 无人问津

或许有一天就可能永远消失在了哪里

也许 我想要去找一个对象 最大的原因

还是想要与这个好似不属于我的世界

留下更多的羁绊 更多的牵连 更多的想念吧

也许是从高空坠下前最后一秒闪过的倩影

也许是永远闭上双眼前脑海飘过的最后画面

道理我都懂 但 起码到现在为止 一切都不值得

# It’s rainy

迷失了野望 看不见前方的光

那便停下 不去寻觅 不去幻想

我依然会去羡慕 每一个勇敢的人

每一个看清了现实还在奋斗的人

终究不会也不敢再去期待些什么

我不是做题家 也不配成为做题家

可在这个做题家们内卷着的时代

我又有什么办法去面对呢。。。

只能这样 无路可退 无路可逃


我希望 有朝一日花会开

我希望 梦想不要成了白

我希望 对这世界还有爱

我希望 结果能顺应期待

我希望。。。

罢了

一切都仿佛过去了很久 却又好似方才

什么欧亨利式风格 情理之中意料之外

现实就是现实 不会有小说那么的精彩

# 晴·雨

大约会是我人生中最废的一段时光吧

无所事事。不报念想。满怀绝望。

天空是蔚蓝色 窗外有千纸鹤

陪我弹琴写歌 每一分每一刻

晴朗无风的夜 未必是好的预兆

就像现在 长夜明朗 不见星辰 明朝仍有雨

甚至 雪的踪迹 犹未可知


村里又一次下起了淅淅沥沥的雨

忽然发现日子过得还挺慢

但 好像 又挺快的

记忆恍惚间回到了2020年8月24号

那天的雨 好像也是这么大

但是 有些人应该是不会再看见了

北门外的油菜花海没了

但似乎东门外的山上依然是金黄一片

所有美好的珍视的事物可能都会消失吧

但消失过后 也会有重新开始的一天

晚安 世界

# 旅行者·瓯江

众所周知 沪苏杭都是传统的江南水乡

但温州 算江南吗？

滔滔瓯江从城市的一侧流过 汇入大海

给城市留下了密布的河网 与大海的清新

在离市区六十公里外的洞头岛上

近海的浑浊和泥沙与东海的蔚蓝

筑起了一道肉眼可见的分割线

惊涛拍岸 是海的味道

而在城市的中心 河网密布

走在五马街和南塘街上

又仿佛漫步苏州 走过观前与山塘

美景 美食 美人 相得益彰

这里终究也是江南吧 一种别样的江南

却也是 我所期许的江南的模样

就像每一个我认识的温州人一般

勤奋而聪慧 美丽或帅气 潇洒且善良

但终究 是时间太过的短暂 却无暇好好欣赏

如果以后有机会 我会再一次来到这里

但这时 我不得不选择离开了


每当我的人生遇到了瓶颈与困难

我总是想要去看一看大海

在大自然的磅礴力量前

一切的一切都只不过过眼云烟

爱与恨的林林总总 也亦无足挂齿

纵使确实放不下 却又何如

也终不能一直这么苟且下去

# 电设告别

电设的2021春招落幕了 我的春招也落幕了

2017年加入合工大 2018年加入电信科

2019年加入电子设计创新实验室

2020年参与第一次电设新生选拔

2021年了 未来终将属于更新更卷的人们

也许不久的将来 406网络会所将不再属于电设

但我仍会记得 一个又一个在这里度过的日子

单片机 电赛 ican 秋招 考研 春招

那些与电设相关的人 相关的事 或爱或恨

尽管我来得太晚 无缘见证电设曾经的辉煌

但我大约能够看见 电设越来越好的明天
