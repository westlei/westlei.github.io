# 牵丝成戏

# 光遇

阳光倾落在综一的走廊上，感觉暖暖的。

墨水挥洒在数学的草稿上，感觉凉凉的。

看到一句话：遇到事情的时候，

有的人硬刚，有的人逃避，

还有的人会选择曲线救国。

我总是会下意识去选择逃避，

可嘴里仍然说着我是在曲线救国。

但结果是骗不了人的。


你说，你为什么对我这么好。

我说，因为我对每一个人都很好。

你回了一个表情，说晚安。

我也回了晚安，说祝你幸福。

雪白的屏幕上，仅存的一条记录格外显眼。

突然又来了条消息，有人来问问题。

我随手敷衍一下，然后便划掉了记录。

我是说过我对每个人都好，但我只能看见你。


生病了，心情特别不好，控制不住情绪。

你还是一如既往地会问候一下。

我踌躇了许久，点开又退出，退出又点开。

终于还是假装什么也没看见。

等着大火变成小火，小火变成无火。

再像什么也没有发生一样。

说一句，最近有点忙。


一旦过分亲近了，关系总会变质的吧。

那样对谁都不好。像现在这样，就很好。

我默默地看着你，却假装看不见你。

习惯性地排斥，排斥一切的亲密关系。

也只有足够疏离了，才能足够长久吧。

如果你已经选择了去浪迹天涯，

那就不要说为什么你的旅途如此孤独。


孤身打马南屏旧桥边过，

恰逢山雨来时雾蒙蒙。

想起那年伞下轻拥，

就像躺在桥索之上，做了一场梦。

梦醒后跌落，粉身碎骨，无影亦无踪。

# 牵丝成戏

我和你 最天生一对

你憔悴 我替你明媚

问世间 哪有更完美

愿谁记得谁 最好的年岁

每350个中国人里面

就有一个人在考研

作为1/350中的一份子

谁还不是这喧嚣尘上的内卷里

时代的一粒炮灰呢

梦是要做的 但也是会醒的

把该读书的时间用来做梦

等到失败的时候 就能更加体面

# 巴巴托斯

我好像以前不是现在这样子的

我是什么时候变成现在这样子的

当年发生了什么 摧毁了我什么

其实这一切我都已经有些记不清楚


大约从那时起 我就已经开始只会混日子了

没有对美好未来的憧憬 只有日复一日的摸

现代社会的变化真的太快 但这并不可怕

怕的是你已经不敢去面对 这个魔幻的世界了


人总是要做人上人 所以奋斗只是为了不做人下人

这就是为什么这么卷 也是为什么这么累

所以或许不要做人会好一点 或者早早去世吧

我已经死了 七百五十多天前就死了 只是还没埋


谁考研不是为了想上岸呢

只不过 想不想是一回事

能不能是另一回事罢了

我依然向往一个光明的未来

希望有那么一天

树在 山在 大地在 岁月在

你也在


或许 在亿万时间线的某一条里

终究是有那样一个更好的世界的吧

但 我注定也见不到了

干点正事吧 巴巴托斯！

# 无色海

最近状态很不对劲

有的想法有些

人不能 至少不应该

该逃避的时候还是要逃避的


海 没了记忆 沉睡进时光里

风声静止 天光散去 等不来归期

无论如何 过去的终究都还是过去了

未来的也用不着多少期待


课设恍惚间也算是结束了

如果按课表来看 这学期总共的四学分修完

这个学期也似乎就即将告一段落了

终于也不得不正视自己蜕变为纯粹的打工人

没有钢铁般的意志 却沉迷在虚幻的古风世界里

想要一个人去一个绝美的地方 看一场大雪

过去和未来 俱与我无关 人间不值得


在世间 本就是各人下雪

各人有各人的隐晦与皎洁

你说了 她会懂 她不说 你也明白

终究各自散场 徒留下不知所以的悲伤

也许几年后 再回想起考研的生活时

我记不清新安的风 敬亭的雨

想不起实验楼午后阳光的温暖惬意

却单记得一抹影子 若有若无 如梦似幻

# 离场

一切林林总总的祸端总结起来无非俩字感情

没有疯狂的爱自然也就不会有无端的恨

20年代了怎么还会有这种狗血的故事

为了一个爱而不得的女生争风吃醋

仿佛回到了琼瑶剧武侠片当道那个年代

武德充沛 不像现在很多年轻人 不讲武德


所以既然选择了跑路 那就该干啥干啥吧

比赛开卷 成绩很卷 考研特卷 什么都卷

可以有梦想 但那不是成年人该有的东西

20岁的人应该想想自己的以后 怎么活

这样的话 想着想着就知道自己不配活着

自然是点到为止 先考虑明年后能混几年


所以有的话说出来以后 其实就很释然了

不管究竟有没有 被应当听到的人听到

所以有的决定做了以后 其实也很轻松了

不管究竟会不会 像我猜想的那样变简单

转眼中秋国庆已经是差不多一个月前的事了

月亮又一次圆了起来 期待已久的秋游快来了

庐州月光 洒我心上 牵挂的人儿们 又在何方

有的人就是只适合当朋友 不管她们再漂亮

我也不会生出 任何一点非分之想

但有的人 可能一开始想法就不单纯了

# 不如结束

相比得到别人的爱

努力变得更优秀反是一件更简单的事

所以大家都努力变得优秀

大家都在卷

可是努力变得优秀又是给谁看呢

反正不是我自己 我不需要

卷王和躺平人生活在同一个世界里

前者总是有意识忽略后者的存在

他眼中只有卷


但事实上 这个世界上还有很多种生存方式

这一切都只取决于你对你自己的选择

我很不想卷 但我没有后路

如果后退 只会面对比现在卷一万倍的现实

要么前进 去未知的方向 找未明的出路

要么 结束这场让人无法愉悦的游戏

# 打工人上人

打工人坐上了黑车

准备开始打工的旅程

意志不坚定的打工人

扛着各式各样的行李箱

随时准备跑路

而意志坚定的打工人

一个背包走天下

打工人绝不跑路


有的人她每天都笑着

谁也不知道她经历着什么

也不知道哪一天

她可能突然就疯了

# 既错即止

莫名心情好了不少 考得上考不上的也无所谓

学啥学 难得来一趟总得寻找点快乐

昨天跟学弟聊了挺多 绕翡翠湖校区一圈一圈一圈

人生又何尝不是一圈一圈一圈

循环往复 永无休止

不管你怎么走 怎么想 都逃不出你所在的那个圈

有圈 所以卷

梦想 情怀 誓言 信仰 都卷的魂飞魄散 无影无踪

人可以去尝试 但终究还是应该安于自我

做自己擅长的领域 放弃自己努力也做不好的东西

明知不可为而为之 只会让一切都不可为

感情无分对错 只有有无

但是有和无的本身 就分对错

既错 即止 莫复错

# 从不圆满

老区的日子 安宁而平静

斛兵塘的秋水 春华路的落叶

抑或淮河路的繁华或盛景

期待的终点无非如此

世界本无所谓圆满

但心中圆满 世界便也圆满


渐渐地 她也没了生息

在某一刻 迎着光沉寂

相比可能是更加残酷的现实

我宁愿相信 有的人只是陷在了梦里

然后出不来了 那不如让她再睡一会儿

幸运字符 也许会带来幸运 也许不会

但我还是希望列表里这些朋友们

都能拥有足够的运气

虽然有的人 走着走着就散了

不知道从何时起 谁不再理会谁

可能的未来 谁也不再认识谁

# 清言未语

有的人 未见 即会心动

有的人 遇见 便是幸运

有的人 再见 或许永别

心怀的向往 也许并不是真正的向往

心怀的喜欢 也许并不是真正的喜欢

心怀的期待 也许并不是真正的期待

午后 傍晚 夜深沉

想起曾经一个个在翡翠湖奋战的日子

合柴 罍街 宁国路

是夜色为寒风披上了行囊

还是寒风给夜色一层光芒

故事 是巧合中的巧合 还是命中应有的劫数

清笔未落惊风雨 语出无声自动人

也许 这就是所谓 喜欢一个人是藏不住的吧

有的人兜兜转转 最后还是走到了一起

有的人一往情深 最终还是输给了岁月

或许当初 勇敢点 不去纠结那么多有的没的

也就没有上面这句话了
