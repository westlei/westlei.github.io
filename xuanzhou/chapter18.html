# 光明

当移动互联网悄然走进了时代，当后浪们拥有了获得信息的自由时，殊不知，大数据与算法这只看不见的黑手，却在一点一点将着自由的光芒蚕食。当你看到的都是你爱看的，你听到的都是你爱听的，这时候的你，又会变成什么样子呢？

早在战国时期，邹忌就已经明白了“兼听则明，偏听则暗”的道理。“吾妻之美我者，私我也；妾之美我者，畏我也；客之美我者，欲有求于我也。”唐太宗李世民也深谙这个道理，他重用说话不中听的魏征，就是为了“兼听则明，偏听则暗”。可是随着时代的变迁，这些中华民族的传统却在互联网的大潮中节节败退。优良的传统在埋没，在丢弃。算法和大数据在不知不觉间做到了封建统治者几千年都无法做到的事——让民众自愿的去选择去相信他们所看到的一切。因为，他们所看到的一切，都是他们爱看的。

随着互联网时代的进步，我们也能发现一个现象，整个互联网世界的戾气都在不断变重。当你每天看到的都是一样的东西的时候，你就会日渐盲目地迷信它。就像题目中写到的同声相应，同气相求。人们总是关注自己喜爱的人和事，久而久之，就会被同类信息所环绕、所塑造。你已经不再是你自己，而是被你接受的信息，接受的环境塑造出的你。就像我们不能理解肖战粉丝们的狂热举动一样，他们也不能理解，为什么这个世界上还会有跟他们意见不一样的人，还有不喜欢甚至诋毁他们的“哥哥”的人。就像我们不能理解为什么特朗普一条推特发出去，就有不在少数美国愚民喝起消毒液，他们总是盲目相信，大统领说的一切都是对的。这种感受，是一种潜移默化的过程，外人难以理解，更无法体会。

在智能互联网时代，你的未来真正掌握在了自己手中。多去看阳光正能量的内容，你的世界里就会充满阳光和正能量；看多了阴暗和丑恶世界的人，内心也终究会变得阴暗丑恶。但，这种改变，是从哪里开始的呢？是一次从心所欲的浏览，还是一串惺惺相惜的点赞？是一回情不自禁的分享，还是一场突如其来的感动？在现代互联网的世界里，当你看到的世界是光明的，你的世界便是光明的；当你看到的世界是阴暗的，你的世界便是阴暗的。

所以，与其去陷入黑暗，不如去拥抱光明。我们要看到，我们的世界里不仅仅有那些只会推诿责任一问三不知的干部，还有李文亮医生，钟南山院士这样不畏生死勇往直前的人。我们要看到，我们的世界里不仅仅有那些网络上怼天怼地的键盘侠，还有罗翔教授，李子柒这样的正能量博主。我们的世界里不仅仅有阴暗，还有着更多的光明。所以，去拥抱光明吧，后浪！

# 京城往事

当T65次列车越过西四环边的丰台站缓缓驶进城里，眼前的一切变得模糊又熟悉。奢华的北京南站并不是铁路的尽头，龙潭公园和东二环的车水马龙氤氲在眼帘里。嘈杂的人声鼎沸在广场上，拖着行李越过天桥来到那胡同里给窗口递上10元，把许久未用的市政交通一卡通激活。122路的老巫婆和403的小财神挤在那里，却久久没有发车的意思，等了许久才听着柴油机隆隆的响，便刷上四角钱，数着一个个熟悉的站名，去见，那许久未见的外公外婆。北京，一个既非我出生非我成长的地方，却伴随了我十余年的记忆，在每一个重要的时刻里，从未缺席。

十四年前的冬天，我还是一年级的小萌新。从重庆出发坐了一天的火车，来到了北京。崭新的西客站旁，是新婚不久的小姨的新家。宽敞明亮的大房子里，承载着我童年的记忆。动物园和天安门还是那么的新鲜，国旗飘扬在风里。龙潭公园的庙会也是第一次去，人潮如织里感觉也甚是新鲜。神州大一路那时还到马官营，1462次硬卧也还是木质的床铺。这，便是遥远的京城记忆的开始。

十二年前的冬天，当南方还在大雪的覆盖里，刚刚转学回南京的我便再一次来到了北京。刚刚出生不久的表妹还在襁褓里，张灯结彩的冬日里感受着团圆年的气息。冒着寒风去北四环工地里看着不远处的鸟巢初具雏形，也到什刹海的冰场里尝试过冰上的刺激。而半年后再一次到访时，奥运的人流却已挤满了城市。还记得军事博物馆和首都博物馆的壮观，也记得世纪坛旁中央电视台的老楼。天安门已经不再新鲜，但后海的胡同依然吸引着我的注目。清华大学的食堂廉价而美味，可渺小的我早已认清了同他的距离。

一转眼又过去几年，我升入了初中。国庆节时，我又再一次来到北京城里。依然是熟悉的122，依然是轰鸣的老巫婆。只是四路已经不再，大一路也随之迁往了马官营。刚刚修成的北京西站南路上开过双层的72路，大观园池塘的鸭子充满活泼可爱的气息。京沪高铁的开通拉近了时空的距离，前门大街现代的铛铛车渐渐把传统抹去。飞驰的动车越过山东大地，也意味着一个时代正离我们远去。

再一次来到北京是七年前的夏天了，从大草原上的赤峰坐了一夜的火车来到西直门，再辗转13号线去往外公外婆在望京的新家，可随即便迷失在望京西的出站口，找不到去往花家地的路。从城西南到了城东北，现代化的望京让人迷失在高楼大厦里。南锣鼓巷的人声鼎沸里，少了些什刹海老胡同的烟火味道。不远的798街区里，是现代艺术的辉煌。

渐渐的，去北京不再是为了游玩，更多的是过年时一份家的味道，一种大家团圆的感觉。过年去北京住几天，看看年迈的外公外婆。熟悉的T65/66次江南明珠号，熟悉403路开往环行铁道。还记得一五年的冬天，北京下的一场雪，很美，很美。

可随着高中的课业压力不断增加，再到北京却已是高考后的暑假。在旅途中突发了急性肠胃炎的我拖着虚弱的身躯从沈阳来到北京，渐渐康复，渐渐能够骑着车远行。已经不会再在望京的大街小巷里迷路，也能够在脑海镌刻北京的地图。在又一个春节里，看尽了故宫的天空，和冬日的焰火。去北京读研深造的想法，渐渐成熟，却渐渐沦落。

转眼又是两年，曾经辉煌的T65/66已经化为了绿色的垃圾桶，轰鸣的403路财神京后也已变身各种电动。时光在变，北京也在变，可能不变的，只有那历尽沧桑的紫禁城，和那承载着繁荣与梦想的，北京青年们。

还记得八月长安橘生淮南的小说最后，当洛枳和盛淮南重聚在景山的山顶，看着那日暮时分华灯初上的北京城的场景。北京是什么？是清华园意气风发的学子？是金融街衣冠楚楚的白领？还是后厂村深夜时分成群结队的码农？他们都在北京，他们又都不在北京。

# 百景·江南

外面的雷雨依然一阵又一阵，没有停歇的意思。今年的梅雨季似乎格外的长，又格外地暴躁。暴风骤雨，电闪雷鸣，狂风阵阵，洪水涛涛。这不像我印象里的江南，但确是我生活着的江南。这生活了近十八年，却从来没有喜欢过的地方。

风到这里就是粘，粘住过客的思念。雨到了这里缠成线，缠着我们流连人世间。在一切的诗歌词曲间，江南都是美好富饶的地方。应天府、扬州府、苏州府、松江府、湖州府、杭州府、太平府、宁国府，是扬一益二，是无徽不成商，是上有天堂下有苏杭，是苏湖熟天下足，是买不尽的松江布收不尽的魏塘纱。经济发达的长三角，是文人墨客的梦，是无数人向往的江南水乡。

然而，这里的夏天也是酷暑难耐，这里的冬天湿冷彻骨。淅沥的小雨是诗意，可连绵数月的梅雨则是折磨。江南就像是一位外表美艳的姑娘，她以诱人的外表吸引着你，却也有着属于自己的大小姐脾气。就一如江南的女子，婉约而不失傲气，清丽而不失潇洒。令人一见倾心，却难以得到她的回眸。台下人走过，不见旧颜色。台上人唱着，心碎离别歌。情字难落墨，她唱须以血来和。戏幕起，戏幕落，谁是客。

六朝的脂粉气氤氲了秦淮河的烟波，大运河的交错织出了扬州城的繁茂。园林绣坊装点了苏州府的奢华，水漫金山书写着润州城的往事。也曾独自在江南的大地上走走停停，却总与某位网友相交在未知的巧遇。烟雨之中的洞庭西山，紫色绚烂的薰衣草谷，瑞雪缤纷的城隍庙里，灯火璀璨的七里山塘。旅途里见过的无数浪漫，终究只剩似梦非梦的回想。

但终究，人还是要回归于现实。再多的浪漫，也需要有人同你分享。一个人的旅途，收获的总是突然而然的感动，它可能关乎自然，但却更关乎于内心。也许是情感焦虑里的一份释然，也许是学业压力下的一份超脱，也许是沉郁之后的豁然开朗，也许是忙碌工作间的心绪恍惚。但终究，离开了那份心境，纵景致仍是那份景致，可感动也不再是感动。

江南百景，妙笔生花。是人铸造了江南的风景，还是江南的风景塑造了人？这似乎本就并不重要，因为最美的风景，是人。

# 九万字

当坊间最善舞的女儿死了，京城就该有一场大雪。

世上九万字，情字最伤人。

飘泊的雪，摇曳回风。诗意灵魂，更迭情人。总管用轻浮的茂盛，掩抹深沉。

网易云之所以为网抑云，是因为人世间总有那么多遗憾和不圆满，而伤情的音乐总让人相会在热评之间。也许白天散尽了快乐，深夜里剩下的总是忧伤。人总有两面性，每一个熬着夜的灵魂里，都浸润了忧郁和悲伤。

有谁不是，少年热诚。孑然一身，爱一个人。望尽了毕生温柔眼神。写得出最刻薄的字文，以讥诮这庸尘。却不忍，斥你毫分。

曾经十七岁的我，喜欢上了一个注定不属于我的人。只因北京街头看到的一眼，却再也放不下了心中的念。后海的惊鸿一瞥，半场一载的缘。所谓埋在心底的喜欢，终究还是会被有心人发现，却不如早做坦白。可是我太年轻，可是我什么都没有想好。给了我太多的机会，却终究也没有去珍惜。没有什么好去说的，错的不过是那时的自己。

我也算万种风情，实非良人。谁能有幸，错付终身。最先动情的人，剥去利刃，沦为人臣。我爱你苍凉双眼，明月星辰。不远万里，叩入心门。一个孤僻的唇，摘获了你首肯，献上一吻。

此后来到计算机系的两年里，我又认识了许许多多的人。也曾一次又一次动过念想，却从不敢把朋友的关系再进一步。沉溺在虚拟的世界，着迷于有趣的灵魂。但喜欢永远是卑微的，先动情的人，把心都都交给了对方，自己便啥都不剩。所有爱情的悲剧，总是来源于痴情者。因为他们的心里，她就是世间的唯一一抹绝色。世间的其他一切，都不再重要了。

越繁华的地方，独自走过的人就越感到孤独。不论是中山路珠江路南还是察院场观前街西，不论是万达金鹰还是苏州中心。当你走在这偌大的世界里，身旁是熙熙攘攘的人群，而你与世界的联系，却只剩了一部区区数寸的手机。独自一人的旅行，未必算是旅行，也许应当是修行。修行的意义，在于寻找一份独属于自己的感动。

喜爱在山川湖海间穿行的，或许是风吧。像风一样，无牵无挂的来，无牵无挂的去，不留下太多的踪迹。每一个孤独的人，或许会拥抱白天，但更热爱的，定是黑夜。黑夜里，世界只剩下自己，钱塘江上的空气，西子湖畔的落雨。松江清晨的第一缕朝霞，金鸡湖边最后的一丝牵挂。就连风，也有着那么些许的惬意。

可我还是放不下你。

# 宣州·晴

七月的尽头，越过石臼湖，南望牛头冲。

间歇性的伤感总是在所难免，但可以救赎。感情的惆怅在深夜间酝酿，却也不会在白天展露。该努力的时候还是得努力，尽管有的时候努力本身可能并没有过多的意义。

重情之人的情感是个枷锁，在找到下一个合适的人之前，这份镣铐会一直佩戴着。心中只有一个人，也永远只会有一个人。就是一份传递的执念，没有新的人来化解，它就会一直存在下去。

总有人劝我去说，其实大可不必。人在大四，未来大概率还是去远方发展。现在所争取的一切，到未来，谁也不知道会怎样，但大概率不美好。既然能想到结局，那为什么还要去开始呢？

就让这些点滴埋藏着吧。也许未来再相见时，提起曾经，无喜无悲，仅轻轻说道：那时我也曾喜欢过你。这样的结局，或许比永远不再是朋友的以后来的美好些。

阴雨过去，阳光出来了，未来还是会明亮的吧。

其实表白失败不一定做不成朋友，但我显然不是。内心打成了死结，残存的高傲也会使我主动去远离。那不如，珍惜做朋友的美好。

重情却又放不下残存的自尊，这可能是我未来失败悲剧的预兆吧。毕竟每一个拥有这样特质的男主，似乎都不曾有过好的结局。

# 虚无之梦

小雪时节放起的寒假，在处暑时节才终会落下帷幕。清明雨上兴起的幻想，在夏至未至时便无影也无踪。当这个世界被改变的时候，每个人都不过只是时代背影下渺小的一粒沙。无力去抗拒，也阻挡不了历史的车轮滚滚向前。就像美国的五百万感染远在天边，但大盘的绿色近在眼前。

考研更多的也只是一个梦了吧，抑或是一个幌子，一个摸鱼的借口。我以为我卸了游戏就能抵制了诱惑，但我发现我想的太多，总有东西能够吸引着我。不论是无知的幻想还是确切的迷茫，都无法把我空空如也的脑海点亮，不敢去想，以后怎样。

当我看到巨佬们靠着刷SSR在一个暑假里挣上了高达五位数的财富，当我听说神佬把两年半的创业项目卖给蚂蚁金服转手就是一个七位数。我看看蓝蓝的单元格里黑黑的名字陷入了苦涩。执着于太过无所谓的东西，就会像蓝桥杯的校赛一样，只有一个人的赛道，不过是比了个寂寞。

有的大佬逐渐走出了考研的赛道，在多彩的世界里散发光芒与热量。乌龟在考研的赛道上缓慢地爬行，看着后来的兔子一蹦一跳地从身前跑过。它想要退出这条赛道，却又怕错过了兔子睡着。它小心翼翼地打算抄小道，绕了半天却又回到原来的路上。看着睡眼惺忪的兔子揉了揉眼重新奔跑，失望的乌龟在树下睡起了大觉。
