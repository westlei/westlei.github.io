# 年号令和

# 流离

流离在现实之外的人，甚至难以感受到消逝的时光。

不经意间的季节变幻，岁月重又交错在开始的地方。

这消极而厌倦的日子，每一天都平凡，沉默与荒唐。

昨天，今天，明天，后天，其实并没有什么不一样。

既然追求刺激，那就贯彻到底，把品如的衣服穿上。

早就知道后果，却仍然执迷不悟，追寻那一丝希望。

渐忘却了，为所有爱执着的痛，为所有恨执着的伤。

直到弹起那心爱的拉普拉斯，一切空剩下大梦一场。

# 肥宅

无论犯了无数的错误，又触发了多少警告。

仍然写着风花雪月的散文，唱着自己填词的歌曲，

用脚步和车轮丈量土地与风景。

依旧喜欢去谈论着某一朝代的历史，

或者讲一讲政治环境和经济的形势。

翻一翻国家地理，看一看世界军事。

各色的规划图中，是不一样的城市。

搜集一些感兴趣的资料，挖掘不为人知的密辛。

敲上几百行花哨的代码，没有bug便是小确幸。

不再热衷比赛或是寻爱，不去希冀小姐姐的理睬。

真实的世界发生的一切，也许显得比小说更精彩。

但却让人不忍去回望的，那笼罩着的伤感与无奈。


肥宅没有女朋友，但他们拥有着快乐。

而我不配拥有爱情，更不配拥有快乐。

# 舔狗

这个世界上本没有什么一见钟情，

有的只是舔狗欲的发作和荷尔蒙的悸动。


等到舔狗欲过去的时候，

一切都将终结于一无所有。

只留下所谓悲伤的故事，

那，是属于舔狗的悲哀。

# 亏欠

越是不愿意亏欠的人，最后总是亏欠的越深

越是不愿意伤害的人，最后总是伤害的越深

越是不愿意见到的人，越是会和你如影随形

越是不愿意认识的人，越是会和你纠缠万分

人生总是充满着巧合，巧合里却充满了悲伤

不是每件事都有开头，也不是凡事必有结尾

更多的时候，

她不知不觉地来，又不知不觉地去

不掀起一丝波澜，只在心头留下一串涟漪

就像那爱情

为人称颂的美丽下

作用在灵魂深处的吸引

# 令和

令和的第二天，并没有比前一天更好些。

凌晨搞废的单片机还在bb着电梯门的开关，

新鲜出炉的课设报告也依旧是格外的烫手。

关掉电脑并不能忘掉烦恼，

正如骑上车不能奔向自由。

时光在不知不觉间流走，岁月是那最无情的神偷。

但荒废的日子里，总也许有那么些光明。

新港的人多的超乎意料，但小姐姐仍是那么可爱。

国购巡茶排队令人苦恼，但奶茶的味道确实不错。

生活是一味苦涩的中药，但你可以往里面加糖。

这样，它就会有点甜。

也许，这便是喜欢的感觉吧。

其实本没有什么喜欢，

你快乐，我也快乐，

那便是最好的幸福。

# 立夏

立夏已至，夏至未至。

喧嚣的时光里，内心的浮躁愈演愈烈 。

一波未平一波又起的琐事，

把有限的时光置入了无尽的黑洞。

半个月一晃而逝，恍若梦里，一事未就。

忙碌使人忘却，忘却时光，也忘却悲伤。

有的大佬说，大学最幸福的事情，

就是把自己热爱的东西做到极致。

有的大佬说，大学最幸福的事情，

就是陪自己所爱的人一起学习成长。

他们是忙碌的，他们也是幸福的。

闲暇使人空虚，忙碌使人充实。

这便是，属于大佬的幸福。

又要到520了，祝福列表里面每一个有所陪伴的人。

愿你们，和你们的另一半幸福。

无所期望，便也无所谓失望。

无所喜欢，便也无所谓忧伤。

无所有，无所无，无所是，无所非。

# 忘记

有的东西，我以为我忘了，却发现我还没忘

有的东西，我以为我还记得，却发现早忘了

快乐都是别人的

悲伤才是自己的

# 无常

今年的5月18没有暴雨，但C语言的测试仍然继续

今年的英语书依然不懂，但却已经没有必要再看

今年的5月19又到来了，但我依然没有找到女朋友


没有悲伤，没有失落，就连泪也不曾再流。

忘记学习，忘记自我，寻不到的梦里花落。

喜欢的人，喜欢的事，都已尘封在岁月里。

不再提起，不再伤痛，时光匆匆只愿错过。

# 六月·高考重忆

又到了高考的时候了，想写些什么，

却发现，什么也写不出来。

在时光里磨灭了的记忆，最终只剩冷酷无情，

就像，一台沙雕的复读机器。


又到了学长学姐，学弟学妹们即将毕业的季节。

又有着一部分人，要与那熟悉的一切挥手告别。

又会让些许爱恋，因着距离的远去而灰飞烟灭。

727天前，高考落下帷幕；

727天后，毕业论文答辩。

不记得那个女孩到底有没有喜欢过，

也不知道如今是否有值得追求的人。

未来已经不远，日子却还在平平凡凡的过；

前方一片迷雾，却没有动力让我重新振作。

终究成为了学弟眼中的学长，

也终会变成如今学长的模样。

是一无所有？是应有尽有？

未知的现在，未知的未来。

列表里没有学弟和学妹，高考加油便显得虚伪。

相比之下，或许更应该为之加油的，却是自己。

以为来到了人群中心，其实却一直在边缘徘徊。

既已做了错误的选择，那也不要再去为之后悔。

那怯懦者无谓的逃避，不过为了故事快点结尾。

# 霞光

天鹅在空中自由翱翔，远去的霞光下一抹斜阳。

笔下生花的锦绣文章，在看不见摸不着的远方。

一颦一笑萦绕在心房，期待是不切实际的幻想。

曲终人散迷失了过往，往后余生里愿不再思量。

# 中场

2019年的上半部分，就这么平淡地结束了。

一无所知，一无所成，一无所为，一无是处。

翻开相册，看一月一号那天立下的FLAG：

不再有所喜欢，却也找寻不见真正的自己。

心态越来越崩，脱发熬夜和裸考成了日常。

话是越说越多，可真的做事却没做了几件。

暴富无可期待，只剩愈发的贫穷环绕着我。

不像二零一八，却是因为比二零一八更惨。

可能唯一的小幸运就是认识了这么多人吧，

优秀的，漂亮的，聪明的，可爱的，温柔的...

但唯独，就是没有找到，我将喜欢的那个人。

As a Virgo，爱上一个人很难，但忘记更难。

就像，心裂开了，需要很久很久才能愈合吧。

再也不去迎合任何一个人，我选择随心所欲。

也许曾经对你很好，但也一定会有一天厌倦。

也许本就不曾做人，也许只是未曾学会做人。

有一天若是突然消失在你的世界，请别惊诧。

并没有什么发生，只是突然厌恶了生活而已。
