# 年关将近

# 末日传说

距离那个世界末日的传言，已经过去九年了。世界没有毁灭，也没有变得更好，只有日历从12/12/21翻到了21/12/21。

九年的时间看起来很长，实际上却又很短。2013年的北京地铁14号线从张郭庄到西局，2021年底的14号线西段还是只有这么长。九年前我手上可能只有个50块，九年后也只有个50k。再过九年我30岁，还不知道会怎么样呢，希望能攒个50m吧。按说人家随随便便出手都是多少亿，我这个理想应该不是特别离谱吧。

今天是冬至。下了一天雨的深圳，比江南还要更冷几度。来深圳半年了，难得有一点想家了。虽然我也不知道家是什么，家在哪里，我只知道这里不是家。生活不是很忙也不是很闲，就是精神压力有时候挺大的。疲于应付无处不在的甲方，不知道干什么的外包和没事找事的测试，并不是工作的乐趣。

但人生总归也没什么值得快乐的，不是吗。

# 2021年终总结·逆水行舟

在上一年的岁尾，我曾经写下这样一段文字：

当你在逆境里，光让自己往前走就已经要竭尽全力的时候，看太远只会让自己掉沟里。顺境眺望远方，逆境立足当下。每一个人的2020都有着或多或少的不幸，但我们应该相信，我们会有一个更好的2021。

2020年的最后一段时间，大抵是我近几年精神上最难受的一段时间。感情的挫折夹杂着考研的压力，结果反而是彻底的放纵。结局其实是确定的，所谓的努力只不过是一种强作镇定的伪装。最后一场考试提前半小时交卷离场，我的考研生涯告一段落，落魄的如同一个路人甲。

一场大雪洗净天空，也给我的心情刷上了明亮的白色。翡翠湖的年末聚会，让年末所有的遗憾渐渐消散。在大学的后两年时光里，合肥一直对我来说是一个意义十分特殊的地方。那里有一群对我很重要的朋友，一群无关时间和距离的朋友。我所珍视的，莫过于朋友与家(406)了吧。

2021的日子，也就这么开始了。一切的境遇仿佛理所当然般的顺畅，就好像一双冥冥之中的手抓着你走向了既定的道路。而努力和挣扎，并不会造成什么改变。既然如此，不过如此，躺平罢。

# 2021年终总结·十方世界

高德地图年度出行总结告诉我，这一年我去过十七个城市，但实际上肯定不止。南京，镇江，溧阳，仪征，苏州，合肥，宣城，黄山，杭州，温州，上海，青岛，长沙，深圳，东莞，惠州，广州，珠海，佛山。从长三角到大湾区，从胶州湾到伶仃洋。心随神至，心旷神怡。

我喜欢在路上的感觉，尤其是一个人在路上的时候。当肉体离开熟悉的人群，灵魂脱离尘世的喧嚣，地球online就成为了真正的开放世界探索游戏。客船划过新安江，是春天千岛湖的灿烂。若隐若现的黄蓝分界线，是洞头外海与东海的边际。如愿来到苏州博物馆，在金鸡湖畔的诚品书店收获了如愿以偿的好消息。从世博园漫步到外滩，看黄浦江上夕阳渐暮。世上从不缺少诗和童话，只是生活遮住了我们的眼睛。

我吹过青岛的海风，与黄山清晨的雾海相拥。我漫步珠海情侣路，与惠州西湖的暮色相逢。享受长沙和顺德的美食，走近南浔与塘栖的古镇。徒步三十里解锁南漪湖的风景，横穿马峦山打开背仔角的日落。无论生活多么艰难，在这十方世界之中，总能发现，那独属于灵魂的美好。

# 2021年终总结·烟花易冷

2021年里对我来说最大的一件事，大约就是走出校园和新的集体。走出校园，意味着离开旧的社交环境，无论有意或无意，主动或被动，总归要和一些人疏远了距离。而新的集体，也意味着我要认识一些新的人。

然而，新的集体并没有给我带来什么新的朋友，新的朋友依然来自于牛头冲。疲惫的打工人之间只有虚伪的职场社交。正因为他足够虚伪，所以甚至不能评价关系的好坏，只知道在非工作时段里，我不想和他们发生任何一点交集。

在深圳的旧有的朋友，大抵也随着时间的流逝而多了一层厚障壁了。仰慕多年的小A同学走进了文科的最高学府，专业领先的L同学也同样步入了南国C9的殿堂。数学建模好心收留我的H同学辗转来到深圳，成为了我的同事。因缘际会认识的J学弟完成了自己的梦想，也在现实的鞭打中认清了生活。或许我们还能一起聊天一起吃饭，但距离终究愈来愈远。人与人的关联，本就如此脆弱，就像我做的前端展示页，无论展开了多少级密密麻麻的连接，在维度的放缩和筛选后，都剩不下什么。

毕业前差不多是最后的晚餐，我邀请了两个对我来说十分重要的人。尽管我对待人与人的关系一向比较随意，也还是有我所需要珍惜的东西的。我终究需要这样一个人，能在凌晨听我絮絮叨叨四五个小时，能在我需要能量时给予我需要的能量。我会去小心避免越界的感情，释放溢出的内存和堆栈。对我而言，最好的期许，也就是维持现状而已。还有一些人，尽管不舍，但也终究是疏远了。不必纠结是因为距离还是误会，结局如此， 便是如此。

# 2021年终总结·一晌清欢

406的欢笑渐渐离我远去了，取而代之的是工位的忙碌、扯皮和高压。工作，终究是有许多不快的。但，这就是现实。

校园离我远去，但又没有完全远去。工位的日子和在实验室的日子本身差别并没有很大，无非是为任务做事还是为兴趣做事罢了。但任务也未必尽然没有兴趣，兴趣背后也不尽然没有任务的影子。单纯的工作和学习说不上谁更辛苦。以长期视角而论，一个是为现在的自己，一个是为未来的自己。

但我对自己未来的预期一向是悲观的。无论是遥遥无期的升职加薪，还是各种毫无头绪的工作。尽管现有的任务还算顺利，但是名下还有一大堆连是什么都不知道的工作。今年的表现还算良好，但是连方向是什么都不知道。

中秋节那天，我骑车去给两个朋友送月饼，回程的路上摔伤了膝盖。一瘸一拐的回家的路上，我认识了一个很有意思的人。深圳的人潮人海与荒僻的宣村，反差的很奇妙。因为荒僻而拒绝，并不是绝对明智的选择。但认识久了，也会发现，这是一个挺有意思的人。

# 时光循环

时光，在怠惰里流失与浪费。人情，在忙碌中消散和疏远。当时间的指针走过了2021的尽头，反思或迷茫，沉沦与沮丧。没有用的。失去了勇气的我，弱的不堪一击。

2022的开头，伴随的似乎是好运。普池单抽，剑光如我，牛杂斩尽。但回顾过去，我身上总是伴随着一次又一次的周期律。大约26个月一次，4个月平淡，6个月衰退，4个月困顿，4个月绝望，4个月重生，4个月顺遂。我接受定律的安排，也将信心寄托在定律的效用下。无论遇到什么困难，随着时间的推移，周期总会进行到有利于我的一边。拖延，也是一种合理的手段。

从19年11月开始的上一个周期到现在为止大抵是结束了，如果定律依然如此，我会度过一个相对平淡的上半年，然后年底会过的挺艰难。不一定像前两个周期一样困于感情，也可能是其他的原因。不过我怀疑依然可能是感情，毕竟人总是会在同一条河里来回翻船。。。
