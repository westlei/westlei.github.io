# 心无所向

# 迟暮

本以为，最为迷茫的时节已经过去。放下曾经怀有过的幻想，就得以翻开新的篇章。但现实却是，在无尽的时光里奔忙。没有向往，更没有了前方。盘旋在山道上，心无所伤。敬亭宛溪，夏渡向阳。找不回昔日骑行的欢畅，抛弃了曾经放纵的疆场。感谢在这个月里关心我的所有人，让我在最软弱的时刻里选择坚强。纵浑身是伤，仍不想绝望。

天色已然迟暮，太阳收敛了光芒。深邃幽蓝的空中，繁星特别明亮。还有那么一片温暖的，是白月光。流连在群星璀璨里，却再也找不到心之所向。也不知，那迷惑了视野的，是那月的清朗，还是那星的微茫。抑或着，还在回想，那日未落的时光。

只好选择，在天台孤独地守望。念念不忘的，也许终究会有回响吧。

# 重来

生活，就此走向新的起点。故事，没有过去也没有将来。无疾而终的念，如奔流的滔滔江水，一去，便再不复回。再莫谈及那些过往，再莫回想曾经故事。所有人都是好人，除我以外。我不是坏人，但更不是好人。

人生，是一个人的旅途。一场经过，一场因缘聚散。看荣枯流转花谢花开，哭哭笑笑，纷纷扰扰。命运所赐与我们的，有责任，有义务，还有无数的考验。每一分执念，都是一种束缚。

但，随着长大，人会慢慢变得平和。随缘来去，一切皆坦然自若，内心清凉。但不曾丧失的热情，不曾放弃的梦想，这种执著，可以让生命更加多彩。命运永远真实，但你可以拥有，独属于你的不二安详。

# 交错

每一个季节交错的时光里，总有那么一个荷尔蒙躁动的时日。因为，现在的人所缺乏的，不仅仅是头发，还有爱情。

二月十四日，冬的尾声里点燃了春天的火种，情人节便在这时降临了。沉寂了一冬的春心开始复燃，想着，有个对象，该多好。

五月二十日，春风里已经传来了夏的气息。这个谐音我爱您的日子，让多少荡漾着春心的单身狗无语凝噎。逐渐炎热的气候让人急躁，但缘分未到，注定仍是孤身。

到了七夕的时节，火热的夏里已微微透出秋的凉意。有对象的秀着属于自己的恩爱，没有对象的看着有对象的秀着恩爱。日渐淡然的心底，却也有着那么几分落寞与惆怅。

而到了双十一，瑟瑟的寒风和冷雨给秋的一地金黄披上冬的寒霜。心底里已经无可奈何的接受了单身的事实，自嘲之于只好投身阿里的卖场。心若死灰，待明年复燃。

# 清明梦

整个春天不过是一场幻梦，一场始于2018终于2018的幻梦。泡沫般绚烂的一切，背后亦如泡沫般脆弱。过去不曾是朋友，未来也不曾是。不知是空中楼阁，还是海市蜃楼。

泡沫在夏日里绚烂，背后那喜欢的一切却在悄悄发酵。相信泡沫总有破裂的一瞬，却期待泡沫破裂时那最绚烂的花火。忙碌着，闲暇着，充实着，空虚着。却见，无数的小幸运，把生活点缀。

秋日的暖阳下，泡沫炸开了缤纷的花。幻梦如愿以偿地走到了结尾，生活归于了另一条轨道。已经不清楚自己喜欢什么，也许直到你脱单了，才会意识到对你动心。

双十一过去了，冬的序曲已然奏响。默然，相守，寂静，喜欢。而你，却是在哪里？

# 深渊

当你凝视着深渊时，深渊也在凝视着你。

从不曾向现实低头的人，从深渊之上走过，无所觉察，亦不曾沦落。屈服于命运的人，俯视深渊，被无尽的深邃吸引，陷入其中，便无法自拔。

深渊永无止境。当你向命运低头的那一刻，就是陷入深渊的开始。待到愈陷愈深时，即使恍然醒悟，却已不得脱身，只能沉沦。

也有的人在短暂的沉沦后重又振作起来，无视那深渊的凝视，抬头远望，从深渊中爬起。但更多的人，却无法抗拒深渊的凝视，而坠入了深渊。

# 雪月

雪，落在了宣州大地。纷纷扬扬的精灵，飞舞着，盘旋着。白茫茫的大地与呼啸的寒风，大雪绘就出梦幻的冬。纵几多相似，纵几重相思。尽化作那该当忘却的故事里，永远不再的记忆。

没有散场的灯光，可大幕已悄悄拉上。没有播完的结局，也无人能够想象。只有时光，仍在流淌。

只如今，茫茫大雪之中等着谁回顾，明知无人回顾。谁能初心不负。

# 岁末

2018，即将走到尾声。从开始到结束，一事无成，一无所获。只剩了匆匆逝去的日子，还有那匆匆逝去的年华。

还忆得彩金湖畔的碧水青天。一池碧水，守望着宁静的天地。一山之隔，隔绝了牛头冲的熙熙攘攘，断开了尘世的一缕喧嚣。默然，相守，寂静，喜欢。目视湖心那片轻浮的落叶，也许有所喜欢，便是心灵最好的慰藉。

还忆得摩天轮下的碧绿草地。柔和的阳光撒在脸上，微风轻抚，忘却世间多少烦恼，留下一个甜蜜而美好的梦。既无所念，便忘却那敬亭宛陵，心所往，唯新港彩金。

还记得年初，那场纷纷扬扬的大雪。洁白的天地里，金陵褪去了浮华，只剩一份优雅。故事已经结束，却仿佛未曾开始。似乎淡却的伤感之中，又添了几分不一样的东西。帝都风华，洗去已久的尘埃，回归豁然与平和。

待到一年将了时，一场大雪不期而遇。纷纷扬扬的白，落在宣州大地，洗落一片心空。凌晨的杭州，寒风微雪，卷过沉睡的西湖。雪落的上海，美如仙境，还上海滩一片安详。甪直古镇，小桥流水，尘世的喧嚣不再。姑苏，又回到了记忆中那个美丽的城。曾因人而喜欢，也曾因人而厌倦，终于在大雪飘飞的时节，回归了从前。

爱那风，也爱那云。爱那雪，也爱那月。观风云变幻，看雪月交辉。不再望那一山一湖，也不再记得湖中曾有过，那飘飞心上的一片树叶。

梦里梦到醒不来的梦，红线里被软禁的红。所有刺激剩下疲乏的痛，再无动于衷。

岁月的尽头处，一切不复当初。光阴的流淌里，长梦不曾结束。

2018的最后，匆匆回顾，想一切如故，不曾如故。
