# 不适合一般人的旅途

回到南京最大的意义，大概是我过年的时候在家里的身份从编外人员降低为了客人。除夕夜回去吃个年夜饭，然后就回到我租的小屋待着自个儿跨年，年初一再去嗑嗑瓜子聊聊天补补春晚，就算是过年的环节结束了。红包什么的也没有了，最多就是我妈初一早上慰问完下属以后剩了一个一百的红包给我。

过年的两天里最大的成果是，初步跟家里达成了关于资产的一致意见。在2025年第一个合同结束以前，家里再努力一下看看能不能把资产提升回8位数的量级(不包括房车，当前大概是0.6k净资产+0.2k的负债)，如果失败的话就把负债全部清空。与此同时，我不进行任何大宗消费，包括但不限于房、车等，然后在此期间对于谈不谈对象之类的问题，他们不再过问，之后再说。

24年的开局跟23年相比最大的不同在于，我对于我、我的工作和我的未来都没啥信心，所以今年的主要态度应该是摆烂为主。然后由于我比较主要的一块社交圈已经毕业并且开始了读研或者打工的帕鲁生活，所以我的社交范围已经大幅减少。健康问题日益恶化，社恐问题日益加剧，这都不是什么好的现象，但都是今年应该想办法去解决的问题。

当前的躯体状态有些缺乏活力，这对于安排行程来说是一个很大的问题。在2023年的时候，已经出现了很多次躯体疲惫值过高阻碍行程的现象。从我的个人角度来看，只有躺平看管人的状态是轻松的，剩下的哪怕是抓帕鲁干活对我来说都有些劳累了，而像肝原神这种非常高强度的体力活更是显得过于艰巨。剩下诸如积极社交和谈恋爱什么的，简直超出我的能力范围了好吗？

于是年初二我就出来玩了，从杭州，到珠海，然后澳门、香港、深圳。整体行程很赶，但是从另一个方面来说，其实一开始根本没有定要去些啥地方，所以非常咸鱼，走到哪算哪。相当多的地方其实都是想起来就去一下，也可能仅仅因为有些累了或者来不及了就不去了。这也是我现在不太喜欢跟别人一起出去的原因，一方面人多了就不够灵活了，另一方面既能跟上我的体力也能跟上我的脑洞的人确实没咋见过，所以跟别人一起出行对于我来说不是什么特别愉快的体验。正常人是不会早上斗门下午hk的，也不会去macau先去黑沙沙滩的，但我会。

# 当一年多出了一天

恍惚间突然就忆起了上一个2月29日。 上一个闰日还是特殊时期，但是大的灾变已经趋于平和，人们已经从惶恐之中恢复平常心。海外的风险开始发酵，大解体的齿轮在冥冥之中开始了转动。大洋彼岸疫情开始泛滥，川建国无奈下野，经济落入圈套，在疯狂上涨了一年之后开始爆雷，落得一片死气沉沉。

闹的欢腾的香港人在疫情的治理下停止了运动，然后被重拳出击，将香港化作东亚金融中心遗址公园。新加坡吃掉了大部分的财富，而随着横琴的封关，剩下的一部分似乎也要被横琴夺走。两边都不再认可香港作为中立方不站队/站队不彻底的行为，选择由更确定的力量来主导未来的经济发展。

曾经想要获得这个位置的角色可能是上海，但是22年灾害之后的上海元气大伤，无力去承担这样一个角色。各个产业几近永久的撤离，对城市乃至区域而言都是无法逆转，无可挽回的。一家公司里哪怕我这种小人物一年都要上缴两万多的税款，这对维系财政而言是重中之重。也正因此，政府可以免费给公司地皮用于盖基地，条件就是纳税达标。

距离22年底第一次探访练秋湖基地已经过去了十四个月。昔日古朴破落的小村西边已经建起了安置房社区，而东边则是气势磅礴的水乡基地。但是原本村庄的两条街道已被封锁，街上的店铺更是尽数关门。本就人烟寥寥的小村更加显得人迹罕至，只剩下大卡车碾过的尘土飞扬。

排除各种各样附加于工作与生活之上的不快，我还是很喜欢这个地方的。苏州于我而言有一种特殊的吸引力，我喜欢这里的山水，这里的人情。而水乡客厅则是水乡之集大成，曾有我一切的期许与幻想。从基地向南跨越太浦河到嘉善的丁栅，不远便是西塘，再溯西塘大道-苏同黎公路而上，便至黎里、同里。我曾经想，倘若余生一直生活在汾湖，找一个合宜的对象，结婚生子，平平淡淡的过日子，也不失为一种幸福。

苏州的山也清楚，水也温柔，没有上海的精英感，倒有些烟火与市井气。前几天刷到一个小up主，做吴语翻唱的，看评论区应该是江阴人。翻唱的《琴师》特别好听，循环播放几百遍了。忽然发现一个挺离谱的问题，通宵充电+单曲循环b站视频(哪怕用仅声音)发热都特别厉害，甚至有时候会因为过热保护中止充电，但是晚上放网易云什么的不会这样。

# 城际飞行与消费主义

打工以后，时间似乎变得比之前快了很多。哪怕如今失去一切上进心的我过着闲散的975生活，四季与岁月的变换也不过须臾之间流逝。生活无所谓锚点，无非夏天半枝莲冬天寒江雪。

但江南的春天还是应时而来了，伴着顾村公园盛放似雪的樱花，伴着嘉兴路应许之地特别公演的落幕。小偶像们为了梦想出走半生，最后带着伤痛和一无所有离去，或转生皮套，或缘尽世间，也不失为一种幽默。

自爱国路而下定海路，就能抵达曾经的国棉十七厂，如今的上海国际时尚中心。各色各样的国际服装品牌半死不活的开着，甚至不如门口的烤肠摊生意好。保安戴着N95口罩，四目无神的看着门，完全看不出昔日此地保卫科干事的勇猛之气。老杨浦的工业遗产半死不活的树立江边，在新的世纪里再也难以发挥他们曾经的意义。

从龙阳路坐上磁悬浮去往浦东机场，以300公里的时速飞越张江，体验中国最短十条航线之一的沪宁航班。十五分钟起飞，十五分钟平飞，十五分钟降落，300公里不到的距离也得飞45分钟。当然如果只是45分钟那还好，起飞之前一个小时就得在飞机上坐着了，这才是煎熬。高铁对三五百公里以内的短途是碾压性的优势，飞机是无可比拟的。

今天有人问我存款有没有七位数。

我打开个人所得税app，把21-24年的收入合计全部加一起，也就844724.7元。然后呢，税应该交了五万上下，五险一金交了差不多快八万。也就是说，过去1000天，我实际到手的工资应该是70万左右。

开支方面最大的花销应该是房租，数额应该跟五险一金差不多也是八万上下。基本生活开支算五万，那也就十三万。也就是说，如果维持一般水平的基本生活，我应当可以存57万。

但是实际上存了多少呢？我的银行卡里只有十万。加上某理财产品定存的20万，加起来也就30万。再把外债和大额捐赠一共十万也算上，距离57万也还是差了17万。17万去哪了呢？那只能是陷入消费主义陷阱了。17万换算成648的话大概是262个，也就是差不多212万原石，折合13000多抽，130个人物或武器，按满命满精算的话也可以有18个人物或者26把武器。

这样看好像也还是挺多的，那么换一种消费主义算法。17万平摊每一周的话大概是1200块，大概是一个周末去1000公里以内的城市旅游的花销。哪怕每个周末都去一个1000公里以内的城市，那么两年多也就花17万。给b站主播上32个月的舰长是4400多，提督是6.4万，17万可以给差不多40个主播开32个月的舰长，或者给3个主播开32个月的提督。

如果不看这些我日常的消费主义开支，转而去欣赏真正的消费主义，那么17万就显得非常渺小了。江西彩礼的平均值在10-20万之间，17万也就将将够付一次彩礼。小米SU7的报价预计在20万以上，17万的话买不起一台小米SU7。上个月南京郊区失火的那个小区是比较便宜的地段，近期成交价大概1.4-1.5万一平，17万可以买12平左右，也就一个十分逼仄的小房间。

所以，消费主义又怎么样呢？结婚、买车、买房随便那一样都薄纱大部分日常的开支了。至于这十七万到底去哪了，似乎也不是很重要了。

# 夏天半枝莲冬天寒江雪

曾经我一度以为，在那个自在的夏天之后，我应该不会再回到这里。但是又一个春天，在老东西们的邀请下，我还是回到了这里。

地方依然是熟悉的地方，但熟悉的人越来越少了。在东大门办完手续进入学校，双腿自觉的踏上去综合实验楼一的道路。可惜406已经人去楼空，甚至连门牌都不曾剩下，而隔壁的405也就只剩下实验室的牌子幸存。计算中心楼302A成为了电设的新家，更大的工区为实验室吸引了更多的卷王。在竞赛加分大萧条的背景之下，坐拥AB类比赛若干的电设招新伊始人数达到了400人的新高。但环顾四周，除了圣遗物之外，只剩一个还算熟悉的人。

wdr几经流浪，在综一的2F有了新的处所，但面积已经大大的缩小了，一堵柏林墙更是把本就不大的空间压缩的更为彻底。与之相对应的，则是岌岌可危的财政状况以及各系之间越来越低的合作力度。新一批电设精英不再为电控提供支持，无人机实验室也不再提供力所能及的帮助。rm渐渐回到了他一开始的样子，变回了机械系自己的游戏。就像物联网撤走之后，电设也回到了电信科绝对主力的日子一般。

学校西边抵达彩金湖的通路，时隔多年终于全部完工。还记得第一次去彩金湖时从文景路西的断头处翻越土山，再后来从绕城路涵洞穿过关庙社区农庄，创业路修建后取道创业路，最后文景路西延水泥路直通湖畔大堤。但无论周边如何发展，湖一直在那里，作为一片去城市化的净土，与湖畔的老木屋一起，氤氲着沦落的美。不远处的高档小区已经交付，120平仅需八十万，可依然滞销严重。就像沪国人均二百起步的高消费门庭若市，皖南人均五十不到的平价开销却少人问津。金钱不能说明一切，但需求可以。

网友见面让社恐人士感觉有些慌张，但相比于被迫啃一大口狗粮的尴尬，与网友见面似乎算不上啥了。其实还是聊的还挺开心的，一些不必要的担忧终归是多虑了。

七年前初到宣村，填的第一首词就是cover当年最火的消愁，可惜这么多年过去，当年的填词却已不知道消失在了哪里。

# 花语明前

24年一季度的最后一天，南京的天气有点阴沉，清早起来去殡仪馆给我逝去四十多年的亲爷爷扫个墓。虽说是我亲爷爷，但是鉴于他逝去的年份过于久远，以致于我不仅是素未谋面，就连他大概是个什么样的人都一无所知。

由于这些年我一直都不怎么在南京，所以上一次来看望可能还是四年前的2020年春天，亦或是2017，再或者更久以前。严格来说，他的灵位这四十多年来都只是以寄存的方式居于殡仪馆的某一个格间之中，等待着我奶奶过世之后合葬或者撒入江海，但是我奶奶今年八十六岁了，依然健在。

四年前的冬天，他多了一个邻居，是他的战友，我的舅爷爷。他身体比我奶奶硬朗很多，而且作为离休干部有几乎无限的医疗资源。但也没用，一场小病没扛过去，在病床上度过了痛苦的一年，最终还是走了，世事无常大抵如此。

祭扫时在格间中翻出了我小学还有高中时写的祈福卡，当时完全不能理解这些，也不知道写些啥，就爸妈说啥我写啥。这么多年过去了，我大概有了些可以自主选择愿望的权力，但是我却也不知道能寻求些怎样的保佑。升职加薪似乎与我无关，找对象则与我更无关。这些是爸妈的愿望，但不会是我的愿望。

高淳桠溪的国际慢城，油菜花开的正是时候。上一次油菜花时节来桠溪慢城还是九年前的2015年，大概是一个周末，然后还在桠溪淋了一场很大很大的雨。好像我也很少会专门看油菜花，毕竟金黄色的原野总让我想起疫情前春天的皖南，学校北门对面金黄而灿烂的油菜花田。

S9以80的速度慢慢开过石臼湖天空之镜，但是阴沉的天空并不能照应出湖水的蓝。一晃高淳通地铁都7年了，而城区的老白下至今依然没有地铁降临。开工的第八个年头，南京地铁5号线终于以一种残废的姿态加入了地铁大家庭。全程9个站12.1km，运行时间18分钟，末班车21点就结束，鸡肋程度和目前分成三段的7号线不相上下。7号线的南/北段也只开了9/10个站，里程10.2/13.5km，运行时间19/21分钟，但是末班车起码到了23点。

怎么说呢，估计未来还是会有离开南京的一天吧。不知道会是什么时间，也不知道会去哪里，也不知道会是因为什么。希望是因为有了一个真正想要追随的伴侣吧。
