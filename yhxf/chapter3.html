# 行在鹏城

# 海的那边

生活，不仅仅是生，还应该有活。生存只是生命里微不足道的一个小部分，似乎更有必要的，是去寻找一些鲜活的东西。因此，尽管是一个人的七夕，但这也不应该是不去寻觅浪漫的理由。

深圳不出意外的下着雨，来到蛇口的我，突然就有了去海的对面看一看的念头。短短一个小时之后，我就出现在了珠海的情侣路上。情侣路是珠海东边的滨海大道，其中最精华的一段便是情侣中路段——从九洲港到海天公园的7.8公里。骑行其间，看情侣如织，天高海阔，或许也不失为一种体验。

短暂穿行了珠海市区，去珠江江畔的广州塔看一眼夜景。灯火辉煌的珠江两岸，高楼鳞次栉比。游船穿梭其间，仿佛有回到了外滩的感觉。广州塔上的灯光打出了七夕的字样，似乎有那么一点感觉了。骑行过珠江，穿越天河的大街小巷。无数外卖小哥载着满车的鲜花去向各处，它们的归宿又在何方？

晚上十一点半，回到坂田。吉华路上的人流依然熙熙攘攘，几步就有一个卖花的摊位。七夕节最后的半小时，九块九一束的鲜花已然无人问津。当节日过去，太阳重又升起。还有几人会去铭记，那已经成为过去的七夕？

如果昨天的我喜欢你，那么今天的我也会。

# 五和路口

一个月其实挺快的。

大湾区的日子一晃而过，东莞、深圳、珠海、广州。说近不近，说远不远。仿佛神游物外，又仿佛出离樊笼的畅快。仿佛每个工作日的繁忙并不存在，work & life，该工作就工作，该生活就生活。马马虎虎的过着每一天，寻找自己的方向，不过找不到也无妨。

人是环境的产物。十年的阴影，让我已经开始畏惧光芒。尽管经历了四年还算不错的时光，但心底里对光明的惊惧已经成为了习惯。我习惯于一个人的自在，却也欣然于集体的约束。我有着极强的自我，但又好像早就失去了自我。

我写下的文字，从来都和我的人一样，夹叙夹议，形散实散。你很难说它到底是一种文体，还是纯粹的意识流呻吟。现实解构了一切，失去了土地的新一代农民工，在格子间的工位辛勤劳作，铸键为犁，化鼠为锄。而不远处的山野农田，早已无人耕作。

这里是坂田，坂中之田，四面环山。塘朗山-银湖山隔绝了与城市的联系，而西北面的羊台山和东北方的不知名山脉则更进一步加剧了这种撕裂。若是进城，东则单走布吉，西则需出西丽。写字楼混杂在工厂里，繁华隐没在荒芜里。不尽是城市，不尽如乡村。

大湾区plmm真的很多，可我莫名的没有了多少兴趣。我一直以来都期待着精神的认同和情感的交互，但很遗憾，在慕强的社会里，不需要我这么一个起码是心理上的废物。我觉得我更需要的是安慰和陪伴的知心姐姐，而不是一个崇拜和依赖于我的小迷妹。但我又不喜欢御姐风成熟风的那种气质，唔，所以其实很无解。

还有26天就满21岁了。届时的我，大约也是一个人度过吧。尽管在深圳也算是有那么几个认识的人，但过生日的话，还是不必了。虽然很想有一个认识很久的朋友陪我，但是，还是算了吧。距离看起来只是区区十二公里，骑行一个小时而已。但中间差的，是远不止十二个月的距离。

# 社畜生涯

已经进厂差不多一个月了。。。

早上八点骑上自行车匆匆忙忙出地下车库，然后看公交站台上的人行色匆匆，厂里的班车停在旁边，但我不能坐，因为那是去东莞的。

公交车的玻璃上映出一张张人脸，拥挤的车上没有给他们多少喘息的空间，一张张工牌表明着他们的厂狗身份。无数毕业于985、QS50的硕士博士们，每天匆匆忙忙的上工，匆匆忙忙的下工，一天工作十二个小时，却只能拿一点微薄的薪水。这就是厂狗。

办公楼下的7-11是解决早饭的好去处，就是略有点贵。厂狗们买了早饭就去工位上开始一天的打工生活。到处都是机器的声音，中间夹杂着各种各样的电话铃声以及随之而来的welcome to join the conference。透过窗帘看窗户外面的蓝天白云，天气真好啊，就是有点好。

中午拿出折叠床，戴上耳机，睡上一个小时。总觉得睡得不好，下午还是很困，但是得坐起来，然后在机器面前接着昏昏欲睡。机器一直在工作，而我没有在工作，但睡又睡的不香。

日子就一天天这么混着。晚上八点半拿完夜宵下工，回到租住的二十六楼一百二十平的房子里，打开二十平的小屋，躺着刷刷手机，啥也不想。明天，好像又是新的一天了啊。

但明天，跟今天，似乎没有多少不同。


深夜里，总是做梦，总是多想。

过去四年时间里认识了好多好多的人吧，尽管大多数人的接触都不多，为数不多的接触经历算不上多好。很多我心目中很厉害的人，最后都做不成朋友，即使做了朋友也做不了多久，还是有些遗憾的。不过被厉害的人嫌弃对我来说也算是常事了，谁会去关心一个事实意义上的社会废物呢？

我经常水群，但大多数时候，水群是因为完全找不到想要跟我说话的朋友了。水群的时候太想要获得关注，却总是引起更多人的嫌弃。大概这也就是一个恶性循环吧，迫使我一次又一次改变自己的环境，但怎么也逃不出这样一个过程。

我很羡慕大佬们会那么多东西，有那么多关注。我在努力尝试着去做一个大佬，学各种各样的东西，帮各种各样的人。我以为这样就会有很多朋友，但是结果只能去饰演一个工具，甚至都算不上工具人。当我试图向我认为的朋友倾泻情绪时，我发现并没有一个合适的人，并没有什么人愿意听这些东西。我只能日复一日的码一些没有人看的句子，然后再默默删掉。我已经不期待有人能把我救出来了，大概就是彻底的绝望吧。

上上个周末的晚上，我一个人在蛇口看海。暮色下的海湾很好看，我想找一个人分享这种感觉。但翻了整个列表，一个可能的人都没有。怀着没多少的时候期望去发了一个人，果然一直也没有理我。在那一刻我突然有些破防。或许我就是这么一个失败的人了吧。

说起来，我这一路走来也还算幸运的。不管是多艰难的时候，都有挺多人在支持我，安慰我，鼓励我。他们会信任我，给我机会；会相信我，找我组队。但这么久过去了，我还是很菜，感觉有些愧对他们。

微博收到一条私信，说的是学习和工作哪个更累。我觉得吧，压力什么的都是自找的。你想轻松，就可以轻松，没有人能拦着你。学习压力大吗？但是如果你不想考研保研，甚至可以无视挂科延毕，就想着混完日子继承家业，这样压力其实也不大。工作压力大吗？但如果你不想升职加薪就混混日子，也没那么难。

说起来我大学过的还挺轻松的，比赛啥的就是玩，成绩混个及格而已，考研都没考上。要是说累，可能就期末考前几天累，考研前两三个月累，被感情伤害到心累。但工作其实更轻松啊，任务慢慢的做，只要效果做的还可以，就不用很累。作为部门学历最低技术最差的垃圾，我也没对自己有啥要求，不过好在年轻，应该没那么容易被淘汰，所以就混着吧。

去年，听说实验室的学长考研失败去了小米，不加班，过的很悠闲自在。然而今年，学长辞职去考研了。真的有那么多人工作之后又回去考研嘛？我不懂，也不想懂。

那就。。。继续混日子？只要不考虑感情问题，这日子过的还是挺舒服的啊。

我喜欢过很多人，但她们都逐渐消失在了我的世界里。有几个我已经完全想不起她们的模样了，但还有几个我依然喜欢着。

但是喜欢是这个世界上最没用的东西。

# 八月长安

八月快结束了。我的二十岁也快结束了。

我曾经以为，INFP的人生注定是一场悲剧。但现在发现，或许还是有不悲剧的结果，但结果注定跟我没啥关系。INFP进阶是很难的，大约需要无敌的天赋，优良的条件还有足够强烈的支点。缺乏哪一条都很难成功。换而言之，我应该还是注定悲剧的。

以普遍理性而论，我现在在一家还算不错的大厂里，付出不算过多的工作时长和工作精力，就能拿一份个人认为多于我应得的收入。目前做的事情也是我感兴趣和比较拿手的，入职第二周就凭借改进的demo就得到了跟大领导汇报的机会，第一个月结束的时候做的东西就得到了大领导的部分认可，并开始接手更加复杂和艰巨的工作。

得益于双休，我有闲心在城市里骑行，去逛热闹的老街，去看湛蓝的海滨。一个月的时间，我探索完了100%的深圳，在银湖山顶看繁华的城市风景，在大亚湾边看古朴的海景风光。一个月的时间，我走过了大湾区的三面，感受了伶仃洋的波涛，品味了情侣路的午后，徜徉于珠江边的夜色，流连于广州塔的璀璨。旅行者逐风而行，而大海天生浪漫。

INFP看起来是正常人，但又不完全是正常人。似乎融入在社会里，却又独立于世间。逐风的行者，追寻浪漫，却也希冀于归宿。但这个世间，并没有什么人愿意去倾听。

我是喜欢你的罢。

但你是不会喜欢我的。

那么。就酱。

# 不见长安

突然就9月了，列表里好多人要开学了。上了16年学，以为自己适应学生生活。但出来才发现，适应只是因为没有选择。我这种人可能天性就不是很喜欢念书，但是我很随遇而安，所以其实也无所谓。

画界面、做模型、写体验、听培训、肝考试，似乎挺忙的，但其实好无聊。INFP最大的悲哀，是容易陷落在自以为是的悲哀里无法自拔，把每一个生命里的过客都看的太重，却把自己看的太轻。仿佛不是为自己而活，却又并没有为谁而活。

喜欢大约是一种感觉吧。感动是构成喜欢的一种要素，或许对我而言，如果你做了什么让我感动的事的话，甚至只是某一瞬间让我有感动的感觉，我都可能会很容易喜欢上你。但这并不意味着感动和喜欢就对等。

INFP似乎很容易喜欢上一个人，但是这并不意味着就会怎么样。对于INFP来说，喜欢仅仅意味着心里有了一个人的影子。INFP太多的时间都用在了精神内耗里，经常会陷入荒诞的逻辑怪圈之中。比如我喜欢一个人，那么这个人必然足够优秀，优秀到我配不上。那么她大约又是不可能看上我的，所有尝试注定要白费的。那为啥不一开始就放弃呢?

于是我就这么一直单着下去了。

# 一心净土

稻妻主线走完了，第一次想说说剧情。不涉及太多具体内容，就说说感受。说实话，很无厘头，而且有一点压抑。更难受的是，我发现，影身上有很多我的影子，像极了我最讨厌的样子。

因失去而执着于永恒，像极了执着于被爱的我。但我真的能被爱吗？把真正的自己封闭在一心净土之中，让虚假的人格在外展现。最终就是越来越分裂，越来越仓皇。

我想不通我是为了什么在奋斗，我没有需要呵护的人，也没有需要负责的人。社交链里的任何一环，缺了我都不会有什么变化。不会有旅行者走进我的一心净土，更不会有八重这样的老朋友。遗弃了这个世界，这个世界也遗弃了你。

虽然刚刚充了钱，但是稻妻图实在不怎么能玩的下去的感觉。虽然现在并非没时间玩游戏，但稻妻的环境布置实在是很难让我有好心情。也就珊瑚宫的布景稍微好看一些，别的地方跑图的时候总觉得不那么阳间。

心海的剧情完全莫名其妙，尽管漂亮但没有一点想互动的感觉。距离感太重了，就像我现在组里那唯一的女生，太过于生分。唔，单从剧情而言，除了神里，我还是更喜欢八重神子吧。正经的时候正经，有趣的时候有趣，漂亮而聪明，会去布一个局，去设法帮朋友开解心扉。

一二三，三二一，啊啊。唔，醒了，今天周末，该去哪里呢？

# 碎碎念碎碎念

上回书说到，老板在给我画人上人生活的大饼。并明确的向我指出了实现人上人生活的途径：做超过你能力的事，你可以升级加薪。不断实现能力范围之上的事，你就能成为年薪百万大专家。我厂给老员工的待遇是远超市场平均水平的，只要能一直待下来什么都会有的。。。

但作为一个INFP，搞钱和钻技术都算不上我的理想吧。很多东西我承认我是喜欢的，但我也不至于天天去做。喜欢旅行，但天天旅行也会累死，偶尔躺着也不错。喜欢看书，但很多小说套路看多了也很腻，没啥新意。统计模型、地图规划这东西我是有兴趣的，但要是看多了我也会很困。尽管兴趣不少，但是天赋一点没有，不管做啥也都是三分钟热度罢了。

为什么会经常提INFP呢，其实INFP跟什么处女座、00后之类的一样，就是一个标签罢了。把自己隐藏在标签后面，就好像我的问题不是我的问题，只是这个标签，起码是标签中一部分人的共性问题。既然是共性问题，那就不是我的责任，亦或者说不是我能解决的问题，我可以心安理得的躺平这样子。

当然嘴上说着躺平，事实上怕是躺不了。虽然每天真实工作时间估计就一小时多，但丝毫不影响这周得例行加班。加班挺好的，毕竟快到发薪日了，没剩多少钱，也没法出去玩了。在公司带薪摸鱼，虽然不是特别舒服，但其实也还不错，起码会有一种生活很充实的错觉。吃饭睡觉，玩玩手机，一天就过去了一半，剩下就是应付差遣和开各种听君一席话如听一席话的会。

我不怎么相信谁，更不怎么相信我自己。因为啥都会一点，所以拿到的活很多，但是工作量都不是很大，每天的日子主要就是迷茫的玩手机。躺平的生涯很快乐，但是不知道该怎么躺才能躺的比较平滑。

再然后可能就是舍得与舍不得吧。

# 你好，国法

21岁了。新的城市，新的环境，新的身份，新的开始。

校园离我远去，但又没有完全远去。在工位的日子和在实验室的日子差别并没有很大，无非是为任务做事还是为兴趣做事罢了。但任务也未必尽然没有兴趣，兴趣背后也不尽然没有任务的影子。单纯的工作和学习说不上谁更辛苦。以长期视角而论，一个是为现在的自己，一个是为未来的自己。

工作同样还是需要学习的，学习让自己变得更强的技术和方法，学习无数前辈们的经验和智慧。工作中的学习，需要的是能让自己快速从繁重或棘手的任务里脱身的方法。可能不需要太多的智慧和深入探索，更多的是一些技巧性的东西。或许是经验+思考得出的创新，或许是经验+技能得到的策略。

但愿在这条路上我能走的远一些吧。
