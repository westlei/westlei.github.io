# 叶落归心

# 江南水乡

曾经，有过那一个属于江南的梦。然而，这突如其来的四月之旅，却让那美好的梦境，化为一个个五彩的肥皂泡。破碎了，便一去不复返了。

那美好梦境的主角，便是，那姑苏城，那我曾认为，最能代表江南的城。然而，来自上海滩的喧嚷，已盖过了曾经美好的一切。形犹如此，而神已不再。正如那长大成人的女孩，虽然漂亮的外表依然如故，但不复存在的，是曾经眼神中那干净与清纯。

长江与太湖的两水相依，让苏州，有着似水的温柔。吴侬软语，昆曲婉转。这是曾经梦里天堂般的江南水乡，这是那理应只由姑苏独享的荣光。但如今，婉转与温柔犹在，可那只是矫揉造作的饰品。姑苏，已不复天堂。

人世的苦痛不外乎贫穷疾病孤独死亡，抑或是爱欲不能圆满。曾经那太湖之滨的古村，村口的大槐树下，能让这一切顿时消散。使人回归心灵的净土，不再去想物欲的沉沦。但如今衰败中夹杂着商业气的古村，已不再是能让心灵平静的地方。当天平山已不再寂静，木渎古镇便只剩下了，喧嚣无尽。

已经没有多少再值得爱慕，没有多少再值得怀念。梦碎时分，江南不再。忘却属于这一座城的梦，忘却属于这梦里的一个个人。那融入在这梦里的温柔女孩，已淡却在记忆里；那从这梦中走出的清丽姑娘，却仍使我迷茫着。

曾经，这是我心心念念的姑苏，悠然神往的江南。现在，她只是苏州，一个有着清丽外表的现代之城。梦，破碎了。而何处，是前方？

# 清明一梦

春的四月，梦醒于江南。美好的，都是假象，就像，那虚无缥缈的，不知来自谁，也不知对谁的爱恋，只是一场，空怀的妄想而已。

风到这里就是粘/粘住过客的思念/雨到这里缠成线/缠着我们留恋人世间

姑苏，那江南烟雨的朦胧，曾是缥缈烟云的雾绕。雨落古街，似是，那三生三世之间一次又一次的相遇。也许，这一世，还不曾遇见。

你在身边就是缘/缘分写在三生石上面/爱有万分之一甜/宁愿我就葬在这一点

也许，曾过于注重于一人的我，很少去想到，那温柔恬静的她。唯美的女子，去到那水乡里温婉的城，想必是美好的。相见不曾动念，动念不再相见。也许，这便是所谓的无缘。

圈圈圆圆圈圈/天天年年天天/的我/深深看你的脸/生气的温柔/埋怨的温柔/的脸

也许，是因为经历的多了，错过的多了。江南，已不复那曾经我认为的美好了。幻梦的泡影，终究把那关于过去和未来的愿景破灭。回归现实，不再做那夜里都不应有的梦。

不懂爱恨情愁煎熬的我们/都以为相爱就像风云的善变/相信爱一天/抵过永远/在这一刹那冻结了时间

曾爱的故事，永远地停在了南京。曾爱的幻想，消散在太湖的烟云。也许，这世上本没有爱。也许，只是它还没有到来。相比那山的伟岸，还是更偏爱，那水的轻柔。

不懂怎么表现温柔的我们/还以为殉情只是古老的传言/离愁能有多痛/痛有多浓/当梦被埋在江南烟雨中/心碎了才懂

又是一年春。消去了曾经的一切幻想，抹掉那关于过往的所有回忆。不再抱有各种幻想，让那所有的身影从我世界的每个角落消失，不去念想，也就当作，都不存在。

# 叶落归心

花开了又谢了，雨下了又歇了。风吹了又住了，云近了又远了。情生了又散了，心动了又静了。江南的春，就这么匆匆而来，又匆匆而去了。

人生也是如此，在历史的长河里，你的一生，也许只不过是一瞬间就会消散的东西。那么，那些所谓的爱，却又是多么渺小的刹那呢？其实，多数的爱本身也许只是一种矫揉与造作，以爱之名，来掩饰那对欲的冲动与对孤独的逃避。那，也许并不是，真正的爱。因为，真正的爱，虐别人，更虐自己。真正的爱，不仅仅停留在花言巧语，不单单止步于相互依偎，而是相伴相守一生的责任，是一世的呵护与关怀。

如果你只是想追求一种体验，满足一下内心的欲望，那么这不足以称之为爱的东西，姑且就叫做情吧。情通常是浓烈的，会让人失去理智，会让人忘记自己真正想要的东西。无边的宠溺与包容，不尽的情话与缠绵，让世间的一切都黯然失色，如同七彩祥云笼罩的梦境。但梦终究会醒，会让你心酸，让你痛苦，久而久之便会麻木。不可否认有很多的情变成了长久的爱，但更多的是深情之后戛然而止的薄幸。但，却仍然有很多人前仆后继，只因那恋爱之时，那有如吸毒般令人上瘾的感觉。

但，无论是爱还是情，都是属于那些有缘之人的浪漫。先是相见，再是投缘，才有了那许许多多的故事。属于那些相见而无缘的，是注定悲剧的妄念。尽管初见时有那心动的感觉，但确认过眼神，那不是对的人。纵策马出征，终究也是泪奔。江南四月的月光，照进这山城。再怎么去跟，只剩伤痕，何必去一往情深。但妄念既生，却也难以放下。那，便默默关注着她，寄一份浅浅的情在上便好，没有必要去说明，更没有必要去点破。只要纵使在那最艰难的时日里，想起她时，也能轻轻一笑，让内心重归于豁然开朗。这，也就够了。

深情不是错误，错误的只是人。遇上对的人，用情再深也不为过；但如果你走向的是南墙，那么用情越深，受伤就越深。外表的吸引让人沉迷，而灵魂的契合方令人回味。找不到那灵魂的伴侣，不如孤守一生。爱情可以不必轰轰烈烈，只要互相知道，你喜欢我，而我喜欢你便好。孤独也不意味着与世隔离，与三五好友相伴，谈一些不着边际的话题，谁喜欢上了谁，谁又向谁表了白。暂时忘却前方的苦闷与烦恼，让幸福充盈在那自己内心深处的一方天地之间，也是极好。

敬亭宛陵，新港彩金，山水的相遇给宣州华丽的外衣。曾经恋的，是那江南的空灵与梦幻，在山与水之间自由放纵的心灵。江南的梦碎在了姑苏，淡却了那山水的热爱。便，坐在那图书馆的窗前，捧一杯温热的奶茶，看窗外风起云涌。想到那心动的姑娘，嘴角向上微微弯起，然后，便继续在那高数的怀抱里了。这，又何尝不是，小小的幸福？

迷茫，焦虑，恐慌，挣扎。待到放弃时，却发现全是自作多情的为难。其实一切都很简单，英语，C语言，高数，都很简单，不像感情那么艰难。贝叶斯纳什均衡的求爱博弈，选择取决于对人好坏的判断，也许，我把她想象的太美好，把自己想象的太糟糕。

说不上爱别说谎，就一点喜欢。说不上恨别纠缠，别装作感叹。就当作我太麻烦，不停让自己受伤，我告诉我自己，感情就是这样。

# 纳什均衡

贝叶斯纳什均衡的可怕，在于你不知道他什么时候会发生，又会以什么样的方式影响你的生活。露骨的文字之间，是行将放弃的不舍；迷离的表白之外，透露着抛下的意味。默默喜欢，默默关注，却又因着对方的优秀，而选择了放弃。在临放弃的时日，吐露真实的情感，从此，你是你，我是我，不在心间。走上了这条路，却发现身后却有着奋不顾身的跟随。平淡时光里，那纠结与矛盾折磨着自己，何必？不如选择放弃，或是，那我所不敢选择的，表白与恋爱。

莫名疲惫，莫名心累。沉浸于大佬的伪装，却深知自己渣的本质。生活如同酸糖，只能尽力从酸涩中回味甘甜的味道。半个春日的放纵，只为让那心沉沦，抛下所有，不切实际的想法。我是怎样的人，我其实很清楚。也许只有整日里泡在手机里，去解决人们层出不穷的问题，才找得到些许存在的意义，其实不必。敬亭宛陵，新港彩金，找不回曾经悠然的心境，看不见心旷神怡的美景，只因心已难静。贝叶斯纳什均衡，或许我从未了解过它的真谛。但我知道的是，若不把心转移，则快乐难觅。

孰真？孰假？是戏如人生，还是人生如戏？试探的最后，终究是一无所获。但也许，这反而是，期待的结局。切切深情，凄凄思慕，伤情事，恨无缘。

其实世间的一切，并不曾如想象的那般不好。鸟仍语，花尚香，天空湛蓝，阳光灿烂。我所追寻的到底是什么？也许，不过是一份情感的寄托罢了。钟情一人，则她，以及她周围的一切，都是美好的。心有所寄，则可安宁。也许，人要学会忘记，忘记淮阳，扫尽，那心中落叶满地。湖涛波影，而光粼粼。心无所向，而自凄凄。叶绿枝头，而终落去。花开四月，未闻花名。也许这只是个梦，一个淡淡忧伤的梦。但那梦里的人，仿佛，就是我的影子。

也许犹豫和纠结才是一切失望和落寞的根源，打败自己的也只是虚妄的幻想。未曾成熟而的吐露的喜欢，对谁都是一种伤害。爱是循环的轮回，开始与结束同在。这更像是一个不平等的棋局，你知道我，而我不知道你。

心火未平，初心难定。本已决定要逃避，却忽被步步直逼。不知是说的太多，还是有人点破，毕竟心思本是错，为何不能擦肩而过。湖畔晚秋，心生恋慕，叶落梦中。曾经的一望成欢，而今的一念倾心。但终究还是要承认自己的弱小，承认外界的一切说法都是假象，承认自己没有能力也没有勇气，去爱，或者去拥有爱。想知道她究竟是谁，毕竟，能对我有好感的女生，我还从未遇见。

随心而行的旅途，从波光潋滟的宛陵湖，一路北行。熙熙攘攘的小镇狸桥，人流如织的小城高淳。再是由南向北遍历金陵城的地铁，最终抵达，城市的中心。当钱，只剩数字的时候，如何才能保证，那爱情的真挚？人生，一所巨大的围城，没有时想得到，得到时想远离。幕后玩家，想的是害他的命，最后却让他走出了围城，得到了真正的爱情。

# 后来的我们

其实挺羡慕林见清的，他一直逃避着自己对小晓的感情，但最后小晓还是捅破了窗户纸选择了跟他在一起，即使他不是她心中的模样。爱其实很简单，只要缘分到了，就在一起了。但是爱又很难，当他的梦想因她的梦想而改变，才恍然发现，一切都不复曾经。她给了他机会，可他却被自己的畏缩打败，只能说一句对不起，却没有意识到，自己究竟错在了哪里。感情随心，一个人的坦诚和勇敢，不一定圆满。在那家乡的小菜馆前，伊恩找到了凯莉，世界重又有了色彩，但伊恩和凯莉已习惯于不再并肩，即使互相爱着，也只会互相说一声，再见。

五年穷困潦倒，五年飞黄腾达。五年相爱相守，五年各自纷飞。陪你同甘共苦的前任，并不能陪你走到最后；陪着你享受成功的现任，并不是那你心中最挂念的人。后来的我们，可曾能心平气和地面对那刻骨铭心的前任，可再能坐在一起，回忆往事，然后道一声再见？该说的，别说了。你懂得，就够了。真的有，某一种悲哀，连泪也不能流，只能目送。没有句点，已经很完美了，何必误会故事没说完。也许还是不要开始，不要承认喜欢过。反正后来，总是会去后悔的。有些人，一旦错过就不再。

既然快乐太难，那我祝你平安。生活已如此辛苦，何必再把你拖累。林见清的身上，有着我的影子，有着我的胆怯与逃避。所以即使有一天，有一个像方小晓这样的可爱的女孩，选择了向我表白，结局是不是也会像故事之中的一样，世界变得黑白。爱她，却未必能真正了解她。想要明白她爱的是什么并不容易，而真正意识到的时候，已经是后来。后来的我们，什么都有了，就是没有了，我们。

曾经喜欢一个人，她数学不好，当她遇见困难时问我，那小小的幸福，仿佛拥有了全世界。后来，没有了她的身影，但若有人问我问题，我依旧会耐心回答，只因想去，找回那时般，小小的幸福。然而，如今的我们，都已盖上了生疏的隔膜，不再是曾经同甘共苦的日子，不再有那时简简单单的喜欢。

心头之病，药石无医。老去想着人格的变幻与撕裂，不如多去结识一些积极的人，让他们来感化内心的抑郁，找回失落的美好。生活总有无奈与悲伤，总有阳光洒不到的时刻。永远保持积极很难，我们所能做的只是调节自己的内心。人生难得坦诚相待，但也不是所有事情都能说开，有的时候根本不知道怎么说，譬如围城。里面的人想出去，外面的人想进来，围城围的是爱情，也是人生。没有人是真正可恶的，都是围城之中可怜的一份子。以为爱着谁，想要进入那人的心，但进去才发现那不是自己所爱的，想要出来。以为想要什么，得到了才发现不是想要的，想要出去。真正知道自己爱着谁，想要什么的人不多，于是我们每个人，就都成了围城之中的一份子。迷恋上优秀的人，要么选择更优秀，要么就选择放弃。此时看到你，忽然觉得，到了决心放弃的时候了。打破围城，尽管会有遗憾。但这种遗憾，只与我自己有关。

当心的导弹瞄准了目标，便无所顾忌的追逐着前方。直到突然间它失却了方向，便浸淫出无穷无尽的迷茫。孤单的时候，看着那指缝的间隙，那是我的手指安然停留之所。确认过眼神，是苏锡常镇人。我为你转身，而心中泪在奔。姑苏城外的江水涛涛过那城，我不想再跟，不付情深，不愿心中留伤痕。水中的我看不见自己的眼泪，但水能感受到我的眼泪在它心中。她就像谷歌，不是因为知晓一切，而是因为她有着所有我所搜寻的，而一道无形的墙却把我隔绝。

当曾经占据心房的人在时间和距离的驱赶下远去，敬亭和宛陵给了心的宁静，新港和彩金包容了困顿的心。没有纠结没有思念，没有心的混乱与变迁。还能静下心去，沉迷于学习。然而心渐渐被身影填满，烦躁和痛苦从那黑暗的角落重又出现，便怠惰了，放弃了学习。竭力还原内心，却已不复曾经。
