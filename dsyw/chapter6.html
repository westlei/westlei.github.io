# 华子六百天

从第一次迈入三丫坡的大门算起，我在华为已经度过了633天的时间了。633天，433个工作日，33个加班日，37天在培训，30天在远程办公，请假2天半。去除以上林林总总，我有397个日夜在工位里无谓的内耗着。在深圳坂田H1、H3、D2，在上海万创、国展。华为或者海思，上研或者坂田，没有什么不同，也不应当有什么不同。

PC上永远开着Idea、Pycharm、WeCode，把一块屏幕留给五彩斑斓的代码。另一块屏幕不是写着不明所以的PPT和文档，就是开着不知所云的会议。永远喧闹的工位充斥着百十人的嘈杂吵嚷，从不熄灭的大灯掩盖了白天与黑夜的隔阂。白天亦是黑夜，黑夜也亦是白天。非黑非白，无日无夜。

这便是大厂中人的围城，论薪资，称微薄大抵不当，曰丰厚亦有弊病。 时间在不知不觉中流走，只留下日复一日的困倦。昔日所谓立足于华为而在国运升腾时得以分一杯羹的期待甚是可笑，只剩下国运衰微华为败退时何以自保的惊惧。不过说起来毕竟是到了大而不能倒的地步，最次大约也就是变身国企而告终。

近来流行的特种兵旅行，于我而言，早已熟稔。昔日手中钱银无多，行四方之时不得不收紧预算，屡次往返于半夜的宣杭线上。而今钱包有余闲暇不足，依然需要晚间高铁与红眼航班来补足远途行程所耗的时间。再辅以早八晚十，花尽十四个小时周游，避开排队良久的景点，大多城市一天半即可游遍。所谓旅途，不过是离开自己厌倦的地方，去看看别人所熟悉乃至厌倦的生活。尽管步履匆匆，但亦不失旅行真意。

然而，日行五十里四万余步，于我而言不过寻常，于旁人而言甚是煎熬。故遍访四方之时，少有人与我同行，形单影只，以为常事。良久如此，社交恐惧渐生。一切集体活动与我无关，一切新人我皆拒绝相识。互联网上我重拳出击，线下我唯唯诺诺假装空气。不过如此，本当如此。

曾经，我有一个机会，去获得一段感情，但我最终选择去将它埋葬。习惯于孤独的人，会为难得的温暖而感激，却不懂得如何去回馈那一份温情。我所有的一切，不过冷冰冰的余额，或者下班之后的疲惫、厌世和负能量。让一个无辜的人承受了这一切，我很遗憾，却也无能为力。

初次来到提瓦特至今的939个日子里，我只有674个活跃天。须弥而后的版本里，日益趋于答辩的剧情和完全不对胃口的角色消灭了大部分玩的想法，但剩下的极少数仍然值得我去掏一点money去赞助。就像我跑到七公里外的万象城，面对着八点之后空空如也的必胜客，吃完一份并不怎么好吃的单人餐。我也不知道我是什么感觉，但怎么也不是啥好的感觉。

# 浦江落雨

醒来就听着窗外淅淅沥沥淅淅沥沥的小雨，一直到下午接近傍晚才放晴。当然也可能早一点，不过我午觉睡醒已经四点了。​想着还是去凑个热闹，虽然有预期，但是外滩的人人人人人人人从众还是叹为观止。

不是很喜欢坐地铁，感觉71路就很好，沿着沪青平公路延安路直达外滩还挺快，毕竟BRT专有车道基本不堵。然而今天从人民广场到外滩堵了半个钟，车厢只能说挤的比沙丁鱼罐头还挤，才意识到这段原来没有专用道。

避开南京路方向的汹涌人潮，一路往下走过十六铺到复兴东路渡口。往杨家渡方向的入口主打一个门可罗雀，渡口全是去东昌路方向的游客和招呼去外滩的拉客三蹦子。上船时已经暮色渐起，看远处的天边已经晕染了一层粉色。

过江就没什么人了，外滩的繁华已经远去。不远就是塘桥渡口，夜色之下已经收渡。旁边是塘桥新路枢纽站，上海最长的地面线浦卫线从这里出发，去往金山石化汽车站。全程八十多公里一百多站单程三小时，站牌直接摆大烂，奉贤往后的几十个站基本不写了。

塘桥以下就是南码头了。从南码头往下到世博这段的江滩很像杭州滨江，南浦大桥和卢浦大桥横跨黄浦江，一如西兴大桥和复兴大桥横跨钱塘江，唯一的区别似乎只是钱塘江上没有这么多船来来往往，以及西兴大桥允许行人通过。夜色下桥的绚丽光影洒在江面，恍惚如此，不过如此。

# 淀山遥望

青浦本没有海，但湖足够的大了，亦可视为海。

但海水有着强大的净化能力，能将大部分污秽溶解或者沉入深海，甚至可以净化观海者的心灵。湖大多不行，来自这里或那里的污秽在湖水中沉淀，发酵，腐化变质，最终成为湖底深不可测的淤泥。若是水网丰富的地区，表层的活水可以借大气的水循环不断刷新，让湖水深层的腐化不那么明显，但终究是腐化了。

这些年的工作于我而言，便是一种持续不断的腐化过程。变化的工作地点、工作内容、绩效激励就仿佛不断循环的活水，掩盖了因级别薪酬长期固化而引起的心态腐化。腐化后的我社恐孤僻、胆小怕事、敏感多疑，让领导更不敢安排承重。如此而来，未来自然是可以预期的糟糕下去。

今天是cp29的第一天，大约是我这些年来见到异性最多的一次。无数plmm戴着各色装束出没于会场，而我只关心我的4G上不了网。于是我回到了空无一人的工位蹭网刷手机，甚至还去看了看工作相关的事情。当我离开工位时，我大抵已经明白，漫展这种现充活动是不适合我这种赛博鼠鼠的。

于是我漫无目的的离开了会展中心，向西行去，穿过高楼林立而无人问津的西虹桥商务区，穿过人群熙熙攘攘的徐泾镇区，穿过风格奢华的徐泾西部别墅区，然后向北折往天空之城。所谓天空之城，大抵是居于地铁高架线与高架快速路之侧，看天空中川流不息的车水马龙。或许我后面会搬到这里，又或许不会。

从青浦西部的漕盈路枢纽出发，拐两个弯就进入了江苏昆山的淀山湖镇。淀山湖镇因湖而名，且拥有淀山湖东北的几个主要景区。时值五一，自各处而来的车停满了淀山湖大堤。湖岸的草坪帐篷林立，空中风筝飞扬，不时还有烧烤的烟熏味。但总的而言，人还是少。

沿湖进入上海的桥被挖断了，进入上海需要绕行三公里走检查站。无论再怎么临沪，必要时也不过是挖断个桥封闭个检查站的事。沪国终归是沪国，特立独行的沪国，就像今天天气似乎应该很适合看日落但并没有日落一样。

每年的五月都要去面对入夏的第一关——手臂和脖颈晒伤。年年如此，自然也无所谓习惯不习惯。对于身体相关的一切不大不小的疼痛，我都不是特别在意。长期亚健康的情况下，没有个这儿疼那儿疼都是不正常的。

16personality的问卷进行了更新，我再一次从INFP-A变成了ISFP-T。依稀记得几年前我就是ISFP，但是自我认同上我还是更认可作为INFP的我。或许我本就不是什么典型人格，属于精神病患者的现实载体。

本来应该去贝洛伯格待一天的，但是这种雨要下不下的天气实在是闷的太厉害了，于是觉得还是应该出去转转，看看星穹铁道古典版。然而cp29的巨大人流让徐民东路变成了二次元阅兵仪式现场，徐泾东进站口的队伍长达几公里。我瞅了一个缝插队钻了进去，但依旧未能抢到一个始发站的座位，被迫一路站到城区的江苏路。

张江有轨已经在这座城市运行了十三年，而随着共享单车的普及、地铁覆盖的增加，这破车已经变得无人问津，现在似乎就真的只是旅游列车了。作为纯进口的列车，工厂破产让本就艰难的运营进一步雪上加霜，最后沦落到现在只剩两辆车，一天只跑四个来回的摆烂结局。这破车的运营也丝毫不令人报以希望，刚开一站路起步就熄火。张江地区的道路本就不宽阔，两车道上蹲一个移动路障无疑是当地司机的巨大折磨。

张江地区的路名也是以名人为主，华佗路、蔡伦路、毕升路、李冰路、张衡路、沈括路、祖冲之路、郭守敬路、李时珍路，达尔文路、爱迪生路、哥白尼路、法拉第路、牛顿路、居里路、哈雷路、高斯路，像极了曾经待过的深圳坂田。想起曾经在贝尔路、隆平路、稼先路、张衡路、冲之大道、居里夫人大道徘徊蹉跎的岁月，不禁有些茫然。

如果说网名是一层赛博皮套，那我的这个皮套也已经用的太久，人皮一体，也到了应该转生的时候了。细想来，这个身份也已经随我走过了八个春秋。八年间，大多数时候的我，就像这个网名本身一样的荒诞而无意义。换个ID，抛下些素未谋面的赛博朋友，也算是一种重新开始。

# 螺旋而下

大晚上看奶绿直播b站周刊历代top1视频，从16年一直看到21年，就看着逐渐消失了二次元和鬼畜的b站渐渐变质，18年以后满世界的各路尸体和赛博丁真。有些东西就是当时觉得好笑，但是现在看真的烂完了，比如什么对对子这类的。一晃都这么多年了，我也成了老东西了。

每年到这个时候就有一种赛季末奖励结算的感觉吧。这个赛季转了个职业，约等于啥事没干，赛季奖励跟上个赛季也差不多，没增没减，日常奖励等级也一样啥变化没有。但是BOSS越来越难打，下旬又要换主管了。说是要把南京那边加班到十二点的风格弄过来，那我可顶不住啊。

最近删掉了一些列表里长期失联的以及一部分完全不熟的人，十分困惑于为什么我曾经会那么热爱去认识陌生的后辈，或者说，困惑于为什么我在这个赛季里变得社恐了。有时候与一些自以为比较熟的朋友联系，才会发现人家对我的印象几乎全部来自于某些我的所谓的“朋友”，一些一辈子不想再去接触的“朋友”。当对方说出这些话的时候，我可能就再也不会想去跟这个人做朋友了。他们眼中的我大抵不是我，而我拒绝与一切不认同我之于我的人有交集。

但全面的冷淡，感觉大抵还是与疫情有那么些关系吧，在那个寒冷的冬夜里，加班到十二点过才下班的我，给同事送完药回家，一个人躺在房间里，发起不知道多少度的烧，关掉空调却比开着空调还热。一场病并没有多么影响我本就稀烂的健康，但是让我本来就趋于闭塞的内心更加拥堵了。原来亲密关系什么的似乎并不适合我，疏离才是该有的样子。不过如此，早该如此。

所以现在沉迷原，沉迷舟，沉迷穹轨，沉迷看v。一切后现代互联网文化的糟粕纷至沓来，也许我本来就应该在活在虚拟世界里，但现实还是要继续。想开摆但不能，也是无数痛苦之中的一种罢了。就像上周晒伤的地方开始脱皮了，而蜕皮只会带来些许痛苦，并不会带来新生。就像up池里的第二次出金变成了克拉拉的星魂，而大家都说克拉拉很弱。不过脱皮之后起码不会二次晒伤，抽卡手游也主打一个休闲。我是不会去忘却之庭坐牢的，就像我打原神几乎从不下深境螺旋一样...
