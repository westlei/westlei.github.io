# 彼岸花开

# 电子客票

当南京段的带鱼冲破大胜关大桥的晨曦

把东南沿海的欢声笑语带进巴渝的夜里

当南局的狮子在宁芜铁路上拉响了汽笛

带旅客一路西行望向那彩云之南的天地

坐上清早的快一一零九把皖南春色开启

让山水之间那片徽州的梦境投射进心底

凌晨一点的钱塘江西兴大桥上凉风习习

看远处的快三四八带来飘向北方的记忆

太湖之滨的小城长兴散发着落魄的气息

远行的湖州人却还得不辞辛苦来到这里

快一四三八在巢湖站台静静地停下休息

只为了赶到那合肥城里时太阳刚刚升起

来自遥远南疆的快五九三午时来到此地

让阿克苏与杭州缩短了时间空间的距离

芜湖站的首班高铁在马鞍山东异常拥挤

无数的旅客们奔涌向了上海苏州或无锡

夏夜的南京站候车厅里早已开足了冷气

绿色的长虫是日常晚点的动集动七零七

V字型线路的金凤凰需在城站转个座椅

让宁杭沿线的人们去上海的路更加安逸

夏夜的溧阳站坐上那大地铁动二二二一

北纬三十度的乡音融化进那西湖的水里

南局的大红枣目视着金山卫的朝霞升起

唤醒了夏天的上海带来逐渐灼热的空气

宁东段的狒狒深夜里奔走在苏南的大地

同合段的狮子一起扰动着那旅人的安息

铜陵长江大桥上广铁红海豚潇洒又飘逸

艳羡的带鱼却瞅见一旁卑微的二十五B

老旧的双层车在早晨离开南京开上宁启

在白色和绿色世界里装点着蓝色的绚丽

无数的车票记载了多少遥远的快乐回忆

又听见宁芜铁路上依稀响起蚂蚱的汽笛

# 端午

大佬在墙外的世界里敲打着高深的函数

菜鸡在白嫖的字里行间苦苦思索着出路

介乎两者之间的我看着套了层壳的目录

却发现对于仿真的尝试卡在了最后一步

不乐观 却也不悲观 不黯然 却也不淡然

喜欢一个人 是什么滋味 我知道 也体会

不想再遭遇伤悲 在压力之下 心碎 酒醉

是应该趁着现在把念想全部都烧作飞灰

还是留一缕念想来给低谷里的自己安慰

其实也只不过是自己 过于了卑微

# 终末感言 (Cover:杨超越)

对不起詹总和WL姐

我就是干啥啥不行 混日子摸鱼第一名

我前阵子压力实在是太大了

我就是转下这个专业

我没想过我能学电子两年

你不知道就是大早上被群聊喊起来

他告诉你要去实验

我说我不行的 他说LWX你可以的

然后其实我这两年经历了很多的争议啊

他们说LWX怎么可能转专业

他要能转成我都可以

我想说真的 我给你们做了一个很好的榜样

你们看老天它不一定爱聪明的人

它万分之一也会宠幸到我们这些笨小孩

所以不要放弃平庸和笨的自己

说不定老天就是喜欢你

它就是让很多人都喜欢你

你就配拥有这些爱

但是这些爱太重了 我每天都要爬起来实验

我每天都好焦虑啊 我学不动了就要挂

但是我想说我真的练了好久 我一考试我就错

我也不知道为什么 哎我气死了

我两年了我终于放弃了

我以后就没办法实验给你们看了

我真的是我学不好

我也没有专业大佬给我去白嫖了

我一学不好就挂的太明显了

然后 我哎 哭成这样

点心科铁fiveLWX 再见

我现在是LWX 未来请多指教

呜呜呜我不要学电子了

# 合杭高铁

期待终究失去了期待 就像那CCS的仿真毫无色彩

等待终究不值得等待 就像HFSS的演算漫长难耐

热爱终究没有了热爱 就像单片机的课设令人心塞

存在终究并没有存在 就像是一人的念想泛滥成灾

# 六月下

微信仿佛一个支付工具般毫无存在感

搜个早晚安零星记录还来自同一个人

南京到宣城的高铁学生票也要51元了

甚至不如在芜湖转车还能再便宜一点

成绩单的字里行间满是废物两个大字

想玩一会儿内心都要怀着些许的愧疚

反复横跳的期末终于以线上考试收场

似乎还有两周大三生涯就彻底结束了

好好反省反省网课半年我学到了什么

好像认识了几个网友 而后一无所有

# 尊严

到电信科的第二个年头差不多快结束了

可惜刚来电信科的第一个学期就学废了

从此就自我放弃不再想着成绩这回事了

别人复习的时候我都在玩 考前几小时才翻翻书

这样又怎么可能考得好呢 怎么会不是刚及格呢

但是我依然有我作为一个学渣的基本尊严

那就是所有的实验课设都必须要自己来做

即使有时因为理论水平很差可能不大会做

但也不肯就甘心就完全去白嫖一份这样子

我至少也得做一点自己的东西自己的痕迹

大小毕设前的最后一个课设应该算结束了

以前的课设全优看起来应该也只是侥幸吧

我不喜欢模拟电路 更不喜欢高频模拟电路

我喜欢的是数字电路 喜欢74触发器锁存器

我喜欢凭我自己的印象自己的理解来做题

虽然成绩惨不忍睹但是确实是真实的自己


但是这一次 我似乎好像必须得去努把力了

即使失败 也要失败的体面些 失败的有光彩

你不是一个人在努力 还有那么多人支持你

更何况 这里面还有 你喜欢的人

你要成为 她眼中那个 最好的你

# 怀

期待它本身并不值得去期待

所有的色彩混在一起成了白

口口声声的喜欢并不意味爱

越穷的时候越会幻想着发财

理想和现实并不可以做买卖

天天做计划心里并没有未来

满嘴跑火车的人实际上很菜

暗地吞咽了泪水伪装着开怀

# 没有如果

如果我

离开了5.99英寸的手机，15.6英寸的电脑

告别了移动4G网络，200M电信光纤宽带

我是不是就和这个世界，完全失去了联系


好像是的

不会有人知道我在哪里，或者在做些什么

灵魂安放在可可西里，身体则在野兽腹中

再烧毁所有的资料证件，那就永远是个谜


想念牛头冲的二百零六个夜

仿佛已经失去了所有的一切

现实生活与网络已完全割裂

越是卑微越喜欢去自称为爷

打算出门走走却看日头正烈

或是一头暴雨浇得神魂俱灭


只好去做个不会说话的哑巴

天天在屏幕前打字胡乱表达

飘在九重云端里看戏和吃瓜

游戏中动不动就大意失亲妈

要见没见过的人期待却害怕

网络上经营的伪装全都崩塌

终究不过是个笑话


好想爱这个世界啊，

可是世界却并没有那么值得。

在乎的人能有多少，

又有多少人会真的在乎我呢？

然而，越如此，越害怕。

孤单行走，不过是逃避世界的一种表达。

我可以随时去到什么地方，

或许我也随时可以去到另外一个世界上。


也想过去爱一个人，也想去爱一个人。

可是这个疫情。

情感在长久的生活中厌烦，

又在长期的隔绝中变淡。

在家中受不了父母不停的唠叨，

走出家门却又不知去哪。

翻遍列表，要么太久没见，要么并不熟悉，

能够一起玩的却又隔着天南海北的距离。

生活了十几年的地方，

感受最深的，却是这无处不在的孤独。

# 彼岸花开

生活不只有兄弟和闺蜜 还有永远无法实现的愿望和理想

以及不知道什么时候就停滞了的各种计划

本以为能够充实的暑假 还是荒废了

选择越多越趋于迷茫 终无所适从的慌

想忘记现实的一切 相信总有所谓的未来

可我已经不是大一的我了

三年丧失的不仅仅是信心和学习能力

还有那种对未来的憧憬和向往

翻翻列表 并没有谁能够去打扰

认识的人越来越多 交情越来越淡薄

渐渐地 谁都没了声息

只剩黯淡下去的名字

彼岸花 开彼岸 花开叶落几世缘

谁说魂魄终相伴 奈何桥头却不见

没有谁能陪伴你到最后 再亲近的人也不行

有人问我是不是有喜欢的人

也许 大概吧

但我也不知道算不算喜欢

但这么多年的经验告诉我

不确定 就是不喜欢
