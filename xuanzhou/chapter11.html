# 日薄西山

# 仓皇

看着学弟学妹们踏入了转专业的不归路

听见那远处在争夺奖学金名额的号角声

预备考研的钟声在前方一遍一遍地敲响

似乎没完没了的比赛掩盖了黯淡的迷茫

总是会无意识地伤害到自以为亲近的人

总是会忘记自己究竟需要的是什么东西

总是会在同样的错误之中循环而又往复

总是会栽在自己所珍视的每一件事物上

别人的故事里，有着快乐，也有着悲伤，

但也许你只看到了快乐，却看不见悲伤

自己的故事中，有着选择，却没有悔过，

你为你做下的一切负责，永远不能逃离


我累了

但我想逃离这一切时，才发现无处可逃

# 明天

从愚昧之巅走向绝望之谷的谷底

从人山人海迈向灯火阑珊的流离

逐渐找到一点前进的方向和动力

却有人说我在做的一切毫无意义

不曾想过前方的光明出现在哪里

仿佛一个只会怼人还暴躁的笨逼

终究会在挣扎的末尾去寻求放弃

那故事的开始所谓追寻又是何必


十八岁的最后一天

厌倦眼前的一切 期待未知的明天

# 输赢

当失败的铁拳一记又一记地落在我的头顶

当现实的利刃一道又一道地戳进我的心灵

仿佛神魂仍在那梦里，不曾醒，也不愿醒

仿佛宣州秋日的细雨，不曾停，也不必停

就让我一直在梦里，无需面对现实的困境

就让我还活在过去，依然拥抱美好的曾经

若再来一次，必绝不犹豫，总好过，困于情

若再来一次，必坚持自己，何必想，输与赢

# 抑郁

终于似乎忙完了手上的事情 只感觉累

不知道是连绵十天的课伤尽了元气

还是突如其来的寒潮冻结了热情

也可能是单纯的心生倦意

最近抑郁症的话题逐渐火热

我也在想 是什么支撑我走到现在

好像过去的绝大多数时间都不尽如人意

但似乎也没有糟糕的极点的地步

一年多噩梦般的经历 失落在渺茫的岁月里

迈着颤抖的步伐 从天堂走向谷底

似乎活跃的表象 掩盖着日渐封闭的自己

心灰意冷 为现在所拒绝 为过去所抛弃

也许说 曾经努力的动力 多么荒唐可笑

再看来 再荒诞的缘由 总胜过于没有

为何会去死亡 似是 找不到活着的理由

无所牵挂 无所念想 无所欲望 无所寻求

心空灵 神沉寂 心痛的感觉 崩溃的瞬间

感谢忙碌 感谢忙碌让我避开了空虚的想

感谢酒精 感谢酒精在难受的时候麻醉了我

也感谢艰难的日子里陪伴过我的每一个人

既已注定了众叛亲离 何不开始就选择自闭

至少 小确丧的日子 没有心痛的抑郁

# 劝学

余自金陵而下宣州两载有余，伤于心，困于命，渐至于今日之败。

每思之，夜不能寐，两载故事犹然在目。

初向学，名每列于众人之先，与诸友相偕，或舞文弄墨，或游山涉水，心康而体健。

后渐迷于色，心有所指而意不足，久而知无所成，遂生去意。

既去，课业多艰，适情有所困，学无所成，心焦而身颓，遂一败涂地。

既败，闷闷而不乐，郁结于心，头益秃，体益肥。

后迷于赛事，辛勤数月，略有斩获。

然学业益艰，向学之心益减，思及往后，则惶惶不可终日，唯以暴食并通宵游乐而自慰。

愿改过而自新，然无力为之。是夜，复思及此，遂书此文，示来者，且以学业为念，勿蹈敝人之覆辙。

# 释然

那个有些遥远的夏天后，生命的颜色些许的黯淡了。没有那么多希望，也没有那么多值得。没有了心动的感觉，消散了人事的纷争。也许，只是单纯的老了而已。也许，是岁月终究磨平了心灵，把未来照进现实的侧影。

冬天若即若离，在各种各样的选择之中犹豫，在各色各样的劳碌间挣扎。爱之一词沦为禁忌，就连喜欢两个字也不敢轻易说出。仿佛一开口，所有美好的东西就消散的无影无踪。

曾经的我，给每一个好友都写过一段话，也许他们并不知道，也不会知道。在心中占据一席之地的人们，渐渐走散，消失在时间的路口，消失在列表的尽头。以前总觉得失去的总会珍惜，可如今，什么都不剩的时候，反而释然了。

没有梦想，孤独的走在不为人知的沙场，在水天一色里流浪。就好像那一个个夜晚的旅途，不知何时开始，累了才会结束。形形色色的人，有一天会遇见，有一天会消失。能相逢于这个偌大的世界之中，本就是一种缘分。记得那最后一眼，殊不知一眼万年。

曾经，有过一个大雪的夜。唱了最喜欢的歌，却再也找不见，不归的旅人。

# 陷落

梦终究陷落在天亮的时候

就像每个故事都有始有终

像不曾报以过的零星期待

像晚霞终究会落下成夜幕

都过去了 一切都过去了

失落而迷离的情感

并一切的休闲享乐

都过去了 一切都过去了

不愿回忆那些往事

不曾请求所谓谅解

如同所谓酒不醉人人自醉

迟来的答案与残酷的现实

这世上本没有一见钟情

更多的都只是一厢情愿

可能，这是最好的一天

也许，这是最坏的一天

# 冬来

今年的冬天，似乎比往年要来得更早些。

还算温暖的白天过后，清冷的夜晚降临。

读着花呗账单数字，内心多了些许寒意。


实验和作业如一把把尖刀，戳进了心口。

但随之而来的无尽考试，才是透心的凉。

看起来，这个冬天并不比上一个更好些。
