# 我的抽象人生

近来总觉得自己的记忆力有些问题

大约是像大模型一样出现了幻觉

发生过的事情经常一点都记不起

没做过的事情却在大脑里像真的一样


认识添翼老哥到现在也有六年了

总能get到一些有意思的知识

还有更多的有意思的生活态度

虽然有时候唠嗑的时候经常会有一种

"这一幕是不是之前发生过"的感觉

但是从大脑里检索又检索不到任何东西


六年前也是六月初我去了谷里大塘金

那时候在谷里看到的薰衣草非常好看

以致于我总有一种

我应该在什么时候又去过一次的记忆

但是具体的时间地点我是一点想不起

甚至于连相关的照片都找不到一张


如今香草谷已经变成了摄影基地

收着价格远超其内容本身价值的门票

谷外的一点点薰衣草田也被铲掉了

不知道是种了东西还是干脆抛荒了

景点似乎就此淹没在荒郊野岭里

在任何地方都看不到什么推荐......

# 匆匆岁月流逝

南京地铁更新了2024版的毕业祝福：

岁月匆匆，惟愿总相逢。南京地铁祝福毕业生毕业快乐，愿大家去更远的地方，见更亮的光。

相聚别离，都是刚好。南京地铁祝福毕业生毕业快乐，愿大家在未来的日子里，前程似海，一路繁花。

从南京南站一路向南，大大小小的学校数不胜数。南航、河海、金肯学院、南航金城学院、南工浦江学院、江苏二师院、南财红山学院、南京机电学院。如果从高淳继续向南，再有六十多公里，还有隐蔽在山间小城之中的牛职院。似乎，这两天也正好是牛职院的离校日。

每到一个临近学校的地铁站，毕业生祝福语便会播报一遍。但毕业于我而言，已经是一个有些古老的名词。三年之前，依稀记得是个有点下雨的日子，从牛职院驱车向北，跨越石臼湖大桥，越过这一座座大学。而随着今天这一批人的离校，在校期间与我有所交集的人也就全部离开了。再回去，似乎也没有什么大的意义了。

毕业以后这么些年，在不同的岗位间周转，在各地研究所里游荡。说混的好也算混的还好，有说得过去的工作和说得过去的收入，起点已经可以比肩不少硕士。说混的不好也混得不好，完全看不到上升空间的工作状态和一片混乱的生活状态让我收获了十分糟糕的精神状态和健康状态。曾经还想说或许有什么契机可以改善一下，如今的我只觉得维持现状就很难了。

闲暇之时，看看大江大湖大海，也不失为一种生活的慰藉。毕竟，除此以外，我的生活只剩下双眼无神的麻木工作与看管人扮傻子了。当然最近也沉迷于剑网三，不过游戏总是一阵一阵的，过段时间估计就咕咕了，就像我不知道多久没打开过的帕鲁一样。

# 赛博桑海宁

之前突然意识到公司WiFi的出口是上海华为云机房，所以上班的时候我依然是赛博上海宁，在各大平台的IP归属都是上海。

在上海待的一年，应该说大多数的时候都不太愉快。但是从华为管理最松散的部门到管理最严苛的部门，对华为底层逻辑的理解更加深刻了一些。我大概不太适合华为，但在目前的情况下，也没什么更好的去处。

去年这个时候，有些期待，有些感慨。去年毕业的那批人贯穿了我大学的下半场，乃至我工作以后的生活。今年又到了毕业季了，但是今年毕业的各位只能让我感到陌生。不知道明年的话又是一个什么样的景象了，或许我那时候也毕业了呢？

这个月的账单有些离谱了，钱如落花流水般散落不知归处。流萤和知更鸟一起也就花了648+128，乱七八糟的东西也没买多少，赶紧把原计划当礼物的东西拿去转手卖了回回血，换一件没那么贵的当礼物。当然其实不送也可以，毕竟我一直也没有送礼的习惯来着，但恍惚记得我好像说过我要送了，再不送也不好来着。

# 梦与仓桥直街

昨晚做了一个非常离谱的梦。梦见我从坂田打车去深圳站(罗湖)赶高铁，然后等了很久很久，高德才有人接单。显示是距离我12公里，从平湖过来。等他来的时候，已经快来不及了。然后我问师傅从平湖过来这么远接单，师傅说他是从塘厦来的，更远。

接上我师傅就往布吉开，我跟师傅说应该走坂银通道，然后师傅完全不听，一直开到了沙湾。这时候差不多到发车点了，于是我跟司机大吵了一架，就感觉血压噌噌的往上升高。最后师傅把我丢在了丹平快速路上，我汗流浃背的醒来了。

昨天下午小转了一下绍兴。南京出发到杭州西两个小时，杭州西坐地铁到鲁迅故里又得两个小时。雨后的小城里，屋檐依然滴答滴答的落着水。闷湿潮热的空气里散发着臭豆腐与黄酒交杂的发酵气味，熏的人有些迷离。

仓桥直街跟江浙沪其它地方的古镇水街并没有太多的不同，最大的不一样大概就是别的地方卖奶茶，到这里全变成了黄酒，连冰棒都是黄酒冰棒。转一圈再到鲁迅故里的时候已经是晚上，免费的预约票早已售罄。门口看见有家面馆，想想还是吃一顿。杭州的面总的来说我还是喜欢的，虽然说浇头感觉怪怪的，不如苏式面也不如南京的面。

# 风雨夏日

24年也过去半年了。

昨天喝了点酒，度数不高，但是喝完第二天早上起来就有点头疼。出去溜了个五公里多点的小圈，脑袋就昏昏的了，果然又是一个躺着半梦半醒度过的下午。

明天就是24H2了，上半年什么都没做，就这么浑浑噩噩的过了。去年底接的项目，半年下来就写了半个demo，倒是进了一些奇奇怪怪的工作组，跟我刚入职时那群级别高资历老的leader们排在了一起。只能说没想到在外面转了一圈回来还是面对这些人，不过位置倒是从弟中弟混成了合伙人的代理人。但是级别没升工资没涨，所以事实上还是弟中弟中弟。

在事业毫无长进的同时，生活也还是同样的混沌着。健康状态几乎没有任何改观，看病检查做了一半觉得麻烦不做了，智齿拔了以后天天卡各种东西，腹泻和呕吐也还是每周必有的常客。对象依然是看不见摸不着的虚空概念，半年里线下打过交道的女生除了同事和前同事好像应该就只有一个，工作以外除了看v就是到处转悠。

铁道抽了阮梅花火知更鸟加流萤，但是练度烂的主线任务都过不去。原神几乎啥也没抽，娜维娅后面就希格雯，因为剧情烂的没有想玩的欲望。雀魂有活动偶尔打，没活动登陆都懒得登。农药的话因为领导打所以我偶尔也打，靠着两个英雄上了波王者，但是基本上也打不动了。最近沉迷剑网三，但是热情消得很快，已经懒得去开地图主线了。

上半年一共26个周末+节假日，含7个单休，三个三休，两个长假。出南京玩的话大概是23趟，可以说每个双休以上都有至少一天不在南京。不过长途只有西安和珠三角，剩下都是长三角的周边城市，其中苏州四次，上海三次，杭州两次。大部分出行没有明确目的，甚至也不一定会造访多少景点，只是出于一种对生活的逃避和厌烦而出行。就像跟朋友说的一样，如果出门没带脑子，那我的双腿会自动把我带到南京南站。

剪了26个plog发阿B，加起来浏览量2500出头，主打一个粗制滥造只给自己看。同步到阿抖的精简版24个，加起来浏览量530，主打一个仅好友观看但没有好友。QQ动态和朋友圈大约各100多条，频率大约是1.6-1.7天一条。微博发了25条，频率在一周一条。welink水群记录估计五万条起步，大概每天上班都要花半天以上的时间闲聊。简单的说，废话是很多的，能聊天的人是很少的。拿夜宵的次数是40次，也就是说加班仅40天，别的日子都是六七点之间下班。

这堆数字好像能说明点什么，但事实上什么都说明不了。大概就是说，现在的我挺闲的，但是很无聊，生活中充斥着无所事事。当然如果你说我一定要做点啥的话，那我肯定是抗拒的。生活就是这样，维持现状总是好的。
