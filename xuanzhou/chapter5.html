# 心无所依

一切的开始，是黑暗的。直到那一天，我看见，那来自天边的一缕微茫。光越来越亮，让我痴迷，让我向往。我追逐光，但光总是背离我的方向。试着去追赶，光却越来越远，越来越暗淡。直到乌云遮蔽了光芒，起风了，放下了。停下脚步，回味那光的美好，却见那天边，是两道彩虹，正闪耀着光。

暮色四合，天空陷入了苍茫深蓝。光的影，终究消逝了。只有那彩色的记忆，依然存在着，不会消逝。也许，关于她的记忆，就像这彩虹一样，会是永远美好的吧。

在我的性格里，本有着诸多的弱点。矛盾与冲突，才是我性格的主旋律。也许我在很多时候都愿意去谨小慎微，但总有那么一段时间不愿去计较是非因果。许多人不喜欢我，更多人挺讨厌我，这也没关系，有人不那么讨厌我就好。

曾经，我也是个不那么活跃和积极的人。爬一爬山，游一游湖，看一看山水之间，那宣州城别样的风景。整理好那似是而非的往事，默默地在心底里喜欢着一个人。淡泊宁静，无人打扰，独自享受，那份天地之间的清寂。在各种各样的比赛里忙碌，有着淡淡的伤感，也有着小小的幸福。

不知道从什么时候开始，我变了。沉浸在虚拟的网络世界里，去跟一个个见过但更多的是没见过的人天南海北地聊着，似乎一刻不看群就不舒服似的。但是实际上，要关注的，本没有那么多。

也许，只是因为对爱情绝望了吧。

# 再逢七七

飞逝的时光里埋葬着过往和曾经，皖南的小城中磨灭了梦想与初心。也许还有几分书生意气，也许还怀多少幻想回忆。一晃，已经快过去三百天了。三百天，不算长，不算短。长到足够忘却自我，短到难以忘却时光。过度修仙，飘忽不定；放纵自我，一片狼藉。尽力不去想，不去念，但终究有所触动，有所牵连。遗憾不算遗憾，叹惋不必叹惋。既已做出了选择，那就不再悔过。前方的路仍然漫长，不如寻找前方的姑娘。

人事将尽，剩下的全听天意。随波逐流，磨灭掉三分傲气。曾经想要努力，也曾喜欢着你。渐落得初心无觅，渐生得几分去意。也许并非不适合经济，也许未必擅长于信息。趁早把一切封存回忆，不管未来是否会惋惜。叹落花有情流水无意，不如闭门来反思自己。若是走远，那就把一切归于沉寂；若仍驻留，就不把念想埋藏心底。一晃已经入夏，一晃即将入秋，一晃又要到那叶子落的季节里。踏过十八岁，面对的，又会是怎样的自己？

七月七日晴/忽然下起了大雪/不敢睁开眼/希望是我的幻觉/我站在地球边/眼睁睁看着雪/覆盖你来的/那条街。七月七日，晴间小雨。睁开眼，看时间，却忽然想起，从前。

十一年前，七月七日，告别了两江环绕的重庆，来到了几分生疏的金陵城。单纯无知的我，并不知道在这座城市会发生什么。还抱着几分天真的幻想，以为世间，尽是美好。

时光渐渐过去，现实让我放弃了幻想，黑暗让我抛下了天真。那是一段多么不愿记起的遥远岁月啊，尽力在黑暗中喘息，没有光明，没有未来。挣扎在岁月里，却看不到明天。

也许从西藏归来时，那高中的开始，让世界开始有了几分光明。但三年前，那个七月七日的离别，却让那份光明蒙上了几分阴影。喜欢的人远了，压力和竞争近了。各色利益，各色冲突，如一池碧水，在表面的清澈之下，却是一片污浊与黑暗。

又到了七月七日。难得快乐的一年时光，难得美好的一载年华。近在眼前的期末，早已不放在心间。七月七日晴/黑夜忽然变白天/我失去知觉/看着相爱的极限/我望着地平线/天空无际无边/听不见你道别

时光的尽头，剩不下多少从容。一年到了终点，竟已无话可说。也许就只能这样了吧，做熟悉的陌生人，在该配合演出的时候演视而不见，管他是不是只能即兴表演。云不淡，风不轻，阳光似火，一影独行。什么时候开始没有了底线，什么时候开始随心所欲表演。没有观众没有剧组，顺着谎言被动就不显得可怜。反正是蹩脚的演员，反正是随心所欲的表演。静静地回味一遍过去，也许一切，本不应该是这样。

# 故事尽头

药医不死病，佛度有缘人。 吕受益，彭浩(黄毛)，刘牧师，还有刘思慧的女儿…他们是幸运的，他们有缘遇上了能给他们带来生机的程勇。但他们又是不幸的，程勇放弃了药品走私，让吕受益因没有药品而走上了绝路；警察的追捕，让彭浩丧生在车轮之下…在生命和法律之间，他们毫不犹豫地选择了生命。绝望，是最大的悲哀。

科学技术的发展，让生和死这最后一丝的公平也化为云烟。钱就是命，格列卫就是慢粒白血病人的命。许多人钱没了，命也没了。于是乎，最终而言，人只有一种病，那就是穷病。我们不能指责谁对谁错，因为没有谁是错的。程勇想救人，病人想活命，药商要维权，警方要破案。生与死，贫与富。这是一部让人笑不出来的喜剧，也是一部让人哭不出来的悲剧。

没有什么愿景，也没有什么期待。烈日的灼烧之下，一切都在黯然间褪色。第一次，可能也是最后一次了。曾经想说但没说的，就不说了吧；曾经想做但没做的，就不做了吧。把时光退回一切开始的那一天，退回那仿佛已经很远很远的从前。无所谓喜欢，无所谓情感。还没有开始的故事，就已经结束了。想想矛盾的普特关系，想想纠结于一个人有什么意义。且恢复平常心，不去作，不去自找罪受。

夏天到秋天，秋天到冬天，冬天到春天，春天又到夏天。十二个月的时光，任回忆被时光抹去。曾经三载的青春，任距离把岁月消灭。走过北京东路没有了感觉，鸡鸣寺路的樱花只剩下了记忆。西家大塘的战斗已经过去很远，碑亭巷口的日子也难得再留恋。车水马龙声一去不复返，只留得牛头冲的阵阵蝉鸣。梧桐絮落如雨的时节里，再也体会不到，蒋公与美龄的爱情。烟花易冷，春水易逝。曾以为那片叶子的爱情纯真美好，但却在意外间埋葬。一念倾心，不留痕迹。

看过城市的繁华喧嚣，也看过乡村的宁静从容。但我的心最终沉浸在，那一抹湖水之中。滚滚的乌云，遍布那金陵的空；片片的山林，吹过那郊野的风。不得安生的心灵，游荡在城东；分明不曾去拥有，却说太匆匆。道路崎岖难行，历经坎坷无穷；旅途一向艰苦，人生从未轻松。叶子该落了，心也该散了，毕竟，方向错了，注定只是徒劳无功。

最后的奔波，随即踏上前途。高温与酷暑，不必算作折磨。正如单相思腐蚀着心灵，烈日的骑行灼烧了肉体。大江东去，滚滚江水带不去点点思慕，也留不下一分儿女情长。坦然面对有过的情感，放下所有虚妄的幻念。浪漫似香榭丽舍的偶遇般的开端，终究会化作浮华过后的泡影。现实，才是对自己最好的成全。也许，我所要的，只是她，只有她，只要她。不知姓名，不知死活。

所谓的不可能的人，也许就是自己给自己设下的枷锁。你认定了，就真的不可能了。得不到的永远在骚动，被偏爱的都有恃无恐。一念倾心，既已倾注，则唯有毁灭一途。弱势的一方总是受挫，卑微的追求让人折磨。放下，则是洒脱。江湖再见，已然无缘。

也许我也应该学会沉默了。心累到啥都不想说。一切都是我的错吧。妄想过度神思恍惚吧。一个人好好的为啥要想那么多？还是安静吧，把时间留给内心吧，想想自己要啥吧。再好看的妹子不是你的，再美丽的风景不是你的，再成功的人生也不是你的。不如承认你就是个loser，在生活中失意，在现实中伤心。这样，说不定还能好受些。

# 西南风月

桂林，一座温柔的城市。山与水相依，江与湖共舞。行走间尽是装点的精致，却也仍免不了留下岁月的痕迹。也许她曾是个清秀的少女，但如今她却已渐渐老去。浓妆艳抹的脂粉气息粉饰出她曾经的雍容华美，穿金戴银的奢华装扮掩饰着她此时的饱经风霜。她，是最早的网红，有着让千万游人沉醉的山水，有着那无数赞不绝口的评价。如今的她，已经充满了世俗气。鲜艳夺目，却已不复当年纯情。

当和谐号CRH2A动车组的白光跨越黔桂边界的崇山峻岭，大山深处的黔南，便浮现在了眼前。纵使晴空万里，也会忽然下起暴风雨，但待到云开雾散，又是一片蓝天白云的风景。都柳江畔三都小城，樟江沿岸荔波古镇。山与水的交融，仁与智的交锋。仁者乐山，智者乐水，大贤无为山水。恪守山的谨严，谨守有形或无形的规则。追寻水的自由，放飞真实或虚妄的自我。大音希声，大象无形，道隐无名。

贵州的雨，一刻不停。一朵乌云就是一片雨，一缕阳光照亮一片天。海拔一千米的黔中，让这个火热的夏日瑟瑟地寒凉。云青青兮欲雨，水澹澹兮生烟。难逢的蓝天，飞湍的瀑流，还有雨，还有风。不求像风一样自由，但愿如水一般洒脱。爱了就是爱了，恨了就是恨了，放下了就是放下了，忘却了就是忘却了。心灵放空，百念既除。

彩云之南，那梦一般的地方。迷蒙的天气，幻化的心情。阳光有一阵没一阵，洒落滇池畔的昆明。水是一汪碧绿，天是一抹清新。距离愈行愈远，心灵愈走愈清。变幻莫测的天气，纷扰了心情。光怪陆离的梦境，模糊了过去。想带你去看星空万里，想大声告诉你我为你着迷。但，往后的一生，不可能是你。把别人看得太重，却把自己看得太轻。时光匆匆，却没有往事；来来往往，也没有感动。再多的说辞都是借口，再多的言语不过说说。单恋，也许是更大的快乐。

丽江，那云贵高原之上离天空咫尺之遥的地方。碧蓝的天空与碧蓝的幻想，投影在那古城的斑驳里。一座山，一条河，石板路的青苔与水车。酒吧的夜夜笙歌，茶社的清静无为。去一棵树下，感受时间的流逝；去一座山上，体会古韵的春秋。雪山不远，藏区也不远。云贵高原的小城，却有着几分雪域高原的味道，也许，这便是这里浪漫风韵的缘由吧。七夕过了，又是一年散发着单身狗清香气息的时光。放下幻想，不怀期望。

苍山，洱海。一个在云端，一个是深蓝。爱雾绕的山，恋梦幻的水，却不喜欢，那山海交际的城。古韵的记忆消失在梦里，现实的山水消散在人群。只剩，三塔，佛门，还有那一条流过古城的溪。并非每一种不知所起的情，都值得一往而深。喜欢她，念着她，纵不再相遇，亦不负深情。

# 花开半夏

本以为，最为迷茫的时节已经过去。放下曾经怀有过的幻想，就得以翻开新的篇章。但现实却是，在无尽的时光里奔忙。没有向往，更没有了前方。盘旋在山道上，心无所伤。敬亭宛溪，夏渡向阳。找不回昔日骑行的欢畅，抛弃了曾经放纵的疆场。感谢在这个月里关心我的所有人，让我在最软弱的时刻里选择坚强。纵浑身是伤，仍不想绝望。

天色已然迟暮，太阳收敛了光芒。深邃幽蓝的空中，繁星特别明亮。还有那么一片温暖的，是白月光。流连在群星璀璨里，却再也找不到心之所向。也不知，那迷惑了视野的，是那月的清朗，还是那星的微茫。抑或着，还在回想，那日未落的时光。

只好选择，在天台孤独地守望。念念不忘的，也许终究会有回响吧。

生活，就此走向新的起点。故事，没有过去也没有将来。无疾而终的念，如奔流的滔滔江水，一去，便再不复回。再莫谈及那些过往，再莫回想曾经故事。所有人都是好人，除我以外。我不是坏人，但更不是好人。

星空的璀璨，在于它的变幻。人生如流星，在宇宙间走走停停，被每一颗星影响，也影响每一颗星。一许流星划破天空，带着耀眼的光芒，让人沉醉，令人倾心。但当它离去时，除去无可奈何，也并无它想。直到一天，你发现一颗与你相互吸引的星，互相围绕，牢牢相依。这，才是所追寻的幸福。

世界不是一个人的世界。你的好心对别人未必是好意，一味的想做好人只会被欺。要遵从自己的内心，也要对得起自己的良心。做好事前先多考虑一下自己，做坏事前先多考虑一下别人。好报和恶报这种东西，都挺虚无缥缈的，谁也不知道它会应验在哪里。不用去想好心有没有好报，恶意有没有恶报。对得起自己的良心就好。每个人的追求不尽相同，有的人追名，有的人逐利；有的人求道，有的人修仙。还有的人求子孙安乐，还有的人求一世幸福。想法不同，追求不同，行为自然也不尽相同。笼统的定义好人和坏人，并不靠谱。在我看来，好人，是他所做好事的影响比坏事产生的影响大得多的人，坏人亦然。我们既不能算是好人，也算不上坏人，我们都只是凡人。

再回宣州，再逢，宣州雨。初到宣州时的点点忧伤已经散尽，而初到宣州时的点点迷茫，却愈发深沉。仍能想起宣州赋的慷慨激昂，但也会忆起消愁的自由与远方。其实，都是假象。离开宣州的日子里，思念宣州，思念宣州雨落，一片凉凉夜色。岁月静好的日子里，把心彻底埋葬。你若有光，便是远方。念念不忘，必有回响。但，这世上，没有什么值得念念不忘，也没有什么值得回响。光，本就在远方。看到了，也就足够了。

当再一次越过宛陵湖来到宣州的城中，时间仿佛回到曾经的初秋。军训刚刚结束时青涩的自己，还不知道什么是春夏什么是秋冬。憧憬与冲动早已过去，平淡的日子需要刺激。做回曾经自己的自己，不要为了他人而掩饰内心。坦然放弃过去，才能迎接未来。

当一切回归了最开始的模样，我不认识你，你不认识我。这是最好的开始，也是最好的结局。进村第三百六十五天，做完能做的事，然后且听天命。见完要见的人，然后径自离去。不得真假，不做挣扎。放下那水群的岁月，放下那哗众取宠的日子，放下那不该再念的人。春风十里，那曾经想念过的人，也许不知不觉，就见过了最后一面。

犹记得上一个9月12日，那是开学典礼的日子。领导们在台上说，我们在台下听。新的生活正式地由此开始，旧的故事渐渐地画上句号。结束的结束，开始的开始。祝福过我的人，我为你祝福。

又一个9月12日，平淡，而又幸运。感谢室友和刘总，感谢两位小姐姐。感谢经济一班，伴随我度过了迷茫；感谢电信二班，往后的日子还需同行。成年了，一切回到最初的起点。过去，告别，道一声再见；未来，您好，请多多关照。

也许一切，就这么结束了吧。

# 烟花易冷

每一段故事最后，总有那需要感谢的人。

感谢你们的陪伴，也感谢你们教我成长。


唯美的爱情只存在于梦想

那些歌颂的美好都是假象

且听风铃儿叮叮当当作响

且看那湖中天鹅自由飞翔

却不知心之所往只在何方

空剩得叶落铺满一地金黄


看路上旅人行色匆匆奔忙

作锦绣文章书写纸短情长

所有的辛勤付出也许泡汤

多少通宵未必会派上用场

清晨的露水不化凝结成霜

纵越过龙门咸鱼又能怎样


不辨方向在大雾之中迷茫

但枪中子弹不知何时出膛

生活给予的并不只是悲伤

有时候， 它更会让人绝望

# 友达以上

不要为了失去一个朋友而哭泣

也不要为了努力的白费而惋惜

你要相信，上天总会公平待你

即使你的付出没得到任何收益

即使你的善意被别人视作可欺

你要相信，有人总会看在眼里

坦然面对人生旅途不断的树敌

成功路上必然会侵犯他人私利

你要相信，问心无愧即是正义

心之所向，素履以往

路在脚下，梦在前方

# 人事纷纷

大梦初醒，已忘却了那梦的故事。

一念初平，已忘却了那念的初心。

无所倾心，无所挂记。小心举止，谨言慎行。

既然你们不曾待见我，那我便决然离去。

无论路上多苦，无论前程多艰。

曾以为拥有的一切，其实都是过眼的云烟。

被美色冲昏的头脑逐渐恢复冷静，

对班级事务的热情也已化为灰烬。

关心我在乎的人，努力做好自己。

不考虑所谓的力所能及，不去随意的施加好心。

反正，不在乎你的人，不会把这些去记。

# 十里宣州 (Cover: 烟把儿乐队 纸短情长)

那是九月的宣州

不遥远的城郊

抹不去的忧愁

陪伴着我的左右

记起你的歌喉

想起那些微笑

飘然物外神游

却是难回头

那是一片美丽

在牛头冲大地

舍不得抛下的

却终要面对的

春风十里啊

不如你倩影依稀

也许错过就是永远的呀

伤已经结起了痂

就不要再看那疤

忘却了她的音容她的笑貌又怎样

春风十里啊
