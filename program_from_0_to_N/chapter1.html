# 从编程小白到生产力带师——（1）数据，程序处理的内容本质

date: 2021-10-25

author: 西风镭

location: 深圳，华为软件中心

# 数据是什么

数字世界的一切都是由数据组合而成的。程序处理的对象是数据，返回的结果是数据，其中产生的每一个中间环节也是数据。数据就像现实世界中的各种粒子一样，无处不在。当然，就像自然界中的粒子最小单位是夸克一样，数据也有着最小单位——比特。一个比特是计算机处理数据的最小单位，仅能表示0或1。多种语言中常见的数据类型布尔型(Bool)即为1比特。8个比特组成1个字节，除布尔型外的所有基本的数据类型均由若干个字节组成。普遍而言，各语言的基本类型如下：

说明：此处仅列举我接触过的几种常见语言，包括C、C++、Java、Python、Go、Swift、JavaScript。

基本数据类型 字节数 适用语言 说明 相关扩展
整型(int) 一般为4 C、C++、Java、Python、Go、Swift、JavaScript(BigInt可以用任意精度表示整数) 最常见的数据类型，表示整数 前缀unsigned/U表示无符号类型，Go和Swift有不同位数的Int8(char)、Int16(short)、Int32(long，常见的普通int)、Int64(long long)用法
浮点型(float) 4 C、C++、Java、Py、Go、Swift、JavaScript(Number默认为64字节浮点数) 最常见的数据类型，表示浮点数 双精度浮点数占8字节，为double，适用C、C++、Java、Go、Swift
字符型(char) 1 C、C++、Java(char为2字节、byte为1字节)、Go(byte)、Swift(Character) 最常见的数据类型，表示字符
布尔型(bool) 1 or 4 C++、Java(Boolean)、Py、Swift(Character) 判断类型，仅有true、false
无类型(void) 0 C、C++、Java(无法操作)、JavaScript(null、undefined) 空类型
指针型(*) 4 C、C++、Go(uintptr) 变量地址 针对内存操作，可以指向数、数组或者函数
复数型(complex) 16 go、Python 复数a+bj go有complex64和complex128表示实部和虚部均为32位浮点型或64位浮点型
字符串型 不定 Python、Swift、JavaScript 字符串 相当于一组字符

然而在真实的编程场景中，我们遇到的数据类型远不止这么多，但所有扩展的数据类型都来自以上的基本数据类型。比如数组、C、C++、Java中的枚举、C、C++中的联合、C、C++、Go中的结构体、面向对象语言中的类。

  • 数组

数组是一组连续多个数据的组合，内存地址相连。在Python等面向对象的语言中，针对不同的应用场景，有列表、元组、集合、字典等更多用法。
