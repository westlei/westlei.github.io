# 烟花易冷

每一段故事最后，总有那需要感谢的人。

感谢你们的陪伴，也感谢你们教我成长。


唯美的爱情只存在于梦想

那些歌颂的美好都是假象

且听风铃儿叮叮当当作响

且看那湖中天鹅自由飞翔

却不知心之所往只在何方

空剩得叶落铺满一地金黄


看路上旅人行色匆匆奔忙

作锦绣文章书写纸短情长

所有的辛勤付出也许泡汤

多少通宵未必会派上用场

清晨的露水不化凝结成霜

纵越过龙门咸鱼又能怎样


不辨方向在大雾之中迷茫

但枪中子弹不知何时出膛

生活给予的并不只是悲伤

有时候， 它更会让人绝望

# 临江仙·冬雪

雾色宛陵深似海，纷纷细雨独哀。伤情只在心中埋。

欲见湖光好，且待春日来。

寒风吹彻宣州外，前路艰险难猜。谁道未来几多灾？

抬眼空望水，举步独登台。

# 初心

不要假装自己努力，因为结果不会陪你演戏。

不要试图忘记过去，因为念旧才能勇于前进。

纵使伪装大佬一名，可是成绩依旧会出卖你。

纵使否认失败过去，可是记忆终究难忘彻底。

抛弃学习拥抱吃鸡，到了考前求生希望难觅。

抛弃往事拥抱当今，到了午夜梦回心伤无尽。

那年花开月圆金陵，倾尽思慕迷茫不解前途。

如今花谢月落宣州，无关风月时光梦幻往昔。

三月春来遍野花开，时光悠然轻言思慕无再。

石城匆归所爱无回，心无片叶只愿初心犹在。

# 守望

感觉生活很没有意义，感觉世界充斥着滑稽。

世事艰难前路无光亮，夜空是那靛青的微茫。

昏暗的路上惨白的光，不知何处照耀着前方。

星河无影惟天地苍凉，夜风吹去是四海八荒。

没有不破的爱情幻想，不如在夜里随风流浪。

孤行向南到湖边怅惘，声色光影照水面金黄。

栈桥春影心不曾荡漾，落雨终归入浩浩大江。

夜幕之下忽心羡叶杨，再多曾经亦当忘则忘。

不曾有过的年少轻狂，理不清楚的满腹忧伤。

世上有多少山高水长，何必把过去念念不忘。

几多故事里几多怨望，几多伤情时几多迷茫。

有故事的人已成班长，抛下过去与幸福守望。

摸不清何处心之所往，岁月静好再扬帆起航。

# 友达

不要为了失去一个朋友而哭泣

也不要为了努力的白费而惋惜

你要相信，上天总会公平待你

即使你的付出没得到任何收益

即使你的善意被别人视作可欺

你要相信，有人总会看在眼里

坦然面对人生旅途不断的树敌

成功路上必然会侵犯他人私利

你要相信，问心无愧即是正义

心之所向，素履以往

路在脚下，梦在前方

# 人事纷纷

大梦初醒，已忘却了那梦的故事。

一念初平，已忘却了那念的初心。

无所倾心，无所挂记。小心举止，谨言慎行。

既然你们不曾待见我，那我便决然离去。

无论路上多苦，无论前程多艰。

曾以为拥有的一切，其实都是过眼的云烟。

被美色冲昏的头脑逐渐恢复冷静，

对班级事务的热情也已化为灰烬。

关心我在乎的人，努力做好自己。

不考虑所谓的力所能及，不去随意的施加好心。

反正，不在乎你的人，不会把这些去记。
