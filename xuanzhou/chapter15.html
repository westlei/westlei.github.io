# 仲夏星空

# 虚无之梦

小雪时节放起的寒假，在处暑时节才终会落下帷幕。清明雨上兴起的幻想，在夏至未至时便无影也无踪。当这个世界被改变的时候，每个人都不过只是时代背影下渺小的一粒沙。无力去抗拒，也阻挡不了历史的车轮滚滚向前。就像美国的五百万感染远在天边，但大盘的绿色近在眼前。

考研更多的也只是一个梦了吧，抑或是一个幌子，一个摸鱼的借口。我以为我卸了游戏就能抵制了诱惑，但我发现我想的太多，总有东西能够吸引着我。不论是无知的幻想还是确切的迷茫，都无法把我空空如也的脑海点亮，不敢去想，以后怎样。

当我看到巨佬们靠着刷SSR在一个暑假里挣上了高达五位数的财富，当我听说神佬把两年半的创业项目卖给蚂蚁金服转手就是一个七位数。我看看蓝蓝的单元格里黑黑的名字陷入了苦涩。执着于太过无所谓的东西，就会像蓝桥杯的校赛一样，只有一个人的赛道，不过是比了个寂寞。

有的大佬逐渐走出了考研的赛道，在多彩的世界里散发光芒与热量。乌龟在考研的赛道上缓慢地爬行，看着后来的兔子一蹦一跳地从身前跑过。它想要退出这条赛道，却又怕错过了兔子睡着。它小心翼翼地打算抄小道，绕了半天却又回到原来的路上。看着睡眼惺忪的兔子揉了揉眼重新奔跑，失望的乌龟在树下睡起了大觉。

梦里，1月8号我依然顺利拿到了驾照

梦里，1月11号我依然会偶遇，定下buaa的梦想

梦里，2月我会回到学校跟队友一起好好肝美赛

梦里，蓝桥杯不会尴尬地只有我一个Python

梦里，我会认真完成大创并且参与创青春

梦里，千岛湖的四月景色，碧波荡漾无比绚丽

梦里，黄佬的项目我也会尽心尽力，做出东西

梦里，我考了CSP，考出了一个不错的成绩

梦里，我的成绩不会像现在这样飘逸，但更平和

梦里，我的智能小车上了赛道，随着音乐而舞蹈

梦里。。。

梦醒后，一切都只是泡影，只听见要答辩的消息。随便投两份简历，剩下的话怎么样全看天意。要是能够拿到offer，其实不考研好像也可以。毕竟，人间不值得。

好想爱这个世界啊。但，再没有一个人值得我去挂念去记忆，再遥远的未来也都只有一个我自己。

# 缘起一面

今天的南京又是一个暴雨倾盆的日子。漫天的水幕遮天盖地，让我的摸鱼划水竟有一丝丝心安理得。每天重复着枯燥无味的刷题阅读背单词，进度却慢到近乎于停滞。没有想好怎么去过现在，更无法想好怎么去过好未来。

一不小心大学生涯就已经来到了最后一年，可能不久的将来就会去往不确定的城市，过上不确定的生活。再多喜欢的潜藏终究会输给时光，所谓奋斗不过是聊以自慰的念想。考研更像是逃避的借口，而不是逃避的过程本身。

有的时候，我们缺少的并不是能力，也不是那一点点运气。差的只不过是几分信心，几分坚持下去的信念。离开牛头冲的半年，或许是二十年来最长久的孤独。朋友和对手在时间和空间的隔绝中消失在生命里，我仿佛变成了一个世界的旁观者。不再参与生活，也无从感受世界。自然，未来也不再是实实在在的压力，而是飘在云里的东西。我的世界里，只有过去，没有未来。

然而，过去的东西也随着时间的流逝化作了清风，消散在云里。没有爱也没有恨，那世界仿佛与我没有了多深的联系。认识的人很多，但他们都有着自己的事情，自己的朋友，自己的忙碌。而我，则像个没有感情的摸鱼机器。仿佛在做些什么，却什么也没有做。漫长的列表不会有人愿意陪我闲聊，终究只得水群来虚耗漫长的时光。

水群的时候谈到了恋爱和婚姻。恋爱是感情，婚姻是交易。感情可以经历风风雨雨，但交易容不得太多的意气用事。大家庭的婚姻，动辄累桌成席，浩瀚的排场撑起无数人脉和关系。但家族长辈无数双眼睛带来的负担，却也总让年轻人难以承受。可小家庭也有小家庭的悲伤，或许婚宴的礼堂，男方的嘉宾连一桌都凑不上。每一个夜里，看着别人合家欢聚，心中总有隐隐的伤。 异地恋终究是一个过于沉重的词，沉重到我不相信我能够承受的起它的重量。终究还是太晚，晚到还没有开始，就可能已经没有了未来。读研在远方，工作在远方，生活在远方。但喜欢的人，她会不会在远方？

“因为我觉得这个世界上不可能会有女孩子喜欢我。既然不喜欢我的话，也就不会有人和我结婚啊。还有我认为我不可能会碰到我喜欢的人，就算碰到了，以后她也会变成我不喜欢的人。我不可能跟我不喜欢的人结婚啊。”

男主用很笨的方法一次又一次试图博得女孩的关注，终究还是将对方打动。可现实终究不是故事，我也没有男主的光环。这个世界上终究没有哪个女孩子会喜欢真正的我，顶多被某一部分的表象所迷惑。如此，纵使我遇见了喜欢的人，那也无济于事。 于是也明白了我和女生为什么只能做成兄弟或闺蜜。若是我不喜欢的人，即使再亲近也只能是闺蜜或兄弟的亲近，总要刻意保持相当的距离。若是我喜欢的人，聊的越多就越珍惜当下的感情，不忍破坏，终究也就成为了兄弟或闺蜜。这，或许就是我避不开的魔咒。

希望能有一天，当我看见那个她时，我就知道，她会是我一辈子爱的那个人。可惜，我喜欢的人注定不喜欢我，就像我想要的天注定抽不到。

# 黑白之间

当生命在黑白和彩色间交替循环的时候，谁也看不出它真实的色彩。当深夜的泪水在枕间徘徊的时候，谁也看不出白天笑着的无奈。当一个人的内心遭受抑郁的破坏，没有一片雪花能够看得出来。

感情从来都只是借口，背后是内心撕扯的破洞。在一念倾心的时候，无数的温暖通过破洞流向了对方，像盐倾倒在大海里的模样。终究只收获了疲惫和悲伤，变成空洞而冰冷的模样。

依旧一事无成的一天过去了，看着小姐姐们的照片满是艳羡的模样。坚持若有若无的念想不知是对是错，但或许报了名走上考场的那一刻我就已经不是完全的卢瑟。可上岸，不过是一种奢望。

听说，七夕又快要来临了。

其实本没有那么多值得表达的情绪，但一个人太久了总是会有些伤感吧。有人劝我去找个女朋友，也有人劝我别去。环顾大学生涯，似乎也并没有一个时机，是适合去认认真真去参与那么一场爱情故事的。活得太赶，过得太忙，却也无所成就。

仍然会像当年一样喊“小雷”的人越来越少，可本质上我依然是两年前卑微的小雷，时光并没有带来多少的改变。张华考上了北京大学，李萍进了中等技术学校，我在百货公司当售货员，我们都有光明的前途。但不管是“我”还是李萍，亦或是张华， 他们可能都没有看到所谓的光明，就在黑暗之中迷失了自我，消失在Utopia的幻想。

以2015年的视角看2020年的世界，一切都是那么的玄幻而不可思议，或许就像2020的我们看2025一样。未来，永远都不是哪一个人能掌握的东西。即使你会时间管理，也比不上操纵时间的人。决赛圈的人越来越少了，第95个年头即将开始了。

# 莼鲈之思

What is homesick?

在浩如烟海的成语中，有一个很好听的词，叫做莼鲈之思。西晋文学家张翰因思念故乡的莼菜汤、鲈鱼脍，一气之下辞官归隐，留下这样一个典故，反映着那时人的乡愁。

中国是传统的农业大国，乡土、乡亲、乡音、乡情一直是刻在中华民族骨子里的一部分。被乡土人情束缚的中国人，即使漂泊在外，百年之后仍要认祖归宗。一方水土养一方人，一方人总是眷恋那一方的水土。

但是，科技的进步，伴随着传统农村社会的裂解，却让这传统的乡土文明逐渐失去了下落。乡愁可能是小时候去玩的一片水塘，可能是小时候常走的一条街道。可在繁华的城市里，一切都变得太快，也许未必过去很久，你就再也找不到儿时记忆留下的痕迹了。

于是终究，乡愁还是得归结于人。大院里相互熟识的左邻右舍，幼时一起长大的青梅竹马，来往频繁的亲戚朋友。即使一切都已经面目全非，只要有他们在的地方，那里就依然是乡愁的落足之处。

可是，城市的高楼大厦里，淡漠的人来人往中，你真的知道你的邻居是谁吗？所谓的青梅竹马，在一次又一次的动迁之中，你还能再次从百万人口的大都市里，找到他们的联系吗？远渡重洋远赴他方的亲戚朋友，终究让空间的距离变成了人和人的距离，再次见面，已经不知是何许年。

当家庭缩小到原子的级别，乡愁失去了他最后的一层依赖，在现代生活的摧残里轰然崩塌，留下了一个个生活在困扰之中的个体人。他们在繁忙的大城市里活着，有着许许多多表面上的朋友。看似潇洒的背后，是更深一重的孤独。

失去了乡土的成长，失去了乡亲的守护，没有乡音的寄托，也无所谓乡情的陪伴。走过了许许多多的地方，并没有一个真正的希冀之中的容身之处。不像南方人，亦不似北方人，少了些东部的精明，失了些西部的洒脱。没有中原大地的豪爽，也没有江南水乡的温柔。也许，我只是我，不属于任何地方的我。

未来，终究会去向何方？上一次的选择交给了命运，命运却给了我一个啼笑皆非的答案。这一次的选择或许不能再去逃避，可我依然不知道去向哪里。就像面对考研院校选择的慌张，就像面对企业意向城市的迷茫。看似总在寻找一个灵魂的伴侣，但其实真正想要的不过是灵魂的归宿。

七夕快要到了，希望每一个19级的同学都有自己光明的前途，也希望其他的人能找到属于自己的宿命，不管是梦想，还是良人。梦宿命的时光，剪余生共惆怅。

# 生而为谁

生而为知府，我很抱歉。

白嫖党注定当不了一个好官，只能选择在狗官中当一个好一点的狗官。可以给城市种上些树，使城市不受污染的折磨。给街道种上些花，让城市看起来愈发的繁荣。人们有公园可以去，有塔可以爬。

应天府里埋葬了我十几年的记忆，而苏州城则安放着我关于水乡和人的梦想。江南有百景，可最美的风景，依然是人。是那寻觅而不可得的，温婉的江南美人。

各种各样的尝试仍在继续，那个冬天里昙花一现的初心动摇着到了如今。命中注定的东西总是难以去改变的，就像有的人总要为相思病而死。但不管怎样的意外，20补天石的简陋坟墓都是他们最终的归宿。十抽换不来心仪的人物，就像隐晦的真心换不来伊人的回眸。

被动的处女座总是不喜欢主动的出击，所有的行为都一定有客观存在的目的。越是心态濒临崩溃越是看起来愈发努力，可一切完全崩溃反而又是豁达的随性。一切也许都应该像我想的那样，成功便是成功，失败便是失败，毫无意外。可是，我永远寻找不到的，是我的世界里，每一个匆匆离去的你。

再见，应天；再见，苏州；再见，江南。

# 友情尽头

友谊是有保质期的。

稳定的生活环境会使得友谊的保质期得到延长，而一直在路上的人却很难得到长久的友谊。或是因为保质期到头而变成故交，或是因为不可抗的力量而发霉变质。再或者，因着莫名其妙的原因直接消失，也并非不可能。

许久不联系的故交相遇，就像是他乡遇故知的兴奋，可兴奋和寒暄之后往往会变成无话可说的沉默。不一样的生活让对话失去了土壤，空洞的寒暄掩饰不住尴尬的气氛。很难寻觅所谓的共同话题，也很难去走近那一个你已经远去的世界里。

友谊也有近有远。疏远的友谊，保质期会随着距离的疏远而愈发短暂。可亲近的友谊，也同样可能会因为感情细菌的发酵而发霉变质。也许曾经无话不说的友人，只因为一些小事，甚至是无关于本人的外在原因，使得一段友谊上起了霉菌，而至于舍弃。友谊本身并不能意识到霉菌的侵染，可霉菌终究会成为友谊的障碍，不如趁早将之舍弃便是。

过于亲近的友谊，总是夹杂了太多的感情。我是一个感情用事的人，复杂的感情有如无数的菌落，谁也不知道是益菌占多还是害菌占多。友谊的尽头，可能是清冽的白酒，可能是醇厚的酸奶。可能是飘香的酱油，可能是酸味的陈醋。但更多的时候，它只变成了过度发酵而无用的糟粕。现实就像氧气，源源不断将有害的菌落散入，让友谊一次次走到尽头。

沉淀和过滤可以让有害的菌落远去，但有时候也会让所有的一切烟消云散。这时，只有对方的坚持，才能把友谊继续维持下去，终究酿成一壶老酒，愈久远，愈香醇。但这也注定了我，大多数时候都只是一个所谓的薄情寡义之人吧。

也许我这样的人，并不配拥有真正的朋友吧。当然，既然连友谊都稀疏，那就更不用知道，什么是爱情的模样了。斯人若彩虹，遇上方知有。可纵使遇上，也只是转瞬即逝的光芒。

可她会是记忆里永远的霞彩，永远的一抹白月光。

# 四六级回忆

上一次考四级好像还是没多久之前的事情，打开日历一看，也就1007天而已。还记得那时候还没有万达，跟辩论队的伙伴们在国购吃了饭，坐上破破烂烂的9路柴油前置非空调破车，去新港国际体验所谓的皖南明珠摩天轮。天很冷，也没有人，玻璃桥很粗糙，风景也一般。在车上对了答案，拿百词斩估分大概500多点。那时候简单纯粹，默默喜欢一个人，谁也不知道。辩论队的伙伴们都很好，可后来渐渐就越走越远。很想回到当时的日子，不过终究，回不去了。

825天前考了第一次六级，心态却已经发生了变化。和曾经的兄弟干了记不得多少的白酒，却不知这是最后一次，互相当对方是兄弟。后来的日子，逐渐起了误会，误会越来越深。我走了，可记下的仇不会走，任时光把一切酝酿。也许，再后来的一切悲剧都本可以避免，如果那个晚上，喝多的我没有水群的话。

当643天前我再一次踏上六级的考场，被补课折磨的我已经跳下了学业的高崖。连绵不休的考试使我的身心都已经疲于奔命，只记得那时说走就走的一场旅行。是上海滩头城隍庙里的大雪，甪直古镇的清丽；还是那十里山塘，护城河畔炫彩的光与影。当然，还有那雪夜里的一支歌，谁能初心不负。为了梦想踏上了不能回头的旅途，祈愿，幸福。

待到一切都已经崩溃后，考六级已经变成了游戏。为了体验口语的魅力，于是便来到了科大赏雨。第三次到合肥的我，早已不再陌生于路况。从三里庵漫步到罍街，在小雨淅淅中看合肥的城景。彼时的我尚未成为所谓的佬，也不认识本部的朋友，不过是平凡的自己，在平凡的岁月里。

于是乎，461天的最后一战也失去了他本来的意义，没有过的现实，让我反思我到底做了些什么。约上经济的老朋友们，回到了熟悉的南京城，是与过去的遇见，还是与未来的道别。毕竟，后来的我们，终究不能再像曾经般亲近。渐渐的，永远的515也终究让渡给未来的生活。

一转眼，就来到了大四。生活只剩下乏味和压力，有时候回忆曾经，不禁去羡慕过去的自己。在这过去的三年，我得到了什么，却又失去了什么？似乎，得到的不过是一些浮于网络的虚名，却失去了一些本来最值得珍惜的，友谊和回忆。

# 苏香·工大

庚子年八月初八18:08，主教的大门缓缓打开，蓝色大巴闪耀着大灯开进了南一环的滚滚车流，没入了合肥的黄昏中。十分钟后，它离开合肥城的繁华，拥抱肥东县的原野，从高速里横跨皖中南，飞跃长江，最终于深夜开进皖南小城荒凉的牛头冲里。

时隔344天的重逢，仿佛是时隔半个世纪的聚首，又似乎有那么些劫后余生的幸运。寒冬里骤然而至的疫情与隔离，网课和禁闭，终于捱到了解封的日子。

可此时，曾经年轻的我们都已经多了一轮岁月的洗礼。生活里不再只是当下的岁月，更有着未来的隐忧。但是，我们依然年轻，依然朝气蓬勃。

同一个江苏，同一个工大。We are family， we are always family。愿每个人都能有着光明的未来，愿归来时，都还是少年。

从合肥回来的第二天，世界显得淡然和平静。生活与之前的每一天并没有多少不同，依然是北区、实验室和南区三点一线的日子。但是似乎又有什么不同，或许不同的便是心态吧。未来自有他的运气，不必太过苛求于努力。

忽然意识到大四了才第一次去看军训汇演，看着那些酷似曾经的我们，却不感到任何的亲切。他们是他们，我们是我们，前浪有前浪的运气，后浪有后浪的幸福，只有韭菜一无所有。作为四年级的老大哥，我只能说，很有精神。

终究没有走上过程序设计竞赛的路，也终究只能在他面前败走无救。绞尽脑汁不过是一半的用例通过，知道怎么做和战胜tle之间依然是横跨宇宙的距离。打表和暴力测试也无法让A过的题更多，只能让罚时流失的更加快速。最终还是只能用上rand的魔法，用随机战胜随机。

保研的名单不出意料的公示了，并没有任何意外的气息。高达89.1的保研均分高举着内卷的大旗，隔壁专业85出头的保研均分里氤氲着香气。加分失败的朋友终究失去了一切，本来就没有过希望的我反而也没有失望的痛苦。就如同我现在的失败，一切本来就都是理所应当的。

今年的月饼没有一丝一毫的惊喜，甚至比去年的月饼更加无法吸引我的注意。虽然是最后一年，但也失去了收藏的意趣。就像工大本身，值得记忆，值得留恋，值得珍惜，却不值得收藏。

下面是著名哲学家汪直久的一段话：

他们是他们，我们是我们，你是你。谁都难以走入谁的世界，谁都难以分享谁的痛苦。但是，我们依然站在这里，透过玻璃展窗，我们会因你的快乐而笑，也会因你的悲伤而叹，我们无法分担情绪，但是能够感受，即使只是几万分之一。希望你也能听见，玻璃外的我们，发自内心为你祝福，即使只是几万分之一。

很多洒脱的人，当你靠近他、了解他之后，发现他其实也是个凡人，也有他的爱恨情仇和一地鸡毛，只是从不向他人诉说，自己承受痛苦、背负迷茫、消化情绪这些罢了。叔本华说:“要么孤独，要么庸俗”；而在你看来，是“要么痛苦，要么庸俗”。 很可惜，我曾经试图一个人这样，但痛苦发酵了之后只会更痛苦，那不如走向庸俗，随性的庸俗。
