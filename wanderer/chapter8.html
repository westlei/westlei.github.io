# 无数个寻常的奇迹

# “高”端社畜

一晃日历只剩最后两页纸了。

成为了鸿蒙next+iOS双持用户，两个手机两个平板，两个苹果两个鸿蒙，家里是一点安卓生态都没有了。讲道理鸿蒙系统手机端没有想象中的难用，基本功能基本上都有。

24年的体检也算平稳度过，各项指标没有变好但也没怎么变坏，维持在一个马马虎虎的状态。医生说不戒掉小甜水大概好不了，但我觉得让我戒掉吃饭都比戒掉小甜水容易。唯一值得庆贺的是身高又涨了一公分，现在我是身高183的“高”端社畜了。

keep在微信朋友圈和公众号页面放了一堆广告，脑子一热就下单买了一套灰原哀徽章。keep支持直接同步华为运动健康的数据倒是挺省事，跑完直接短信告诉我徽章发货。但是真到手了也是基本上看一眼就直接扔进二次元储物袋。我大概不是一个非常合格的二刺猿，毕竟我好像只是花点钱，而并不像真正的二刺猿追这个追那个。

听说这周末的紫金山里塞满了赏秋的游客，可现在南京连叶子都没有黄。从丹阳丹北慢慢溜到常州孟河，耳机里放着《一首想不通的古风》，似乎这首歌也已经有不少年头了。但你要问我这到底是什么时候的歌，我又想不出来。就好像之前看某个v聊到锦鲤抄已经是十一年的歌了，突然意识到时间是一把杀人不眨眼的利剑。曾经只是作为一个小众圈子的古风歌手，如今也大多踏入了准娱乐圈的生活。曾经爱听古风的小孩们，如今也已经是即将迈入而立之年的中坚社畜了。

如果算起来，我听古风应该也快差不多十年了。手机缓存的1000多首歌里，起码有个七成是古风和古风的翻唱。I人大多应该是喜欢这个风格的，当然E人不一样。东莞的同事来南京出差，另一个同事带我们去夫子庙老门东瞎转，最后走进了一家live house。几个小时的时间里，基本上就是民谣和摇滚在轮转，并且越往深夜越是激昂。我这辈子估计都欣赏不来摇滚，这玩意跟我的性格差的太远了。

天气渐渐变冷，冬天又要来了。

# 闪光足迹

截止上周为止，我总算完成了整条沪宁线五座城市的100%探索度打卡。这件事本身其实没什么意义，对于开车的人来说甚至也没什么难度，但是出于一种爱好，使用完全的公共交通去探索每座城市的犄角旮旯，大约也有一种独特的趣味在。

很久以前的高德地图似乎只有点亮城市的功能，全国371个城市，我在过去的1900多天点了116个。坐上一趟长途高铁就能点上不少地方，所以东南版图在很短的时间内就差不多点完了。随着我长途出行越来越少，点亮的城市数目也很长时间没有增长了。目前来说排除直辖市，完全点亮的省份只有浙江和福建，安徽和江苏则各有一个城市没点亮，而遥远的祖国边境省份则几乎一个城市都没有点亮。

相比于刷城市探索度而言，2020年上线的角落探索度功能更符合疫情时代的特色，也符合一个没什么钱也没什么时间的穷鬼在市内瞎转悠的需求。我点亮的第一个角落是2020年1月20日打卡深圳南山街道，第一次完成100%探索是2021年10月7日打卡深圳坑梓街道。在深圳完成100%打卡相对而言是一件比较容易的事情，因为深圳行政范围里没有事实上的农村，最算得上农村的大鹏南澳作为一个整体打卡点直接占了7.85%的城市面积。所以对于深圳而言，沿着主要交通线去打卡还是相当方便的，不像别的城市多多少少有一些犄角旮旯的地方相当的难以打卡。

第二个完成100%探索的城市则是上海。从2021.4.9点亮安亭镇开始，到2023.2.26点亮横沙乡，总共是花费了两年多的时间。上海是一个充斥着各种犄角旮旯的地方，最后的十几个角落每一个几乎都得单独刷探索度来完成。在金泽古镇老街口坐上上海最长的1元公交，单程32公里的金泽1路，到上海最西端的陈新支路南新渡口站，走一点路就能看到横跨省界的东垞港桥，下桥就到了周庄古镇东门。从崇明南门坐上去往上海地理最北端新村乡的南红专线，在新村乡最东边的新卫站下车后步行两公里，跨越仍未拆完的省界隔离墙，到达位于江苏南通海门区海永镇的永临汽渡坐船过江去海门城区。还有几乎没有任何公共交通的崇明东滩、完全与大陆隔绝公交都需要轮渡上岛的横沙岛。这是最原始而荒凉的上海边缘，但同样是上海。

过去的一年里，我陆续完成了南京、澳门、香港、苏州、无锡、常州、镇江7个城市的100%探索。完成南京100%探索的时间是去年的11.11，最后完成的打卡点是高淳的桠溪、东坝和溧水晶桥。溧水晶桥在无想山的东南，既没有产业也没有旅游，甚至连主要的交通线也不经过这里。如果排除打卡目的，我会去这个地方的概率可能几近于0。澳门和香港则因为过度的小而很方便走遍全境，但要达成全境探索还是得去一些游客很少会去的地方，比如黑沙沙滩、海怡半岛、西贡将军澳，甚至入离境走的都是港珠澳大桥、深圳湾口岸这种相对不常走的线路。

苏锡常镇则完全是一个点一个点的慢慢攻关的结果，从张家港站取道常阴沙去通沙汽渡，然后在南通坐大巴过苏通大桥回苏州来完成常熟碧溪街道的打卡。从宜兴太华镇翻越阳羡山间的茶场到达溧阳南山竹海，然后分别去刷社渚和埭头上黄的探索度。无锡最后完成的是城区最偏僻而令人遗忘的角落东北塘，以及交通极度闭塞但很有名的阳山。镇江最后完成的是丹阳最北边的丹北界牌和最南边的珥陵导墅，从界牌镇上走了六公里到常州西北的孟城，从皇塘坐挤到吊门的常金巴士88路进常州城。有意义吗？没有，但这不重要。

# V圈末日

这两天身体不适，各种各样的药都用了一遍，抗细菌，抗过敏，抗感冒，我也不知道抗什么。有一点用，但并不完全有用，眼睛依然痒依然红，鼻涕也依然时不时的流。于是我选择了打开空调，温度25度，一会儿房间热了起来，感觉似乎啥毛病都好了。其实没什么大不了的问题，无非也就是寒气入体，寒气驱散了，也就快好了。

vtuber们在这个秋天死的有点多，也有点快。不能说猝不及防，只能说倒也正常。贴吧里的投资从业者分析asoul（三人团版本）预计年收入838万，成本420万，利率50%。而同比折算的话丸子这种年收入可能只有asoul的1/3，成本近两倍，也就是收入280万成本840万，一年净亏损600万。这么一算，字节把锅甩给乐华，b站把锅甩给vspo，然后再是今年的毕业潮，就变得非常合理了。

国v的3d势并不多，比较有名的也就as五人、eoe五人、丸四人、a二期2人、e二期哆啦、丸二期istella3人、腾讯的星瞳扇宝雪绫、蹭3d的伊万九夏吉诺儿、沙东女团bs两人，然后男团量子少年四人、男团afaer四人，总共36人。到2024年开始时，所谓蒸蒸日上的国v圈里，这36个人只少了a的珈乐和量子少年的三个，加上彼时还没出道的哆啦，还没复活的九夏，倒是有30个人在抢这点不算多大的市场。截止11月22日，a剩3+2，e剩2，is剩2，星瞳，伊万，bs时月，afaer四人，30人只剩下了16个，其他人大多倒在了这个秋天。

虽然死的大多是3d，但2d的v日子也不好过。完美五人团普遍需要外出兼职，虚研社号称工资都发不起，海对岸的阿夸也离开了holo转生结城昨奈，活着的还行的只有个人势和堪比个人势的vr。经济继续下滑，资本逐渐离开，男女对立加剧，阿b加速摆烂。这个圈子还能活多久大约是个问题，但是也并不是什么很重要的问题。

两年前的我并不关心v圈，彼时的我应该还是半现充、op以及刀客塔，有一份看起来还比较高端的工作，有一个关心我，可以分享生活的人。我的空间还有很多人，我会经常抽奖，经常找人玩。有的时候我会经常的怀念深圳的蓝天白云，怀念那时候的日子。尽管有疫情，尽管有做不完的核酸，我依然觉得那时候好像还挺好的。后来我明白了，那个时候的我有而现在的我没有的东西，叫希望。

# 十年大梦

多年以后，面对HarmonyOS18.0的发布会，作为退役工程师的我将会回想起给终端云投递简历的那个遥远的下午。

近来时常有梦游的感觉，大体表现为不知今夕是何年的混沌感。走行在南京南站漫长的换乘通道，走行在大行宫地铁站的壁画旁，亦或是走行在新庄站、武定门、雨花门...原来三号线已经开通快十年了啊。

十年，是从懵懂的高一生到社畜的第二个合同期的距离。对于少年时的人来说似乎是天翻地覆，但对于社畜来说只不过漫长劳碌岁月的1/4，甚至是1/5。虽然很多人的职业生涯可能也就这么长，但确实从来也没有人规定职业生涯的结束可以意味着不工作。

一般而言，人之所求无非钱与闲，亦或者说叫财富与自由。或许有人会有第二阶段的人生目标，譬如说事业与爱情，但对于庸庸碌碌的我这般愚人，事业这种东西离我太远，爱情这种东西太不现实。我对金钱的要求不高，虽然很大程度上是因为我没啥钱，但我确实也没想到有钱会给我带来多大好处，所以拥有想做什么就做什么的自由时间反而更可贵了。很遗憾，我感觉我可能没有机会拥有了。

十年过去，没有了开往迈皋桥，没有了开往油坊桥，但经天路还是经天路，ICU还是ICU。人生自有天命，ICU迟早会收你。那么，在被ICU收走前，把日子过好一点吧。

# 夜航四千里

农历冬月十五，戌时末。夜航的班机缓缓飞出昆明，向着东北方向以八百公里的时速疾驰。窗外是一轮明月，在机舱玻璃上折射出如一克拉钻石般的光泽。这是2024年的最后一次月圆之夜，9500米的高空之上，看不见地面峰峦，只有月光相伴。

繁忙的第四季度在又一次护网零失分之后濒临了尾声，而一趟跨越千山万水的两天极限旅途则是给这一切最好的收尾。下关风、上关花、苍山雪、洱海月，在十二月的大理古城里交织在了一起。粉红的樱花与金黄的银杏一起灿烂在古城的街头，红色的水杉和白色的海鸥一起共享着洱海的水波。湛蓝的天空之下，洱海碧波荡漾，苍山白雪皑皑。

不算充足的时间，只够我在古城周边徘徊。来不及去爬上苍山的山顶，来不及去洱海的那一边，甚至来不及去喜洲感受一下有风的地方。只能沿着苍山大道寻找每一处山脚之下的樱花成林，骑着小电驴从苍洱大道飞驰而下直达洱海边的龙龛码头。看夕阳西下，月色初升。看朝霞云雾，海鸥成群。看古镇的上午人流熙熙攘攘，看才村的码头花车巡游四方。

相比于计划周全，提前许久开始筹备的旅途，我的旅途大多是随性而为的。一个人走走停停，走到哪里，再去寻找下一个目的地。作为一名永不停歇的社畜，周末随性旅途的终点无非是周一早九务必要回到的那一方工位。路上遇见什么，都可以视为一种上天馈赠的礼物。

作为当代I人中最典型的那一批，我的旅途会尽量避开一切需要和人打交道的部分。我不会去包车、不会去砍价、不会去订无法从互联网渠道预定的酒店。我会去尝试找几个熟悉的朋友，但如果没有找到也不会影响什么。这在相当程度上影响了我的行动范围，也让我的活动区域限制在了网络相对发达的城镇里。

我也尽可能去避免请假、避免早起，因为请假需要的阐述理由也属于人际交往麻烦里的一部分，而早起则是社畜的不可承受之痛。随着身体素质每况愈下和体力的日益杂鱼化，熬夜赶路也逐渐从我的选项里排除。可以预见的，我所能去的地方大约是越来越少了。

或许，等我什么时候买了车，我的旅行范围就只剩下江浙沪的短途漫游了吧。虽然，其实现在也差不多了。
