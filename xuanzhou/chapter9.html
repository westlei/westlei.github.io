# 数据结构

# 自勉

少说话，少做事。缄其口，慎其行。

不在其位，不谋其政。拒绝冲动，拒绝放纵。

少管闲事，少说闲话。少讲义气，少怀好奇。

三思而后言，三思而后行。

淡定看世界，平心对人间。

记得自己的初心，把自己沉入人海。

尽力让自己低调，避免恃才而放旷。

不该帮的忙绝不帮，不该管的事绝不管。

尽力维护所谓底线，尽力保持所谓尊严。

# 梦

浮生若梦，梦染凡尘。

黑暗间短暂的黎明，却仍不得喘息。

忙碌一旦开始，也许就没有了尽头。

早已忘记，是什么让我选择来这里。


走出五号楼的喧闹，融进四号楼的僻静。

告别那逝去的心情，斩断了过往与曾经。

忙碌从此没有了尽头，如同黑夜里难觅光明。

不后悔，因为故事没有结尾。


那一直在躲避的人和事，也许永远也无法直面。

正如生活艰难，但仍要学会去勇往直前。

等到什么时候看到，不会下意识躲避的时候。

才算是告别了忧伤，释放了心结。


萧瑟的秋风吹尽落叶，暖阳划破乌云的天际。

在那湖畔孤单守望着，却不抱希冀。

不懂得什么是喜欢，那便孤独下去。

孤傲的灵魂啊，永远寻不到的伴侣。


轻松悠闲的日子，就这么一去不复返了。

引以为傲的一切，就这么化作了空气。

忙碌在考试和作业的压迫里，难以呼吸。

自己选的路啊，跪着也要走下去。

# 忘情

清歌一阙，琴瑟无声。

似情之所往，若念之所向。

爱与恨，不知所起，不知所终。

不如，做一个没有情感的复读机。


世事无常，不能不相信运气。

缘分有始有终，请把美好珍藏。

不怀妄念，才没有伤害。

待到缘尽时分，情已终结。


永远记得，这花开的盛夏。

在这最迷茫的日子，温暖了心灵。

寒夜里，物是人非处。

回头望，点点尽是光明。


愿你有一个光明的未来，

愿有情人终成眷属。

目送着远去的人，

余生里，不再有的温柔。


未曾理智的爱，只是伤害。

未曾明白，便不宜去恋爱。

任心游走，在浩瀚宇宙。

看一朝月落，云起长空。

# 数据结构

一场细雨，冷落了清秋。

梦犹未醒，淡却了记忆。

往事如链表，有始无尽。

删去了头节点，便不再是自己。


记忆的指针往往随意，

如那生活一般不可预期。

即使你把那回忆片段的连接抹去，

记忆依然不时指向那里。


愿悲伤和烦恼沉入栈底，

愿队列里没有它们的踪迹。

或者去做无限递归的复读机，

嘻嘻哈哈沙雕般笑个不停。


看着人际的广义表日渐庞大，

熟悉陌生的名字交错复杂。

门前的二叉树又发了新芽，

选择困难症患者还在纠结于枝杈。


念想的人，可能只在图里。

拨开路径，不知何处边际。

寒夜之中，任凭代码相伴的孤寂。

往后余生，她是不会再有的回忆。

# 愁

想做一个快乐的沙雕

整天嘻嘻哈哈大笑

发着各种表情包

没有丝毫的烦恼


想做没有感情的复读机

不用纠结话语的意义

只用不停地按加一

直到没电了才停息


但即便是沙雕熊猫头

也有那悲伤与难受

戴上了面具之后

泪水伴随着忧愁


但即便是冷酷的复读机

也要经历砸坏和修理

一个个寒冷的夜里

无可复读悄无声息


宛陵风 宣州雨

飘渺了红尘一曲

尚未落幕的戏剧

不知真假 不知结局


敬亭雪 新安月

辉映那清歌一阙

挥手告别的决绝

或有阴晴 或有圆缺

# 忆·金庸先生

问世间、情为何物，只教生死相许？

天南地北双飞客，老翅几回寒暑。

欢乐趣，离别苦，就中更有痴儿女。

君应有语，渺万里层云，千山暮雪，只影向谁去？

武侠的世界，有着国仇家恨，有着快意江湖。

时代在笔下风云变幻，

驰骋沙场保家卫国的豪迈里，却也有着儿女情长的温柔。

襄阳城的血战见证了侠之大者的为国为民，

绝情谷的一跃也把那份痴情演绎到了极致。

可惜，这一切都不在了。

金庸先生走了，属于武侠的时代也已经没落了。

# 深渊

凝视着你的，不是深渊本身，

而是那无数已经沉沦在深渊里的灵魂。

时光匆匆，从来也不曾有梦，一切成空。

茫茫白雾，笼罩天地的孤独，尽是虚无。

爱过恨过，不留遗憾的错过，只未悔过。

无论如何，自己做出的选择，也许值得。

# 纪念日

——纪念侵华日军南京大屠杀八十一周年

八十一载的岁月，不曾磨尽伤痛，不曾流干泪水。

理应铭记的历史，仍然沾满鲜血，仍然留下伤痕。

一二·一三，记得这个沉重的日子。

古都金陵，白骨成堆，血染长江水。

三十万同胞，在日寇的屠刀下死去。

铭记历史，只是为了，更好的前行。

不要以为战争有多么遥远，

也许和平的终结比我们想的更快些。

至今不愿正视历史的日本，

还有虎视眈眈向中国下手的美利坚。

珍惜现在的岁月静好，珍惜先烈们为我们争取来的和平。

勿忘国耻，不要原谅，砥砺奋发，振兴中华。

# 江城子·宣州雪

清辉一片雪纷扬。弃过往，却情伤。

宣州多事，一载叹时光。

从此相逢不相识，勿思量，只相忘。

浩瀚山野总苍茫。身彷徨，心流浪。

物是人非，叹世事无常。

天涯沦落又何妨，拾行囊，向前方。

# 岁月尽头

迈过了18岁的台阶，推开了电信科的大门。

忘却了伤感的故事，放下了过往的风尘。

心底里一直忘不掉的影子，丢不下的几分倔强。

终于丢盔弃甲，终于化作风尘。

曾以为喜欢山，却发现偏爱的是山巅的云彩。

曾以为喜欢湖，却发现偏爱的是湖畔的清风。

像风一样自由，像云一样起舞。无所谓喜欢，只是偏爱几重。

来到过凌晨四点的西湖畔，也见过夜晚八点的敬亭山。

击毁了骄傲，余下了放纵。沙雕的生活，也不是不好。

却见，雪下得那么深，下得那么认真。
