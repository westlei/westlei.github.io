# 花神之舞

# 奢侈旅行

去年下半年的时候，马鞍山下面的当涂县因为穷，公交开线买不起新车，从深圳这边拉了一批跑了五六年的老车过去，随便刷几个字就上线了，都是贼长的城乡班线。

比如这次刷到了有人坐606路，从当涂客运站一直开到高淳边上的湖阳镇，全程62公里，说石臼湖段风景很不错。然而这车全程票价两元就很离谱，深圳这边差不多长的833路全程得十元了。

这几年大力推动城乡公交发展的额外效果就是养活了一群从上海甚至广州坐公交去北京的up主。但仔细分析一下线路就会发现他们都是沿海行进，城乡公交这种东西对于内陆还是太奢侈了，又贵又烂的乡村客运班线才是常态。

又想到前几天聊到的K101这种破烂绿皮车的现状。尽管温州早已通上了高铁，但是很多北方客商依然需要K101(北京)/K347(沈阳北)/K551(佳木斯)三兄弟南下温州。沈阳到温州，高铁要1000+，绿皮车硬座只要300-。但是高铁只要14个小时，绿皮车要开36小时。

但即便如此，绿皮车也依然时常一票难求。出行对于大多数人来说，还是一件奢侈品。不是每一个人都有能力跨越半个中国来回，还有很多一辈子都没有走出过自己所在的市的普通人。

下午突然有点想去扬州，看了看要开十个钟头的高铁，再看了看票价一千多时间贼差的飞机，还是算了。钱目前来说不算什么问题了，但是没有时间对于我来说也是一种痛苦。我想像大学那样自由自在的出去玩，但我却悲哀的发现，我没有时间了。

# 平庸生活

在深圳的词典里，从来也没有什么生活一说。来到这片土地的人，为事业、理想与财富奔忙，忙忙碌碌的加班，吃着并不廉价的快餐，闲暇时更倾向于窝在那个姑且称之为家的出租屋里打游戏刷剧。

深圳很美，但深圳的美并不属于此地的打工人。早出晚归的打工人没有时间也没有心情跑来跑去，他们顶多抬头看一眼夕阳——不，更多的人在不分昼夜的格子间里，连夕阳也看不见。去寻觅深圳的美，似乎并不是一般的打工人所能做到的。

宝安西乡红树林的落日，南山山顶的蛇口夜色，深圳湾繁华的盛景，莲花山下人来人往的福田中心区。华强北穿梭不停的人群，罗湖东门老街的古早深圳味儿，淘金山绿道下深圳水库的磅礴，梧桐山顶精彩的日出。盐田海滨栈道带着咸腥味儿的海风，溪涌沿岸美不胜收的湛蓝，南澳海滨新鲜的海产和渔获，大鹏所城最古老的商街。我走遍深圳的每一个角落，度过我事实上的gap year。这里很好，但我不属于这里。

庸庸碌碌的凡人所渴望的，无非金钱，事业和爱情。大厂能够解决金钱的问题，也能带来一点事业的成就感，但面对爱情无能为力。我以为我能在这里寻觅到爱情，但...

我似乎寻觅到了，不过，不是在这里。

距离战略预备队还有12天。距离上海海思还有约27天。距离前往上海还有约70天。距离恋爱还有...

undefined。

# 隔世久违

当你被困在了梦里，却意识不到梦的存在。每当午夜的钟声敲响，一切归于缘起。你做了一个漫长的梦，梦醒之后的你，什么也不记得，但总觉得眼前的一切有些熟悉，就仿佛在哪里见过一样。

清醒或是梦境，无法定义，模糊清晰。一如预言家口中2025年东海的大灾难，未知其真，难辨其假。或许你我相遇在梦里，但梦后的我却失去了记忆。再相遇时，却只是：初次见面，多多关照。

银临的新歌《日出前起飞》。一如去年的玫瑰与泪，直接扣在了内心深处最敏感的地方。我期待着相会在下一个秋天里，期待着隔世之后的久违。

# 杀毒软件

一夜之间，深圳好像又一次进入了疫情的中心。七个街道封控五天，其他地方人人自危。这次流行的毒株叫BF15，是什么BA5.2.1的子分支。反正感觉看病毒更新就像软件更新一样，版本号随便起。据说现在的毒株可以十次核酸测不出来，那还要核酸干啥呢？当然这不重要，毕竟抗疫软件还是当年那款，版本一直没更新。

是什么让我想要离开深圳的呢？最直接的原因可能就是三月那场疫情吧。隔离的时间让我意识到我所做的工作并没有什么实际的意义，甚至于可以说有一点浪费时间。当然更重要的是，让我意识到一个人在深圳的脆弱，真有什么事情发生，我甚至没有一个能帮我做些什么的朋友。

当然，疫情年代特有的脆弱，也让我又有了想要去找一个陪伴我的人的想法。但是曾经的一些过去，让我很难去信任一个人。我其实不在乎她到底喜不喜欢我，或者说对我好不好。我希望的是哪怕有一天她出轨了，劈腿了，起码跟我说一声，不要让我自己去揭开真相的纱布。所以我需要积攒足够的信任，去解开所有让我不信任的源头。

有些东西，尽管直说的话对我自己不好，但我不喜欢瞒着。我从来也不会是站在正义立场的，我只代表我自己所支持的立场，不管正还是邪。

大不了，remake也不失为一种选择。

# 预备队生活

感觉就是很累，非常累。一种让我不断有想要离职冲动的累。

不是那种单纯因为体能训练或者早上六点起床晚上九点下课导致的疲惫。而是一种完全不知所云，难以理解的痛苦。预备队里基本上都是华为各个领域里摸爬滚打十几年的精英，也是经常出入预备队换岗的老油子。这批人对世界的理解似乎与我，或者我常接触的研发或者it人员，差异巨大。我跟不上他们的思路，也不能理解这个环节的意义。讨论的议题动辄就是华为多少年的历史孵化出的各种政策，以及对这些政策的理解，那我完全不理解，我也不知道我怎么理解，我只能纯摸鱼。好在这个似乎对出队转岗不影响，那就硬混吧。

我一向觉得我对环境的适应能力不是很差，但是预备队的经历让我意识到在经历、资历、生活场景等各方面背景完全不一样的场景下，想去适应是多么艰难的事情。再加上十几门专业课的存在，让我心神俱疲，两次多错一题导致擦边挂科，实在是一种很彻底的折磨。好在新的部门对我还是不错的，至少我觉得是个挺纯粹的技术部门，能让我学到很多东西，至少让我在痛苦的边缘，不会变得更过分的痛苦。然后就是新部门认识的第一个兄弟居然是牛头冲的学长，这运气也是没谁了。

最近有好些人很关心我的感情问题。但事实上的感情状况只能说是非常的力不从心。我很感激某位为我所做的一切，但我真的很难去理解如何经营感情。或许是因为工作导致感情的漠然，也或许是我本身就没有这方面的能力。理性和感性的冲突是无可避免的，最后剩下的就是疲惫。掺杂了感情的交往让我有了许多的拘束，让我不可能真正的放飞自我。这也就是我卑微而失败的根源吧。

我应该不是一个喜欢讲道理的人，也不是一个说话很无趣的人，甚至于说应该是个经常玩梗喜欢发癫的人。但不知道为什么，感觉一切的一切，在面对感情问题的时候就失效了。就好像我现在一无所有，但好像也确实一无所有。我现在大约是有一点钱，但不是很多；大约是没有什么负担，但也没有什么可挥霍的。

所以我现在没有什么压力，也没有什么向往。爱情什么的，我期许，但不强求。

# 中秋节的絮絮叨叨絮絮叨叨

是一年中秋节了。按农历生日来算的话，今天我应该22岁了。不过考虑到这个日子的特殊性，我一般过的是公历生日。我不是很想急着变老，但是日历却又无时无刻不在提醒着我这件事的发生。

一周的战略预备队体训算是结束了，剩下就是若干考试的清理，然后就是面对新的工作和新的生活。最近左眼皮跳的异常厉害，大抵是说明最近可能有一些意外惊喜。我是没有理解什么是意外惊喜，好像我现在也不缺什么。

上一个中秋节的时候，我爬了长沙岳麓山，然后赶到广州海心沙直播抽心海。时隔一年，今年的中秋节居然又是心海复刻池刚开，不得不说也是一种奇妙的安排。

我很喜欢心海这个角色。尽管大家对这个角色的评价很不好，但我觉得很不错。大约是因为我足够的缺乏安全感，所以每当血量不足的时候只要心海开大站场，就能让我很安心。再加上海染套的buff以及水元素本身的万能适配，伤害也并不算低。

但心海的剧情表现才是真正戳到我的。尽管大家公认心海的剧情十分糟糕，但是在我看来，似乎就是我本人的写照。我在外包装出的能力很强的形象，其实是靠非常多的经验和阅读积累以及搜索技巧来实现的，就仿佛“算无遗策”的心海和她究极漫长的锦囊。

心海的所有活动都会消耗能量，我也是。或许与人交往消耗能量就是I*FP系列的一个常见的特征，我会喜欢去接触一些人，但不论是谁，都会让我的能量衰耗，陷入疲惫。不管我表现的有多么社牛，但是本质上还是一个社恐，这是无法改变的。

心海看上去是一岛之主，是富裕的海祈岛现人神巫女，但战火让海祈岛并不是看上去的那么富裕，反而十分贫穷。我看起来家庭条件优越，工作薪水很高，但是家里的财产并不能多大程度上给我以便利，工作的薪水稳定性也有些堪忧。尽管现在看起来有钱了，但本质上还是感觉贫穷。

心海仿佛一面镜子，让我认识到我自己是一个什么样的人。当心海被大家当成小丑的时候，我觉得小丑似乎也是我自己。突然又想起来上一个让我觉得很像我自己的人。我喜欢她的原因就是她的很多想法和行为让我觉得像我的镜子，但最终交恶的原因似乎也源于此，因为镜子的感觉不过是幻像，而远非她本人。

所以现在的我试图去寻找一个能带给我光明和温暖的人，她能够在一定程度上去弥补我所有的不足。她不需要交际广泛却热爱交流，感化人心；她不需要懂得太多却实事求是，能够做到最好；她不需要十分富有但可以精致，让人为之倾心。她像明月一般照亮我，却又不让我感觉太远而无法接近。

但我终究还是过于的理想化了。明月的光辉穿过云雾窗台将我照亮，她却并不能感受到我的存在。我照镜子，镜子也会反射出我的模样；我照月亮，月亮却还是月亮。我单方面的接受了一切，却不曾去洞察，对方需要什么。

皓月当空，清辉洒落，留下斑驳的影。前面的路是杂草还是荆棘，在月光之下并不清晰。或许，月夜下的行者，不应该急着赶路。是时候抬起头，好好看看，那一轮明月了。

# 月色入怀

一晃就22岁了。

回想在深圳的这一年，其实有挺多东西想说的。但我表达能力像我的人际关系与身体素质一般般退化的迅速，写了很多，却又好像不知所云。

南海之滨的深圳无疑是美的。深圳的空气质量在全国主要城市里大约都是数一数二的好，大鹏湾沿岸金黄的沙滩与湛蓝的海水治愈麻木于工作的心灵，西湾和前海的落日余晖舒展困在城市山林中的身心。山海，是属于旅者的浪漫。在旅途里忘却属于工作的烦恼和焦虑，在骑行间感受风的自由。

但这座城所带来的焦虑，却也是无处不在的。无论是暗无天日的工位，卷王一般的同事，还是各种无形却凝重的仿佛成了实体的考评压力，散在坂田的寒气里，压在每个人的心里。第一年所幸做出的些许成绩，给我带来了超额的金钱回报，却也让我的压力更多了几分。所谓理想和愿景显得可笑，只有实实在在的kpi和okr才是真。

感谢深圳岁月里每一个新认识或者一直陪伴我的朋友。在迷失自我的时刻呼唤着我，在陷入迷惘的时刻拯救着我。或许有时候的我显得过于淡漠，但并非刻意如此，只是被工作磨灭了些许的精气神。感谢赠予我的鲜花与礼物，前路漫长，我们还有足够的时间，可以慢慢来。

仍是中秋时节，明月高悬空中。那么的近，又那么的远。我还在期待，期待可能的未来，月色入怀。
