# 从单片机到嵌入式开发——(3) LED灯与数码管

date: 2020-11-28

author: 西风镭

location: 宣城，合肥工业大学宣城校区

前文介绍了基本的51单片机C语言编程部分。C语言的更多知识此处不再介绍，代码能力还是需要在实践中磨炼。

下面我们主要介绍单片机常用的一些外设。单片机最基本也是最常用的外设就是发光二极管LED。LED一端连接高电平，一端连接低电平时即可导通，因此通过最简单的语句即可点亮。

P1 = 0x00;
1

通过使用软件延时，我们可以制造出各种动态效果。

void delay50ms(void)   //误差 0us
{
   unsigned char a,b;
   for(b=190;b>0;b--)
      for(a=130;a>0;a--);
}

void main (void)//主函数main
{
   P1 = 0xfe;
   while(1)//无限循环
   {
      delay50ms();
      delay50ms();
      P1 = P1 << 1;//循环左移
   }
}

LED数码管本质上也是一组按特定方式排列的LED灯，因此也可以按照LED来进行理解和使用。点亮特定的LED灯，即可得到合适的数字显示。

P1 = 0x82;
1

同理，我们也可以通过使用软件延时，制造出数字显示效果。如图为0-f显示，每100ms刷新一次（仿真时间略长于实际时间）

unsigned char code num[] = {0xc0, 0xf9, 0xa4, 0xb0, 0x99, 0x92, 0x82, 0xf8, 0x80, 0x90, 0x88, 0x83, 0xc6, 0xa1, 0x86, 0x8e};
unsigned char i = 0;
void delay50ms(void)   //误差 0us
{
   unsigned char a,b;
   for(b=190;b>0;b--)
      for(a=130;a>0;a--);
}

void main (void)//主函数main
{
   P1 = 0xff;
   while(1)//无限循环
   {
      delay50ms();
      delay50ms();
      P1 = num[i++ % 16];//循环显示数码
   }
}

通常来说，当我们需要显示不止一个数字时，我们依然只有一个数据端口可以使用。这种情况下，我们需要使用视觉暂留的方式来实现多个数字的同时展示。

视觉暂留技术，顾名思义，利用人眼对高分辨率的刷新难以识别的特点，令所有数码管都处于高速连续的刷新状态，从而让人眼可以看到不同数码管上显示不同的值。我们采用每1ms刷新一次位置和各位置上的数，从而令数码管能够显示多个不同的值。

unsigned char code sit[] = {0x01, 0x02, 0x04, 0x08, 0x10, 0x20, 0x40, 0x80};
unsigned char code num[] = {0xc0, 0xf9, 0xa4, 0xb0, 0x99, 0x92, 0x82, 0xf8, 0x80, 0x90, 0x88, 0x83, 0xc6, 0xa1, 0x86, 0x8e};
unsigned char i = 0;
void delay1ms(void)   //误差 0us
{
	unsigned char a,b;//unsigned char类型
	for(b=4;b>0;b--)
		for(a=130;a>0;a--);
}

void main (void)//主函数main
{
	P1 = 0xff;
	P2 = 0x00;
	while(1)//无限循环
	{
		P2 = sit[i%8];//位置
		P1 = num[i%8];//显示数
		delay1ms();
		P1 = 0xff;//清屏
		i++;
   }
}

如图所示，这种方法显示八位数码通常需要两组IO端口的所有引脚，这对总共只有40个引脚的单片机来说有些艰难，因此，我们通常会使用一个3-8译码器来实现使用位数的减少。

unsigned char code num[] = {0xc0, 0xf9, 0xa4, 0xb0, 0x99, 0x92, 0x82, 0xf8, 0x80, 0x90, 0x88, 0x83, 0xc6, 0xa1, 0x86, 0x8e};
unsigned char i = 0;
void delay1ms(void)   //误差 0us
{
	unsigned char a,b;//unsigned char类型
	for(b=4;b>0;b--)
		for(a=130;a>0;a--);
}

void main (void)//主函数main
{
	P1 = 0xff;
	P2 = 0x00;
	while(1)//无限循环
	{
		P2 = i%8;//位置
		P1 = num[i%8];//显示数
		delay1ms();
		P1 = 0xff;//清屏
		i++;
   }
}

由于74HC138选通端输出0，所以使用时需要接反相器74HC04。如图所示，效果与之前一样，但只使用了P2.0-P2.2三个引脚，引脚需求大为减少。

将以上两种方式相结合即可进行数码管的任意显示。
