# 卷王之王

下午写概率论的时候睡着了，做了一个关于未来的梦。一觉醒来意识到一个很严肃的问题，我们对于个人未来的一切设想都是以当前为背景，可是就仅仅十年以前，2010的人能想象到 2020的人过的是什么样的日子吗？

外循环经济体(生产者≠消费者)：全球生产普遍过剩→生产方降低生产成本→996以及各种内卷化现象(更高的劳动以及更低的收入)→消费方借贷消费→完成经济运转。崩溃临界点：消费方债务爆炸→无力消费→生产端无法完成售卖→雪上加霜→进一步压低成本→公司普遍破产→大量坏账积压→大萧条。

内循环经济体(生产者=消费者)：国内生产严重过剩→消费方无力消费→鼓励消费者借贷消费→消费者高还贷压力→对工作容忍度下降→进一步加剧内卷→生产继续过剩。崩溃临界点：生产者无法继续压低生产成本(内卷尽头)→工作不仅没有报酬还要负债→公司破产个人躺平→大量坏账积压→大萧条。

感觉内卷的日子快到头了，只是不知道是以哪种情况而结束的。内卷过程本身就是一个恶性到无以复加的恶性循环，不知道是自然终止还是烈性终止，生产力发展瓶颈是个死结。所以眼下读研或者出去工作都免不了挨一刀，那怎么办，只能躺平。

20岁以后对未来越来越悲观了，也许这就是内心一直存在的悲观吧。经济学适合业余学一学，但是真的投入去学的话就觉得，人生的前途这么无亮，为啥不现在就打开窗户跳下去。我们看见了很多很多的问题，但我们却没有能力解决任何一个。这也许就是一个非理性观察者的悲哀。其实一切都只是永远的围城，不过如此。

# 一路·一城·一人

2020年10月11日，全国铁路实行四季度运行图。K645/6/7/8次列车改经由宣杭-沪昆-衢宁线至福州运行，T151/2次列车缩短南京始发终到，K155/6次列车停运。从此，皖赣铁路省界仅剩余T151/2次列车通过，百年皖赣线终于陷入了无可奈何的衰落。这同时这也意味着，宣城站出发往江西方向仅剩余经宁铜-铜九线运行至南昌的K1220/17次列车，经皖赣-沪昆线运行的T151次列车，以及经合杭-沪昆高速线运行的G1391次列车。

江南铁路建设构想始于1905年，1906年芜湖-宣城段首次开工，建成32公里。后于民国时期1933年正式建设并完成南京(中华门)-宣城(孙家埠)段。1936年，出于战备需要，宣城-贵溪段开始加紧建设，并于1937年落成。然而由于随之而来的侵华战争，该线路建成即被拆毁，未能发挥出其作用。建国后，几经风雨，皖赣铁路于1985年完成重建。该铁路在相当的时间内都是皖南的唯一一条铁路，也是南京南下的主要通道。后来，随着淮南铁路芜湖长江大桥，宣杭铁路等线路落成，华东二通道成形，南京可经由芜湖连接合肥、阜阳，从而快速通达武汉、重庆等沿江地带，也可经由芜湖、宣城快速抵达杭州，从而绕开繁忙的上海枢纽，节约时间和里程。铁路运输逐渐取代了传统的长江航运，芜湖、宣城也随之成为皖南重要商埠。小时候，家旁边的宁芜铁路线从早到晚火车络绎不绝，汽笛轰鸣，十分繁荣。我还记得07年开通重庆经合肥、南京至南通直达列车时，车上那满载的人流。

可时代进步的速度也让这条饱经沧桑的铁路赶不上时代的发展。大明路、晨光路道口每天排成的长龙让宁芜铁路南京段的搬迁不得不早早提上了日程，却受制于各种原因一直未能执行。也因此，宁芜铁路的复线修建和电气化改造也一直遥遥无期，皖南的铁路也逐渐落后于时代，经由宣杭铁路运行的列车逐步又改回了距离虽然远速度更快的沪宁-宁杭线路。铜九铁路的修建也给了皖赣铁路一记重拳，位于山区的皖赣铁路在雨季时经常遇到的线路坍塌损毁难以运行的问题，使得越来越多的列车改经由铜九-京九线运行。尚未发展便已衰落，是皖南的悲剧，也是时代的落寞。

此后不久，高铁的东风席卷了全国，宁杭高铁的开通让南京-杭州的时空距离急剧缩短，经由芜湖-宣城运行的列车又一次失去了距离优势。国家级干线合福高铁避开芜湖宣城直取黄山，给本就衰落的皖赣铁路又一次暴击。宁安城际的开通进一步瓦解了宁芜线的短途客流，杭黄高铁则让黄山的发展一定程度上摆脱了安徽的掣肘。当然，合杭高铁的开通，则是压垮一切的最后一根稻草。可是，芜湖和宣城已经失去了过多的机遇。一条在新冠纪元里姗姗来迟的高铁，又能为两座城市带来什么呢？

宁芜铁路南京城区段改线工程在近20年后终于提上了日程，可这时的宁芜铁路早已经没有了昔年繁盛的模样，稀稀落落的汽笛声，仿佛仍在追忆以前繁茂的年华，而中华门地铁站高架之下的中华门火车站，却早已变成了历史的尘埃，连带着周围发展停滞的老城南。皖赣铁路替代品的宣绩高铁也开始了开工的准备，可宣城的落寞已经成为了一个板上钉钉的现实。先后于15、18年消失的芜湖、宣城老站，在2020年的夏天展现出新的模样。可这似乎，也来的稍晚了些。

上午10:35分，最后一班K155次列车驶出了宣城站。开行了二十一年的经典列车自此闭幕。而十分钟前开出，同样去往昆明的G1391次列车，则象征着一个新时代的帷幕。

上一个消失的记忆，是T65/6次江南明珠号，但它仍然以D715/6的形态存活着。而这一次消失的K155/6，则是永远的消失了。

人有的时候不必太过感伤于过去，但站在时代的十字路口上，总还是有那么些迷茫。或许明年这时候，我去到了一个未知的城市，在一个未知的角落里开始了新的生活。但是，在皖南这片土地上留下的回忆，依然值得去拥有，值得去怀念。

那么，就散了吧。沉迷在过去，还是执着于不可能发生的事，都只能带来忧伤和迷惘。过去的生活里注定不会有喜欢你的人，只有未来才有可能有那么些希望。

放弃幻想，准备投降。

# 长夜漫漫

可能是很久都没有认认真真对待过世界了，已经没有办法认真对待这个世界了。复习感觉似乎好像还挺好的，但是根本没法面对即使类似考试的环境，心态崩了，反而豁达了。反正注定失败，还是想想明年怎么办吧，又不敢去想。

表面的光鲜可能只是纸糊的一戳就破，可我现在连表面的光鲜都很难去维持了，运气真的很重要，前提是有实力的基础上。真的很羡慕身边优秀的人，他们各有各的归宿。有幸在这么几年里，能与他们或她们擦肩而过，其实已经是运气了，还能去要求些什么呢。

一直以来所渴求的所谓脱单，不过是脱精神的单。让我能够意识到，我不是一个人在这世界上。从而有能力去拥有梦想，或者去追逐梦想。我没有梦想，也没有理想，考研是无奈的选择。我可能更想，在最好的年纪看过自己想看的风景，然后选择一个无人问津的角落，默默远去。

我一直觉得我和这个世界联系其实很微弱，微弱到了什么程度，就是消失很久，可能也没人意识得到，这样一个程度。没有什么固定会联系的人，哪怕是家人，也经常一个礼拜才会微信说两句这样子。家人之外，就更没有什么特别值得亲近的了。有时候特别喜欢水群，是因为感觉，群才是家。

每个人都有自己的生活，自己的理想，像一条条空间里的直线，向着一个个远方。 而我只是一段段线段的集合，与一条条直线交汇于空间的不同平面之中，却很快就会走散。甚至于什么时候，消失。

一年前的五月，就像是大学的一个分水岭。往前是现实，往后是虚幻。在那之后认识的朋友，纵使打过比赛组过队，也只是互联网上，一点点微弱的联系。在现实的世界里，从未见过，也从未了解。

维系着我现实的生活的，是各种各样的情结。就像处女座一如既往的强迫症一样，经历过也许就是圆满，结果反而并不重要。所以我做硬件也做软件，写计划书也写论文，看起来什么都做，其实似乎什么也都做不好。我尚未发现我有什么能力，可以让我生存下去。

有的时候我会觉得，世间的一切都很简单。敢于在人生地不熟的地方独自探险，也敢于在大雨的深夜开车在高架狂奔。可却又明知这个世界上更多的东西很难，我会习惯性不敢去面对这些棘手的问题，不管是考试，还是感情，亦或者难以定性的关系。习惯性的冷漠和逃避，尽管是下下策，但是有用。

就像我也不知道我为什么会报buaa，尽管我找了一堆理由，也不过只是假的。一月份和一个陌生小姐姐无意的谈话，才是九月底做出这个决定的根本原因。我喜欢在一个隐蔽的角落观察这个世界，而世界也会以隐秘的方式操纵着我自己。无意识的一句话，或者偶尔谈起的一段往事，都可能是我之后一个重要决定的直接原因。

面对别人的问题，我总能想办法分析的头头是道，即使超出我的能力，我也能给一个合理的解答。可一旦涉及到我自己，大脑就彻底失去了活性，情感上的抗拒，让本就艰难的生活更加艰难了。很多人都觉得我不至于此，但我真的没希望了。人与人的悲欢互不相通，可起码吵闹好过无聊。

晚上总是很现实，焦虑的一天天睡不着觉。白天总是在做梦，梦里时间却又过的特别快。我有过许多个喜欢的人，可是我啥也不敢说。三年前的事情教导我，遇事先想配不配。而现实的大山一次次告诉我，什么是不配。曾经还有些朋友，能陪我聊到天亮，会听听我有多少难受，尽管给不出任何办法。现在再翻一翻列表，大家都已经越走越远了。

每个人都有自己的理想，都有自己光明的未来，可我什么都没有。甚至于觉得，自己没必要存在。存在即被感知，或许，并没有多少人能够感知，那这样事物在大众眼里，它也许就不曾存在吧。

# 凛冬将至

雾色宛陵深似海，纷纷细雨独哀。伤情只在心中埋。欲见湖光好，且待春日来。

寒风吹彻宣州外，前路艰险难猜。谁道未来几多灾？抬眼空望水，举步独登台。

三年前所填的一首临江仙，放在三年之后其实还是一样的感觉。等到下一个春天，十个海子会全部复活，每个人都会找到他们期待已久的归宿。

冬天好像还很长，但对于有的人，已经不长了。最寒冷的冬天还没有到来，但我们更应该去期待春天。等冬天的一场大雪，与大雪里不会回来的人。

当你觉得你被世界抛弃的时候，请记住，可能还有那么一个人，在一个你所不知道的角落，默默仰望着你，卑微地喜欢着你。

愿你眼里有光，心里有爱。雾霭终有一天会散去，光总会照进来。这世上，有阳光，也有影子。光与影，相斥，却又相生。

就像曾经我有过的，许许多多不切实际的梦。我经常去幻想他们发生，却也明知仅仅是幻想。但说不定有朝一日，幻想可以成真，失去的都会圆满。

一切也许又会像半年之前那样，向好的方向变化起来吧。当一条咸鱼突然有了梦想，那生活可能就没那么难了。希望来年的时候，一切都没有太迟，一切都是刚刚好的样子。

列表扎着堆儿奔着二十岁去了。

有些东西总是写的很隐晦，希望有的人能看懂又不希望有的人能看懂，不过暗喻明喻什么的，事后总是会看出来的。

打怪升级了。切换了地图并解锁了新场景，新场景下好像plmm挺多的，可我却已经没啥兴趣去了解一下了这样子。

牛头冲最后一年了，不想认识什么新朋友了，就像现在这样，安安稳稳混过剩下俩月就好了。我以为我对考研的理解已经很独特了，但是总还有更独特的。整天想着都是找自习室的pljj然后去表白，我只能emmm。

说起来似乎是因为感冒加重引起的乏力

无非是为几重心病找的一重借口

感冒药会让人陷入深沉的睡眠

熄灭屏幕 在这静谧而安详的日暮之中

过去 现在 和未来 都交融于一体

化为一种短暂而永恒的沉寂

但也或许不会
