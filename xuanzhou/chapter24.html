# 光遇

阳光倾落在综一的走廊上，感觉暖暖的。

墨水挥洒在数学的草稿上，感觉凉凉的。

看到一句话：遇到事情的时候，

有的人硬刚，有的人逃避，

还有的人会选择曲线救国。

我总是会下意识去选择逃避，

可嘴里仍然说着我是在曲线救国。

但结果是骗不了人的。


你说，你为什么对我这么好。

我说，因为我对每一个人都很好。

你回了一个表情，说晚安。

我也回了晚安，说祝你幸福。

雪白的屏幕上，仅存的一条记录格外显眼。

突然又来了条消息，有人来问问题。

我随手敷衍一下，然后便划掉了记录。

我是说过我对每个人都好，但我只能看见你。


生病了，心情特别不好，控制不住情绪。

你还是一如既往地会问候一下。

我踌躇了许久，点开又退出，退出又点开。

终于还是假装什么也没看见。

等着大火变成小火，小火变成无火。

再像什么也没有发生一样。

说一句，最近有点忙。


一旦过分亲近了，关系总会变质的吧。

那样对谁都不好。像现在这样，就很好。

我默默地看着你，却假装看不见你。

习惯性地排斥，排斥一切的亲密关系。

也只有足够疏离了，才能足够长久吧。

如果你已经选择了去浪迹天涯，

那就不要说为什么你的旅途如此孤独。


孤身打马南屏旧桥边过，

恰逢山雨来时雾蒙蒙。

想起那年伞下轻拥，

就像躺在桥索之上，做了一场梦。

梦醒后跌落，粉身碎骨，无影亦无踪。

# 牵丝成戏

我和你 最天生一对

你憔悴 我替你明媚

问世间 哪有更完美

愿谁记得谁 最好的年岁

每350个中国人里面

就有一个人在考研

作为1/350中的一份子

谁还不是这喧嚣尘上的内卷里

时代的一粒炮灰呢

梦是要做的 但也是会醒的

把该读书的时间用来做梦

等到失败的时候 就能更加体面

# 巴巴托斯

我好像以前不是现在这样子的

我是什么时候变成现在这样子的

当年发生了什么 摧毁了我什么

其实这一切我都已经有些记不清楚


大约从那时起 我就已经开始只会混日子了

没有对美好未来的憧憬 只有日复一日的摸

现代社会的变化真的太快 但这并不可怕

怕的是你已经不敢去面对 这个魔幻的世界了


人总是要做人上人 所以奋斗只是为了不做人下人

这就是为什么这么卷 也是为什么这么累

所以或许不要做人会好一点 或者早早去世吧

我已经死了 七百五十多天前就死了 只是还没埋


谁考研不是为了想上岸呢

只不过 想不想是一回事

能不能是另一回事罢了

我依然向往一个光明的未来

希望有那么一天

树在 山在 大地在 岁月在

你也在


或许 在亿万时间线的某一条里

终究是有那样一个更好的世界的吧

但 我注定也见不到了

干点正事吧 巴巴托斯！

# 无色海

最近状态很不对劲

有的想法有些

人不能 至少不应该

该逃避的时候还是要逃避的


海 没了记忆 沉睡进时光里

风声静止 天光散去 等不来归期

无论如何 过去的终究都还是过去了

未来的也用不着多少期待


课设恍惚间也算是结束了

如果按课表来看 这学期总共的四学分修完

这个学期也似乎就即将告一段落了

终于也不得不正视自己蜕变为纯粹的打工人

没有钢铁般的意志 却沉迷在虚幻的古风世界里

想要一个人去一个绝美的地方 看一场大雪

过去和未来 俱与我无关 人间不值得


在世间 本就是各人下雪

各人有各人的隐晦与皎洁

你说了 她会懂 她不说 你也明白

终究各自散场 徒留下不知所以的悲伤

也许几年后 再回想起考研的生活时

我记不清新安的风 敬亭的雨

想不起实验楼午后阳光的温暖惬意

却单记得一抹影子 若有若无 如梦似幻

# 离场

一切林林总总的祸端总结起来无非俩字感情

没有疯狂的爱自然也就不会有无端的恨

20年代了怎么还会有这种狗血的故事

为了一个爱而不得的女生争风吃醋

仿佛回到了琼瑶剧武侠片当道那个年代

武德充沛 不像现在很多年轻人 不讲武德


所以既然选择了跑路 那就该干啥干啥吧

比赛开卷 成绩很卷 考研特卷 什么都卷

可以有梦想 但那不是成年人该有的东西

20岁的人应该想想自己的以后 怎么活

这样的话 想着想着就知道自己不配活着

自然是点到为止 先考虑明年后能混几年


所以有的话说出来以后 其实就很释然了

不管究竟有没有 被应当听到的人听到

所以有的决定做了以后 其实也很轻松了

不管究竟会不会 像我猜想的那样变简单

转眼中秋国庆已经是差不多一个月前的事了

月亮又一次圆了起来 期待已久的秋游快来了

庐州月光 洒我心上 牵挂的人儿们 又在何方

有的人就是只适合当朋友 不管她们再漂亮

我也不会生出 任何一点非分之想

但有的人 可能一开始想法就不单纯了

# 不如结束

相比得到别人的爱

努力变得更优秀反是一件更简单的事

所以大家都努力变得优秀

大家都在卷

可是努力变得优秀又是给谁看呢

反正不是我自己 我不需要

卷王和躺平人生活在同一个世界里

前者总是有意识忽略后者的存在

他眼中只有卷


但事实上 这个世界上还有很多种生存方式

这一切都只取决于你对你自己的选择

我很不想卷 但我没有后路

如果后退 只会面对比现在卷一万倍的现实

要么前进 去未知的方向 找未明的出路

要么 结束这场让人无法愉悦的游戏

# 长梦不醒

昨天夜里做了一个很长很长的梦，从深夜零点一直到清晨六点。中间似乎数次醒来，却分不清现实还是梦境。六年生涯以一种魔幻的方式交织在一起，有的人已经模糊到记不清面容，而有的人却仍然深深铭刻着。就仿佛这六年来的生活，离散而凌乱。琐碎的情感揪扯着心，分割又撕裂。只能去若远若近的所在，寻求若即若离的安宁。

2020的第四季度，重新看曾经的年度计划。美赛卑微的S奖，终究无能参赛的智能车，疫情之中萧瑟的春天，去不成的千岛湖。考点过完了也忘完了，花时间重新捡起来。身体在持续的亚健康，考研状态也不好。或许见到了值得的爱情，却不值得去争取。一个人来到了潜山，又被暴雨阻挡了脚步。依然照着心中的白月光去追寻所谓的理想，但终究也没有能考成CSP去成翡翠湖。不过，最后的最后，合肥的朋友们都很好。

唯一一首发在朋友圈的填词，是很多人喜欢的那首《我有一个道姑朋友》。也许有人能看出些什么，但除了我自己，终究不会有人再懂。

# 长夜未眠

可能有的人还没入睡，然而有的人可能已经醒了。睡眠质量莫名的糟糕，不是睡不着就是睡不安生。虽然知道考不上是大概率会发生的事情，但还是免不了特别特别焦虑。白天就很想找人聊天，聊啥都行，像个话痨。晚上就要么比较抑郁，或者陷入古怪的梦里。可惜大家都很忙，也不会有空听我絮絮叨叨。

恍惚间过了一年，什么也没有做，但又好像，什么也来不及做了。

起初，神创造天地。地是空虚混沌，渊面黑暗；神的灵运行在水面上。神说：“要有光。”就有了光。神看光是好的，就把光暗分开了。神称光为昼，称暗为夜。有晚上，有早晨，这是头一日。 ——《圣经·旧约·创世纪》

无数次纠结于要不要放弃考研直接打工，直接快进到成为一名光荣的打工人。但又想想2021这行情，大厂肯定没我的份，中厂小厂可能把我卖了我还得帮着数钱。人的需求是有限的，而资本家的供给是无限的。资本家为了让打工人心甘情愿为他们奋斗，不得不搞出各种各样的玩意诱使打工人消费。双十一的庆典开始了，资本家又会盆满钵满，而打工人的当下和未来，将全被剥削一空。

天好冷，感觉一天比一天冷，冷到读不进书。打工实现不了财富自由，读书也不能。精神、肉体和财富，哪种自由最难？可事实上，无论哪一种，都与我无关。

# 倒计时间

2020年10月28日，考研的倒计时迈入了第五十九天。在历年真题与张宇八套卷的爆锤之下无力挣扎的我，早已放弃了生的希望。曾经的一切期待在现实的鞭打之中，悲伤而无奈。不愿意面对现实，又不得不面对。

曾经的我，总喜欢一次又一次删除过去。或许以为，删除了空间的历史，抹掉了过去的痕迹，曾经的伤痛就能够不存在，曾经的错误就能够避免。但是时间不能倒流，当大学的时间片流转到下半部分，恐惧的不再是过去，而是未来。习惯了逃避的我，可以用眼不见为净来躲避过去，却又如何躲避未来？

我的空间封面已经很久没换过了，依然是“说一说我们的梦想”。可是，梦想这样一个遥远的词汇，早已成为了不敢也不愿提及的悲伤。从大三开始至今，我保留了120余条动态，似乎不算多也不算少。重载一下，这四百多天的日子。希望未来的人，不要再犯下如我一般的过错，铸就考研失败的恶果。

一念心动，一生不悔，一朝爱慕便落得卑微。一缕执念，不敌岁月，化作了灰，但愿寒夜里不再流泪。

一念心动，一生不悔，一朝爱慕便落得卑微。一许流年，尽数化作，浊酒一杯，回首再尝那苦涩滋味。

招生办出乎意料地公开了实时赔率，我押的注赔率是不高不低的6.55。相比相关专业最热门的30.5着实不多，相比同专业课冷门专业的1.32却也不少。

实验室里的SYN6288传出了内卷的声音，让无心学习的我心里又多了三分沮丧。想如何，又如何，失败了，还能如何。再不济，码农也不至于失了工作。

二一年的事情，二零年的人想不到，也不必想。
