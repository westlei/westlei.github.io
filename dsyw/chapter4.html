# 周期性失败

2023年开始了...

虽然我一直嘴上说今年一切都会好起来

但是眼下的日子怎么也不像是好的起来

我很坚定的相信我自身的运势周期规律

大概是26个月大周期 13个月小周期

现在是2022.1-2023.2的下行小周期


2020.12-2022.1的上行小周期

我从考研失败的低谷里爬起

然后在无疫的春天 拿到了大厂offer

并顺利成为优秀新员工 奖励不断


2019.11-2020.12的下行小周期

我在艰难的课业里选择摆烂

然后在该准备考研的时候 碰上了疫情

并经历了一段体验非常糟糕的感情线


2018.10-2019.11的上行小周期

我从声名狼藉的环境中脱身

然后在课业完全失败后 开启竞赛生涯

并加入了一个让我变得成功的实验室


2017.9-2018.10的下行小周期

我来到了一个全新的环境里

然后始终无法完全适应 专业的生存状态

并招惹了一群若干年后还在骂我的人


2016.8 2015.7 2014.6 2013.5 2012.4

...以上周期至少可以追溯到2012年

按照历史经验来看 下行周期的最后

我的事业和感情都会陷入崩溃

而我也明显的在下行周期里一路狂奔


下行周期的开始 是上行周期的顶点

上行周期的成功之后 我还能安生一会儿

但做啥啥不行 干啥啥失败的挫败会伴随

下行周期的四个月后 一段感情线会开始

然后在周期的终点 感情就会陷入崩溃

与学业或事业一起 失败到无可挽回


距离完全失败只剩下最后一个月了

失败的征兆和已经初步展现了出来

从一开始就十分混乱的方案和分工

在关键时间点之前忽然跑路的组长

而下个月的回片 就会是崩溃的终焉


感情线的失败往往都来的很突然

在一切真正发生之前 我都猜不到为什么

但如果如周期般 那也就只剩一个月了

下个月是她的生日 会发生什么呢...


虽然我知道崩溃即将到来...

但我似乎什么也做不到...

这个世界上 有很多事情

当一切失败的条件走向成熟

又如何去挽回呢...

# 塘桥夜话

来上海快三个月了。

到深圳三个月的时候，我已经差不多走完整个深圳了。然而高德100%的深圳探索度，也不过1961平方公里，上海目前86%的探索度，就已经达到了6511平方公里。上海，之所以被称为大上海，自然是有其原因的。

深圳所谓的远，不过沙井松岗，不过光明公明，不过观澜平湖，不过坪山坪地。顶多也就大鹏南澳，也不过六十来公里。深圳所谓的远，要么是好山好水，比如光明，比如大鹏；要么是工厂林立，比如沙井，比如坑梓；当然也有两者兼有的，比如龙城，比如观澜。因此即便是深圳最偏远的角落，也能感受到一种城市的气息，也能看到高的离谱的房价。

而上海不一样，上海的偏远，大抵是真正的偏远。毗邻昆山的安亭、白鹤、盈浦、金泽，有地铁的需要耗时超过一个小时才能进入市区，没有地铁的更是完完全全的原始村落。破烂不堪的乡镇道路上阿叔阿婆闲聊度日，甚至没有什么年轻人的影踪。不远处的昆山高楼林立，宽敞整洁，一时比上海更像上海。

青浦嘉定的偏远乡镇大抵是幸运的，毕竟还有11和17号线能够照顾一下。远离上海又没有强市毗邻的南部诸镇，几乎是把田园两个字写到了地图上。朱泾、廊下、金山卫，柘林、海湾、芦潮港，狭窄破旧的乡村公路将他们与上海相连，很难分辨这里是上海还是浙北某个小镇。

种植自然农作物都田地是不会自然生长出天价楼盘的，种植了人口和产业的才会。而出生率低迷到仅高于黑龙江的上海，并没有足够的人口可以填充到城市的边缘，没有足够的产业可以照顾到这些远离上海的角角落落。金山卫和奉贤海湾的楼盘不过一万多一平，但无论人们多抱怨上海的房子买不起，也没多少人考虑这些价格只有市区1/5甚至1/10的地方。

无论再怎么修饰措辞， 房地产终究也只是一个投资标的物，没有价值的房产并没有什么人会去力捧。无论再怎么说自己“刚需”的人，都逃不过这个现实。同样，所谓求学、工作、结婚、生子，无一不是生意，无一不需要算计。求学算计学成能否值回学费，工作算计涨幅能否覆盖预期，结婚算计对方能否有所助力，生子算计养大能否提升阶级。一切都是算计，一切都是投资。

在当前的经济周期下，如果投资的标的物只是财富，忽略其他外部因素，那么因为活着就要消耗财富，活着就会财富缩水，所以一旦停止获得收入，就应该立刻remake。但是任何正常的人都不应该用单一变量来衡量自己的人生。人生本就是一个组合优化问题，而我现在显然是需要优化的一个状态。

或许，工作不重要，生活才重要。但我似乎没有什么生活，或者说，我在逃避生活。我逃避生活以及与之有关的一切，似乎只有在工位上debug垃圾代码的时候，我才是我。因此即便我不太想做现在的工作，我还是得做，因为我没有理由说服我自己不做。

路过塘桥，突然想起之前听的一首不才的《塘桥夜话》。虽然此塘桥非彼塘桥，昔年的塘桥在如今的闵行七宝古镇。但终究也是上海滩的故事。

旧上海街头拥吻

九龙渡海小轮

七十年后浮生绮梦难记清

只夏夜星空几寸

似他眉眼动人

牵动我贫瘠魂魄一刻伤神

也许，如果有机会，我也想这样，闲下来，与爱的人相伴相守吧。但我真的，有这个机会吗？

# 一些毫无逻辑的春节后碎碎念

历史是一个很玄幻的东西。严格按照唯物主义来说，早于当前所发生的一切都可以算作历史。但如果从唯心的角度上来说，只有古老到无法记清细节无法翻案的东西才可以称为历史。

满江红的所谓历史，既不属于前者，也不属于后者。他没有在真实历史上发生，故不满足唯物的历史。他的细节来自于杜撰，清晰可辨，也不满足唯心的历史。总而言之，他应该是一个很新的东西。

这也是我对这部片子的印象，既不是历史正剧，也不是主旋律作品，也不怎么满足什么爱国主义教育。严格的归类，大概是悬疑，或者说喜剧。喜剧视角下，所有人物各司其职各负笑点，但放在悬疑视角下，就有一种为了反转而反转的感觉，上一次产生这种感觉还是原X的3.2主线。

这种东西适合看一乐，但不适合细想。大量行动细节去铺画整个计划的宏伟，只会暴露出各种各样的瑕疵。计划永远是有缺陷的，没有plan bcdefg以及随机应变的计划不是一个完整的计划。但是这一切都不会写在剧情里，剧情永远只有所谓的妙算如神。

庙算无遗，靠的不是智慧，而是《珊瑚宫大锦囊全书》这种无尽的穷举。正如我的一切行动都不怎么计划，结果也并不怎么计划一般，我如今的生活，也不是某种计划之中的产物。我当前的情况，不在任何一种计划之内，自然也没有任何相应的应对。

对于目前的工作我说不上多喜欢，但似乎也没有什么我特别喜欢的岗位，那就维持现状就这么干着。对于...我也说不上多喜欢，但似乎也没有什么我特别喜欢的，那就将就着感情基础这么来往着。我知道这不是什么很好的人生态度，但总归说不失为一种选择。

如果算虚岁的话，过完年，我也就二十四岁了。二十四年，不管怎么说，也是一段挺长的时间了。就像家里那些二十四年前的家具管道，也都因年久失修出现各种问题。不要说二十四年，暴雪进入中国十四年，也塑造了多少人的回忆，改变了多少人的命运。熟悉的事物渐渐消失，这也是我们所不得不面对的现实。

去扬州的时候，我妈说她二十岁时候来的扬州是怎样怎样的，但那也已经是快三十年前的故事了。那时候，我妈才刚刚遇见我爸，而一晃，我也都二十多了。我爸和我妈算是校园恋爱，而且是不顾我外公外婆的反对毅然成婚的。因此他们也从来没有经历过相亲、催婚，也没打算给我安排这些。

但我既没有我爸那时候的财大气粗，也没有我妈那时候的才华横溢。爱情故事，似乎发生在我这里并不现实。此后的一些日子，我大抵会更加自闭一些。如果有什么的话，那就只能是，再说吧。

也得感谢最近聊天一些朋友的问候和鼓励。但是我也不得不说，这样的我，可能只会令你们失望吧。2023，奋斗哒咩，进取哒咩，该怎样就怎样吧，能干多久就干多久吧，反正也没什么值得在乎的人和事了...

今天看到一句话，是这么说的： 如果工作环境对碳基生物不友好就换份工吧，年轻人大可不必这么卑微...

今天又看到一条微博，是这么说的： 一上来就把伴侣在心中置有极重要的位置是不可取的，那种喜欢不单单是对那个人，也包含了自身压抑了很久的对爱情的渴望以及占有欲...

所以结合一下就是我现在的情况： 虽然我知道连上八天班对碳基生物不友好，但是还是卑微地去加班了。毕竟别人都有伴侣，我只能用加班去压抑对爱情的渴望以及占有欲，去抹除她在心中占有的极重要的位置...

# 癸卯年

过完正月十五，这个年就算过完了，时间正式步入了癸卯年。

生活并不因为新年的到来而发生了怎么样的变化，唯一变化的可能就是朋友更少了。现实生活与我脱节的愈发厉害，每天就剩下上班、原、舟还有各种包括但不限于战鹰柯洁连笑和九转大肠等抽象主播与烂梗。

很难说这是一件好事还是坏事。曾经看到一种互联网克苏鲁说，也就是跟抽象互联网世界在一起久了以后人会魔怔，就像被古神凝视后疯了一样。但是现实世界真的没有什么意思，哪怕几乎每周都会出门接触现实，我也并不愿意去跟现实的任何人打交道。

面对现实无非是让我的人动了起来而已，一切仿佛8d现实电影，面前的人可能是某种数字生命，或者也可能是人，但都无所谓。他们各自按部就班做着自己的事，与我无关，当然也不必有关。

站在程序员的立场上，如果有朝一日数字生命成为现实，那我一定是数字派的一员。没有人的文明毫无意义，我都活成这个样了还要个鬼的意义。现实的程序员其实是个很尴尬的职业，干着硅基生命的活，好让硅基生命有朝一日替代自己。挣着比碳基生命平均水平高出不是很多的工资，让更多碳基生命的钱变得不值钱。

这个行业本身就是为了卷而卷的，是一种资本主义高度发展后生产力自发迭代需求的产物。之所以程序员的工资还可以，是因为程序员毕竟在先进生产力的最前沿。自程序员以上，都是脱离了生产者角色，能够掌握管理和分配的食物链上游。而程序员以下，则是随着生产力进步而逐渐贬值的传统生产力代表。

就像曾经第一次工业革命时代的机械工人，他们可以用机械造出更高效的机械来提升生产力，他们让手工业者逐渐走向淘汰。就像曾经第二次工业革命时代的电气工人，他们通过引入电作为能源和控制单元极大的提升了电力本身的生产能力。而第三次工业革命时代的代码工人用代码实现了精密加工，实现了代码自进化，占领最新的生产力前沿。

但生产力发展的过程永远是短暂的，生产力原地踏步甚至是退步才是人类社会的常态。这也就是工人怎么样都只是工人，除非能够走到食物链上游，掌握管理和分配。当然，消费也是分配的一部分，所以掌握了消费端的互联网行业能够收获超额的利润。

氪金手游和直播打赏，本质上是抢了赎罪券和香火钱的生意，网购外卖和打车住宿，那就是更古老的中介行业了。无论怎么制造概念，消费端新事物的本身一定是某种旧事物的迭代升级。但生产端的新事物，大抵一定是个新的事物。

或许，没有人的文明，才能真正的进入共产主义吧。无论生产力如何发展，只要是有人的文明，都不可能做到让每一个人都享有无限的资源。因为在一部分人看来，另一部分人本身也是资源。而数字生命，才可以做到一个除你之外全是npc的世界，满足人类所有的欲望。

众所周知，人类有三大欲望，食欲，X欲，睡眠欲。三者分别可以模糊对应到生存、发展和享乐。生存意味着要占据尽可能多的资源，并阻止其他人获得资源。发展意味着其他人要成为你的一种资源。享乐意味着什么也不用做就能获得资源。有人的文明，一定会因以上的欲望爆发争斗，使得人类难以长期可持续进步。

对人类的管理，最终也会落实到对欲望的管理。曾经的部门天天画饼，抓住发展的欲望进行各种各样的pua，而最终升级加薪的大饼也没几个人吃到。现在的部门从不画饼，但总是用各种方法渲染出一种随时会开除你的氛围，制造生存焦虑。而不管哪一种，干六天每天十二点走的人都不见得能比干五天每天六点走的人拿得多、升得快。

之前做过一个乱七八糟的小测试，反正就是答一堆题给个结果那种。结果是一个横纵坐标上的点，横轴是从同性到异性，纵轴是欲望弱到强。我的位置是最右下角，也就是100%异性，0%欲望。

我不关心别人是不是同，甚至于我们部门还有个大哥(或者应该叫大姐)是药娘。但是反正不要在我这整什么性别别卡太死啥的，反正我不接受。至于二次元老婆还是三次元老婆其实无所谓，我都没啥想法，现在看涩图都不怎么能冲了，大概就是，没啥欲望吧。

# 沙漠书『仇敌仿佛众水翻腾』

新开的卡池跟我关系不大，毕竟胡桃和夜兰我都有了，前两天又歪了琴。但是看米游社发现沙漠书『比勒琪斯的哀歌』发布了最终章，据说有大西王之风，还是要看看的。

然后就发现这任务依赖的前置任务一打，不得不肝了三个钟终于启动了任务。开篇就是传统诬陷剧情，直接让婕德杀性大发，干掉了一堆愚人众，差点也想干掉爷。最后不负众望执行了三光政策，把主母一刀剁了。

讲道理剧情也就那么回事，但是婕德挺可怜就是了，好不容易找到一个似乎是家的地方，却又被利用了这么久。当然派蒙的煞笔对话还是想让爷直接给他下锅炖咯。愚人众经常也挺倒霉的，经常老老实实打个工就给人砍了，或者干些不可能完成的活。

抛开一切客观因素，我还挺喜欢婕德的，当然可能考虑中间亮血条，所以造型设计不怎么样就是了(讲道理沙漠的npc和镀金旅团大部分都比婕德好看)。「我希望我们下次相见，会是在一个阳光更好的日子。」

就像，我以为我应该已经被删了。但是我依然去祝她生日快乐，并且没有看见红色感叹号。也许，还能做朋友吧。也许，什么时候又会再见呢？
