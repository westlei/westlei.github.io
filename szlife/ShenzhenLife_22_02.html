# 深圳的每一天

2022年2月篇

# 2022-02-01

过年了，祝大家新年快乐啊。

好不容易没疫情了想好好逛逛，这下赶着年关又来了，00:09准时上星。

新年收到了一束花，写错了地址，害得我不得不从坂田跑到横岗去拿。

# 2022-02-02

继续更，过年疫情没敢乱跑，市里随便转转

初二从坂田出发，翻过银湖山进城，然后再爬莲花山，穿过市民中心购物公园到福田红树林，最后走到下沙回家，一共走了三十公里。

银湖山

莲花山

福田红树林

# 2022-02-03

初三歇了大半天，傍晚去爬了南山，荔林地铁站到海上世界地铁站。南山上看蛇口夜景真的不错，就是好冷还飘雨。

南山

海上世界

# 2022-02-04

初四去打卡知名的吊扇西站和平南铁路。作为一个铁路边长大的人对这种穿越城市的单线非电气化铁路还是很感兴趣的。南头到西丽到塘朗，晃晃悠悠半小时到坂田，然后走甘坑平湖出深圳。不过家门口的宁芜铁路已经提上了改造进程，平南铁路应该也命不久矣了。

南头火车西站

吊扇西候车室

复古风站台

五和大道

甘坑

东莞站（石龙侧）

到东莞打卡了一下东江江边和地铁二号线，最后虎门站返回。

东江江滨

地铁二号线蛤地

虎门高铁站

# 2022-02-06

火车上看到甘坑小镇挺漂亮的感觉，于是初六去了趟。

甘坑古镇

古镇水街

其实这种商业化的“古镇”在江南一带早就被玩烂了，于是去了隔壁平湖生态园转转。不得不承认深圳的自然环境还是保护的很好的。

# 2022-02-12

继续更，周末捞了一个学弟陪我一起逛

南头古城蛮无聊的其实

中山公园，每个城市都有的东西吧

深大计算机院，一个本鼠鼠高攀不起的地方，对面腾讯滨海大厦（羡慕啊，我也希望有朝一日能去，但当初面试没过，太难了）

对岸确诊一天四千，都快赶上最严重时候的湖北了，太惨了。

深圳湾公园随拍

皇庭广场吃饭，看到有人遛。。。羊驼？ 不是玩具，是活的。。。

这是什么新宠物吗？

# 2022-02-27

二月最后一更，实在是疫情闷坏了想出去走走，又不敢去人多的地方，那就南澳随便溜达溜达。

对面疫情一天两万多了，所有沿海地区都严防偷渡，沙滩全都不开了。

听说坪山聚龙山的樱花很漂亮，季节也差不多了，可惜完全没开。随便拍几张。
