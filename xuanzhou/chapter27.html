# 无梦无念

# 乱

日子一天天乱七八糟莫名其妙的

总有一些乱七八糟莫名其妙的人找我

我并不想认识乱七八糟莫名其妙的人

也不想掺和乱七八糟莫名其妙的事

可我自己就是个乱七八糟莫名其妙的人

做着些乱七八糟莫名其妙的事

操着乱七八糟莫名其妙的心

未来就好像在写字楼里轻松能望见的

那一抹滚滚流去的长江之水

但是一条扬子江大道的存在

让咫尺之遥的距离变得分外遥远

就仿佛我和她 和她 和她 和她。。。

生命的本质就是这么的滑稽

纵使心向明月 明月多半还是照耀沟渠

每一个真正意义上吸引着我的人

大多都像我一般 不是什么所谓的好人

太单纯善良的人 我可能更愿意去保护

而缺少一种所谓的感情在里面

真真假假假假真真 我分不清真假 谁又不是

物质的世界里吵吵闹闹 MONEY TALKS

而我对钱不感兴趣 没有梦想 钱也没用

也许我应该努力去做些什么 可我不愿意

看过了这个世界的黑与白 我愿活于混沌

看过了这个世界的光与影 我愿活于清风

暗淡如我 从来也不曾可能会发哪怕一点点光

但我羡慕明月 它拥有本该属于其他人的明亮

# 觅风

山与水 是所有自然风景的灵魂所在

山 或险峻 或奇伟 或黄沙满布 或风吹草动

水 或平静 或澎湃 或惊涛拍岸 或碧水蓝天

所谓千种风景 万般变化 皆寓于其中

自由的风 吹过山野 吹遍江湖

纵使狂风大作 山巍然不动 而水早已波涛汹涌

即所谓无风不起浪 风雨不动安如山

我自乘风而来 而又乘风归去

可风的偏爱一文不值罢了

# 似梦非梦

当熟悉的韵律在列表里再次响起

思绪仿佛又回到了夏天 阳光与暑气

江水没过圩堤 而我隐在山间 躲在云里

难以逾越的 不只是那不到两英里的距离

我只能 忘却岁月朝夕 隐去过往记忆

我不再是我 也不会成为曾经的我自己

用平凡的碌碌无为的社畜时光 来麻痹


当一个人身在梦中的时候

他如何才能知道自己正在做梦呢

或许现实的一切都是假象

只有梦里发生的才是真正的现实

现实里会丢失了梦的记忆

就像在梦的时空也会忘却了现实

是现实中梦 是梦中梦 还是梦中现实

天空是蔚蓝色 窗外有千纸鹤

虚虚实实 实实虚虚

现实的我可能早就死了 可梦里还活着

# 社畜

早七的天色蒙蒙亮 走出家门等待公交

穿过城市的中心区 终点是另一端城郊

看着车厢越来越满 是新街口或夫子庙

但很快又逐渐空旷 早晨不会那么热闹

每天早上朋友圈里 总是有人会起更早

单词打卡勤耕不辍 这就是自律的大佬

优秀的人发愤图强 而我只能卑微弱小

感情路上坎坷无助 期待的人她不够好

转眼已经到了2021年的二月了

多事的庚子年即将过去

每一个二月 都既是结束又是开始

憧憬新一年的爱情 遭受新一轮的伤害

不管十八岁还是二十岁沉溺在期待幻想的我

还是十九岁或者现在失去了期待向往的我

都是卑微而失败的我 是无能为力的我

# 是福非福

2020年的说说 现在看起来都充满了滑稽的意味

天天熬夜的我 也不过是为了未知的未来而困惑

身在“福”中不知福 是我了 没毛病

江南百景图松江府要上线了 然而仔细一看要37级

我只有36级 杭州还没咋开发 就很难受

友谊的保质期就是很短很短

希望也总是很快就会变成绝望

明月高悬 我却已经无力去追求去寻找

大约希望 所有我喜欢过的人 余生幸福

是我不能带给她们的那种幸福

昨天跟一个朋友聊了聊

执念和伤害永远是相关的

也许年轻的我们还应该不顾一切冲一次

但我早已没有了年轻的心态 瞻前顾后

为所有一切的未知而恐惧

但是 太害怕受伤害 往往最后真受伤的还是自己

所以还是对自己好一点吧 想干嘛就干嘛吧

或许 会有那么一天 真的有一个值得的人 落入心上

如此 她会值得一场隆重的表白

“每一段我认真地过的感情 我都会尽力的去挽留

然后最后无计可施 潇洒离去 不为别的

只是为了以后回想起的时候不会后悔

当初没有回头望一望 努力一把

我的人生 可以有错过 但不会有遗憾”

# 新年快乐

又到了一年一度的璃月港海灯节(×)中国传统农历春节的时候了。

过去的十五年里，我有明确印象在南京度过的春节，不到五个。为什么会很少在南京过年呢，一大因素就是亲戚特别特别少，以致于会有那么点无聊的感觉。南京禁放烟花爆竹似乎也有七八年了，比较冷清的家里，大家看着无聊的节目，似乎也就这么回事。

上一次在南京过年的时候，好像还是五年前。对上次过年最大的印象好像是年三十的夜里，从理工大里奶奶的老房子走回秦淮区这边的家，一路上数着一座座小区里点亮的灯火。星星点点的，大约一小半吧。此外似乎就没啥了，一切都平平无奇。

那年好像是要小高考了吧，寒假安排我们去南大补习小高考科目。好像下过一场挺大的雪，坐车路过紫金山里，漫山遍野银装素裹，煞是好看。我喜欢这雪，但也知道这雪终究要化。大约也像是我当时有过的一点卑微的喜欢，喜欢隔壁班的那个整天欢欢乐乐的女孩，却知道永远也不可能有结果。

看起来感情是我生命中十分珍贵的一部分，但是珍贵的前提其实是稀少。我的感情总是淡薄而疏远，顾虑多于执着。但偏生我又是一个感性重于理性的人，多数时候被这稀薄的感情所控制、所影响着。非理性状态下的我，是我自己都难以理解的我。

来到大学的我，一开始想寻找的，其实是她的一面镜子。一厢情愿的我，依然觉得，只要能找到一个她在这里的投影，似乎一切就应该会是顺理成章的。但后来的事告诉我，这样一个光芒四射的人，她不会成为一个能够保护你的人。她的人生会很精彩，但却无关于你的无奈。

于是，我怀揣着我碎裂的前程和梦想来到了珠海。在这座南国的小城里，骑着车看着海，逐渐释然了心态。我认识了越来越多的人，也因着各种各样的机缘巧合而萌生好感。有的是有距离感的好感，作为朋友的喜欢与感动。有的是没有距离感的好感，作为兄弟的亲切与陪伴。我珍惜这样的感情，却很难维系这样的感情。很多感情变质了，但更多的感情，走散了，再也看不见了。

诚然有一部分的异性朋友，我打认识开始就有了些别样的想法。但对于绝大部分的朋友，感情都是慢慢才开始变质的。变质的感情不一定不可延续，漫长的时光也有可能会回复到未曾变质的模样。但更多的，只能无力地消散。

渐渐地，我也忘了什么是喜欢的感觉。我只知道，凡是我起了心思想要去追求的人，总会在第一时间与另一个更值得的人脱单。直到我遇见那个，陪伴了我大半个2020的人。我以为我找回了喜欢的感觉，却只是一场属于我一个人，不，一只小丑的马戏。我以为我可以一直等下去，可我终究还是接受不了一个属于我厌恶的人的结局。

也许我真的应该认清自己，认清自己不配也不能去喜欢任何人的现实。最后半年了，回想我的大学生涯。似乎除了充实，更多的还是滑稽。我喜欢的人，值得更好的人；我等待的人，配不上我的等待。而我自己，只配单身。

于是纵使想过千言万语，也只会也只能变成四个字：

新年快乐！

# 梦亦无欢

当山峰没有棱角的时候

当江河湖水消失不见

当冬天的雷声震耳欲聋

当夏天的雪晶莹透彻

那是一个伸手不见五指的白天

那是到了故事的最后

男女主角决裂 老死不相往来

你可以不喜欢你不喜欢的东西

但你得允许他的存在

你可以喜欢你喜欢的人

但你也得接受她可能与别人在一起

情人节的故事未必都是单纯的悲剧或喜剧

或许看似悲剧的结局才是最好的喜剧

当我看见列表里一个又一个十分优秀的人

倾吐着所谓的单身的情人节的烦恼

她们所渴望的是更加优秀而温暖的怀抱

却与我没有任何一点点关系

哪怕曾经喜欢 哪怕有过心动

但我清楚 我这样的 不配

又是一年二月十四号了

这也算是一个对我来说意义特殊的日子

一次次的开始和结局轮回里

我总会把这一天作为心理的锚点

割舍掉那所有过去的情感纠葛

新的春天 新的机缘

以及可能的 新的 喜欢的人

所爱的人 或许就是梦里也不会存在吧

# 小城雨后

说好的雨未曾相遇 便寻思远赴溧水

却在溧水城里遭遇了一场大雨

雨后进山 又是另一番风景

无色无想 无相无形

是无想 故无想

何必想 不必想

五十公里的距离 似乎已在另一个城市

小城的夜色景象 仿佛那年盛夏的溧阳

夜幕拉开 灯彩四射 河上风光 梦里水乡

未知风动 未知意浓 也不解 是否心动

也许我想得太多 但我只是不愿意看到

本来可以过的更好的人 因为我 而受苦
