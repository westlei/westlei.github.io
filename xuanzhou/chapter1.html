# 无觅芳踪

# 无觅芳踪

入秋渐微凉，繁华已然消亡。我在远方眺望，耗尽所有目光。不思量，自难相忘。秋末，宣州，天已凉。曾经，往事，犹难忘。庸庸碌碌，四方奔忙。发现大学，不过这样。

如今不再是高三，没有了那时的压力，失却了初心和梦想。努力没有方向，奋斗失去意义。身为万千学渣中的一员，侥幸脱身于高考苦海。学习一般，看着遍地的大佬瑟瑟发抖的恐惧。生活充实，因为总是有不知从何而来的忙碌。只得抱着一份平常心，相信得过且过。不再向往所谓未来，觉得不过是一场空。眼下的故事，只有未知和迷茫。

于是，便怀揣一份虚假的自信，抱着几分假装的热情。羡慕着，看身旁成双成对的同学。平凡着，做人群中微不足道的自己。像我这样的人，有什么理由取得成功? 有什么理由站在舞台的中央叱咤风云? 有什么理由在十七岁那年的雨季,拥有一份真摯的爱情? 二十岁是人生最美的时光，但二十岁的我可能已经没有了努力奋进的希望。流年似水,人生如梦。天亮之后总是潦草离场，清醒的人最荒唐。

虽说绝对不能失去梦想与奋斗的初心，韶华易逝少年时，风华正茂，应当去中流击水，欲与天公试比高。但现实让当初的理想淹没,劳碌把曾经的初心摧毁。何苦奋发图强，何苦努力向上？年轻人，终究还是too young。

# 奈何纷乱

流年似水，君莫虚度年华。虽然现实很残酷，但梦里还可以美好。寒冷的天气，困顿的日子，把自己沉醉在梦的海洋。仍不时去想起那长江尽头的故人，仍不时去想起那碑亭巷口的日子。看似黑暗的三年时光，因着她的存在而充满了阳光。如今，她已彻底知晓了我曾经的爱慕之情，却也让这些许思念，永远没有了结局。

当一段情感长存心底，却在行将表达之时没有了倾诉的目标。当一段相思久居心头，却在面临消失之际暴露在了光天化日之下。前一种，给人留下的，是久久难以磨灭的伤悲与痛苦，一如那不知名的花朵的调谢。而后一种所带来的，却是另一种难以言说的复杂情感。情至深处,若不曾更进一步，则必然凋谢。然而未放之花的凋零，或是自然，或是人为。那被人从技上掐下的未放之花，它的苦，却有谁人能知。

隆冬的号角裹挟在萧瑟的秋风中悄然来临。一阵又一阵的寒风，吹透我的身体，直扎那内心最柔软的部分，将它冷冻成冰。冬，终究是来得太快，来得太急，以致于没有足够的时间给那柔弱的心包装出那伪装的一层厚厚的壳。脆弱的的伪装，纸糊的坚强，在寒风的面前，只有一触即溃的绝望。七月上，我心伤；至今日，未绝望。心头，仍抱有过那么几分虚妄的幻想。爱慕，是我心头始终无解难舍的假象。凉凉，荒荒，苍苍，茫茫。星辰满天，没有一分是属于我的微光；叶落遍地，处处皆为我这般的凄凉。

曾经那个花开的三月，曾经那抹娇笑的倩影，都已沦为了曾经的往事，沦为了记忆之中那试图毁去的点点滴滴。天已凉凉，爱已茫茫。爱一个人，感受她所有的痛苦，为她烦忧。爱一个人，心疼她一切的低落，给她关怀。爱一个人，享受她在怀的快乐，但更重要的，是理解她，化解那所有负面的情绪，让阳光充盈着她的心。爱，已经远去。爱，不会回来。

清冷白月光，萧瑟晚风凉。独游青山上，不思扬子江。且让往事随风，在时光的冲洗下一去不复。然而，世事无常，奈何纷乱。想忘却的，并不那么容易忘却；想忽略的，并不那么容易忽略。愈忙，愈想，愈是情伤。

# 朔风呼啸

熟悉的面容不再，但那熟悉的声音却仍不时耳畔回响。依旧身形高挑，仍然长发飘舞。可那不是你，永远不是，也不会是。她，不过是我记忆中那永远难以磨灭的那一抹印记。一山，一水，一宣州；一人，一路，一心忧。人，还是应该轻松写意的活着，注重眼下的点滴孤单和幸福。该当它不存在的，就当它不存在吧。

虽然，她，跟她，真的好像，好像。从声音，到性格，到外表，都很像，很像。但是，我不会再如曾经般义无反顾的投入。累了，倦了，伤了，远了。过去的已经过去，何必追念？她，像她，而不是她。过去的，已经不会再回来了。

风凉凉，心亦凉凉。薰化路的勇士，浪完之后，得直面工图惨淡的人生，还有高数淋漓的鲜血。四野荒郊，大雾茫茫。朔风呼啸，心已凉凉。爱已经远去，孤不期而遇。天边泛微光，灿烂现红霞。前路漫漫。越过黎明前的黑暗，放下脑海中的相思。不去怀念伊人，也不去想那些相似的人。前途有多远，这路有多长？且待我走到尽头。

# 一城 一景 是温柔

转眼，入主宣州已有多时。眼瞅着夏尽秋来，秋过冬至。然而宛陵湖，仍在那里。树叶，渐渐变黄；芦苇，不时飘荡。一山，一水，一宣州。一城，一景，是温柔。独自来到这熟悉而陌生的江南小城，熟悉的，是气候；而陌生的，则是人。天南海北的人汇聚于此，是缘分。不同的人，代表着不同的风土，不同的人情。而大学，如同这湖水，包容着一切，逐渐消弥着差异与隔阂，汇聚成一个集体。曾经多少世态炎凉，曾经多少黯然神伤。感恩过去，感谢现在的你们。那些情伤，那些心伤，能过去的就让它过去吧。可是，过去，又谈何容易？

清冷白月光，萧瑟晚风凉。独游青山上，不把高数想。且让一切随风，在时光的冲洗下一去不复。然而，世事无常，奈何纷乱。想忘却的，并不那么容易忘却；想忽略的，并不那么容易忽略。愈忙，愈想，愈心伤。心飞扬，惨死在那高数的沙场；身透凉，魂散于那成绩的悲伤。人惆怅，灭不掉那长久的迷惘；天苍茫，说不尽那永恒的绝望。荒唐。

喧喧，闹闹。冬悄悄的来到，欲要寻求那温暖的怀抱。匆匆，忙忙。想那数月来心头的感伤，不忘高数那彻骨的寒凉。一切忙碌渐渐归于平淡，一切繁华渐渐走向消散。在一场场考试中徘徊，在一次次比赛间游走，自由到难寻几分属于自我的闲暇，这，既是大学。

# 越忙 越想

从未得到主流的认可，从未曾在人群中快乐，敏感，多疑。十载金陵生涯，心，饱经风霜；身，无处安放。远去，兜兜转转却又回往了江南。那远去的倩影，三载的暗恋，一切相思，尽在那人，一手终结。也无妨，逝去的逝去，人总要成长。放不下的那些记忆，总会在时光的冲洗中褪色，在世事的变换中消却。

也许，繁忙能掩盖那内心的伤，可惜，越忙，越想。永恒的宇宙，无尽的世界，每个人都是过客，每个人都会留痕。指望着让自己完全消失在众人的视野里，毫无可能。那么，便日复一日地水群，仿佛水多了群自己就很有几分存在感。那么，便月复一月地辛劳，仿佛做多了事自己就很能够得到认可。习惯了孤单，熟悉了寂寞，想找一个朋友不易，欲求一群知己更难。大学的班级，似乎没有曾经的那么紧密，不如过去的那么熟悉，但是大学，少了几分急功近利，多了几分人情气息。在这样一个地方，有了这么一群嘘寒问暖的同伴，有了这么一群互相帮助的朋友，我，其实很幸运。让回忆里的阴霾过去，看前途中的阳光明媚。

巍巍敬亭，幽幽宛陵。山水宣州，人杰地灵。前路漫漫，无论多少坎坷，我心归处，即是天堂。书山苦行，常常坠入深谷；情海欲渡，往往浪滚舟覆。语文，英语，思修，高数。大机，军理，政经，工图。多少沟谷，尽在不远处。欲海情路，逢海啸，天翻地覆。剩下的只有思慕，心中痛苦。寻寻觅觅无处，怎得倾诉？一切罢休。在宛陵湖畔，心中空无；往敬亭山中，感受孤独。忙忙碌碌，似乎事情无数。却消不了相思情苦，忘不了音容笑貌。她的声音，多么像那人。这，让我如何忘却旧事？一颦一笑，不令我倾倒，而是勾起无尽的思念，唤醒心头的相思。凄凄，思慕，心碎到奈何，剪不断纠葛，越想忘记越深刻。忘了，醉了，以为放下了。却在梦醒后，想起你，泪滑落。阳光消失的夜里，心里，却仍是不住的伤。

# 淮阳多病

淮阳多病偶求欢。多病时，每忆起那淮水之阳的姑娘，想想求欢未得，病愈发深沉。三载烟消云散，想金陵，秦淮，钟山；玄武，江畔，无处寻欢。如今宣州，唯有宛陵湖水，化解一心忧愁。独，驰骋在青弋江大道的路口。孤，消失在那乡间小道的尽头。她的声音那么像，让我阵阵心痛。又到十二月，想三载前那萌发的情意，想三载间那不灭的思念。可惜海啸摧毁了一切，只余陌路。心烦忧。冬，充其量只应是身的寒冬，不应让心也一片冰凉。真的不想听见她那让我心中微动的声音，却又不得不听。难忘。情伤。心凉。既想，却又不想听见她的声音。那声音如此相似，催人回想往事。我既想远离她，以免引发心痛；又忍不住亲近，获得些许愉悦。我到底该怎样？是接近，还是远离？答案，无从寻觅。

转眼间又到了这霜落的时节，白茫茫的大地与呼啸的寒风，绘就了肃杀的冬。此景犹在，此情仍存，只可惜，芳踪无觅。世事无常，奈何纷乱。几多相似，几重相思。

寒风冷雨夜苍茫。感情伤，结愁肠。宣州久住，故人何时忘？相逢何必曾相识，身颓废，透心凉。历尽情劫与沧桑。心飞扬，再启航。物是人非，思念又何妨？独为天涯沦落人，不思量，去流浪。芳踪无觅，那昔日的一切往事定格在了那六月的尾声。如今所见，再多相似，不过徒增烦恼。那心，一片树叶，却是已然飘落。

金陵太大。百步之遥，寻不得思念的她。宣州太小。全城踏遍，仍旧是孑然一身。六月里，绕过紫金山去那三十里外的淮阳住处，停留，举头望那伊人的方向。冬日中，骑上三环路走过宣州城郊的荒郊野湖，驻守，别无他物只余下孤独。罢了，看往事随风，叹一切成空。敬亭紫金，宛陵玄武，几多相似，几多相思。远眺湖心，叶落片片，终不是我心归处。山间四望，缓缓凋零，空叹得奈何纷乱。

敬亭远望，缓缓凋敝。宛陵湖心，叶落无痕。时移事易，不变的，只有那一山，一湖。二零一七，即将步入尾声。这是最好的一年，这也是最坏的一年。这是奋发的一年，这也是颓废的一年。这是上进的一年，这也是堕落的一年。远离了旧都金陵，奔赴了山水宣州。离别了曾经旧友，结识了几多新朋。斩灭了旧时情爱，留下了些许情伤。得到过，失去过。满足过，遗憾过。爱恋过，心痛过。再过两周，翻开新的一页。二零一八，宜，做出改变。

# 流离和幻影

三个月的比赛，还是走到了尽头。原来以为的苟延馋喘，最终也没能实现。墨菲定律有云：如果事情有变坏的可能，不管这种可能性有多小，它总会发生。事实果真如此，从第一到淘汰，就是这么简单。唉，忘却比赛，忘却逝去的爱，忘却，这即将逝去的一年。淮阳，永远不再。二零一七，永远不再。

厌倦了一成不变的薰化路，何必去游人如织的宛陵湖？越过山丘，看看静谧的彩金湖。荒山野岭处，喧嚣闹市无。前路多险阻，唯我一人孤。人生几多险阻，终究无人陪伴。一切皆空，永远的只有孤独 。失败是注定，缘分是天意。湖畔独坐，心中迷惘。叶落湖水，情消意却。一切的尽头，是凄凄思慕。眼已模糊，远处，似是淮阳飞舞。

或许不知梦的缘故，流离之人追逐着幻影。或许，那只是青春的伤痛与梦幻罢了。美丽的宣州，彩金湖畔的日子，究竟会不会遇到所爱的呢？一切，依然只是未知。尽管是萧瑟的冬日，阳光依旧，美好仍存。前路，纵然无人陪伴，仍旧精彩。湖有千般不同，心之所爱可变。叶落可以再生，心伤亦能愈合。祈愿，平安。

# 一重年关

佛系青年，或许，说的就是我这样的人吧。想做一件事，就去做吧，不去想什么利弊，不去想什么后果。不想做一件事，就不做，没有什么理由，也没有什么推脱。心情不好了，一个人出去走走，发条说说。心情好了，那就这样吧。从一种心情切换到另一种心情，并不容易，所以大部分时候，总是充满纠结与矛盾。学习？好，那就学习吧。学什么，怎么学？看心情吧。或许曾经有过什么看的重的，例如感情之类的吧，现在觉得，也不过就是那样。一切都不过如此，挂个科什么的，没办法就挂了吧。2017，走向佛系的一年。渐渐看淡学习，渐渐看淡各种，最后看淡感情。一切都随性，一切不走心。总有人以为我是大佬，那可能只是一个误会吧。像我这样的佛系之人，怎么可能是大佬呢？不过，佛系之外，也许，还有几分强迫症，一旦想去做什么事，还是会去认真做的。好吧，我就是这样一个人。也许过于高调，那也不过就是一些不走心的伪装。到了该暴露的时候，终究还是会浮出水面的。

自由，曾经我觉得是我热爱的方向。因为生活不止眼前的苟且，还有诗和远方。过了三年，我才发现，所谓的自由，其实只是一些不切实际的幻想。真正若是自由，前程如大雾般迷茫。没有约束的不自由，何来目标与方向？雾也茫茫，前程亦茫茫。

转眼间，又到了那日历本新旧交替的时刻。窗外，一片雾海霾踪，使人仿佛身处幻境。多么希望，金陵十载，不过南柯一梦。终于，有暇回望从前。三载，旧都，已然发生了许多改变。可不变的是，那最初的起点。一切发生，在那碑亭巷口，在那织造府边。爱恋，从开始到终结，归于无言。情窦初起，情欲盛开，情伤难解，往事如烟。这一刻，正式结束了，这一年。过去的一切，都已翻过一页，定格在了昨天。寄希望，新的一年，前途得以豁然开朗。寄希望，新的一年，遇见怦然心动的人。寄希望，新的一年，平和，超脱。继续，孤独地走下去。

霾未散，而新年伊始。动车呼啸着冲进那一片白色，让所有的迷茫在视野里日渐清晰。你若有光，便是远方。念念不忘，必有回响。前程的光芒，必定，坐落在远方。既然，选择了远方，那便只顾，风雨兼程。只要有光，便有远方，便寻得到，那前进的方向。然而，四周，只有一片白茫茫。何处寻觅温柔乡？三生三世，改不了的，是凉凉。湖风冷，心已寒，叶落尽，欢不再。雾锁江南。祈愿，春暖花开时，伤心不再。
