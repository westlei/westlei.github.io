# 网课日常

好想回去看看皖南的山 看看皖南的水

在春暖花开的时节 看尽新安江的秀色

再到那东海之上的小岛 听浪涛的喧嚣

可现实只有雨课堂 无尽作业带来忧伤

或许这便是网课的日常 感觉考研无望

智能车近了又远了 早就不再报以念想

牛头冲近了又远了 仍怀揣开学的指望

看春天近了又远了 疫情下慌张而迷惘

未来也近了又远了 渐渐忘记那位姑娘


Optimistic Award; Fabulous Award;

Mysterious Award; Helpless Award;

It seems good, but I choose Sleep Award.

# 似是而非

虽然大家都说她们并不相像

但我还是有了些熟悉的感觉

也许是确实有那么几分神似

可那便是我所喜欢的样子吧


其实客居的日子本就是这么无聊

就像单片机和嵌入式的网课一般

一潭死水般毫无生气也毫无活力

牛头冲的花开了 想念那里的一切

也包括每一个见或未见的小姐姐

# 旅途

二零二零的深圳旅途终究走向落幕

一切的改变皆源自突如其来的变故

单片机嵌入式的作业变得愈发恐怖

网课的岁月朴实无华且枯燥如经书

大型群友线上聚会只为把那法来乳

考研词汇所剩无多前方可见那尽处

没有电脑只得论文写在手机备忘录

智能车的规则姗姗来迟却改动无数

魔幻现实如同小说 情节离奇难以预估

岁月静好时光匆匆 不见未来成功之路

# 喜欢你

似乎 这就清明了啊 也许 该出去走一走了

回忆 大一那时的清明 和经济的同学一起

在苏州淋了一天的雨 然后又深夜里爬上紫金山

大二那时的清明 只剩自己一个人

清早启程去徽杭古道 从昌化兜兜转转到西湖边

原来的计划里 从歙县到深渡 千岛湖的游船

再从金华到温州 去看洞头的海景

可是没有可是 疫情重构了世界 也重构了生活

喜欢自由也喜爱陪伴 却避免不了沦为单身狗

喜欢代码也喜爱文学 却避免不了沦为老垃圾

其实 喜欢和喜爱都只是借口 什么也说明不了

就像 我喜欢你 你也不一定会和我在一起

错了 是一定不会

# 容颜

我喜欢你的容颜 那动人心魄的娇艳

那天与地之间 你是最美的一抹亮色

一见钟情不需要理由 就像我之于你

就在 这个难忘的季节和难忘的春天

这或许就是 一种叫喜欢你的感觉吧

# 梦落

因为音乐 因为剧情 因为风景 而喜欢剑三

从苍山洱海到昆仑之巅 看大漠的冷月高悬

从龙泉冰河到东海茫茫 看扬州城似锦繁花

独自一人行走在江湖的自由 就像是一场梦


谁念这天地玄黄 算不到动如参商

第四个皈依后 沉默让树下的人感殇

你眼中有柔情千种 如脉脉春风冰雪也消融

原来拈花一笑成了孽 令你这般误尽华年

若踏错 一步间背离终点 永不还 此生永不还

只如今 茫茫大雪之中等着谁回顾


喜欢抒情 喜欢古风 醒了很久也还是很感动

喜欢小鞠的歌 喜欢她独特的嗓音 治愈心灵

花与雪相遇 停住的四季 编织冬日的序曲

唱着 未来 可期

# 人生如寄

曾经 特别喜欢眉间雪

只道是纵明知无人回顾 自己初心不负便好

总记得那个冬天的日子 那个落雪的季节

一个人走过外滩和西湖 看古镇甪直七里山塘

转专业最无助的时节 埋藏心里最深沉的感动

那一首歌 那一些人 活在记忆中 却离开了现实


后来才明白 真正的人生 如寄

是不是再多念想捱不过 时光利刃

终究化成寒暄 三两声

曾经亲近的人们最终都成了记忆里的过去

曾经感动的故事都只留下自己一个人的回忆

再多的念想都捱不过时光的利刃

所能剩下的 最多不过是寒暄三两声

许多人怪我清列清的太频繁 清的太绝情

但我更不愿看见一些名字 渐渐消失了痕迹

也许终究还是要渐行渐远的 那又何必流连

一个人走了 他再也不回来 就如同寄出的信

即使再次遇见 也不过是寒暄三两声 而已


听人说后来我们都变成 江湖传闻

茶楼酒肆里说完一生

江湖里渐渐多了些故事 亦真亦假 亦是亦非

细打听才方知那故事主角 似乎是我的名字

浪迹江湖的岁月短暂 却在江湖留下了印迹

也再没有听说过前尘往事 终究断绝了历史

既然如此 那一切的一切 便都随他去吧


你只道昔年情谊皆封存 何必有悔恨

愿前路安稳 不再为旧事执念深

# 拖延

只要你sleep够久 你的任务就可以不用做

总会有Interrupted Exception e让你解脱

即使一切变故都没有发生 世界正常运转

直到最终的ddl来临 实在避无可避的时候

你也可以选择 从容地抛出new Date();

让ddl的倒计时 重新回到那一切的起点

但其实 任务早已经不是原来的任务

而线程 却依然还是原来的那个线程

sleep的这段光阴 叫做挥霍的岁月

在挥霍的岁月里 即使悄无声息淘汰了你

你也不会发觉 这个时代已经不再是属于你

# 百鬼笙歌

生命里总有着许多不可承受之痛

在这生老病死的凡世间 一切皆有可能发生

不可预期的生和死 不可预期的种种天灾和人祸

还有更加不可预期的人心

就像这波澜诡橘的岁月里 疫情的风云变换

就像这危机重重的金融市场 花样的熔断

抑郁在所难免 而死亡成为了一种解脱

可是 在那人间以外的世界里 你就一定能幸福吗

你在第二层 你以为我在第一层

我说我在第五层 可我其实在负十八层

人其实也可以是鬼 鬼其实也可以是人


百鬼笙歌里 趁着它 夜未尽。。。

# 纵月人间

老带学生往往经常性的会遇到连锁打击

毕竟当你一倒霉起来 都是一件接着一件的

于是乎就会让人觉得说

年纪轻轻的 说话总是有一种很沧桑的感觉

这是生活 这又不是生活 生活才不是这样的呢

要不就是我喜欢这种生活 可谁会喜欢呢

挫折会来 也会过去 热泪会流下 也会收起


有时她枕着梦/有风流来相逢

说着爱慕情衷许多种/最后偏又来去匆匆

有时她沉入梦/等久别的重逢

若有一点心动需珍重/深情有始无终

INFP人格 不易动感情 又极易动感情

沉沦在纠结和挣扎里 害了别人也害了自己

# 不能说的喜爱

发现其实我根本什么都没想好

没想好怎么坦然面对接受过去

没想好怎么应付还是拼搏现在

没想好怎么迎接或者遭遇未来

所谓说想考的学校不过是心中的白月光

告别学校时遇上的一段无足轻重的过往

却因着这特殊的岁月而把它记在了心上

就像在这个春天里不期而至的隐约喜欢

但也许那只是个更优秀的自己的影子吧
