# 心之霉变

回南天着实是广东最令人难受的季节。虽然只有27度，但整个城市包裹在水汽里，所有放置不够妥善的东西都在快速的发霉。在这样的季节里，人的内心也是一样会发霉的，比如我，就已经发霉了。

本轮疫情已经40天了。深圳快清零了，希望一切都能有机会好起来吧。当然，需要好起来的有很多，不只是深圳。最近找到了一些很好的机会，一些哪怕是华五的本科生也很难进入的机会。然而，以我的能力，可能只能放任机会流失。不过，总归还是应该努力尝试一下的。

有些东西，要说不难受肯定是假的，但是总归短痛比长痛好。人与人的相遇，是缘分，也是运气。在合适的时间遇上合适的人，对于绝大多数人而言，就像年薪百万一样只是一种奢望。我似乎做的确实过分了一点，但是我自己都已经这样了，还怎么去奢望自己能照顾其他人呢？

看个人所得税app的话，我入职以来的税前收入已经达到了快十九万了，但是去掉各种零零碎碎的花销，我存下的钱却也没多少。有的人穷的只剩钱了，而我穷的不仅只剩钱，甚至钱也没多少。社畜嘛，当着当着，渐渐的，就没有社了，只剩畜了。

# 平安经年

今天是一个有一点纪念意义的日子，正好是拿到华为offer的一周年纪念。一年前的今天，我按下了确认的按钮，开启了远赴深圳的岁月。从黑粉到看客，再转变为参与者。我想谈谈，这一年来，我认识的华为。

作为我们专业最有名气的对口单位，华为，自然是投递简历的一个必然的选择。然而作为一名米粉，厌恶华为自然是一种必然选择。所以在一开始，我是拒绝的。但是架不住认识的人包括父母的推荐，我还是给华为投去了简历。毕竟，身边有华为背景的人，还是多了些。

由于接近于没有的知识储备，秋招的过程基本上可以理解为一轮游。包括华为在内的多家公司连笔试关都没过就没有然后了。然而当考研明明白白的寄了以后，春招又一次摆上了我的事务表。虽然说我的父母并不想要我去找工作，但是对考研产生恐惧的我绝对不想再一次迈入考场。于是华为又一次成为了一个春招的选项。

得益于实习经历和刷题，各大厂春招的机考接近于畅通无阻，除了某哈游上遭遇了一点挫败，其他厂都顺利的进入了面试环节。然而作为一个技术栈杂乱方向混搭的人，各个互联网大厂专业化的螺丝钉岗位让我这种半路出家根基不深的杂牌码农倍显尴尬。这时不挑方向的华为反而成为了一个不错的选择，我可以入职后再自由的考虑做什么方向，而不会受到岗位的限制。

从面试风格来说，华为是最随意的，完全不用紧张，就当聊天就好。金山和小米更多考虑你的意愿，会针对你的情况量身选择适合的业务。字节的面试官非常专业，一场面试下来像听了节课一样学到了许多。腾讯和网易更趋近于压力面，面试过程持续向你施加压力，让你高强度的难受。对我而言，面试体验是相当重要的一环。起码在这一环，华为还是很不错的。

然后玩着玩着就毕业了，休假一个月，便来到了深圳。开始的日子还挺轻松惬意的，每天登登山，骑骑车。业务场景半懂不懂，但是在甲方们的指导下搞出来的东西效果还不错，领导们觉得很赞。那段时间挺忙的，有时候改bug改到十点多，但是挺充实也挺快乐的。当你有明确的目标时，加班其实也并不是一个很痛苦的事情。最大的痛苦，其实是没有事情却又不能走的时候。

华为的文化除了众所周知的狼性，更大的程度在于集中火力。凡事说自己要攻克的东西，总归能靠堆人来解决，虽然很多时候所谓的解决看起来很啼笑皆非，但某种意义上确实是解决了。当然，凡是对外宣传的文案，最好还是一个字也不要信。绝大多数时候，宣传和实际的差距都大得离谱。华为是一个不怎么考虑人文关怀的地方，他从不展现自己伪善的面目，把资本家的剥削血淋淋的揭开给你看。一切奖励皆为物质。

随着接触的人的增多，我意识到领导们还是很有水平的，满怀愿景与期望，给的方案也很有可行性。可是由于屁股不同，下面的人执行起来总要去夹带自己的私活，给自己的队伍谋取利益。若是掺和其中，参与越多，压力越大，死的越快。当我意识到这一点的时候，我所剩的选择已经不多了。由于表现有些突出，我似乎已经成为了一个靶场，各种奇奇怪怪的锅随时有可能甩过来的那种。所以跑路也是一个有些无奈的选择。

但愿，未来顺利吧。

# 海岛假日

阳光，沙滩，海岛，夏日里理想假期的一切要素。推开门看清澈的海水，赤脚走在松软的沙滩，在礁石上听着音乐吹着海风，真是梦幻的生活。如果有可能，我希望这样的生活能更久一点，社畜的生活远一点。

但当橘色的轮船再一次靠近码头时，我知道，这短暂的一日份体验卡结束了。我不得不继续面对头疼的工作，面对各种各样烦人的汇报和材料，继续饰演一个努力工作的员工，尽管我可能很快也不需要再去演了。

不过，一般意义上的海岛度假都会有一个女主角一起，这自然是我所触及不到的领域。精神分裂之下的我，一部分试图找一个对象，而另一部分选择了拒绝。而如今，不愿意找对象的那部分灵魂，已经逐渐完成了对我的控制。

其实，我似乎并不真正需要一个对象，只是需要一个与现实世界的稳定联系罢了。

就像我现在做的产品并没有什么实际上的意义，只是为了给领导们一个交代罢了。

所以我要选择去离开，因为我终究还是一个需要梦的人，我希望我的工作起码能创造那么一点价值。

# 人生赌局

日落时分，伶仃洋的海面上，一艘往深圳蛇口港的客轮高速的驰骋着。翻滚着，摇晃着，起伏不定着。就像我的生活一样，看似古井无波，实则起起伏伏。

人生本身就是一个无时无刻不在赌博的过程，你拿着两张纸票换来十个筹码，一度输到只剩最后三个，够最后上一把牌桌；一度赢到十五个筹码，净赚五成；可最终离开的时候，还是十个筹码，换回两张纸票，怎么的来，怎么的走。

而所谓的贫富差距，仅仅决定了你的开场有多少筹码。开局筹码少的人，更经不起失败的打击，一场失败就能把你赶出赌局。而筹码更多的人，会有更大的机会耗到翻盘。

正如曾经那个已经快被我忘却的四月一般，我再一次走在了转折的路口上。押海思，押终端，或是继续押网安。但仔细想想，这其实也就只是一场简单的赌局罢了。跟风押注，输赢各表，倒无所谓。回想一下最初的目标，无非落沪而已。既然选择了在沪深直接押沪，那剩下的选择，不过一个顺带的赌注罢了。

再一次来到澳门，虽然计划的仓促，但曾经未达成的心愿倒是或多或少也实现了。登上澳门旅游塔的塔顶，坐一次新濠影汇的摩天轮，进一趟赌场感受一下所谓的纸醉金迷，去一趟苹果店去感受下免税区的奢侈品物价。不像上周海岛的梦幻之旅，这一趟纯粹只是猎奇之行罢了。

当我从导师的精神压榨中解脱出来时，我突然意识到此前对感情的执着，不过只是一种本能的情感压力转移和释放罢了。导师对我释放了让我招架不住的精神压力，迫使我需要一个能倾吐与发泄的对象。那既不是一种亲密关系，也不是一种平等的诉求。

在正常的感情中，无论是哪一方的卑微都很容易让感情变质，更别说双方共同的不对等。或许，如406般的感情对我来说才是最好的，有陪伴，有守护，还有能化一切不开心为快乐的源动力。这样的人，大抵是不会存在于这个世界的吧。

我永远喜欢小木曾雪菜。

# 三千点保卫战

四月在浑浑噩噩中不知不觉的就过完了。上一次双休还是4月10号的事情，而下一次双休起码要等到5月14号了。单休挺不爽的，但是也还不错，总比无休好些。转岗的齿轮缓缓的转动了起来，逐渐不必在care导师给的压力，也不必执着于绩效意义上的成就。安心的享受工作里的乐趣，在内网里翻阅感兴趣的知识去学习，倒也挺好。

四月的深圳仿佛疫情的孤岛，任外界无尽喧嚣，城里安详的仿佛没有疫情一样。上海遭遇的一切苦难，在深圳看来仿佛有些不可思议。外卖正常运转，核酸触手可及，每天抽空去做个核酸已经成为了生活中的一种习惯。当做核酸如呼吸一般自然的时候，自然也就觉得去哪里都是畅行无阻了。

四月的深圳似乎已经到了夏天，却并没有那么的炎热。二三十度的天气里，海风吹拂，也不失为一种享受。外伶仃岛度假、澳门畅享游、徒步小三洲，在海边的石头上晒晒太阳，娱乐场里来两把百家乐，走行在山间公路的高架桥吹着山谷里的清风。一个人的走走停停，对我来说，或许也不失为一种享受。

四月的股市绿的生机盎然。坚守了半天的上证指数终于在下午迎来了跌破3000大关的时刻，随后的跌幅便如滔滔江水一发不可收拾，收于2928点，创近46个月的新低。2022年对于投资而言大抵是艰难的，这个世界的熵增已经到了一个不可收拾的地步，一切终究将会以一场大规模的混乱结束，只是我们难以知晓具体的时间与形式罢了。

四月的列表到处都是爱情的酸味，脱单与秀恩爱的一个接着一个。我羡慕他们，却还是选择去把感情埋在泥土里。我在潜意识里终究还是抗拒感情的，我不知道怎么去面对，也不知道怎么去接受。我不了解什么是爱，也不了解什么是被爱。

未来不可期，但现实的美好就足够了吧。毕竟，那也是未来的美好回忆啊。

人嘛，永远是失去了以后才会知道珍惜的。

# 面子or里子

为数不多一个关系还不错的同事要离职了，应届毕业两年半，目前待遇跟我差不多，属于是被倒挂麻了。明天考勤截止，下家字节跳动，总包开55w。

华为的卷比起学校里来说只能是有过之无不及，尤其是当你的上级是大领导的舔狗，但是大领导不喜欢他时。大领导随便放个p都要让下面的人去舔，下面的人还抢着舔。但p终究是个p，做好做坏都是个p，领导反正一通开骂。反正搞来搞去最后就一团乌烟瘴气，又卷又麻。

现在回头想想，我这一生很多选择，图的就一面子罢了。因为自己真实生活一地鸡毛，所以总得找面子，绷场子，立些成功人设感觉自己牛逼。转专业话都放出去了，不敢硬着头皮上计科，不敢赌失败了回原专业的后果。因为要撑场面，所以不敢赌概率。

签华为也是这么一个道理，起码最后去华为了这个人设能苟住，串一个完整的故事就热爱竞赛的学渣考研失败但终究进了大厂。面子保住了不一定意味着这个选项就很合适，但出于面子的考虑这个选项就最好了。

于是也就有了这一轮的转岗，依然基于面子只能去上海不能回南京，还得找听起来有面子的部门。对不对口或者待遇怎样这些问题其实都不重要，只有面子重要。

可面子终归是给人看的，我挣这么些面子，又有谁看呢？

# 记忆回转

一年一度的就业指导，一年一度的学长讲堂。从大二第一次被导员忽悠去听，到大三自己经历这门课，再后来大四听自己的同学吹牛逼，最后到毕业后看下一届的学弟给下下届传授经验。我好像离开了，却又好像没离开。我依然关心着牛头冲里的大事小事，吃着翡翠湖畔斛兵塘边各色的瓜。

当你走进这欢乐场，背上所有的梦与想，各色的脸上各色的妆，没人记得你的模样。这是2017年夏天最火的一首歌，可五年后的今天，我才明白这首歌的意义。当你迈出那一扇门以后，渐渐的，大家不会再记得你是谁，也不会再知道你是谁。

按惯例的给即将离开学校的这一些人按去处分一下类，然而大多数人我并不知道他们将要去往何方，甚至他们可能早已不认识我了。我之所以把他们留在列表，或许只是存留一段很久以前的记忆。毕业后的日子，几乎没有什么现实社交，也不需要什么现实社交。繁华落尽，一切也归于无情。

Tim的聊天记录最长可以保存两年，人的记忆也是如此。两年前的事情，已经只能记得一个大概，而难以知悉曾经所谓的因果。所以，我所真正有共同记忆的人，也就不过上下两届而已。一切终究是会放下的。待到23届毕业的时候，若是疫情止息，我大抵是要回去看看的。作为一切的起点，作为一切的终焉。

夜深，渐不知所言。
