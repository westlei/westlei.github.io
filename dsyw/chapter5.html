# 沙河西路向北

最近的生活大抵是一个开摆的废物状态。上个月短暂有过的一些微乎其微的雄心壮志在现实面前以最快的速度抛之脑后。辛辛苦苦做了一个月的垃圾需求方案有严重漏洞，还没上线就紧急下线。现在的状态就好像行走在沙河西路上从西丽向北——白芒。

我发现我虽然嘴上一直在摆烂，但是身体却很诚实的在卷，不断突破卷的下限，来的最早加班最晚，迫使大家也不得不一起卷。但我又是为什么要卷呢？其实我并没有什么一定要上升的愿望，我也支持不卷，但我没啥选择，我也没啥自己的生活。

所谓社会规劝或者家庭对我来说渐渐没啥意义了。它们并没有什么值得我认同或接受的东西，尤其是自从2023年开始以后。去年的日子里，我发现归根结底谁也帮不了我，一切的一切都得自力更生。软弱无力的时候，哪怕有钱，也一样什么都做不到。

有的时候很羡慕一些多年的情侣，我觉得以我的现状，完全没有什么期待可言。相对于我的条件，我能做出的选择依然几近于0。何况，我现在根本不想面对现实。。。

# 画饼充饥

最近一段时间上海海思的大领导齐聚上海汇报工作，于是最近的日常就是三天跟三个21级部长吃了饭。领导们表示生产问题得到了解决，但是23年会过的比22年更艰难。新的芯片来不及大量上市，而旧的芯片库存已经卖完。简而言之，一是亏损，二是忍耐，三是前途光明道路曲折。

不过这些东西跟我的关系不能说很大，只能说完全无关。底层小喽啰既不会因为盈利而多拿多少马内，也不会因为亏损而直接背C走人。毕竟，干活的人总是缺的，低薪劳动力总是越多越好的。

花了五个月的时间达成了上海的全探索度。从嘉定的嘉北郊野，宝山的美兰湖，到浦东滨江森林，南汇新城滴水湖，再到奉贤海湾森林，金山城市沙滩，还有枫泾古镇，金泽古镇。再是长江之外崇明的西沙明珠湖，东滩湿地，长兴岛，横沙岛。

从上海最西端的金泽镇南新渡口步行过桥去往周庄，从上海最北端的崇明新村乡去到江苏境内的永临汽渡坐船到南通海门。从金桥基地向东经过曹路合庆川沙去看迪士尼的烟火，从南京西路坐三个小时的南新专线辗转南汇老港书院最后感受滴水湖畔的晚风。

以上都不像是正常人会做的事情，但放在现在的我身上就很合理。我发现我现在越来越排斥和人接触了，不管熟悉的陌生的，亲近的排斥的，只要跟人一起，就会有一种尴尬的感觉。排斥社交又不想打游戏的后果就是只能去当街溜子，今天这里明天那里到处晃悠。

但我现在觉得这样挺好的。毫无规划随心所欲的游走，出发时间视啥时候睡醒而定，不小心坐了个小交路就在长江南路下车去旁边复旦溜一圈......不用考虑需要等谁，也不用考虑有没有意思。这好吗？这不好。这不好吗？这又很好......

# 沪国樱开

又到了sakura绽放的时节 ，沪上花开，白樱似雪。

想起上一次看樱花，还是疫情第一年的南京。那年的春天来的静悄悄，第一波新冠带来的恐怖尚未化解。台城下的樱花绚烂着，却没有太多的游人驻足。

一晃三年过去了，我们见证了疫情的起起落落，看见了解除封锁之后又一次遍布各地的游人。可惜有的人已经不再是原来的样子，而有的人再也回不来了。

所谓历史，无非是多数人所经历的值得铭记的大事的合集，而真正事件的发生，大抵不过一瞬。历史长河之中，白色的三年大抵也如此刻樱花的绽放，只此一瞬罢了。

虹口足球场出站口外，满树的樱花在阳光下起舞；共青森林公园园中湖畔，桃花与樱花铺就春日璀璨。世纪大道浦东南路交汇，陆家嘴厨具三件套隐没于樱花之中；南浦大桥引桥之下，大桥的灯光洒落在站场旁的樱花树里。

此行所为沪上樱花，而所见者非唯樱花。在沪青平公路上等待71路时，即将降落于虹桥的飞机压过天空。乘轮渡越过黄浦江头，望见浦东高行满目疮痍的工业遗迹。迷失在徐家汇的高楼林立里，而宛平南路却在临近。随心而随行，此即旅者之自由。

一周六天困居于工位之中不得解脱的人，逐渐丧失了发现与接受爱的能力。所谓的浪漫本不是浪漫，而是一种生活本应该有的样子罢了。社畜生活下畸形而错误的意识混乱，误以为这些所应得的东西反而不可得。继而沉迷贴吧与B站冲浪，寻烂梗，观皮套，成分复杂，频频发病。究其所以，不过道心破碎、本性丧失罢了。

大约，宛平南路600号才应该是我的家。

# 自我和解

听闻旁边的同事接连在上海买了房。有的人一毕业就房贷车贷俱全，有的人毕业快两年还在未知的未来里迷茫。

当年把户口迁到深圳，大抵是为了切断回南京的后路，从而彻底结束这么多年以来对南京这座城市复杂而矛盾的心情。如今看来，也确然如此。但凡未曾足够的潦倒与落魄，我都是不愿再回南京的。

多年来的我，一直生活在自我的矛盾里。鄙夷着传统，跳脱于约束，却又接受现行秩序所带来的一切，默默的用更卷的态度和作风，来践行所谓整顿职场。

我也从未尝试过去与自己和解。就好像如今的工作，与其说是锻炼能力，不如说是自我折磨。每天十二、十四，乃至更长时间的工作，既没有让我学到多少有用的技术，也没有让我获得薪酬、级别上的提升。我只是单纯的自我折磨，挂机水时长罢了。
