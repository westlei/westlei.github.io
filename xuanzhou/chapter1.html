# 流离和幻影

三个月的比赛，还是走到了尽头。原来以为的苟延馋喘，最终也没能实现。墨菲定律有云：如果事情有变坏的可能，不管这种可能性有多小，它总会发生。事实果真如此，从第一到淘汰，就是这么简单。唉，忘却比赛，忘却逝去的爱，忘却，这即将逝去的一年。淮阳，永远不再。二零一七，永远不再。

厌倦了一成不变的薰化路，何必去游人如织的宛陵湖？越过山丘，看看静谧的彩金湖。荒山野岭处，喧嚣闹市无。前路多险阻，唯我一人孤。人生几多险阻，终究无人陪伴。一切皆空，永远的只有孤独 。失败是注定，缘分是天意。湖畔独坐，心中迷惘。叶落湖水，情消意却。一切的尽头，是凄凄思慕。眼已模糊，远处，似是淮阳飞舞。

或许不知梦的缘故，流离之人追逐着幻影。或许，那只是青春的伤痛与梦幻罢了。美丽的宣州，彩金湖畔的日子，究竟会不会遇到所爱的呢？一切，依然只是未知。尽管是萧瑟的冬日，阳光依旧，美好仍存。前路，纵然无人陪伴，仍旧精彩。湖有千般不同，心之所爱可变。叶落可以再生，心伤亦能愈合。祈愿，平安。

# 一重年关

佛系青年，或许，说的就是我这样的人吧。想做一件事，就去做吧，不去想什么利弊，不去想什么后果。不想做一件事，就不做，没有什么理由，也没有什么推脱。心情不好了，一个人出去走走，发条说说。心情好了，那就这样吧。从一种心情切换到另一种心情，并不容易，所以大部分时候，总是充满纠结与矛盾。学习？好，那就学习吧。学什么，怎么学？看心情吧。或许曾经有过什么看的重的，例如感情之类的吧，现在觉得，也不过就是那样。一切都不过如此，挂个科什么的，没办法就挂了吧。2017，走向佛系的一年。渐渐看淡学习，渐渐看淡各种，最后看淡感情。一切都随性，一切不走心。总有人以为我是大佬，那可能只是一个误会吧。像我这样的佛系之人，怎么可能是大佬呢？不过，佛系之外，也许，还有几分强迫症，一旦想去做什么事，还是会去认真做的。好吧，我就是这样一个人。也许过于高调，那也不过就是一些不走心的伪装。到了该暴露的时候，终究还是会浮出水面的。

自由，曾经我觉得是我热爱的方向。因为生活不止眼前的苟且，还有诗和远方。过了三年，我才发现，所谓的自由，其实只是一些不切实际的幻想。真正若是自由，前程如大雾般迷茫。没有约束的不自由，何来目标与方向？雾也茫茫，前程亦茫茫。

转眼间，又到了那日历本新旧交替的时刻。窗外，一片雾海霾踪，使人仿佛身处幻境。多么希望，金陵十载，不过南柯一梦。终于，有暇回望从前。三载，旧都，已然发生了许多改变。可不变的是，那最初的起点。一切发生，在那碑亭巷口，在那织造府边。爱恋，从开始到终结，归于无言。情窦初起，情欲盛开，情伤难解，往事如烟。这一刻，正式结束了，这一年。过去的一切，都已翻过一页，定格在了昨天。寄希望，新的一年，前途得以豁然开朗。寄希望，新的一年，遇见怦然心动的人。寄希望，新的一年，平和，超脱。继续，孤独地走下去。

霾未散，而新年伊始。动车呼啸着冲进那一片白色，让所有的迷茫在视野里日渐清晰。你若有光，便是远方。念念不忘，必有回响。前程的光芒，必定，坐落在远方。既然，选择了远方，那便只顾，风雨兼程。只要有光，便有远方，便寻得到，那前进的方向。然而，四周，只有一片白茫茫。何处寻觅温柔乡？三生三世，改不了的，是凉凉。湖风冷，心已寒，叶落尽，欢不再。雾锁江南。祈愿，春暖花开时，伤心不再。

# 新年伊始

来自漠北的寒风夹杂着大雪，跨越千山万水，却在宣州的群山前耗尽了最后几分气力。最后落下的，只有些许，雪的残影。三载间那些关于雪的记忆，关于人的思念，也在这时空的交织变幻里，如那瑞雪般，逐渐消失难觅。是不是，每种感情都不容沉溺放肆，交心淡如君子；只道是，那些无关风花雪月的相思，说来几人能知？眉间雪，雪中人。在那大江尽头的彼岸，南通，也许，正是大雪纷飞时。斟酒独酌，细雪纷纷覆上眉目，清寒已然入骨。忘却那些早该忘却的，让一切，重新开始。去寻觅，令我怦然心动的身影。来覆盖，那一切，有关金陵的雪，和人，的记忆。

雪的印记无处寻觅，留给宣州的，只有那一阵冷似一阵的凄风和冷雨。2018，宛陵，仍是宛陵。冬的萧瑟，在三九天里格外凸显。风的咆哮，在宛陵湖畔更显凄凉。伤而不可忘的，是那淮阳。寻而不可得的，是温柔乡。但愿戊戌之年，可有佳人为伴。

天寒地冻。迎面而来的，总是牛头冲那一刻也不停歇的风。阳光洒落，却不见一丝冰寒少却。也许，这冬日里，阳光本没有温度。身伤，在冬日的肃杀里难以愈合；情伤，在萧瑟的寒风中郁结成冰。橘猫，纵贪恋宿舍的温暖，也不得不离去；而我，纵流连佳人的倩影，也不敢不放弃。昔日伤未解，今朝恐风流。愿在那丙申之年结束之际，可彻底放下过去。时空流转，且将那过去的情与爱，封印进，那忘却的记忆。待到那时，温柔乡，犹可觅。

# 声色光影

光与影，交织在宛陵；声与色，变幻于无形。情与爱，已失却踪影；动与静，皆化作空灵。冬日下午湛蓝的天，心向那长江口黄海岸边。抓不住那逝去的爱恋，只可惜情已消散水云间。早应该放弃那思念，没必要再把她心牵。冬日暖阳青天碧水重现，一晃却已过去了三年。寻不到的缘。

寻不到的缘，浇不灭的念。世事本就像个圆，一圈一圈仿佛回到起点。看似一切没多少改变，可是时间逐渐走远。能令我怦然心动的俏颜，时光流转有朝一日终究浮现。不过年复一年。图书为伴，不觉宣州日暮。夕阳西下，远眺宛陵黄昏。烟雨宣州，美景有之，佳人亦有之。赏美景，心静若水。赏佳人，心潮起伏。心静而后心凉，故赏佳人；起伏而未心动，故赏美景。人，终究达不到无欲无求之境，那便活得平凡。岁月静好，独行宣州，看日升云起，鸟雀高飞；书海遨游，抬头四望，是嫣然一笑，长发飘飘。不起功利之念，不生非分之想，那便一切安好。

当函数的极限被打破，微分和积分都成了错。高等数学便无从去做，试卷上只有零分的落魄。当情感的前途已经寥落，再多的思念也没有结果。苦心维持空引发灾祸，斩灭一切心田里花朵。人是三维下的四维生物，大脑里蕴含平行时空无数。美好幻想只在脑海重复，梦醒时分回到现实仍苦。人生在世前程漫漫长路，左右环顾不知佳人何处。心中怅惘一人享受孤独，叶落身旁坐看宛陵一湖。尽是空无。

为何对错总要分出？为何答案区分正误？工图，高数。考罢凉凉，只觉心中苦。泪在心头，却难哭出。时光匆匆，又是一年，高考一模卷出。细看时，满篇不会，慌张无助。惊忆起，今非昔日，高考已成往事，无需回顾。爱情已然成空，学业荒废，光阴虚度。此时之我，无异于废物。人生漫漫，既已至此，谈何思慕？宛陵湖心叶落处，敬亭楼里人亡路。政经之后，假若此周未度。似是大梦初醒时，仿佛，一切如故。

夜色已深。冷风，越过那敬亭，拂过那宛陵，吹散，那牛头冲下一片安宁。思念成河的是凉凉天色，半年时光终究学业不成。也罢，那便就此踏上归途。没有什么是时光不能改变的，不管是物，还是人。但也没有什么是时光能够彻底冲刷殆尽的，似那对曾经伊人的几分流连。孰知迈尧路，可一如往昔？物是人非，纵景依旧，访亦无心。从天边的第一缕红霞，到日暮的最后一丝微光；从春日的第一朵鲜花，到秋末的最后一片落叶。所有的美好的单纯的想念，终究会在匆匆时光里消逝，无影无踪。

# 长冬忽过

长夜漫漫，总会念起，那白天的阳光；长冬漫漫，总会忆起，那春日的温暖。情的深夜，念曾经伊人；爱的寒冬，忆昔日淮阳。然而终究，夜会过去，春会到来。天，渐渐明亮，看一轮红日缓缓升起在东方。春，终将来到，那嫩绿新叶悄然出现在枝头。而情的长夜何时过去？爱的寒冬何日终结？或许，终究会有那么一天，会出现那么一个高挑的身影，一头飘飞的青丝，微笑着，来到我的眼前。

我们总是在抱怨，漂亮的皮囊太多，有趣的灵魂太少。然而，追求漂亮的皮囊只要一双发现美的眼睛，而寻觅有趣的灵魂则需要一颗够聪慧的心灵。发现美的眼睛易得，够聪慧的心灵难觅。但当你终于拥有了二者，却发现那相貌灵魂二者俱全的姑娘早已投入了别人的怀抱。那么便失意，那么便怅惘。不如归于平凡，消失在，那人海茫茫。此处，金陵，大佬云集之所。像我这样的渣，还有什么可追求的呢？

也许，每个人心中都有那么一个面码。但是，他们有的是波波，有的是雪集，有的是仁太。而我，或许就是仁太。在离别之际说不出口的爱恋，到了四个月之后终于吐露。仁太永远忘不了的面码，一如淮阳。思慕，深植心底，难以除尽。物是人非，景色一如往昔，而人不再。但至少，仁太是幸运的，还有那么一个人，依然爱着他。深夜，观未闻花名，泪落滚滚。故事，终结于此，而人物的命运，犹未可知。思慕，犹未终结，要忘却那往事，还需几时？

当厄尔尼诺的哭声终结在南太平洋的海底，中华大地上响起了拉尼娜轻柔的吟唱。片片雪花在金陵的空中飘舞，赤色的苍穹下装点着雪后银白的世界。依稀记得，上一次鹅毛大雪还是六年，或是更早以前。那时的我还没遇见她，也不知道，未来的某一段时间，我会痴迷于她。多久没有过这么大的雪了，然而，风月无情，缘分却早已走到了尽头。淮水之阳，她的故乡，一望成欢，即是淮阳。那关于阜阳的记忆里，永远有着一份，属于你。

连绵多日的雪，终于在此时走到了尽头。瑟瑟寒风的吹拂，给雪后的金陵，带来了万里晴空。银装素裹的世界渐渐褪去，一切开始恢复本来的面目。此刻，深冬。找寻十载金陵的记忆，还原过去恬淡的时光。爱与恨，都已经过去。且收拾记忆，展望前方。

失去了，才懂得什么是珍惜；离开了，才懂得什么是怀念。当一切的情与爱都想明白，过去已不能重来。失去了的已经失去，那么就抛下所有余下的东西，只有一切都归于零点，才不会有所想念。离开了的已经离开，那么就忘却所有曾经的故事，只有将过去流于回忆，才可以展望前程。寒假过半，时光在指间挥霍，终究一事未就。时光啊，就这样蹉跎着远去；雄心啊，就这样逐步地磨灭。过去的路，一人独行；前方的路，可有同往？

# 梦回帝都

寒假的岁月，在怠惰的时光中渐渐逝去；金陵的时光，在银白的记忆里模糊不清。时隔半年再到帝京，却早已不复曾经。那时熟识的人们，已经离我远去，无处找寻。那时陌生的人们，却渐渐走入了我的生活，从天南海北来到一起。那时的奉天街头，英俊的小伙，也许曾有照面，但谁能知晓那日后，得以一见投缘？那时的后海街角，漂亮的姑娘，也许曾有擦肩，但素不相识的你我，怎知日后重又相见？新年将至，再不必怀念过往，是时候翻开新的一页。

从前的日色变得慢，车、马、邮件都慢，一生只够爱一个人。但那别离的旅人，也许一世都再难得相见。如今时代变迁，纵使大洋彼岸也不过半日的距离。但缘尽的人啊，纵使离得再近也难得相见，纵使人群擦肩也不复从前。她的十八岁终究谢幕了，有幸在那她人生中最美好的三载时光中，遇上了美丽的她。一切，此时终结；一切，今朝谢幕。帝都，新旧交融，古朴的深宅大院，现代的艺术风景。多少人来去匆匆，多少人心归无处？后海街头，已没有了佳人的痕迹；金陵城里，已寻不到伊人的影踪。往事已矣，新年将至。只愿那前方的路途，再无思慕。
