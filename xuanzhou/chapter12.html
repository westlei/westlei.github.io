# 岁末嗟叹

# 运气

过去的十年间，我曾有过无数个选择的机会

但我每一次都能，选中那不是太正确的一个

也曾失落，也曾后悔，但终究没有勇气去改变

只得接受命运的安排，挣扎于得失之间

寻求稳定的背后，是充满胆怯的内心

人生多苦难，我从未乐观


从遥远的西部山城来到长三角的繁华地带

既不适应，也不习惯，在毫无归属感的城市

没有伙伴的七年孤单，封闭了内心的一切

直到中考如愿以偿的落榜，本该把过去消散

第一道选择的关卡处，我却选择了怯懦

或许，去了南航附中，未来可能精彩


度过快乐的一载，直面高考的无奈

本更适合去文科，却走上物生的不归路

勾心斗角尔虞我诈，是高三的主旋律

厌倦了眼下的一切，想去远方看看

却不知，兜兜转转，命运使然


经济本应该是一个不错的归宿

却恰逢时移世易，内忧外患

怯懦的自己带着卑微的喜欢逃窜

为了必过的结果而选择放弃理想

即使他本已近在咫尺之遥

不归的旅人，还能为谁而歌唱


前方又到了选择的关口，选择拼搏，还是退缩

当熟悉的人已经在为梦想而拼搏

却还沉浸醉生梦死的现在

仿佛看见了，失败的未来


只如今

茫茫大雪之中等着谁回顾

明知无人回顾

谁能初心不负

# 珍惜

或是不喜欢的

有的人

他在时不觉得

消失才会意识到可贵

不懂得

那珍惜的值得

就不会理解人生百味

# 昭愿

人生就像是一台破破烂烂的自行车

当你行驶数百公里的旅途后 车胎可能没气 刹车需要修理

这时 前方的路途便崎岖难行

每一处坎坷都变得难以逾越 每一个上坡都需要费劲艰辛

你累了 你走不动了 你想放弃了 但当你找到那么一个所在

修好了车 打足了气 前方的旅途又变得容易起来

牛头冲的第814天 有点累 有点丧 有点疲倦 有点艰难

但愿能够重新打足气 继续前方385天的旅程


这是缘 亦是命中最美的相见

别恨天 笑容更适合你的脸

再一遍 记起从前的一滴一点

就从此蒸发人间

但愿 归来时还仍是少年

# 车站

这里是宣城站西广场 2017.12-2019.12

待到那时 却也快到了离开的时候

# 落雨

干涸了半年的皖南大地，甚少听闻雨的讯息。

落寞的湖水不见了小溪，懒散的日光仍依稀。

21世纪10年代的最后一轮圆月高悬空中，

把清辉一片洒落在这寒风拂过的世界里。

在持续了一个夏天的忙忙碌碌之后，

碌碌无为成为了这个冬天的主旋律。

心如止水般地寂静，逐渐脱离社交和人群。

厌恶专业课的学习，拥抱毫无意义的游戏。

每个月都会去想着做一个有意思的项目，

但碰到一点困难就会因懒惰而选择放弃。

仍然想要再来一场说走就走的旅行，

可入不敷出的荷包让我终止了念想。

破旧的自行车上，看彩金湖畔夕阳。

安静的实验室旁，见暮色洗礼了窗。

# 破阵子·紫金

江东门诵悲歌，紫金草开金陵。

八十二年屈辱史，城下卅万忠魂泪。

未闻战火声。

虎踞龙蟠仍在，满腔壮志犹存。

千里江山曾浴血，百年风云换今朝。

不见昔日人。

# 跨年

2017年的冬天 初见彩金

震撼于那一方天地 沉浸于那一池碧水

一见倾心 是心动的感觉

2019年的冬天 再至彩金

似仍是当年的模样 却没有曾经的感觉

光阴荏苒 回不到的从前


转眼 21世纪10年代踏入了尾声

年终了 盘点一下这落落起起落落落的一年

也有过短暂的高光 但更多的还是暮色的昏黄

心态逐渐老去 远离现实 远离人群

却远离不了纷繁复杂的专业课

当初的选择 已经分不清正确还是错误

只知道输了当下 却不知赢了什么


回到时间轴的开头 当2019年的元旦钟声敲响

在专业课的苦海里沉没 痛苦着面对现实

还记得电信科的元旦晚会

也还记得与学姐的短暂旅行

二十一周的屯溪路有点冷清

在南国的海滨释放疲惫与忧伤


待到阳春三月来临 熟悉的程设课堂上

似找回昔日的感觉 如没有天书的烦恼

春暖花开的日子里 徽杭古道清风飘扬

筹备着大创的立项 不慌不忙不急不躁

越过谷底重又向上 述说着未可名的想


五月号角太湖吹响 看姑苏古镇的熙攘

金鸡湖畔初见的人 江南大学漫步寻春

单片机竞赛过去了 六级口语却又来了

期末们来了又去了 闲暇时光校赛摸鱼

六月之中寻访金陵 回味那初夏的痕迹


当暑期的大幕开启 锡山古镇小雨淅淅

蠡湖之滨风和日丽 钱江夜色独具风景

芜湖铁画绝伦精妙 江畔明灯照亮愿景

水阳江头小船轻摇 南山竹海天目清凉

走过清晨的田子坊 迈步上午的五角场

那实验室里的奔忙 与翡翠湖畔的星光


当将那快乐的盛夏 在机房训练中收场

中秋圆月无福消享 为数学建模的繁忙

春华路上叫声熙攘 是ican大赛的卖场

西津渡前金山之下 东关街头走过小巷

东海之滨滩涂涌浪 西子湖畔人群浩荡

宁国路口龙虾飘香 沉迷小学姐的美貌


待到冬日终于降临 考试一周便是一场

没有闲暇没有时光 丢了成绩失了理想

寒风吹彻一年回想 不记当年何人心伤
