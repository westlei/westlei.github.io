# 海语西风

# 海语西风

最近突然想换个名字，某种意义上与过去做一个告别吧。上一个名字用了八年了，某种意义上已经成为了我在互联网上的名片。但是过去的我毕竟不是现在的我，于是变换一下名字，以此表示重新开始吧。

在深圳的日子，我喜欢去海边，倾听来自海风的声音。海大多不言不语，只以沉默的海浪回应。但当落日时分，傍晚西风的吹拂之下，海总是有它的话要诉说。在工作之中遇到困难的时刻，去海边，看看夕阳，吹吹海风，似乎也没什么过不去的坎。

上海旁边就是海，而黄浦江的风里并没有海的味道。在营销号的推广之下，我决定南下台州，去到滤镜下无比美好的石塘半岛渔村。但石塘大约确实是个渔村，除了渔村以外并没有什么可以看的，网红照片甚至需要挂在路标上才知道哪里是哪里。

那就回家吧，回到曾经的406。曾经的406依然是406，一切都还是过去的样子。可是大家终将四散而去，从今往后的日子，406还在那里，但也不再是家。

想见的该见的，似乎也都见了。该结束的故事，也该送入回忆了。心结散去，也该去思考未来的路了。远在青岛的WDR战队，也创造历史的取得了战队生涯之中的两个胜场。尽管依然无法出线，但是一切都在变好。那么，今年的赞助，再翻一倍吧。

# 关于WDR的故事

有些忘记知晓WDR是多久以前的事情了。不过起初，这是一个与我完全没有关联的故事。草创期的它似乎并不叫这个名字，也与计算机系毫不沾边。

它与我的生活发生交集，来自于某位自动化先驱的引流。电子设计与机器人的碰撞，擦出了电控的火花。专业电控&赛车团队的技术力加持，让新生的WDR开始有了一些成熟的模样。恍惚记得当年在406给WDR调电机PWM，那还是没有疫情的曾经。

当疫情初期的半年恐怖过去，我的大学生涯已经走进了倒计时，余下的时间也并不允许我去投入这一项事业。WDR的一切开始走向正轨，有了稳定的经费和307的大house，我所熟悉的RMer们也正式走上了舞台中央。酷炫的工程机器人载着传单跑过牛头冲的大街小巷，吸引无数的目光。

入职培训后来到深圳的第一天，去春茧体育场看望参加单项赛的队伍。第一次现场观摩超级对抗赛的激烈，兴奋之余却也感叹。彼时的WDR虽然初具规模，但距离一场对抗赛的水平依然遥远。已经有了属于自己收入的我开始了对WDR的第一笔微薄的投资，总计价值2000出头的STM32F407开发套件三套，用于新生的培训。不多，但大约有用。

打工的时光一晃而过，时间来到了第二年的夏天。WDR登上了南部赛区超级对抗赛的舞台，在美丽的厦门海滨开始了真正的实战搏击。彼时的WDR，刘老板、紫涵姐、炎王、贝贝，缺席的蛋卷，负责宣管的祖彤姐，组成了我大约最熟悉的一届队伍。没有实战经验的后果无非意外频发与遗憾离场，那天厦门突降的暴雨，为初出茅庐的队伍画上了最好的注脚。酒足饭饱之余，我支付了对WDR的一万元投资。希望它的未来，蒸蒸日上。

当我再一次回到牛头冲，看望即将离别的406时，远在青岛的WDR开始了新一次的挑战。积累终于开始有所收获，面对强敌，他们依然拿到了两场堂堂正正的胜利。尽管遗憾倒在了出线之前，但依然是创造队史的辉煌。也许，今年再追加两万投资，明年我就能在春茧看到WDR了吧。

随着毕业季的来到，刘老板终于也要离开属于他的舞台，拥抱小米的工作。炎王也如愿进入了计算所，开始了科研之路。20的诸位也要开始寻找自己的未来，但像芷兰姐这样，属于21、22的新生血液也在源源不绝的涌入。这是属于他们的热血，也是属于我的青春。而我所在其中所扮演的角色，大约更像是一个不求回报的投资人吧。看着我所投资的项目开始有了那么些成果，也是一种成功吧。

# 11x11

苏州地铁11号线开放公众试乘了。顺利解锁花桥-狮子山=130min地下坐牢，解锁苏州轨道交通最西端——苏州高新有轨电车一号线西洋山站。

先吐槽一下到昆山兆丰路的青安线是真破，青浦乡村破中巴就这水平没办法。但是青浦从市区兜一大圈去嘉定也太绕了。

北太湖地区的荒凉程度不比青浦好多少，有轨电车沿太湖大道一路狂飙车速感人，然后到了北太湖大道这一看就这？ 只能说啥也没有湖都不怎么看得见是这样。

于是回主城顺便看看好几年没去的山塘街。呃，我的评价是这儿就主打一个人挤人挤人，我寻思今天下这雨也算不上很小吧...


大雨倾盆的端午d3，苏州地铁11号线正式开通了。本来昨天在烟台的时候是准备D781卧铺去苏州然后去唯亭蹲11点苏州的首发的，但是在烟台被晒化以后，觉得还是回家去躺着比较好。于是在给铁总贡献了100多退票费以后改了当晚的D2154，在破带鱼上坐牢七个小时滚回上海了，第一次知道啥叫屁股坐麻(2A本来座位就不舒服人还巨tm多，动都不敢动)。

在暴雨和崧泽大道车流混杂的白噪音(讲道理还挺好听的)下舒服的睡了一觉，然后起来把端午礼盒里最后俩粽子蒸了吃了。本来一开始还想赶花桥11点的首发的，但是等我出门到徐盈路坐上17号线的时候已经10点半了，等从市里溜一圈到花桥都12点出头了(17号线10分钟 2号线20分钟 11号线60分钟)。11号线出了上海到兆丰路一帮人上来发卖房广告，只能说昆山的房价是真的快蚌埠住了。

苏州11号线不是第一次坐了。但是与上周末试运行时不同的是，这次好长一段都莫得手机信号，属于是准备失联了。花桥站出发到盛庄站为止13公里多，只有夏桥站附近有一丢丢信号，大概是设备坏了吧。

可能是下雨而且中午的原因，苏州地铁根本没有人。11号线看新鲜的人也就勉强坐满，3号线几乎一直稀稀拉拉的，4号线人多一点但也没到站人的地步。花桥到同里差不多两个半小时(11号线60分钟 3号线40分钟 4号线40分钟)，metro大都会无感换乘直接刷出不了站，还需要手动开通一下。扣款是上海苏州各扣各的，上海段62km，计价是9块；苏州段应该是89km，线网最高价13块，但好像苏州这边适配还有点问题，或许是11号线计价还没录进去吧。

两个半小时地底之旅结束，吴江也是大雨倾盆。同津大道上的积水贼深，大卡车驶过溅起的水能有一人高。15点在同里站坐上吴江临沪专线，在旱地行舟之中去往汾湖客运站。37km一票制五元，全程大概70分钟左右，从同里出发到汾湖32km刚好是一小时。其实汾湖城建水平挺不错的，工厂企业什么的也不少，但是就是离哪里都太远了。

由上海青浦巴士运营的示范区1路于16:20从汾湖客运站发车。全程19.1km一票制两元，在青浦属于典型的良心线路了(对比20km分段五元的青安线之类)。全程经康力大道/元荡路+沪青平公路运行，能看到淀山湖南岸的风景。但是淀山湖南岸基本都被围挡了，没啥可看的。运行全程耗时40分钟，17点到达东方绿舟。从这里坐17号线回到徐盈路的里程是28km，耗时30分钟，计价6元。

苏沪地铁公交环线总计经停123站，里程是230km，耗时7小时，总价35元。对比一下，7个小时的动车大概就是昨天威海回来的时间，而苏沪环线的里程差不多是威海到上海的1/4(威海到上海1003km)，票价是威海到上海的1/10(威海到上海373)。7小时的绿皮车差不多就是前天新沂到烟台的时间，里程上的话新沂到烟台是511km，硬座票价72元，同样的时间，一半的距离，一半的价格，只能说长途性价比一般。

考虑到地铁10min以下的间隔，我觉得苏州地铁11号线已经很不错了。当然，据说未来还有一条恐怖的苏州地铁10号线可以从汾湖到张家港，那我觉得...

# 端午

社畜生涯的第二个端午节来临了。

端午节于我而言，其实很适合作为一个职业生涯的tag。作为一年的中间节点，它既是结束，又是开始。

先师马哥曾经说过，大厂三年不一定比985硕士来的差。当初的我以为这个意思是大厂三年的升职加薪可以很容易达到985硕的起点，却没有想到会是三年后985硕在大厂的起点降到了当初的211本科水平。

上个月被通知毕业的同事在差不多两个月的努力之后终于找到了比较满意的offer，陆家嘴江景工位，加20%涨幅。只能说虽然是行业寒冬，但也并没有完全陷入黑暗。即便离开大厂，机会总归是有的。

单以互联网人设而论，我的交际能力大约还是不错的。但是互联网人设这种东西虚拟感太强，回到现实之后，我还是那种阴暗社恐角色，跟不太熟或者现实没见过的人对话需要做很久心理建设的废物角色。所以我还是更适合一个人活动，什么都不需要准备，突发奇想，想去哪去哪。

从独墅湖高教区的金谷路地铁站坐上二号线，晃晃悠悠去往苏州北站。突然就想去趟海边，于是...

可能是现在生活变得太好了，坐绿皮车卧铺基本上睡不着。顶着惺忪睡眼到了烟台，出站就被30多度的太阳晒麻了。

顶着太阳硬溜了六公里，实在顶不住了。还是坐上舒适的空调车车游海滨吧，顺便去隔壁威海打个卡？

失策了，威海公交没空调。

# QZone VS PYQ

自从全网更名以后大部分社交平台都停止更新了，之前发的东西删的删匿的匿。现在还会更新的也就只剩QQ空间、BB空间、wb和pyq了。

QQ空间和pyq大概是2d一更3d可见，一般两边会对应发，QQ空间发长文多图流水账，pyq发精简9图加简单描述，分工非常明确，不想看废话的请自行屏蔽QQ空间。

BB空间的话一般发粗制滥造的plog，大概就是把十天半个月的图翻出来随便剪剪那种。更新周期一个月1-4次(当然一般是两次左右，懒狗更新不会很勤快的)。

wb是个没啥人看的地方，所以属于想起来发啥就发啥，更新周期从日更到数月更不等，主打一个跟不超过十个人的零散互动，所以停不停更影响也不大。

至于个人blog，感觉更像是内容归档吧。三日可见的东西梳理完以后会传过来，就这样。

# 宁滁

当蓝天在梅雨的阴影之中探出了脑袋，扑面而来的湿热暑气沾湿了每一个路人的衣裳。岁月波澜不惊，而23年已步入下半场。

跟不认识的学弟唠了会儿嗑，发现之前全网改名确实是十分必要的事情，似乎我的名声已经在江湖传播的非常离谱了。好在他们的认知都停留在XX镭上，当这个名字消失以后，一些属于过去的故事就还是沉没在过去里吧。

现在我的互联网人设主打一个阴湿社恐逆天op管人痴兼虚拟旅游博主，每月随机出现在1-4个城市进行一些不知名地点的打卡。不过约我出来一般不太可能，毕竟线下多人活动对于社恐来说还是太艰难了。

很多人建议过我可以去整个单反搞搞专业摄影，或者学学剪辑当一个旅游区up主。但我铁懒狗了，怎么会做这么吃力不讨好的事情呢？就像我五年前就想开始写的小说，三年前就想开始写的编程学习心得，至今也没有开始动笔，应该也不会再动笔了。

全程不经过南京的南京S4线一期开通了。其实讲道理这线也不过滁州，都是在新区和郊区跑着。林楼到汊河新城20公里全是104国无人区，中间仨站俩站没通，70码快速通过，一站盖在荒郊野外，还设了越行线。这线速度还是比较不错的，城区40-60码，郊区能跑到115码左右，全程44公里也就42分钟就到了。

从汊河站出站到越过汊河大桥中线进入南京境内的距离是1.2km，需要的时间在15分钟左右。但到大堤南岸的汊河大桥南公交站还需要再走1.2km，也就是从汊河地铁站到宁滁边界的公交站总共需要30分钟。这并不是一个非常有意义的行为，因为这趟车很难等，平均间隔30分钟以上，而汊河地铁站出站就有滁州来安的公交直通南京林场地铁站。

汊河到林场差不多10km的样子，但因为是快速路，所以公交20分钟左右也就到了。这么算起来从南京林场到滁州高铁站也就一小时多一点的样子，但正如不会有人日常从苏州坐三小时地铁去上海一样，应该也不会有多少滁州人坐两小时地铁+公交+地铁去南京。

类似这样啥用也没有的基础设施，最大的意义在于制造了若干的债务。欠债固然不是什么好事，但是也是经济发展过程中不可或缺的一环。负债让企业有能力去做超出自己能力范围的事情，在起步时期有利于企业快速发展。大环境良好的时候，债务会随着企业快速发展带来的收益而变得不值一提，而大环境糟糕的时候，债务就是企业最大的催命符。从还本付息变成还本欠息再变成还息欠本，一步步走向破产的深渊。

通缩时代的绝望就在于此。国家增发货币最主要的方式就是通过增加银行的负债，从而将增发的货币给到借款的一方。借款方通过购置房产、开办公司来将这笔钱用来发放工资、购买原料等活动，从而将这笔钱发放给更多人。通缩时代里，没有人愿意借，也没有银行敢借给没有盈利预期的人。这一链条断了，市场上的货币发不出去，通缩就这么来临了。

但市面上的反应并不是通缩，而是通胀。因为政府债和大基建的存在，相当一部分资金还是直接发放到了基层。而基层的资金是没有足够长的链条向上游传导的，以致于基层的消费导致了物价上涨。总而言之就是一个中层通缩底层通胀的场景，至于上层，他们不需要考虑这个问题。

所以说，对于现在这个情况，摆烂也不失为一种选择。曾经也算是一种高端消费品的车在新能源浪潮之下已经卷到了不过十来万的价位上，大城市郊区哪怕地铁附近的房价也跌到了一万上下一平的水平。抄底似乎也是不久以后的事情。

疫情以后哪哪都是人，说明其实还是挺多人变得有钱了的。当然，无论是通胀和通缩，总归还是收割了同一波人罢了。一个成熟的社会是不应该有中产这个群体的，现在每天存钱发愁的中产终究会变成开开心心月光度日的底层。那么，不如一步到位吧。

# 夏日山海

当青浦还在暴雨和乌云的洗礼之下挣扎，临港的上空却是蓝天与白云之下的晴空。徐泾、虹桥、天山、静安寺、人广、外滩、陆家嘴、花木、张江、周浦、航头、新场、惠南、书院、滴水湖，92km的距离，仿佛隔开了两个世界。青浦抑或者临港，哪一个更像上海？或许两个都是，或许两个都不是。

洋山专线以稳定80码的时速越过长达三十多公里的东海大桥，到达舟山嵊泗的洋山深水港。晴空之下的蔚蓝色，仿佛南海之滨。发动机轰鸣渐起，航船出港，驶向一片算不上遥远的深蓝。

当斜阳即将落下的时刻，航船抵达了岱山岛西北侧的燕窝山码头。日暮时分的岛上静寂而祥和，大巴穿行山海之间，去往岛东南的彼方。在那里，开往宁波的大巴已经整装待发，准备开启下一程旅途。

从浦东世纪大道上标志性的建筑东方之光向正北偏西的方向望去，一条直接延伸到黄浦江边的道路映入眼帘。前行大约一公里，源深路上最大的建筑——源深体育中心便出现在面前。而你不会注意到的是，它的旁边，便是浦东新区的精神卫生中心。尽管它并不像宛平南路600号那样知名，但它的位置却让它在这个乐子人的版本里有了一些特殊的意义。

很久没有填词了。大约可能只是缺少一些灵感，但也不排除一种看管人把脑子看坏掉的可能。但终究，拖拖拉拉一个多月，还是填出来了这么一首。
