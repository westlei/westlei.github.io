# 黑暗低语

有时候想想上一代的辉煌就羡慕，再看看他们的现状有点遗憾。但再一想想自己是个什么玩意，就彻底摆烂了。或者说，感觉我就只配摆烂了。也许，我啥都不做，都比做点啥好。

家庭和家族的区别就在于，家族是有传承的，有些东西是可以织成网的。而家庭是原子的个体，不管这个家庭曾经达到过什么高度，他毁灭的速度并不会比他兴起的速度慢多少。一个人的奋斗如果不能打造出一个利益集团或者共同体，那他个人的衰落也就意味着他奋斗来的一切都会化为云烟。

如果涉及到与另一个形成集团的利益共同体的对抗，那么个人拥有再大的能量在团体面前也不过是一无是处。一对一的战斗可能很简单，一对十有些人也有一战之力，一对一百、一对一千、一对一万...只要对面人数够多，横竖都是必死之局。但我这种，听不得宏大叙事，站不进主流队伍的，再努力又有什么用。最后一切的一切，还不是一无所有一败涂地。

有时候突然觉得说自己有一种天龙人的优越感，单论条件比周公子什么的好了不知道多少倍。但是家族呢，没有家族，一个人再牛又能怎么样，原子化的个体怎么对抗得过人家的族群势力。

意识到自己不管怎么努力都是必死的局面是不是就可以开摆了。做什么都没有用，单纯依靠自己能做什么，什么也做不了。讨厌阿谀讨厌奉承讨厌结党，相信凭借个人能力总能做到些什么，就是笑话吧。没有群体，个人能干啥，做到了又如何，还不是最后一无所有。

我应该就是小丑吧，还是老老实实当好我的小丑角色吧。我就算能有我爸的能力又怎么样，或者说哪怕是更强一些，又能怎样，我对抗的起几个人。 就只是想做一点微小的工作，就那点卖卡的人都能让我破防。个人努力有什么意义，能改变集体吗，能让集体变好吗？什么是集体，利益才是集体。

我以为我在对抗黑暗，结果发现对于大多数来说我才是黑暗本身吧。这个世界都认可白色恐怖结束了，我们还在做的那些努力，也是徒劳吧。利益集团达成的共识东子想打破，也是徒劳吧。

我永远也不曾奢望我能站在大多数人一边，我的性格决定了我不认同集体。那这也就是必死之局了，困兽之斗也没有意义。

# 沙河寂夜

夜晚，大沙河北岸的滨水步道上，行者寥寥。

这里应当有许多人，在晚风中散步，思考，或者是谈恋爱。

但最近的白色危机引发了一些园区封锁，步道旁的门禁机关闭着，黑色的屏幕无言的注视着河边的一切。似乎与他有关，却又与他无关。

穿过了这一段濒临着“Tsinghua”园区的步道之后，河对岸的图书馆散发出熠熠的光辉。大楼上蓝色的HIT闪着悠悠的光，将河面照亮。

道旁出现了亮起的门禁机。靠近它的瞬间，屏幕上显现出了鼠鼠的脸。门禁机犹豫了一下，吐出了黄黄的错误提示：“噫！是老鼠的味道，滚啊！”

鼠鼠看着前面不高的栅栏，寻思着或许可以翻过去。但是翻过去又能怎么样呢？即使你翻过去了，你没有完全翻过去，那便是你完全没有翻过去。进入了那一片被称为“PKU”的神圣领地，鼠鼠也还是鼠鼠罢了，不会成为PKUer。

鼠鼠终究还是呆立在离人上人的领地不足一米的河边，然后灰溜溜的走了。鼠鼠永远只是鼠鼠罢了，他们离人上人的天堂曾经那么的近，却又那么的远。

# 九门回忆

又要到928了。

一届又一届的保研人去向了遥远的彼方，远离了学校，也远离了我的生活。逐渐变成列表里一个个多年不见的名字。彼方的人有着彼方的生活，与我无关，也不适合去打扰。渐渐也想不起他们的模样，化为陌生。我还会去到他们的城市，但是终究，不便叨扰。

习惯性的再去登一下学校系统，发现账号被ban了，大约是真的毕业了吧。尽管依然活跃在学校的一些社交场地上，但是毕竟已经没有什么熟人了。社会关系终究还是要依赖于人来维系的，当熟悉的人全都离开了，那里也就不是什么值得留恋的地方了吧。

今年的秋招格外艰难，保研也是。海王收割了大部分的offer，众人哄抢着剩下的残羹冷炙，却连边也摸不着。但闹剧总会结束，海王终究也只能选择一处上岸。耐心等待的人们，在一片焦土中挖掘着剩余的果实。努力的耕耘，终会有所收获。

蒙德城的风已经吹过了两个年头，旅行者穿过岩之国度，度过雷之深海，迈入了草的世界。前途，还是让风告诉你吧。风来风往，愿前途无量。

# 幕落而起

928狂欢节结束了。

大部分当事人其实并不会特别的开心，无非是高不成低不就的妥协。与自己努力的三年妥协，与自己无数的面试妥协。大部分旁观者也不会特别的开心，无非是羡慕嫉妒或者一些其他的情绪。人生的新阶段而已，不过如此而已。

我的列表里覆盖的人从14级一直到22级全都有，甚至更早的时候还有过12，13级的学长。但相对而言，列表里19级的52位，相比17级的54位或者18级的70位，意义要更重要一点。

他们是在我踩过了无数的坑走过了无数弯路以后，所面对的一届。我第一次走上讲台授课，第一次认真的带团队。我很尽力去做一些力所能及的指导，期望他们能过的更好些吧。

在接下来的岁月里，我体会了失去和背刺，体会了一无所有。但也收获了关心和温暖，收获了十分珍贵的感情。他们曾夺走我的种种，他们也回报给我种种。

现如今，他们的旅途即将走到了尽头。下一程即将开始，而我所能给予的也止步于此。他们中的大多数都收获了比我优异许多的成就，所能达到的高度也是我所不能想象的。

或许，也有那么一些机缘巧合，列表里素不相识的人成为了同窗或者同门，会让我内心里有一些特殊的感觉。就像三年前走出牛头冲保研北理工的大佬，如今成为了我旁边工位的同事。风来风往，远走的人和事，也许终有一天会再回来。

牛头冲的岁月终将远去，而我也应当去拥抱当下，全新的开始。

# 疫色世界与旅人

恍惚之中，国庆就要结束了。

五天，好像什么都没做，却也走了许多路，看了许多不一样的风景，见了一些很久不见的人，学到了一些不一样的新知识。

华侨城甘坑客家小镇位于龙岗区吉华街道，属于深圳市比较成熟的商业化古建筑群。古镇西北侧田园花海的花仍处于盛放期，金灿灿的向日葵在蓝天下格外耀眼。

黄金海岸·乌山头绿道位于惠州市大亚湾经济技术开发区最东侧，东起华润小径湾，西至霞涌海悦长滩。黄金海岸的沙滩是大亚湾片区我去过的沙滩里最细软的，贝壳什么的也极少，踩上去松松软软，丝毫没有硌脚的感觉。金黄的沙滩与湛蓝的海水，是滨海城市最唯美的画卷。沿木栈道走上乌山头，耳畔海风吹拂，海浪拍打礁石，也不失为一种意趣。

滨海湾花海里位于东莞长安镇滨海湾新区，与深圳沙井海上田园隔茅洲河相望。园内种植大量鲜花，当前竞相开放，霎是多彩。此地濒临宝安机场，飞机低空飞过，与海鸥相遇，构成湛蓝的海天图卷。北侧是vivo全球总部，荒野中耸立的高楼，在暮色之中，也不失为一种意境。

依然是深圳闲逛的一天，香蜜公园和前海石。大概也就是一个新的地点吧，有所不同，但也没有太多不同。主题依然是花与海。香蜜公园内的福田婚姻登记处旁边有一丛玫瑰花海，十分浪漫。在这里走入婚姻殿堂的新人们，应当很幸福吧。

最近一直有点emo，严格意义上也不是emo，不知道是一种什么情绪吧。过的好也不好，差也不差，但也不怎么想主动去联系谁。精神状态不是特别好的情况下，给别人带去更多的都只是负能量吧。患得患失的心态日益严重。有时候觉得，或许另一个人会对她更好吧。

唔，总觉得我配不上现在的我。我不应该是现在这个样子的。

# 社恐与须弥AI

炼丹的准备工作：训练框架一副，核心算法（代码和参数）一份，显卡(算力)若干，训练集若干。在须弥的炼神项目中，愚人众负责提供框架(散兵)和核心算法(雷神之心)，教令院负责利用梦境轮回制造算力，沙漠镀金旅团负责抓捕疯学者并提取神明罐装知识作为神明能力训练集。至于测试集、验证集和迭代算法什么的，本版本没有提供。

当社畜在大厂之中忙忙碌碌，或许也是在为大厂的炼丹炉担任算力的角色吧。长期工作之后的疲惫，与须弥人在花神诞日之后的疲惫，似也无别。

梦醒，今天有一个对我来说很重要的约会。

反正就是，很重要。至于到底是哪里重要，我也说不上来。就，距离很近，但是距离很远。

这就是社恐患者的恐慌吧。与异性的隔阂与疏离感，哪怕是认识了快五年的朋友，已经算不上是很陌生的那种。

做舔狗也是有门槛的。比如说起码得是狗，不是是鼠鼠。狗狗是有人喜欢的，鼠鼠不是。

让生命中仅存不多的美好消失，或许也是一种选择。绝对说不上好的选择，但对美好本身，未尝不是一种解脱。

# 金台夕照

观自在菩萨，行深般若波罗蜜多时，照见五蕴皆空，度一切苦厄。舍利子，色不异空，空不异色，色即是空，空即是色，受想行识，亦复如是。舍利子，是诸法空相，不生不灭，不垢不净，不增不减…

时隔一载，再访斗门金台寺。湖光依旧，山色悠然。入黄杨圣境，则无嗔无痴，无念无想。昔日所求者已然忘却，或事业，或财富，或感情。事业财富俱有小成，而感情茫然依旧。今朝所求者，唯体之康健与心之平和。

很久没有再进入剑三的世界了。大约是没有太多时间寄情于画中山水，驰骋于梦里江湖。但因剑三而寄情的金台山水，黄杨一梦，在往后的岁月里，依然不可缺席。

# 滨海末夏

深圳一年里大约有六个多月的时间都是夏天。但步入十月，早晚的风也还是渐渐带上了些许凉意，而正午的阳光也不似先前毒辣。三十度上下的天气不会令人感觉过分炎热，海风也来的恰到好处。晴好的天气，去眺一抹蔚蓝，大约也是应有之意。

从南山蛇口海上世界防波堤到大鹏葵涌的沙鱼涌官湖沙滩的距离大约是54公里，这同样也差不多是香港新界屯门最西端到西贡最东端的距离。南边的邻居曾经为这座城市带来了多少机遇，如今也就为这座城市带来了多大困扰。其兴也如此，其衰也若此。

也许这是我最后一次眺望这片海，但也许此后依然有若干的机会。离去的时刻，正如高速行驶的旅人，见一抹唯美的斜阳，落在海上。沉浸于其中的美好，想要停下，却也不可能停下。既然已经决定了走向新的天地，那么，也就不应当再后悔了。

# 最后的故事

又到10月17日了。每到这一天还是会有一点伤感吧。

当然不是因为今天是国家扶贫日，毕竟我现在应该也不算绝对意义上的贫穷了。伤感，只不过跟某个人有关而已。

平心而论，她并没有对我做什么，只不过是我有太多的一厢情愿罢了。尤其是在那个十分微妙的时间里，她所提供的一些情感支持，让我有了更多不切实际的幻想。

但幻想不过是幻想而已。她展现的一切假面，终究会在某一刻碎裂。一切化为迷梦与泡影，以及无限的空虚。此时出现的温暖，让我感激，让我倾心。

可是快两年过去了，一切依然没有什么实质的发展。繁忙的工作与逐渐恶化的健康状况，让我间歇性的想放弃一切。但终究，还是什么也没有改变，还是日复一日的平常。

恶化的身体状况，让我逐渐失去自信。种种挫折和失败，让我意识到我曾经骄傲的一切也都是虚假的泡影。选择换一个新的地方，是一种自我救赎的尝试，也是无奈之举吧。

至于不可能的人，已经离我的生活很远了。生活圈子的急剧缩小，让一切都变得顺理成章。四年前相遇的时候，本就源自一场意外。既然是意外，那就不必挂怀。
